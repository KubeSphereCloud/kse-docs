* 业务系统资产 +
业务系统资产管理包含资产状态、资产区域、同步资产、开启防护、关闭防护、资产列表几部分。对云上资产进行全局的监控及管理，及时发现异常资产，减少运维管理成本，提高效率，保障云上资产能够稳定运行。
* 访问控制策略 +
访问控制策略是针对云上公网IP和互联网之间制定内访外和外访内的控制策略，满足云用户对于互联网流量访问控制的管理与安全防护需求。此外，只有资产开启保护后，用户才可以通过安全资源池控制云上的服务器的互联网访问流量。
* 入侵防御策略 +
侵防御策略（IPS策略）可识别访问控制策略以外的未知风险，对互联网上的恶意流量入侵活动和常规攻击行为进行实时阻断和拦截。入侵防御策略包括攻击防护策略、应用防护策略、常规防护策略、最大防护策略。说明：仅支持开启单个策略，如开启新的策略则之前开启的策略将自动关闭。
* 流量控制策略 +
流量控制策略可针对全平台设置内访外和外访内的最大带宽及保障带宽，同时也支持多维度进行细粒度的流控设置，帮助用户更合理的使用资源。
* 黑白名单 +
通过配置黑白名单规则阻断、放行指定IP的访问请求，即设置IP黑白名单。将恶意的攻击者IP设置进黑名单，直接拦截访问流量。将正常的业务访问地址、员工终端地址等IP地址设置入白名单，实现流量直接放行，避免影响业务访问。
* 日志审计 +
日志审计将会记录通过云防火墙所有流量的日志，包括事件日志和操作日志，帮助用户实时审计网络流量，并为用户对可疑流量进行相应的处理提供依据。云防火墙日志审计默认保留7天。	| 为客户提供单个的云防火墙产品能力，产品内部界面青云化，提供更好的使用体验。

主机安全提供主机系统防护与加固、主机网络防护与加固等功能，产品具备业界领先的勒索病毒专防专杀、东西向流量微隔离、补丁修复、外设管控、文件审计、违规外联检测与阻断等主机安全能力，全面提升终端安全防护水平。通过在业务主机安装 Agent 执行主机安全防护策略，对主机进行防护。

* 日志审计 +

* 数据库审计 +
数据库审计产品可以帮助用户实现对进出核心数据库的访问流量进行数据报文字段级的解析操作，完全还原出操作的细节，并给出详尽的操作返回结果。以可视化的方式将所有的访问都呈现在管理者的面前，数据库不再处于不可知、不可控的情况，数据威胁将被迅速发现和响应。通过安装 Agent 采集用户对数据库的访问行为，进行实时审计记录。
