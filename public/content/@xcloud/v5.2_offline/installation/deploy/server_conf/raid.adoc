== 管理节点（KS 节点）

管理节点数据盘需要做成一块大盘，数据盘足够的情况下建议做 RAID5 或 RAID50。

|===
| 存储类型	| 存储介质	| RAID	| 备注

.2+| 所有类型	
| 系统盘	| RAID1（必配）	| -
| SAS 盘	| RAID5/RAID50（必配）| 数据盘
|===

硬件 RAID 是否自带缓存对机械盘性能影响很大。

* RAID 50：必须使用带缓存的硬件 RAID。
* RAID 1：硬件 RAID 可以不带缓存。
* 为提高缓存利用率，需要缓存空间尽可能的大，所以在做 CAS 设备的时候，要避免将一张 SSD 分成多个区使用，而是将整块 SSD 做成共享的缓存设备，分给两个 drbd 使用。
