在安装企业云平台之前，需要依次为每台服务器安装操作系统。

安装操作系统的详细步骤请参考《QingCloud 企业云 ubuntu 操作系统安装指南》。

企业云提供操作系统自动安装镜像和手动安装镜像，**推荐使用自动安装**。

== 安装工具

根据用户实际网络环境，选择一种即可。

* 使用优盘引导安装
+
使用工具制作安装引导优盘。

* KVM 挂载引导安装
+
使用厂家KVM 工具挂载 iso 镜像安装。

== 安装方式

根据安装镜像分为自动安装和手动安装，**推荐使用自动安装**。如果自动安装失败，请先尝试手动安装，如果仍然失败，请联系技术支持。
