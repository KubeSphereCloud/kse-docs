== 注意事项

使用自动安装方式，系统盘默认需要安装在 sda 盘上，所以系统盘的磁盘盘符必须为 sda，否则可能导致安装失败。

include::../../../../../../_components/note.adoc[]

可以尝试把所有数据盘拔出，或者修改 iso 内部的配置文件 autoinstall.seed（系统版本为 ubuntun 20.04.1 时，则需要修改文件 /auto-sda/user-data）。

include::../../../../../../_components/table_footer.adoc[]

== 前提条件

已获取自动安装镜像 QingCloud-X.X.X.X-AutoInstall.iso。
