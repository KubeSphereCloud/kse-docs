---
title: "配置服务器时间"
weight: 50
---

为确保系统正常运行，需要配置时间同步。

* 如果环境已规划单独的 NTP 服务器，则无需执行此步骤，在配置文件中填写 NTP 服务器地址即可。
* 如果环境没有规划单独的 NTP 服务器，在安装过程中，系统会自动选择 VG 节点或 KS 节点作为时钟同步服务器，为剩余的物理服务器和管理虚拟机做时间同步。为确保环境时间的准确性，用户需手动配置 VG 节点或 KS 节点的时间。

== 操作步骤

. 登录 VG 或 KS 节点。
 ** 若有 VG 节点，则手动配置 VG 节点的时间。
 ** 若没有 VG 节点，则手动配置 KS 节点的时间。
. 执行以下命令，查看服务器当前时间。
+
[source]
----
date
----
. 若时间不正确，请执行以下命令，修改服务器时间。
+
[source]
----
date -s “yyyymmdd hh:mm:ss”
----
+
示例：
+
[source]
----
date -s “20220831 12:00:00”
----
. 修改时间后，执行以下命令，保存配置。
+
[source]
----
hwclock --systohc
----
