如果是私有云环境不能访问互联网，需要删除 server.yaml 文件中的 test_domains 配置，否则 BOSS 会有 DNS 相关告警。

. 登录 FirstBox。
. 执行以下命令，修改 server.yaml.ZONE_ID 文件。
+
[source]
----
vi /pitrix/conf/variables/global/server.yaml.ZONE_ID
----
+
删除如下内容。
+
[source]
----
  test_domains:
    - 'www.baidu.com'
    - 'www.taobao.com'
----
. 执行以下命令，刷新 server.yaml.ZONE_ID 到节点。
+
[source]
----
/pitrix/build/build_pkgs.py -p pitrix-global-conf
/pitrix/upgrade/update.sh all pitrix-global-conf
----
