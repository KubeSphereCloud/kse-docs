---
title: "配置存储网络"
weight: 10
---

计算节点采用 Hyper-Sanc 融合部署模式时，NeonSAN 与 Hyper 节点共用服务器，计算节点服务器上需要配置两个网络平面（管理网络和存储网络）。
安装 NeonSAN 前，需要配置存储网络。

. 依次登录所有计算节点。
. 执行以下命令，修改 /etc/network/interfaces 中的配置。
+
[source]
----
vi /etc/network/interfaces
----
. 为所有计算节点配置存储网络。
+
示例：
+
[source]
----
# The loopback network interface
auto lo
iface lo inet loopback

# mgmt network
auto ens1f1
iface ens1f1 inet static
  address 172.31.60.205
  netmask 255.255.255.0
  gateway 172.31.60.254
  dns-nameservers 172.31.60.209

# repl network
auto ens1f0
iface ens1f0 inet static
  address 192.168.0.3
  netmask 255.255.255.0
----

. 配置完成后，建议执行以下命令重启网络，校验是否配置成功。
+
[source]
----
service networking restart
supervisorctl restart all
----
