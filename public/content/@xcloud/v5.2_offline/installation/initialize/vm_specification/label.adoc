系统默认已提供丰富的标签配置，若用户需要自定义一些标签，可以修改标签。

本文以修改“规格ID”为例，介绍如何修改标签。

== 修改标签

1. 登录 BOSS。
2. 选择**运营管理** > **产品中心** > **产品管理**，进入产品管理页面。
3. 在产品目录树中选择**云平台** > **计算** > **云服务器**，进入云服务器详情页面。
4. 选择**标签配置**页签，进入标签配置页面。
5. 在目标标签的操作列，点击更多图标image:/images/ep_offline/installation/icon_more.png[,50%]，选择**编辑**，进入编辑标签页面。
+
下图以修改规格 ID 为例，用户可以根据实际情况增加相应的规格 ID。
+
image::/images/ep_offline/installation/initialize_vm_specification_label.png[]
6. 参数配置完成后，点击**保存**，完成编辑标签操作。
