系统默认已提供丰富的属性配置，若用户需要自定义一些规格，可以修改属性、添加属性。

本文以修改“CPU”属性为例，介绍如何修改属性。

== 修改属性

1. 登录 BOSS。
2. 选择**运营管理** > **产品中心** > **产品管理**，进入产品管理页面。
3. 在产品目录树中选择**云平台** > **计算** > **云服务器**，进入云服务器详情页面。
4. 选择**属性配置**页签，进入属性配置页面。
5. 在目标属性的操作列，点击更多图标image:/images/ep_offline/installation/icon_more.png[,50%]，选择**编辑**，进入编辑属性页面。
+
下图以编辑CPU属性为例，用户可以根据实际情况增加相应的属性值。
+
image::/images/ep_offline/installation/initialize_vm_specification_attribute_01.png[]
6. 配置完成后，点击**保存**，完成修改属性操作。

== 添加属性

1. 登录 BOSS。
2. 选择**运营管理** > **产品中心** > **产品管理**，进入产品管理页面。
3. 在产品目录树中选择**云平台** > **计算** > **云服务器**，进入云服务器详情页面。
4. 选择**属性配置**页签，点击**添加**，进入添加属性页面。
+
image::/images/ep_offline/installation/initialize_vm_specification_attribute_02.png[]
5. 根据规划配置属性参数。
6. 参数配置完成后，点击**保存**，完成属性配置。
