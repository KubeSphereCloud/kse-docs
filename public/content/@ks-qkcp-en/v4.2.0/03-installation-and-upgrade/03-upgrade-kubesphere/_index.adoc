This section only describes how to upgrade from {ks_product-en} v4.1.2 or v4.1.3 to v4.2.0. Direct upgrades from v3.5 to v4.2.0 are not supported.

To upgrade from v3.5 to v4.2.0, please contact {ks_product-en} delivery specialists to get the upgrade documentation and upgrade to v4.1.2.
