After logging in to the {ks_product-en} web console, you need to import a license to activate {ks_product-en} and extensions to access all features.

[.admon.note,cols="a"]
|===
|Note

|
If you installed {ks_product-en} offline, please contact delivery specialists directly to get the license.
|===

== Step 1: Purchase and obtain license

For online installations of {ks_product-en}, visit link:https://kubesphere.com.cn/[{ks_product-en} official website], click **Contact Us**, and inquire about purchasing method for {ks_product-en} and extensions licenses via WeChat or Slack.

image:/images/ks-qkcp/zh/v4.1.3/contact-us.png[, 70%]

== Step 2: Import license

ifeval::["{file_output_type}" == "html"]
After obtaining the license, go to **Component Dock > Platform Settings > Licenses** to add licenses for {ks_product-en} and extensions.
// After obtaining the license, refer to link:../../../04-platform-management/05-platform-settings/02-license/03-add-a-license/[Add Licenses] to import licenses for {ks_product-en} and extensions.
endif::[]

ifeval::["{file_output_type}" == "pdf"]
After obtaining the license, refer to the "Licenses" chapter in the *{ks_product-en} Platform Management Guide* to import licenses for {ks_product-en} and extensions.
endif::[]

== Step 3: Verify activation

If the license page shows {ks_product-en} and extensions as **Authorized**, it indicates successful activation and you can start using {ks_product-en}.
