This section introduces how to enable and disable two-factor authentication for the current user.

After enabling two-factor authentication, users must enter the dynamic authentication code generated by the authenticator app in addition to their username and password when logging in. Two-factor authentication provides double protection for the account by combining a password and a mobile device.

== Prerequisites

Get the username and password for the {ks_product-en} web console.

== Enable Two-Factor Authentication

. Log in to the {ks_product-en} web console.

. Click the current username in the upper-right corner of the page, and then select **User Settings** from the dropdown list.

. Click **Enable Two-Factor Authentication** on the **Two-Factor Authentication** tab.

. Search for "Authenticator" in the app market on your mobile device, download an authenticator app. Google Authenticator is recommended. After installation, click **Next** in the {ks_product-en} web console.
 
. Open the authenticator app, click "Scan QR Code" to scan the QR code on the right. After scanning, click **Next** in the {ks_product-en} web console.

. Enter the dynamic authentication code generated by the authenticator app, and click **Authenticate and Enable** to enable two-factor authentication.
+
--

Now two-factor authentication is enabled. The next time you log in to the {ks_product-en} web console, you must enter the dynamic authentication code generated by the authenticator app in addition to your username and password.

// image:/images/ks-qkcp/zh/v4.1/enable-two-factor-authentication.png[ enable-two-factor-authentication,50%]
--

== Disable Two-Factor Authentication

. Log in to the {ks_product-en} web console.

. Click the current username in the upper-right corner of the page, and then select **User Settings** from the dropdown list.

. Click **Disable Two-Factor Authentication** on the **Two-Factor Authentication** tab.

. In the **Disable Two-Factor Authentication** dialog box, enter the dynamic authentication code generated by the authenticator app, and then click **OK**.
// +
// image:/images/ks-qkcp/zh/v4.1/disable-two-factor-authentication.png[ disable-two-factor-authentication,100%]

=== Unable to Get Dynamic Code

If you are unable to get the dynamic authentication code for two-factor authentication, which prevents you from logging in, you can contact the administrator to run the following command to disable two-factor authentication.

[,bash]
----
kubectl annotate user <USERNAME> iam.kubesphere.io/totp-auth-key-ref-
----
