扩展组件上架后，将为您开通运营后台，方便您查看扩展组件的运营指标、用户订阅以及集群运行情况。

您将在运营后台查看到以下内容：

== 用户看板

数据指标：

- 订阅次数：所选期间内该扩展组件的总订阅次数。

- 订阅用户数：所选期间内订阅过该扩展组件的用户（去重）数量。

- 订阅收入：所选期间内该扩展组件已支付订单的总金额。

订阅明细：

用户的订阅明细，包括扩展组件名称、ID、用户 ID、订阅状态、订阅计划、订阅时长、支付金额、订阅时间、到期时间等信息。

== 集群看板

数据指标：

- 安装用户数：所选期间内安装过该扩展组件的用户（去重）数量。

- 安装集群数：所选期间内安装过该扩展组件的集群（去重）数量。

集群明细：

用户的集群明细，包括扩展组件名称、ID、用户 ID、集群 ID、集群名称、集群状态、安装时间、更新时间等信息。
