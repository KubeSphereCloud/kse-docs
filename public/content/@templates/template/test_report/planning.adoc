本次测试初步规划为三阶段：

* **第一阶段**：功能测试阶段（2020.2.1-2020.3.1）
+
该阶段主要进行重点功能的系统测试，以及个别功能的优化后验证测试。

* **第二阶段**：异常/高可用/性能/稳定性/版本兼容性测试阶段（2020.3.2-2020.3.15）
+
该阶段主要针对集群的异常/高可用/性能/稳定性/版本兼容性场景下，对XXX系统的XXX能力进行验证测试，并汇总XXX系统在不同场景下，能够做出的，最优程度上的应对策略。

* **第三阶段**：BUG回归阶段（2020.3.16-2020.3.29）
+
该阶段主要针对已解决的BUG进行缺陷回归。
