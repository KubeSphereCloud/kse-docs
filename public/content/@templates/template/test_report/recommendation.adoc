== 版本使用限制

1. xxx功能目前默认关闭，如有需要可以打开。
2. 本地化访问功能目前默认关闭，如有需要可以打开。
3. 异常故障恢复后自愈的限制：

** 只有当一半以上节点出现断电故障时，当故障节点恢复正常后，集群（数据库）无法完成自愈，需要手动先恢复数据库，集群才可以继续对外提供服务；除此之外的其他故障场景，集群均可以完成自愈，无须人工干预。
** 集群在故障期间会出现一定程度的服务响应延迟，甚至是中断。延迟或中断时长一般在8s~50s之内。

== 支持升级的版本

支持低版本在线升级至v2.2.0版本，具体支持版本如下：

* 支持1.1.x版本平滑升级到2.2.0版本
* 支持1.2.x版本平滑升级到2.2.0版本
* 支持2.0.x版本平滑升级到2.2.0版本
* 支持2.1.x版本平滑升级到2.2.0版本
