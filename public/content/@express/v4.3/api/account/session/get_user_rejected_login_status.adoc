---
title: 查询用户拒绝登录状态（GetUserRejectedLoginStatus）
weight: 5
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| user
| String
| true
| 用户id
|===

== 返回数据
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| ip_list
| Array
| false
| 拒绝登录的ip列表

| rejected
| Integer
| false
| enums{0,
|===

== 请求示例
[,javascript]
----
{
  "action": "GetUserRejectedLoginStatus",
  "user": "usr-dQFUel6D"
}
----

== 返回示例
[,javascript]
----
{
  "action": "GetUserRejectedLoginStatusResponse",
  "ip_list": [],
  "rejected": 0,
  "ret_code": 0
}
----
