---
title: 初始化用户信息（InitUserInfo）
weight: 6
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| currency
| String
| true
| 货币类型

| passwd
| String
| false
| 初始密码

| user
| String
| false
| 用户id
|===

== 返回数据
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| user_id
| String
| false
| 用户id
|===

== 请求示例
[,javascript]
----
{
  "action": "InitUserInfo",
  "currency": "CNY",
  "passwd": "*****",
  "user": "usr-dQFUel6D"
}
----

== 返回示例
[,javascript]
----
{
  "action": "InitUserInfoResponse",
  "ret_code": 0,
  "user_id": "usr-dQFUel6D"
}
----
