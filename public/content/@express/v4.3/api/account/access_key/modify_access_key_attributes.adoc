---
title: 修改API密钥的属性（ModifyAccessKeyAttributes）
weight: 8
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| access_key
| String
| true
| API密钥ID

| access_key_name
| String
| true
| API密钥名称

| description
| String
| true
| API密钥的描述

| ip_white_list
| String
| true
| IP白名单，限制使用API密钥的源IP
|===

== 返回数据
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| access_key_id
| String
| false
| API密钥ID
|===

== 请求示例
[,javascript]
----
{
  "access_key": "LTMJGBXPHSEZRNVKKPHU",
  "access_key_name": "test_name",
  "action": "ModifyAccessKeyAttributes",
  "description": "test name desc",
  "ip_white_list": ""
}
----

== 返回示例
[,javascript]
----
{
  "access_key_id": "LTMJGBXPHSEZRNVKKPHU",
  "action": "ModifyAccessKeyAttributesResponse",
  "ret_code": 0
}
----
