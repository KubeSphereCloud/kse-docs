删除一台或多台负载均衡器。

include::../../../../../_components/note.adoc[]
* 正在创建的资源（状态是 pending ）是不能被销毁的。
* 删除负载均衡器后，与其关联的公网IP会自动解绑，变为``可用``状态。
include::../../../../../_components/table_footer.adoc[]

== 请求参数

|===
| 参数 | 参数类型 | 说明 | 是否必选

| loadbalancers.n
| String
| 负载均衡器 ID。
| Yes

| zone
| String
| 区域 ID，注意要小写。
| Yes
|===

link:../../../introduction/parameters/[公共参数]

== 返回数据

|===
| 参数 | 参数类型 | 说明

| action
| String
| 响应动作。

| job_id
| String
| 执行任务的 Job ID。

| ret_code
| Integer
| 执行成功与否，0 表示成功，其他值则为错误代码。
|===

== 示例

*请求示例：*
[source]
----
https://api.domain.com/iaas/?action=DeleteLoadBalancers
&loadbalancers.1=lb-1234abcd
&zone=testa
&COMMON_PARAMS
----

*返回示例：*
[source]
----
{
  "action":"DeleteLoadBalancersResponse",
  "job_id":"j-1234abcd",
  "ret_code":0
}
----
