关闭一台或多台运行状态的{instance_product_name_cn}。

{instance_product_name_cn}只有在运行 running 状态才能被关闭，如果处于非运行状态，则返回错误信息。

include::../../../../../_components/note.adoc[]
关闭{instance_product_name_cn}不会保存 RAM 中的数据。
include::../../../../../_components/table_footer.adoc[]

*Request Parameters*

[option="header",cols="1,1,2,1"]
|===
| Parameter name | Type | Description | Required

| instances.n
| String
| 一个或多个{instance_product_name_cn}ID
| Yes

| force
| Integer
| 1: 强制关机，0: 非强制关机，默认为0
| No

| zone
| String
| 区域 ID，注意要小写
| Yes
|===

link:../../../introduction/parameters/[公共参数]

*Response Elements*

[option="header",cols="1,1,2"]
|===
| Name | Type | Description

| action
| String
| 响应动作

| job_id
| String
| 执行任务的 Job ID

| ret_code
| Integer
| 执行成功与否，0 表示成功，其他值则为错误代码
|===

*Example*

Example Request

----
https://api.domain.com/iaas/?action=StopInstances
&instances.1=i-rtyv0968
&zone=testa
&COMMON_PARAMS
----

Example Response

----
{
  "action":"StopInstancesResponse",
  "job_id":"j-ybnoeitr",
  "ret_code":0
}
----
