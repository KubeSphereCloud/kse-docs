修改安全组IP/端口集合的名称和描述。

include::../../../../../_components/note.adoc[]
一次只能修改一个。
include::../../../../../_components/table_footer.adoc[]

== 请求参数

|===
| 参数 | 参数类型 | 说明 | 是否必选

| security_group_ipset
| String
| ID
| Yes

| security_group_ipset_name
| String
| 名称
| No

| description
| String
| 描述
| No

| val
| String
| 值
| No

| zone
| String
| 区域 ID，注意要小写
| Yes
|===

link:../../../introduction/parameters/[公共参数]

== 返回数据

|===
| 参数 | 参数类型 | 说明

| action
| String
| 响应动作

| security_group_ipset_id
| String
| ID

| ret_code
| Integer
| 执行成功与否，0 表示成功，其他值则为错误代码
|===

== 示例

*请求示例*：

[,json]
----
https://api.domain.com/iaas/?action=ModifySecurityGroupIPSetAttributes
&security_group_ipset=sgi-rtyv0968
&security_group_ipset_name=sample
&zone=testa
&COMMON_PARAMS
----

*返回示例*：

[,json]
----
{
  "action":"ModifySecurityGroupIPSetAttributesResponse",
  "security_group_ipset_id":"sgi-rtyv0968",
  "ret_code":0
}
----
