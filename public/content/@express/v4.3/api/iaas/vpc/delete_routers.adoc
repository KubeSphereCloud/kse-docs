删除一台或多台 VPC 网络。

include::../../../../../_components/note.adoc[]
* 正在创建的 VPC 网络（状态为 pending ），是不能被销毁的。
* 被删除的 VPC 网络会被立即系统回收，不具有可恢复的功能。
include::../../../../../_components/table_footer.adoc[]

== 请求参数

|===
| 参数 | 参数类型 | 描述 | 是否必选

| routers.n
| String
| VPC 网络 ID
| Yes

| zone
| String
| 区域 ID，注意要小写
| Yes
|===

link:../../../introduction/parameters/[公共参数]

== 返回数据

|===
| 参数 | 参数类型 | 描述

| action
| String
| 响应动作

| job_id
| String
| 执行任务的 Job ID

| ret_code
| Integer
| 执行成功与否，0 表示成功，其他值则为错误代码
|===

== 示例

*请求示例：*
[source]
----
https://api.domain.com/iaas/?action=DeleteRouters
&routers.1=rtr-s5nqo8mr
&zone=testa
&COMMON_PARAMS
----

*返回示例：*
[source]
----
{
  "action":"DeleteRoutersResponse",
  "ret_code":0,
  "job_id":"j-32c66ja8",
  "routers":[
    "rtr-s5nqo8mr"
  ]
}
----
