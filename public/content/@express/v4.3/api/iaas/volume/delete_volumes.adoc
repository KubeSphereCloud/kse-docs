删除一块或多块硬盘。硬盘须在可用（ available ）状态下才能被删除， 已加载到{instance_product_name_cn}的硬盘需先卸载后才能删除。

*Request Parameters*

[option="header",cols="1,1,2,1"]
|===
| Parameter name | Type | Description | Required

| volumes.n
| String
| 硬盘 ID
| Yes

| zone
| String
| 区域 ID，注意要小写
| Yes
|===

link:../../../introduction/parameters/[公共参数]

*Response Elements*

[option="header",cols="1,1,2"]
|===
| Name | Type | Description

| action
| String
| 响应动作

| job_id
| String
| 执行任务的 Job ID

| ret_code
| Integer
| 执行成功与否，0 表示成功，其他值则为错误代码
|===

*Example*

Example Request

----
https://api.domain.com/iaas/?action=DeleteVolumes
&volumes.1=vol-uydrnlax
&zone=testa
&COMMON_PARAMS
----

Example Response

----
{
  "action":"DeleteVolumesResponse",
  "job_id":"j-bm6ym3r8",
  "ret_code":0
}
----
