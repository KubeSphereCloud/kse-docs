将一块或多块``可用``（ available ）状态的硬盘加载到某台``运行``（ running ） 或``关机``（ stopped ）状态的{instance_product_name_cn}。

*Request Parameters*

|===
| Parameter name | Type | Description | Required

| volumes.n
| String
| 硬盘 ID
| Yes

| instance
| String
| {instance_product_name_cn} ID
| Yes

| zone
| String
| 区域 ID，注意要小写
| Yes
|===

link:../../../introduction/parameters/[公共参数]

*Response Elements*

[option="header",cols="1,1,2"]
|===
| Name | Type | Description

| action
| String
| 响应动作

| job_id
| String
| 执行任务的 Job ID

| ret_code
| Integer
| 执行成功与否，0 表示成功，其他值则为错误代码
|===

*Example*

Example Request

----
https://api.domain.com/iaas/?action=AttachVolumes
&volumes.1=vol-12djpg8q
&instance_id=i-12345678,
&zone=testa
&COMMON_PARAMS
----

Example Response

----
{
  "action":"AttachVolumesResponse",
  "job_id":"j-bm6ym3r8",
  "ret_code":0
}
----
