== 功能描述

在纯虚拟化部署的云易捷环境中，用户可能对于服务器有超分设置需求，配置中心的**全局设置**支持用户对 CPU 和内存进行超卖比设置。

== 场景描述

该章节用于指导用户在进入**配置中心**的**全局设置**页面，进行 CPU 超卖比、内存超卖比以及回收站保存周期进行设置。

== 操作步骤

. 登录{product_console_name}。
. 在控制台顶部导航栏右侧，将鼠标移动至用户名上方，打开**配置中心**，并点击**高级设置** > **全局设置**。

. 在配置中心窗口的**全局设置**页面，支持对 **CPU 超卖比**、**内存超卖比**以及**回收站保存周期**进行自主设置。
+
include::../../../../_components/note.adoc[]
* CPU 超卖比默认为 3，支持最大修改为 100。
* 内存默认不超卖，用户可根据实际情况进行修改。
* 回收站保存周期，支持设置 1~30 天。
