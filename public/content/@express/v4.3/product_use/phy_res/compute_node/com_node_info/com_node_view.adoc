---
title: "查看计算节点信息"
weight: 1
---

. 登录{product_console_name}。

. 在顶部导航栏，选择**物理资源** > **计算节点**，进入计算节点列表页面。该页面会显示当前平台内的所有计算节点相关信息，包括相应节点的 IP 地址、名称/ ID、主机数和更新时间等基本信息，以及每个计算节点其相应的物理资源与虚拟资源的占有量。
+

include::../../../../../../_components/note.adoc[]
* 由于 CPU 的超卖比为固定值 1:5，故若物理 CPU 核数为 24 核，则虚拟 CPU 将自动设置为 120 核。目前在不支持调整超买比。
* 虚拟内存和物理内存数量保持一致，虚拟硬盘总量也与物理硬盘总量数值一致，不支持超卖。

include::../../../../../../_components/table_footer.adoc[]
+
image::/images/express/console/com_node_view_1.png[]

. 点击计算节点名称/ID，打开相应节点的详情页面。当前页面主要展示当前计算节点物理资源与虚拟资源的使用情况、健康状况和系统性能的等监控信息。

+
image::/images/express/console/com_node_view_2_4.3.png[]

. 在详情页面除了可查看节点状态、服务器厂商/型号、机架/位置、CPU 型号/机构/核心数、节点 IP 地址、运行主机数、类型、内核等信息，还可查看当前计算节点的资源监控、网卡监控、节点流排名、主机、硬盘、服务进程以及告警记录相关内容。
