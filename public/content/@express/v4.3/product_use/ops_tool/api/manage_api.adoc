== 场景描述

该任务主要用于指导用户对平台内已存在的 API 密钥进行管理，包括修改属性、设置 IP 白名单、启用/禁用、删除以及查看详情等操作。


== 修改属性

. 登录{product_console_name}。
. 在顶部导航栏，选择**运维工具** > *API 密钥*，进入 **API 密钥**列表页面。
+
image::/images/express/console/api_manage_1.png[manage_api_1]

. 点击 API 密钥所在行右侧的更多操作，选择**修改**，在弹出的**修改属性**窗口中，可对 API 密钥的名称以及相关描述进行修改，点击**修改**进行确认即可。
//+
//image::/images/express/console/api_manage_2.png[,50%]

== 设置 IP 白名单

. 在顶部导航栏，选择**运维工具** > *API 密钥*，进入 **API 密钥**列表页面。
. 点击 API 密钥所在行右侧的更多操作，选择**设置 IP 白名单**，在弹出的**设置 IP 白名单**窗口中，在 IP 白名单内输入 IP 地址、IP 段或 IP 范围，点击**确定**即可。
+
image::/images/express/console/api_manage_3.png[,50%]

== 禁用/激活 API 密钥

. 在顶部导航栏，选择**运维工具** > *API 密钥*，进入 **API 密钥**列表页面。
. 点击状态为``活跃``的 API 密钥所在行右侧的更多操作，选择**禁用**，在弹出的**禁用**窗口中，点击**禁用**即可。
//+
//image::/images/express/console/api_manage_4.png[,50%]

. 禁用成功的 API 密钥的状态为``暂停``。
+
image::/images/express/console/api_manage_5.png[]

. 点击状态为``暂停``的 API 密钥所在行右侧的更多操作，选择**激活**，在弹出的**激活 API 密钥**窗口中，点击**激活**即可。
//+
//image::/images/express/console/api_manage_6.png[,50%]

== 删除 API 密钥

. 在顶部导航栏，选择**运维工具** > *API 密钥*，进入 **API 密钥**列表页面。
. 点击 API 密钥所在行右侧的更多操作，选择**删除**，在弹出的**删除**窗口中，点击**删除**即可。
//+
//image::/images/express/console/api_manage_7.png[,50%]

== 查看 API 密钥详情

. 在顶部导航栏，选择**运维工具** > *API 密钥*，进入 **API 密钥**列表页面。
. 点击 API 密钥名称，进入其详情页面，该页面可查看当前 API 密钥的名称、状态、所属用户、描述、创建和更新时间以及 IP 名单。
+
image::/images/express/console/api_manage_8.png[manage_api_8]
