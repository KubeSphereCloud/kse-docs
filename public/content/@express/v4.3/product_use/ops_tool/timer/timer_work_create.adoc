== 功能描述

已创建成功的定时器，可同时管理多个不同的定时任务。

== 操作步骤

. 登录{product_console_name}。
. 在顶部导航栏，选择**运维工具** > *定时器*，进入**定时器**列表页面。点击相应定时器名称，进入其详情页面。
. 选择**定时器任务**页签，点击**创建定时器任务**。
+
image::/images/express/console/timer_manage_10.png[manage_timer_10]

. 在弹出的**创建定时器任务**窗口中，设置任务的**名称**、*执行操作*、**操作对象**等参数，点击**创建**即可。
+
image::/images/express/console/timer_manage_11.png[,70%]
