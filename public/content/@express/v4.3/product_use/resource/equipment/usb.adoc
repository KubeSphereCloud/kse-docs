== 功能描述

云平台支持 USB 设备透传服务，物理机 USB 设备可直接透传至所有运行至物理机上的虚拟机，从而让虚拟机能够直接使用物理机上的 USB 设备。

当用户使用 USB 设备上传数据到虚拟机上或者虚拟机某些应用需要通过 USB 接口设备进行认证时，用户需要将 USB 设备挂载到虚拟机上。在云易捷的环境中，用户可以将 USB 设备插入任意物理节点，USB 设备列表自动识别到后，可在管理控制平台内向虚拟机挂载相应设备。同时，还支持加密狗设备的识别和挂载。

== 前提条件

* 已获得登录{product_console_name}的账户和密码。
* 已将 USB 设备插入物理机，且设备的状态为``可用``。

== 操作步骤

. 登录{product_console_name}。
. 在顶部导航栏，选择**资源** > *设备透传* > *USB 设备*，进入 USB 设备管理页面。
. 点击相应 USB 设备所在行右侧的更多操作，选择**绑定到主机**。
+
image::/images/express/console/usb_1.png[]

. 在弹出的**选择要绑定设备的主机**窗口中，选择相应的主机点击**确定**即可。
+
image::/images/express/console/usb_2.png[,70%]

. 返回设备列表页面，可以看到该 USB 设备已挂载至指定的虚拟机上，状态显示``使用中``，即表示设备绑定主机成功。
