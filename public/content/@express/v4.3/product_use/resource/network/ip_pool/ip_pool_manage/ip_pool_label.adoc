== 场景描述

该章节内容主要用于指导用户如何在管理控制平台修改指定 IP 池的属性。


== 操作步骤

. 登录{product_console_name}。

. 在顶部导航栏，选择**资源** > **网络** > **IP 池**，进入 **IP 池**列表页面。

. 点击 IP 池所在行右侧的更多操作，选择**标签**，点击**绑定**，在弹出的**绑定标签**的窗口中，选择待绑定至当前 IP 池的标签，点击**绑定**即可。
+

include::../../../../../../../_components/note.adoc[]
* 若当前平台内没有可用的标签，可点击**创建**，即可创建新的标签。
* 可绑定单个或多个标签。

include::../../../../../../../_components/table_footer.adoc[]
//+
//image::/images/express/console/ip_pool_manage_7.png[,50%]

. 点击 IP 池所在行右侧的更多操作，选择 **标签**，点击**解绑**，在弹出的**解绑标签**窗口中，删除待解绑的标签，点击**确定**即可。
//+
