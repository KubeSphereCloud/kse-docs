== 前提条件

需预先在物理交换机上按需划分多个用于云易捷业务所需的 VLAN，然后才可以在 **IP 池**管理功能中，添加相应的信息（包括 IP 网段，VLAN_ID）。

== 操作步骤
. 登录{product_console_name}。
. 在顶部导航栏，选择**资源** > *网络* > *IP 池*，进入 **IP 池**列表页面。
+
image::/images/express/console/ip_pool_create_1.png[]

. 点击**创建 IP 池**，在弹出的创建 IP 池的窗口中，配置各项参数，点击**创建**即可。

+
image::/images/express/console/ip_pool_create_2.png[,50%]
