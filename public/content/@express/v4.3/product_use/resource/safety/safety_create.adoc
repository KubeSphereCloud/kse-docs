---
title: "创建安全组"
weight: 1
---


安全组类似防火墙功能，通过策略配置对单台或多台云服务器的网络访问进行控制，极大地提高一个内部网络或主机的安全性，并过滤不安全的服务，从而降低内部网络或云服务器的风险。

== 场景描述

该章节主要用于指导用户在云易捷控制台内创建安全组。

== 操作步骤

. 登录{product_console_name}。
. 在顶部导航栏，选择**资源** > *安全* ，进入安全列表页面
+
image::/images/express/console/safety_create_1.png[]


. 点击**创建安全组**，在弹出的**创建安全组**的窗口中，输入**安全组名称**，点击**确定**。
+
image::/images/express/console/safety_create_2.png[,80%]

. 在跳转至的**添加安全组规则**的界面，设置各项参数，点击**确定**即可。
+
image::/images/express/console/safety_create_3.png[,80%]
+
参数说明
+
[options="header",cols="1,3a"]
|===
|参数 |说明 
|规则名称 
|用户自定义，便于用户浏览和搜索。 

|预设规则 
|用户自行选择，支持 ping、ping6、ssh、http、https、 ftp、remote、openvpn、pptp、l2tp、gre 以及 ipip。 

|优先级 
|安全组规则的优先级决定规则应用至网络流量的顺序， 数字越小优先级越高。 

|规则 
|* 下行规则可控制 Internet 用户对服务器的访问规则 （如仅向用户开放网站访问）。
* 上行规则可控制服务器可访问的 Internet 访问(如让 服务器仅能访问微信 API)，从而全面的保护业务安全。

|协议 
|安全组支持多种协议设置，常用 TCP、 UDP。 

|起始端口 
|1 ~ 65525，可点击右侧标志，选择已创建的 IP /端口 集合。 

|结束端口 
|1 ~ 65525，可点击右侧标志，选择已创建的 IP /端口 集合。 目标 IP 不填写则表示所有 IP 地址，可点击右侧标志，选择已 创建的 IP/端口集合。
|===
