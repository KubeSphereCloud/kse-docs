---

为方便用户使用自有的 ISO 文件进行虚拟机的系统安装，云易捷提供从光盘镜像部署虚拟机的功能。

== 前提条件

光盘镜像文件已提前上传至镜像仓库，若未上传可参考link:../../../../image/image_register[注册镜像]相关操作。

== 操作步骤

. 登录{product_console_name}。

. 在顶部导航栏，选择**资源** > *虚拟机*，默认进入 **ExpressCloud 主机**列表页面。
+
image::/images/express/console/express_vm_create_1.png[express_vm_create_1]

. 在 **ExpressCloud 主机**列表页面，点击**创建虚拟机**，在弹出的选框中，点击**从光盘镜像部署**。

. 在弹出的**从光盘镜像部署**窗口中，根据操作提示完成**选择资源池和镜像** > **虚拟机设置** > **名称及规格**步骤，点击**创建**即可。
+
image::/images/express/console/express_vm_create_2.png[]

+
**参数说明**：
+
[options="header",cols="1,1,3"]
|===
|操作步骤|参数|说明
.3+|选择资源池和镜像
|资源池
|若当前环境有多个资源池可根据实际情况进行选择。

|平台
a|* Linux：Centox、Ubuntu、Debian、Fedora、SUSE、Arch、Oracle Linux、CoreOS、RedHat
* Winsows
* Unix：FreeBSD

|光盘镜像
a|支持选择**自由镜像**或**共享镜像**。所需镜像文件要提前link:../../../../image/image_register[上传至镜像仓库]。

* 自有镜像：需要预先在镜像资源中进行注册，或者捕获一个虚拟机来自行创建，详情可参考link:../../../../image/image_register[镜像]相关内容。
* 共享镜像：共享镜像为其他用户共享给自己的模板。用户可以基于这些模板来创建云服务器。

.8+|虚拟机设置
|CPU
|支持 1~128 核。

|内存
|支持 1~256GB。

|存储
|支持选择分布式存储或远端存储。用户需提前配置远端存储，详细步骤可参考link:../../../../../phy_res/rg_manage/stor_remote[远端存储]相关章节内容。

|系统盘
|根据用户所选的镜像，系统盘大小取值可根据用户实际情况进行设置，至少 20GB。

|副本策略
|支持 1~3 副本。

|数据盘
|点击**挂载数据盘**，在弹出的**选择硬盘**窗口中，勾选可用的硬盘即可。可勾选单个或多个。

|网络
|支持选择基础网络或 VPC 网络，选择已有其中一个 IP 组后，将为虚拟机分配一个其网段内的网络 IP 地址，支持自定义网段。

|高级选项
a|* Hostname：支持用户自定义。
* 安置策略组：该功能为用户提供了虚拟机分组功能，用户可以通过创建合适关系的安置策略组并将部署的虚拟机加入相应的分组，从而将一组虚拟机实例按照分组关系部署到合适的物理节点上，来提高资源的可用性，业务的连续性。

.3+|名称及预览
|名称
|虚拟机的名称，用户自定义。

|数量
|创建虚拟机的数量，最大支持一次性创建 10 个虚拟机。

|预览
|当前待创建的虚拟机的基本信息。
|===

. 虚拟机创建完成后，返回虚拟机列表，若新创建的虚拟机状态为**运行中**，则表示创建成功。
