== 功能描述

用于查看当前平台内虚拟资源的使用情况，该模块可对平台内的所有主机、硬盘以及网络IP 的使用情况进行统计。

image::/images/express/console/vir_usage_1.png[vir_usage_1]

== 场景描述

用户通过该模块内容直观快速地查看主机、硬盘以及网络 IP 的使用情况。


== 操作步骤

. 登录{product_console_name}。
. 选择**概览**页面，点击**分数图标**可直接跳转至相应资源的详情列表页。
+

include::../../../../_components/note.adoc[]
* 例如，`虚拟机 9/80 运行中` 表示当前平台中共有 80 台虚拟机，其中有 9 台状态处于``运行中``，其余虚拟机处于关机状态。硬盘、网络 IP 亦是同理。
* 此处显示的主机总数是指状态为``运行中``、``关机``以及``已删除``的主机数之和。

== 资源介绍

* *虚拟机*
+
虚拟机是一台配置完善的服务器，配备用户所需的操作系统、硬件及网络配置。一般来说，创建主机的请求可在 10 s 至 60 s 内完成，故用户可灵活动态的按需使用相应的计算能力。

* *硬盘*
+
硬盘（Disk）为主机提供块存储设备，它独立于主机的是生命周期而存在，可独立创建后加载至任意主机上。

* *网络 IP*
+
网络 IP对应于内网 IP 或公网 IP，用户如有配置需求，可通过 link:../../resource/network/ip_pool/intro[IP 池]进行配置。基础网络的使用需通过 IP 池管理功能进行实现，用户可将本地环境物理交换机中配置的 VLAN 网段地址添加至 IP 池管理记录中，用于分配具体的 IP 地址的网段选择。
