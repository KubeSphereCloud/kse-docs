由于不同的网络模式，集群网络的初始化参数配置不同，具体操作如下。

. 登录云易捷管理控制平台（Console 平台），访问地址形如 ``console.xxxx.com``，默认管理员用户名/密码为 admin@xxxx.com/zhu88jie。
+
image::/images/express/delivery/install_delopy_46.png[]

. 在顶部导航栏，选择**资源** > **网络** > **IP 池**，进入 IP 池管理页面。

. 点击**创建 IP 池**，在弹出的创建 IP 池的窗口中，配置网络信息，点击**创建**即可。IP 池创建完成后，平台内创建的所有虚拟机的 IP 均会在该范围内生成。
+
* **网络模式为 VLAN** 
+
image::/images/express/delivery/install_delopy_47.png[]
+

VLAN 允许设置范围 ``1~4095``，交换机预制 VLAN 号禁止使用。
include::../../../../../_components/table_footer.adoc[]
+
**参数设置说明**
+
** 名称：业务网络名称，根据实际情况进行填写。
** 工作模式：选择``虚拟机``。
** IP 网段：业务网络对应的 IP 网段，如 192.168.1.0/24。
** VLAN：业务网络对应的 VLAN ID，如 10。
** DHCP 服务：默然需开启 DHCP 服务，平台内创建虚拟机，将通过 DHCP 服务自动管理 IP 地址。
** DHCP 服务器：指定一个地址给 DHCP 服务器。
** DHCP 范围：指定 DHCP 服务器分配 IP 地址的范围，如 192.168.1.1 - 240。通常可以不把网段所有的 IP 地址留给 DHCP 管理，可以预留一个地址段，用于客户自行管理，如作为集群的 VIP 地址登录等。 本例中，192.168.1.240 - 252 地址就是空闲下来的，留给客户自行管理。
** 物理网卡：业务和管理分离模式时，可以选择业务网络对应的网卡。
** 网关： 指定业务网络的网关。    
** 描述：为了审计需要，可以添加描述信息。 
+
* **网络模式为 MacVTap**
+
适用于测试环境，可将业务网络和管理网络共用一段网络地址。
+
image::/images/express/delivery/install_delopy_48.png[]
+
**参数设置说明**
+
** 工作模式：选择``虚拟机/物理机混合``。
** IP 网段：业务网络对应的IP网段, 如 172.31.41.128/25
** VLAN: 须设置为 ``0``。
** 其它的配置项与 VLAN 模式相同，但要注意，不要和管理网络地址重叠，造成冲突。例如，可以一个 C 段，管理网络占用前半部分，业务网络占用后半部分。 该模式使用是有一定的限制，不支持防火墙、负载均衡器等功能。
