在云易捷安装部署过程中可能用到的软硬件工具，如下所示。

[options="header",cols="1,2"]
|===
|工具 |作用
|Ventoy 
|用于将 ISO 镜像刻录至 U 盘

|Chrome 或 Firefox 浏览器
|云易捷采用 Web 图形可视化安装方式，目前要求使用 Chrome 或 FireFox 浏览器登录CloudExpress Installer 部署管理平台（Web Installer），推荐使用最新版本

|网络直连线
|用于连接个人电脑与云易捷一体机的网络端口

|移动硬盘
|用于临时存储云易捷安装所需软件包
|===
