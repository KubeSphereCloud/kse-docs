. ``在准备对节点进行初始化时，需要使用 PC 直连 eth0 端口，如何确定 eth0 的位置？``
+
Q：eth0 通常位于千兆网口的左边第一个，极少数情况下 eth0 会在其他端口。 


. ``电脑直连 eth0 端口后，无法访问 100.100.100.100:9998 地址？``
+
Q：确认 PC 本机网卡 IP 是否有修改为 100.100.100.xxx；如果本机 IP 已修改，那么请更换连接服务器的网口重试。 

. ``节点初始化完毕后，无法访问 web Installer 页面？`` 
+
Q：请确认 PC 本机是否已接入交换机的管理/业务 VLAN，请确认初始化时勾选的两个管理网口是否接入管理/业务 VLAN。 

. ``Web Installer 中扫描节点时，未扫描到可添加的节点？`` 
+
Q：请检查新增的节点与原有集群间的网络通信是否正常，是否已接入原有的管理/业务 VLAN。

. ``节点添加完毕，但进入 Console 平台后无法创建虚拟机？`` 
+
Q：请检查是否已经在web Installer中导入新的青立方®超融合易捷版授权。

. ``集群初始化完毕后，console.xxxx.com 无法访问？``
+
Q：请检查是否有在本地PC的C:\windows\system32\drivers\etc\hosts 文件中添加 host 信息。

. ``点击创建虚拟机时，系统镜像页面显示暂无数据？`` 
+
Q：请前往 Web Installer 中上传镜像，如果没有镜像，请联系 QingCloud 销售人员索取。 
+
. ``新建虚拟机 IP 池时，提示错误，无法保存？`` 
+
Q：请检查虚拟机 IP 池的 VLAN ID 是否设置为 “0”，虚拟机 IP 池选择 VLAN 模式时，ID 不允许设置为“0”。

. ``如果在安装易捷版 ISO 时出现以下提示，如何解决？``
+
image::/images/express/delivery/install_delopy_51.png[,60%]
+
Q：检查服务器 RAID 是否配置，小容量硬盘的指示灯是否正常。

. ``在 Web Installer 平台添加节点时，未找到 SSD 或 NVMe 硬盘，如何解决？``
+
image::/images/express/delivery/install_delopy_52.png[,60%]
+
Q：终止节点添加，检查服务器 SSD 或 NVMe 闪存是否正常。
