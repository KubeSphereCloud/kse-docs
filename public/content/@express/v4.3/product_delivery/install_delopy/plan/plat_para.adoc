---
title: "平台参数规划"
weight: 3
---

云易捷在安装部署前需规划好平台相关参数，主要包括域名、区域 ID、控制台名称等，详细如下。

[options="header",cols="1,3,1"]
|===
|参数 |描述 |规划示例
|域名	
|青云管理控制台等需要域名访问
|bjedu.com

|区域 ID	
|区域 ID，区域的唯一标识通常使用拼音缩写加数字的组合
|bjedu1a

|控制台名称	
|云易捷管理控制平台（Console 平台）上会显示云资源池的名称
|青云云易捷培训平台

|是否支持 RDMA
|此项依赖于网卡是否支持 RDMA 协议。目前云易捷一体机配置的 Mellanox CX4 网卡支持 RDMA
|否

|网络协议
|目前仅支持 http
|http


|管理网络范围	
|管理节点（虚拟主机、VIP）可以占用的管理网络 IP 范围
|172.31.47.2~172.31.47.127

|邮件服务器
|邮件通知、邮件告警等，目前只支持 SMTP server，用户需要提供服务器地址用户名密码
|暂不配置
|===
