用户在**节点管理**页面可以查看当前环境中所有物理节点的基本信息。


. link:../../login[登录 {product_installer_name}]。

. 在左侧导航栏中选择**节点管理**，进入相应页面。在当前页面用户可查看环境中所有物理节点的基本信息，包括**节点名称**、**节点 IP**、**系统架构**、**节点角色**和**相关配置**等信息。 +
如下图所示，当前环境中的第一个节点名称为 testupgrade01，节点 IP 为 172.31.194.10，系统架构为 x86 64位，无 GPU 配置，有 USB 透传功能，CPU 为 12 核，内存为 128 G，该节点角色为 seed、vbr 以及 snapshot。
