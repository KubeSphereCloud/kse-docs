. link:../../login[登录 {product_installer_name}]。

. 系统默认进入**基本信息**页签。在该页面支持用户查看当前云易捷环境的**基本信息**，详细说明如下表所示。
+
image::/images/express/installer/base_info_1.png[]

+
[options="header",cols="1,3"]
|===
|参数 |说明
|域名
|云易捷控制台在浏览器中最终的访问地址，该参数是在安装部署云易捷环境的过程中进行设置的，支持用户在满足域名格式的前提下自定义命名，例如 express.com。

|区域 ID
|区域的唯一标识，该参数是在安装部署云易捷环境的过程中进行设置的，支持用户自定义，通常使用拼音缩写加数字的组合，如 gd2（广东 2 区）。

|控制台名称
|云易捷 Console 管理控制台的名称。该参数是在安装部署云易捷环境的过程中进行设置的，支持用户自定义。

|网络协议
|默认为 http。

|Express 版本
|当前云易捷产品的版本信息，根据安装部署时所使用的安装包，系统自动获取。

|QingCloud 版本
|青云的 IaaS 版本信息，根据安装部署时所使用的安装包，系统自动获取。

|Installer 版本
|当前 {product_installer_name} 的版本信息，根据安装部署时所使用的安装包，系统自动获取

|Patch 版本
|当前环境中所使用的补丁包的版本信息。
|===
