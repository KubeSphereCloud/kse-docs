== 禁用用户

被禁用的用户将无法登录当前平台，需确认用户信息是否正确。

. 登录{product_console_name}。
. 在顶部导航栏，选择**用户管理** > *本地用户*，进入本地用户列表页面。 点击待禁用的本地用户所在行右侧的更多操作，选择**禁用用户**。
+
image::/images/express/console/user_local_setting_17.png[setting_local_user_17]

. 在弹出的**禁用**的窗口中，点击**禁用**即可。
//+
//image::/images/express/console/user_local_setting_18.png[,50%]

. 返回用户列表，此时弹出当前用户被禁用成功的提示符，且该本地用户的状态变为**已禁用**。
+
image::/images/express/console/user_local_setting_19.png[setting_local_user_19]

== 恢复用户

. 登录{product_console_name}。
. 在顶部导航栏，选择**用户管理** > *本地用户*，进入本地用户列表页面。 点击待恢复的本地用户所在行右侧的更多操作，选择**恢复用户**。
+
image::/images/express/console/user_local_setting_20.png[setting_local_user_20]

. 返回用户列表，此时弹出当前用户已被恢复的提示符，此时该用户状态变为**活跃**。
+
image::/images/express/console/user_local_setting_21.png[setting_local_user_21]
