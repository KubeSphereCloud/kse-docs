== 功能描述

付费型计算大屏支持用户自定义配置，允许用户创建单机业务监控系统用于有针对性的监控当前环境中的物理和虚拟资源。

== 前提条件

当前环境已进行计算大屏相关配置的设置。若未配置，则可参考link:../../../ops_tool/gui[计算大屏]相关章节内容进行计算大屏的配置。

== 操作步骤

. 登录{product_console_name}。

. 点击页面顶部导航栏右侧的**大屏展示**，跳转至管理控制平台监控大屏页面。
+
image::/images/express/console/display_charge_1.png[]

. 点击页面左侧栏内的下拉选框可切换至其他面板。
+
image::/images/express/console/display_charge_2.png[]

. 点击页面内右侧栏内的**综合大屏**，可进入**业务综合系统概览**页面。
+
image::/images/express/console/display_charge_3.png[]

. 在业务综合系统概览页面，选择最右侧的**点击查看系统详情**，可进入**单业务系统概览**页面。

+
image::/images/express/console/display_charge_4.png[]
 
== 参数说明

[options="header",cols="1,1,3"]
|===
|监控模块|监控参数|说明

.3+|资源概况
|云主机
|展示当前平台内云主机的总数、云主机运行中的数量以及其他状态的云主机数量。

|云硬盘
|展示所有平台中硬盘总数、硬盘使用中的数量以及硬盘剩余可用数量。

|服务器总览
|展示平台内管理节点、计算节点和存储节点的总数、状态正常或异常的总数。

.2+|存储资源监控
|IOPS
|展示当前平台内计算节点数据盘单位时间内处理 I/O 请求的数量，是衡量系统盘性能的主要指标之一。

|吞吐量
|展示当前平台内计算节点数据盘单位时间可成功传输的数据量，是衡量系统盘性能的主要指标之一。

.3+|虚拟资源占用率
|VCPU
|统计平台内所有计算节点的 CPU 总核数的平均占用率、以及剩余总数、逻辑总数和物理总数。

|内存占用率
|统计平台内所有计算节点的虚拟内存的平均占用率、以及剩余总数、逻辑总数和物理总数。

|存储占用率
|统计平台虚拟存储空间的平均占用率、以及剩余总数、逻辑总数和物理总数。

.2+|各计算节点 24 小时内压力概况
|CPU
|每个计算节点的 CPU 使用率。

|内存
|每个计算节点的内存使用率。

|网络资源池使用率
|-
|平台内所有网络资源池的使用情况，包括每个网络资源池的名称、IP 总数、IP 剩余数已经所处网段。

.2+|24 小时压力趋势
|CPU
|统计平台内所有计算节点 24 小时内的平均 CPU 使用率。

|内存
|统计平台内所有计算节点 24 小时内的平均 CPU 使用率。
|===
