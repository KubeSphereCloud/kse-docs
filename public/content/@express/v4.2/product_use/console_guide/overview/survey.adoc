== 功能描述

管理控制平台为用户提供可直观展示当前平台内所有资源使用情况和运行状态的概览页面，以便用户迅速对资源的扩容和调度做出相应的决策。

平台概览页面，主要包括以下模块：

* 虚拟资源使用情况
* 物理资源使用情况
* 监控告警和操作日志
* 物理资源使用趋势与排行

== 场景描述

用户通过概览页面，可直观快速的掌握平台健康状态、虚拟资源及物理资源的使用和运行情况。

== 操作步骤

. 登录**管理控制平台**。
. 选择**概览**页面，可查看当前平台内所有资源的状态及使用情况。具体参数信息，详见link:#_参数说明[参数说明]。

== 参数说明

* 虚拟资源使用情况
+
[options="header",cols="1,2"]
|===
|参数|说明

|主机
|显示当前平台内所有主机总数以及运行状态中的主机数量。

|硬盘
|显示当前平台内所有硬盘的总数以及状态为使用中的硬盘数量。

|网络 IP
|显示当前平台内所有网络 IP 地址总数以及已分配的网络 IP 数。
|===

* 物理资源使用情况

** 物理资源使用率
+
[options="header",cols="1,2"]
|===
|参数|说明

|CPU
|统计当前平台内所有计算节点的 CPU 总核数后，计算其总的 CPU 使用率。此外，也可查看某一节点的 CPU 使用率。CPU 核数每 2 小时统计一次，使用率每 5 分钟计算一次。

|物理内存
|统计当前平台内所有计算节点的物理内存总量后，计算其总的使用率。此外，也可查看某一节点的物理内存使用率。内存每 2 小时统计一次，使用率每 5 分钟计算一次。

|物理硬盘
|展示当前平台内所有计算节点的物理硬盘的使用率。此外，也可查看某一节点的物理硬盘使用率。硬盘容量每 2 小时统计一次，使用率每 5 分钟计算一次。
|===

** 备份及物理节点状态
+
[options="header",cols="1,2"]
|===
|参数|说明

|备份存储空间
|展示当前平台内备份空间的总量、已使用量以及备份空间的使用率。

|计算节点
|展示当前平台内计算节点总数、运行中计算节点数、异常计算节点数以及待命中计算节点数。

|管理节点
|展示当前平台内管理节点总数、运行中管理节点数以及状态异常的管理节点数。

|服务进程
|展示当前平台内服务进程的总数、运行中的服务进程数以及状态异常的服务进程数。
|===

* 监控告警和操作日志
+
[options="header",cols="1,2"]
|===
|参数|说明

|告警
|展示当前平台内最新的告警运维信息。

|日志
|展示当前平台内最新的操作日志，如创建主机、加载硬盘等操作。
|===

* 物理吞吐和 IO 趋势
+
[options="header",cols="1,2"]
|===
|参数|说明

|IOPS 监控
|展示当前平台内计算节点数据盘单位时间内处理 I/O 请求的数量，是衡量系统盘性能的主要指标之一。

|物理吞吐监控
|展示当前平台内计算节点数据盘单位时间可成功传输的数据量，是衡量系统盘性能的主要指标之一。
|===

* 资源使用排行
+
[options="header",cols="1,2"]
|===
|参数	|说明

|CPU 使用率 TOP5
|展示当前平台内 CPU 使用率过高的前 5 个虚拟机的相关信息和具体的 CPU 使用率。

|内存使用率 TOP5
|展示当前平台内内存使用率过高的前 5 个虚拟机的相关信息和具体的内存使用率。

|系统盘使用率 TOP5
|展示当前平台内系统盘使用率过高的前 5 个虚拟机的相关信息和具体的系统盘使用率。
|===
