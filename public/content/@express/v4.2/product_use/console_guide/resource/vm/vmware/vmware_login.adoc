== 功能描述

该功能实现对现有 Vmware 资源进行纳管，实现在管理控制平台内对所有虚拟机进行管理。

== 场景描述

该章节用于指导用户创建云链接对已有的 Vmware 资源进行纳管。

== 前提条件

{product_console_name}已安装 Vmware 纳管组件。

== 操作步骤

. 登录{product_console_name}。

. 在顶部导航栏，选择**资源** > **虚拟机**，点击 **Vmware 主机**页签，进入 Vmware 主机页面。
+
image::/images/express/console/vmware_login_1.png[]

. 点击**创建云链接**，在弹出的创建云链接窗口中，输入相应参数。
+
image::/images/express/console/vmware_login_2.png[,60%]
+
参数说明
+
[options="header",cols="1,3"]
|===
|参数	|详细说明
|名称
|用户自定义，便于搜索与浏览。

|vCenter 地址
|vCenter 的 IP 地址，根据实际情况进行填写。

|端口
|远端 Vmware 资源的端口号，根据实际情况进行填写。

|账户
|登录远端 Vmware 资源的账户，根据实际情况进行填写。

|密码
|登录远端 Vmware 资源的账户密码，根据实际情况进行填写。

|同步时间设置
|设置与远端 Vmware 资源同步的时间。

|忽略书签发证
|开启后，将会忽略签发证书。

|协议
|选择协议类型.
|===
. 参数配置完成后，点击**测试连接**，验证云连接是否可用。

+

include::../../../../../../../_components/note.adoc[]
云连接测试成功后，会提示有连接成功的提示标语。否则，提示连接失败。

include::../../../../../../../_components/table_footer.adoc[]

. 测试连接验证成功后，点击**创建**即可。

. 云连接创建成功后，点击 Vmware 列表页面的**同步主机**，远端 Vmware 主机将显示在管理控制平台的虚拟机列表页面。
