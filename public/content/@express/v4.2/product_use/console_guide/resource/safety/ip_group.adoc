用户可以将具有相同特征的一组 IP 或者一组端口设置成为 IP/端口集合，并且在安全组规则中进行添加，实现批量管理功能。单个 IP 地址集合最多允许添加 65535 个地址。如需配置超过 65535 个 IP 地址，用户只需创建多个 IP 集合即可。


== 创建 IP /端口组

. 登录{product_console_name}。
. 在顶部导航栏，选择**资源** > *安全* ，进入安全列表页面。
+
image::/images/express/console/ip_group_1.png[]

. 点击 *IP /端口组管理*，在弹出的 **IP /端口组管理**的窗口中，点击**新建 IP 端口组**。
+
image::/images/express/console/ip_group_2.png[]

. 在弹出的**创建 IP 端口组**的窗口中，设置 IP 端口组的名称、类型、IP 或端口范围，点击**创建**即可。
+
image::/images/express/console/ip_group_3.png[]

== 修改 IP /端口组

. 登录{product_console_name}，在顶部导航栏，选择**资源** > *安全* ，进入安全列表页面。
. 点击 *IP /端口组管理*，在弹出的 **IP /端口组管理**的窗口中，点击待修改的 IP /端口组所在行右侧的更多操作，选择**修改**，在弹出的**修改**的窗口中输入 IP /端口组的名称以及 IP 或端口范围，点击**修改**即可。
+
image::/images/express/console/ip_group_4.png[]

== 删除 IP /端口组

. 登录{product_console_name}，在顶部导航栏，选择**资源** > *安全* ，进入安全列表页面。
. 点击 *IP /端口组管理*，在弹出的 **IP /端口组管理**的窗口中，点击待删除的 IP /端口组所在行右侧的更多操作，选择**删除**，在弹出的**提示**的窗口中，点击**删除**即可。
