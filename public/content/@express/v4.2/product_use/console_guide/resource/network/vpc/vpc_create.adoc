== 功能描述

VPC 网络，即虚拟专属私有网络（Virtual Private Cloud，以下简称 VPC 网络）可以帮助用户在 Express 环境中构建专属的私有网络空间，以及为云平台内创建的资源提供隔离的虚拟网络环境。在 VPC 网络中，用户可以自定义 IP 地址范围，创建子网，并在子网内创建云服务器、数据库、大数据等各种云资源。同时，用户也可以通过管理路由器自定义更多高级管理功能。

== 操作步骤

. 登录{product_console_name}。
. 在顶部导航栏，选择**资源** > **网络** > **VPC 网络**，默认进入 VPC 网络列表页面。
+
image::/images/express/console/vpc_create_1.png[]
. 点击**创建 VPC 网络**，在弹出的窗口中配置 VPC 网络的基本信息和初始私有网络的各项参数。
+
image::/images/express/console/vpc_create_2.png[,70%]
+
* 基本信息
+
[options="header",cols="1,3"]
|===
|参数	|说明
|名称
| VPC 网络名称。支持自动填充及自定义。名称长度为 1~64 个字符，可由中文、英文字母、数字、下划线（_）、中划线（-）和点（.）组成

|IPv4 地址范围
|需选择环境内已有的 IPv4 地址范围，VPC 网络创建成功后该地址范围不可修改

|IP 池
|可选项。点击右侧的选择 IP 池，在弹出的窗口中选择可用的 IP 池即可。用于设定 VPC 网络的基础网络。若无可用的 IP 池，可参考link:../../ip_pool/ip_pool_create[创建 IP 池]相关内容进行创建。

|类型
|* 小型路由：管理流量转发能力为 100 kpps
* 中型路由：管理流量转发能力为 150 kpps
* 大型路由：管理流量转发能力为 200 kpps
* 超大型路由：管理流量转发能力为 250 kpps

|安全组
|选择 VPC 网络需要绑定的安全组。若未创建安全组，可选择默认安全组，也可在下方提示信息中点击**新建安全组**进行创建。
|===
+
* 初始私有网络
+
[options="header",cols="1,3"]
|===
|参数	|说明
|名称
|私有网络名称。支持自动填充及自定义

|IPv4 地址范围
|私有网络的地址范围，需要在 VPC 网络的地址范围内，且私有网络间的 IPv4 网段不可重叠。私有网络创建成功后，地址范围将不支持修改。
|===

. （可选）若当前VPC 网络中需要多个私有网络，点击添加私有网络并配置各项参数即可。
+

include::../../../../../../../_components/note.adoc[]
* 至少需要一个私有网络。
* 最多可一次性创建3 个私有网络。

include::../../../../../../../_components/table_footer.adoc[]
+
image::/images/express/console/vpc_create_3.png[,60%]
. VPC 网络各项参数设置完成后，点击确定即可。
