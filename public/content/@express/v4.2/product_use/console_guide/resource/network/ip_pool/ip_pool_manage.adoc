== 场景描述

该章节内容主要用于指导用户如何在管理控制平台对已创建成功的 IP 池进行管理包括：查看指定 IP 池的详细信息、修改属性、绑定/解绑标签等操作。


== 查看 IP 池详情

. 登录{product_console_name}。
. 在顶部导航栏，选择**资源** > **网络** > **IP 池**，进入 **IP 池**列表页面。

. 点击列表内带查看详情的 IP 池名称，进入其相应详情页面，可查看当前 IP 池的基本信息、已关联资源以及网卡的流量监控。
+
image::/images/express/console/ip_pool_manage_3.png[]


. 在 IP 池的详情页面，点击**关联资源**页签，可查看当前 IP 池内已关联资源的基本信息。点击相应资源的名称，可进入其详情页面。
+
image::/images/express/console/ip_pool_manage_4.png[]

. 在 IP 池的详情页面，点击**网卡流量监控**页签，可查看当前 IP 池各资源的网卡流量使用情况及排名。

+
image::/images/express/console/ip_pool_manage_5.png[]

== 修改属性

. 在顶部导航栏，选择**资源** > **网络** > **IP 池**，进入 **IP 池**列表页面。

. 点击待修改的 IP 池所在行右侧的更多操作，选择**修改**，在弹出的**修改属性**的窗口中输入虚拟网卡的新名称以及相关描述，点击**修改**即可。

//+
//image::/images/express/console/ip_pool_manage_6.png[,50%]

== 绑定/解绑标签

. 在顶部导航栏，选择**资源** > **网络** > **IP 池**，进入 **IP 池**列表页面。

. 点击 IP 池所在行右侧的更多操作，选择**标签**，点击**绑定**，在弹出的**绑定标签**的窗口中，选择待绑定至当前 IP 池的标签，点击**绑定**即可。
+

include::../../../../../../../_components/note.adoc[]
* 若当前平台内没有可用的标签，可点击**创建**，即可创建新的标签。
* 可绑定单个或多个标签。

include::../../../../../../../_components/table_footer.adoc[]
//+
//image::/images/express/console/ip_pool_manage_7.png[,50%]

. 点击 IP 池所在行右侧的更多操作，选择 **标签**，点击**解绑**，在弹出的**解绑标签**窗口中，删除待解绑的标签，点击**确定**即可。
//+
//image::/images/express/console/ip_pool_manage_8.png[,50%]

== 删除 IP 池

. 在顶部导航栏，选择**资源** > **网络** > **IP 池**，进入  **IP 池**列表页面。

. 点击 IP 池所在行右侧的更多操作，选择**删除**，在弹出的**删除**的窗口中，点击**删除**即可。
+

include::../../../../../../../_components/attention.adoc[]
* 未被任何资源占用的IP 组才可以被彻底删除。
* 删除操作耗时较长，期间请勿刷新页面或进行其它操作。

include::../../../../../../../_components/table_footer.adoc[]
