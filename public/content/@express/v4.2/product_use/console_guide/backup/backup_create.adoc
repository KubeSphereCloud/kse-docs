== 场景描述

本章节用于指导云易捷用户基于虚拟主机或硬盘创建全量备份或增量备份。


== 基于主机创建备份

详细操作步骤可参考link:../../resource/vm/express_vm/vm_backup[基于主机创建备份]相关内容。

== 基于硬盘创建备份

详细操作步骤可参考link:../../resource/storage/disk_backup[基于硬盘创建备份]相关内容。

== 创建增量备份

. 登录{product_console_name}。
. 在顶部导航栏，选择**备份**，进入**备份存储空间**列表页面。
+
image::/images/express/console/backup_create_1.png[]

. 点击备份名称，进入备份链详情页面。在备份结构示意图中，点击**创建增量备份**图标image:/images/express/console/add_backup.png[]进行创建即可。
+
image::/images/express/console/backup_create_2.png[]
