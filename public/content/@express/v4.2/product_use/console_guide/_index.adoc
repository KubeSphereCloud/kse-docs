== 概述

当前的企业IT 数据中心建设面临着云计算时代的挑战。传统的计算、存储及网络设备构建的竖井式架构已经越来越不能适应企业IT 业务的发展。{product_name}作为“开箱即用”解决方案，继承了 QingCloud 公有云的最佳实践及灵活的架构设计优势，帮助企业低门槛轻松实现云化数据中心建设的第一步。

{product_name}实现 QingCloud 企业云平台易捷版的软硬一体化交付，在计算、存储、网络等核心资源的融合之上，提供企业级开箱即用的云计算平台。同时，{product_name}也是一款云就绪利器，在企业 IT 环境从虚拟化过渡到私有云甚至混合云架构的必经之路上，以“软件定义数据中心”的方式帮助企业按需获取 IT 基础资源和自动运维能力，为业务快速发展和数字化转型奠定坚实的基础。

云易捷 Console 管理控制平台（以下简称{product_console_name}），为用户提供了可视化图形界面。用户可登录该平台实现对云易捷物理和虚拟资源的管理以及运维操作。其功能包括但不限于：

* 物理和虚拟资源的监控管理
* 资源备份
* 运维管理
* 平台用户管理
* 告警配置等

本手册旨在指导用户合理正确地使用{product_console_name}。
