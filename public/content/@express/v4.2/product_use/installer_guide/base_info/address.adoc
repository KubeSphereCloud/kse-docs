. 登录至 {product_installer_name} 基本信息页面。
. 在**访问地址**卡片页，记录当前云易捷环境的 Console 管理控制台的访问地址以及相应的 host 信息。内网环境下，配置当前卡片页上相应的 host 信息后，按下**点击访问**按钮，即可跳转至 Console 控制台页面登录页面。
+

include::../../../../../_components/note.adoc[]

* 访问地址链接是根据**基本信息**中设定的``域名``自动生成，即为 console.{domain_name}。
* host 信息为云易捷环境安装部署后自动生成。
