weight: 2

. 登录至 {product_installer_name}。
. 查看**基本信息**页面。
. 在**访问地址**卡片页，记录当前环境的 Console 平台的访问信息。{product_console_name_en} 可通过 **IP** 和**域名**两种访问方式。
+
* IP 访问方式
+
点击相应的 IP 地址或直接在浏览器中输入 ``http://xx.xx.xx.xx``，即可进入{product_console_name_en} 登录页面。
+
include::../../../../../_components/note.adoc[]
xx.xx.xx.xx 为 IP 地址，需根据实际情况进行修改。
include::../../../../../_components/table_footer.adoc[]
+
* 域名访问方式
+
将当前卡片页上相应的 host 信息保存至客户端，点击相应的域名地址或直接在浏览器中输入 ``http://console.xxxx.com`` 即可进入{product_console_name_en} 登录页面。
+
include::../../../../../_components/note.adoc[]
* 访问地址是根据**基本信息**中设定的``域名``自动生成，即为 console.xxxx.com。
* host 信息为云易捷环境安装部署后自动生成。
include::../../../../../_components/table_footer.adoc[]
