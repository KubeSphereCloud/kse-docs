---
title: "申请基础网络 IP"
weight: 1
---


. 登录{product_console_name_en}。

. 在顶部导航栏，选择**资源** > **网络** > **VPC 网络**，选择**基础网络 IP** 页签，进入基础网络 IP 列表页面，点击**+申请基础网络 IP**。
+
image::/images/express/console/base_net_create_1.png[]

. 在弹出的**申请基础网络 IP** 的窗口中，配置各项参数点击**确定**即可。
+
image::/images/express/console/base_net_create_2.png[,60%]
+
**参数说明**
+
[options="header",cols="1,2"]
|===
|参数 |详细说明 
|名称 
|基础网络 IP 的名称，用户自定义。

|数量 
|基础网络 IP 的申请数量。

|IP 池 
|点击右侧的选择 IP 池，在弹出的窗口中选择可用的 IP 池即 可。若无可用的 IP 池可参考link:../../../ip_pool/ip_pool_create[创建 IP 池]相关内容进行创建。 

|IP 地址 
|根据所选 IP 池的 IP 范围设定具体的 IP 地址。
|===
