== 功能描述
VPC 网络必须与私有网络配合使用，在创建 VPC 网络的过程中，用户须进行至少一个私有网络配置。另外，在 VPC 网络创建完成之后，用户可根据以下步骤添加多个私有网络进行子网划分，同一 VPC 网络内的子网默认私网互通。

== 操作步骤
. 登录{product_console_name_en}。
. 在顶部导航栏，选择**资源** > **网络** > **VPC 网络**，默认进入 VPC 网络列表页面。点击指定 VPC 网络的``名称/ID``，进入其详情页面。当前页面可查看该 VPC 网络的``基本信息``、``私有网络``、``资源列表``、``管理配置``以及``监控``等。
+
image::/images/express/console/vpc_link_vxnet_1.png[]

. 在 VPC 网络详情页面内，选择**私有网络**页签，点击 “+” 图标，弹出的**创建/连接私有网络**窗口。
+
image::/images/express/console/vpc_link_vxnet_2.png[,60%]

. 配置私有网络的基本信息，点击**确定**即可。链接私有网络的方式包括**创建私有网络并连接**和**连接已有私有网络**两种。
+
* 创建私有网络并连接
.. 在创建/连接私有网络窗口中，勾选**创建私有网络并连接**。
.. 配置私有网络名称、工作模式、IPv4 网络地址、管理地址、DHCP 起始和结束地址后，点击**确定**。
+

include::../../../../../../_components/note.adoc[]
* 工作模式：目前仅支持虚拟机。
* IPv4 网络地址：私有网络的地址范围，需要在 VPC 网络的地址范围内。

include::../../../../../../_components/table_footer.adoc[]

* 连接已有的私有网络
+
.. 在创建/链接私有网络窗口中，勾选链接已有私有网络。
.. 点击选择要加入的私有网络，在弹出的私有网络列表中，勾选单个待加入的私有网络，点击确定。
+

include::../../../../../../_components/note.adoc[]
此处必须选择未加入 VPC 网络的空闲私有网络。

include::../../../../../../_components/table_footer.adoc[]
+
image::/images/express/console/vpc_link_vxnet_3.png[,60%]
.. 配置 IPv4 网络地址、管理地址、DHCP 起始和结束地址后，点击**确定**即可。

. VPC 网络与私有网络连接建立成功后，选择指定私有网络，点击**资源列表**右侧的**+创建虚拟机**，即可基于当前网络配置创建主机。
+
image::/images/express/console/vpc_link_vxnet_4.png[]
