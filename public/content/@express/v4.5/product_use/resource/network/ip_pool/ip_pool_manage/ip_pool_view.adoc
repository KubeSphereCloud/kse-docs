== 场景描述

该章节内容主要用于指导用户如何在管理控制平台查看指定 IP 池的详细信息。


== 操作步骤

. 登录{product_console_name_en}。
. 在顶部导航栏，选择**资源** > **网络** > **IP 池**，进入 **IP 池**列表页面。

. 点击列表内带查看详情的 IP 池名称，进入其相应详情页面，可查看当前 IP 池的基本信息、已关联资源以及网卡的流量监控。
+
image::/images/express/console/ip_pool_manage_3.png[]


. 在 IP 池的详情页面，点击**关联资源**页签，可查看当前 IP 池内已关联资源的基本信息。点击相应资源的名称，可进入其详情页面。
+
image::/images/express/console/ip_pool_manage_4.png[]

. 在 IP 池的详情页面，点击**网卡流量监控**页签，可查看当前 IP 池各资源的网卡流量使用情况及排名。

+
image::/images/express/console/ip_pool_manage_5.png[]
