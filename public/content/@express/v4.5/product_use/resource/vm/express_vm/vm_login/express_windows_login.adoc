title: "Windows 系统虚拟机"
weight: 3
---

从安全考虑，平台上的 Windows 主机默认关闭了远程登录，用户需通过浏览器 Web 方式登录到主机，并开启远程登录功能。

. 进入虚拟机列表页面，打开**远程连接桌面**，连接 Windows Server。
+
image::/images/express/console/express_windows_login_1.png[]

. 在 Web 登录页面，点击左上角 Ctrl-Alt-Del ，然后输入设置的密码。
+
image::/images/express/console/express_windows_login_2.png[]

. 同意内部网络共享。登录后会弹出网络共享的界面，点击**是**，允许 VPC 内部的网络。
. 在 Windows 主机中，点击下方**文件管理器**，点击 *此电脑* > *计算机* > *系统属性*,打开系统属性.
+
image::/images/express/console/express_windows_login_3.png[,60%]

. 在弹出的系统设置页面，点击远程桌面，打开**启用远程桌面**的按钮。
+
image::/images/express/console/express_windows_login_4.png[,60%]
