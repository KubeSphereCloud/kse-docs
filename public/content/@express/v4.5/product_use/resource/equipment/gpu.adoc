== 功能描述

云平台支持 GPU 设备透传服务，物理机 GPU 可直接透传至该物理机上所运行的虚拟机，从而让虚拟机能够直接使用物理机上的 GPU 。

通过将 GPU 卡插入至任意节点上，列表自动识别到后，可在虚拟机管理页面创建 GPU 主机。


== 前提条件

* 已获得登录{product_console_name_en}的账户和密码。
* 已将 GPU 设备插入物理机，且设备的状态为``可用``。

== 操作步骤

. 登录管理平台。
. 在顶部导航栏，选择**资源** > *设备透传* > *GPU 设备*，进入 GPU 设备管理页面。
. 已被平台识别出的 GPU 设备会显示在 GPU 设备管理页面，且状态为 ``可用``。
. 在左侧导航栏中选择**虚拟机**，在虚拟机管理页面可以创建 GPU 主机。

. GPU 主机创建成功后，返回  GPU 设备管理页面，可以看到该 GPU 设备状态显示``使用中``。
