== 限制条件

* 仅**管理员**角色用户可执行删除用户操作。
* 平台内自带的默认用户不允许删除，即平台内用户 ID 为 ``installer`` 或 ``admin`` 的用户。

== 操作步骤

. 登录{product_console_name_en}。
. 在顶部导航栏，选择**用户管理** > *本地用户*，进入本地用户列表页面。 

. 点击待删除的本地用户所在行右侧的更多操作，选择**删除**。
+
image::/images/express/console/user_local_setting_22.png[]

. 在弹出的**删除本地用户**提示窗口中，点击**删除**即可。
+
include::../../../../../../_components/attention.adoc[]
* 用户被删除后，其账户下的所有资源将全部被清除，请谨慎操作。
* 若被删除的账户为管理员角色，则其下的所有子账户也将被删除。
