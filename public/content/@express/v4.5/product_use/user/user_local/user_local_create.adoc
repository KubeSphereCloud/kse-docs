== 功能描述

云易捷支持管理用户在本地用户页面，创建多个不同角色的**本地用户**，且每个本地用户之间的资源是相互隔离的，用户可自主管理和部署其名下各种资源。除此之外，不同角色的用户其针对{product_console_name_en}控制台的权限不同。 

* 管理用户角色的控制台权限包括：概览、物理资源管理、资源管理、备份管理、公有云接入、运维工具以及用户管理。
* 普通用户角色的控制台权限包括：概览、资源管理、备份管理、公有云接入以及运维工具。


== 场景描述

该章节主要用于指导管理用户在创建新的本地用户。

== 操作步骤

. 登录{product_console_name_en}。
. 在顶部导航栏，选择**用户管理** > *本地用户*，进入本地用户列表页面。
+
image::/images/express/console/user_local_create_1.png[create_local_user_1]

. 点击**创建用户**。
+
image::/images/express/console/user_local_create_2.png[create_local_user_2]

. 在弹出的创建用户窗口中，输入登录邮箱和密码，点击**创建**即可。
+
image::/images/express/console/user_local_create_3.png[,50%]
