== 功能描述

第三方登录列表中显示了当前平台内所有第三方登录服务器的状态、类型、服务器地址/端口、基准以及创建时间等相关信息。在该页面用户可以通过用户 ID、用户名称以及状态对列表信息进行筛选。

== 场景描述

该章节主要用于指导用户查看当前平台内所以第三方登录服务器的相关信息。

== 操作步骤

. 登录{product_console_name_en}。
. 在顶部导航栏，选择**用户管理** > *第三方登录*，进入第三方登录列表页面，可查看所有第三方服务器的名称 / ID、状态等相关信息。
+
image::/images/express/console/user_third_view_1.png[view_third_user_1]

. 在列表上方的搜索框中输入用户名称 / ID 或者用户状态可对列表进行筛选。
+
image::/images/express/console/user_third_view_2.png[view_third_user_2]

. 点击第三方服务器名称可跳转至其详情页面。

image::/images/express/console/user_third_view_3.png[view_third_user_3]

. 详情页面显示了该第三方登录服务器的状态、名称、服务器类型、服务器地址/端口、基准、创建时间以及该服务器下关联的所有用户信息。
+
image::/images/express/console/user_third_view_4.png[view_third_user_4]

. 在第三方登录的详情页面，可对当前第三方服务器下关联的用户进行禁用。
+
image::/images/express/console/user_third_view_5.png[view_third_user_5]
