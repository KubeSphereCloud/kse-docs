== 功能描述

配置中心的**全局设置**用于对当前云易捷平台内的某些配置进行全局性的统一设置，主要包括 CPU 和内存的超卖比、平台回收站资源保存周期以及分布式资源调度。


== 场景描述

该章节用于指导用户在进入**配置中心**的**全局设置**页面，进行 CPU 超卖比、内存超卖比以及回收站保存周期进行设置。

== 操作步骤

. 登录{product_console_name_en}。
. 在控制台顶部导航栏右侧，将鼠标移动至用户名上方，打开**配置中心**，并点击**高级设置** > **全局设置**。

. 在配置中心窗口的**全局设置**页面，支持对 **CPU 超卖比**、**内存超卖比**、**回收站保存周期**以及**分布式资源调度**进行自主设置。
+
* ``CPU 超卖比``：用于设置当前平台内 CPU 资源的超卖比，默认为 3，支持最大修改为 100。
* ``内存超卖比``：用于设置当前平台内内存资源的超卖比，默认不超卖，用户可根据实际情况进行修改。
* ``回收站保存周期``：用于设置当前平台内回收站中的资源保存周期，支持设置 1~30 天，超过期限的资源将被彻底销毁。
* ``分布式资源调度``：设置相应数值后，当集群中的某一计算节点的内存资源剩余百分比低于当前值时，云易捷平台会自动将该节点上的虚拟机迁移至其他内存资源充足的计算节点。默认值为 0。
