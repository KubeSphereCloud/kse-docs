将任意数量密钥对加载到任意数量的{instance_product_name_cn}， {instance_product_name_cn}状态须为``运行中``（ running ）或``已关机``（ stopped ）。

*Request Parameters*

[option="header",cols="1,1,2,1"]
|===
| Parameter name | Type | Description | Required

| keypairs.n
| String
| 密钥 ID
| Yes

| instances.n
| String
| {instance_product_name_cn} ID
| Yes

| zone
| String
| 区域 ID，注意要小写
| Yes
|===

link:../../../introduction/parameters/[公共参数]

*Response Elements*
[option="header",cols="1,1,2"]
|===
| Name | Type | Description

| action
| String
| 响应动作

| job_id
| String
| 执行任务的 Job ID

| ret_code
| Integer
| 执行成功与否，0 表示成功，其他值则为错误代码
|===

*Example*

Example Request

----
https://api.domain.com/iaas/?action=AttachKeyPairs
&keypairs.1=kp-12djpg8q
&instances.1=i-12345678,
&zone=testa
&COMMON_PARAMS
----

Example Response

----
{
  "action":"AttachKeyPairsResponse",
  "job_id":"j-bm6ym3r8",
  "ret_code":0
}
----
