启动一台或多台关闭状态的{instance_product_name_cn}。

{instance_product_name_cn}只有在关闭 stopped 状态才能被启动，如果处于非关闭状态，则返回错误信息。

*Request Parameters*

[option="header",cols="1,1,2,1"]
|===
| Parameter name | Type | Description | Required

| instances.n
| String
| 一个或多个{instance_product_name_cn}ID
| Yes

| zone
| String
| 区域 ID，注意要小写
| Yes
|===

link:../../../introduction/parameters/[公共参数]

*Response Elements*

[option="header",cols="1,1,2"]
|===
| Name | Type | Description

| action
| String
| 响应动作

| job_id
| String
| 执行任务的 Job ID

| ret_code
| Integer
| 执行成功与否，0 表示成功，其他值则为错误代码
|===

*Example*

Example Request:

----
https://api.domain.com/iaas/?action=StartInstances
&instances.1=i-rtyv0968
&zone=testa
&COMMON_PARAMS
----

Example Response:

----
{
  "action":"StartInstancesResponse",
  "job_id":"j-ybnoeitr",
  "ret_code":0
}
----
