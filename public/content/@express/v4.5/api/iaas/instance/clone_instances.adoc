拷贝一台或多台机器。

拷贝{instance_product_name_cn}的前提，是此{instance_product_name_cn}对应的镜像可用、没有被删除掉。{instance_product_name_cn}处于运行或关机状态。

include::../../../../../_components/note.adoc[]
拷贝后的{instance_product_name_cn}处于运行状态。原{instance_product_name_cn}的硬盘等也会拷贝并挂载到目标{instance_product_name_cn}。
include::../../../../../_components/table_footer.adoc[]

*Request Parameters*

[option="header",cols="1,1,2,1"]
|===
| Parameter name | Type | Description | Required

| instances.n
| String
| 一个或多个{instance_product_name_cn}ID
| Yes

| vxnets.n
| String
| 一个或多个{instance_product_name_cn}克隆生成的新{instance_product_name_cn}的vxnet或ip地址 +
格式是: 'instance_id\|vxnet_id\|ip_addr'
| No

| zone
| String
| 区域 ID，注意要小写
| Yes
|===

link:../../../introduction/parameters/[公共参数]

*Response Elements*

[option="header",cols="1,1,2"]
|===
| Name | Type | Description

| action
| String
| 响应动作

| job_id
| String
| 执行任务的 Job ID

| ret_code
| Integer
| 执行成功与否，0 表示成功，其他值则为错误代码
|===

*Example*

Example Request:

[,shell]
----
https://api.domain.com/iaas/?action=CloneInstances
&instances.1=i-xrvfi3jx
&instances.2=i-w2kcs8vb
&vxnets.1='i-xrvfi3jx|vxnet-q57edqv|172.30.10.21'
&vxnets.2='i-w2kcs8vb|vxnet-3m9hetp'
&zone=testa
&COMMON_PARAMS
----
