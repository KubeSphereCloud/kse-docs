从回收站销毁一台或多台{instance_product_name_cn}。

销毁{instance_product_name_cn}的前提，是此{instance_product_name_cn}已被删除并进入回收站。

include::../../../../../_components/note.adoc[]
已销毁的{instance_product_name_cn}会直接从回收站销毁，无法再恢复。
include::../../../../../_components/table_footer.adoc[]

*Request Parameters*

[option="header",cols="1,1,2,1"]
|===
| Parameter name | Type | Description | Required

| instances.n
| String
| 一个或多个{instance_product_name_cn}ID
| Yes

| zone
| String
| 区域 ID，注意要小写
| Yes
|===

link:../../../introduction/parameters/[公共参数]

*Response Elements*

[option="header",cols="1,1,2"]
|===
| Name | Type | Description

| action
| String
| 响应动作

| job_id
| String
| 执行任务的 Job ID

| ret_code
| Integer
| 执行成功与否，0 表示成功，其他值则为错误代码
|===

*Example*

Example Request:

[,shell]
----
https://api.domain.com/iaas/?action=CeaseInstances
&instances.1=i-rtyv0968
&zone=testa
&COMMON_PARAMS
----

_xample Response:

[,shell]
----
{
  "action":"CeaseInstancesResponse",
  "job_id":"j-ybnoeitr",
  "ret_code":0
}
----
