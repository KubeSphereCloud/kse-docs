获取一个或多个{instance_product_name_cn}

可根据{instance_product_name_cn} ID, 状态, {instance_product_name_cn}名称, 镜像 ID 作过滤条件, 来获取{instance_product_name_cn}列表。 如果不指定任何过滤条件, 默认返回你所拥有的所有{instance_product_name_cn}。

*Request Parameters*

[option="header",cols="1,1,2,1"]
|===
| Parameter name | Type | Description | Required

| instances.n
| String
| {instance_product_name_cn}ID
| No

| image_id.n
| String
| 一个或多个镜像ID
| No

| instance_class
| Integer
| {instance_product_name_cn}性能类型: 默认为:0
| No

| vcpus_current
| Integer
| {instance_product_name_cn}CPU的核心数
| No

| memory_current
| Integer
| {instance_product_name_cn}内存大小
| No

| os_disk_size
| Integer
| {instance_product_name_cn}系统盘大小
| No

| status.n
| String
| {instance_product_name_cn}状态: pending, running, stopped, suspended, terminated, ceased
| No

| search_word
| String
| 搜索关键词, 支持{instance_product_name_cn}ID, {instance_product_name_cn}名称
| No

| tags.n
| String
| 按照标签ID过滤, 只返回已绑定某标签的资源
| No

| owner
| String
| 按照用户账户过滤, 只返回指定账户的资源
| No

| verbose
| Integer
| 是否返回冗长的信息, 若为1, 则返回{instance_product_name_cn}相关其他资源的详细数据。
| No

| offset
| Integer
| 数据偏移量, 默认为0
| No

| limit
| Integer
| 返回数据长度，默认为20，最大100
| No

| zone
| String
| 区域 ID，注意要小写
| Yes
|===

link:../../../introduction/parameters/[公共参数]

*Response Elements*

[option="header",cols="1,1,2"]
|===
| Name | Type | Description

| action
| String
| 响应动作

| instance_set
| Array
| JSON 格式的{instance_product_name_cn}数据列表, 每项参数可见下面 Response Item

| total_count
| Integer
| 根据过滤条件得到的{instance_product_name_cn}总数
|===

*Response Item*

[option="header",cols="1,1,2"]
|===
| Name | Type | Description

| instance_id
| String
| {instance_product_name_cn}ID

| instance_name
| String
| {instance_product_name_cn}名称

| description
| String
| {instance_product_name_cn}描述

| vcpus_current
| Integer
| {instance_product_name_cn}CPU核心数目。

| memory_current
| Integer
| {instance_product_name_cn}内存大小, 单位为MB。

| status
| String
a| {instance_product_name_cn}状态, 有效值为pending, running, stopped, suspended, terminated, ceased。 +

* pending： 等待被创建 
* running： 运行中 
* stopped： 已关机 
* suspended： 由于欠费, 已被暂停使用 
* terminated： 已被删除, 但处于此状态的{instance_product_name_cn}在2小时之内仍可以被恢复为 running 状态 
* ceased： 已被彻底删除, 处于此状态的{instance_product_name_cn}无法恢复

| transition_status
| String
a| {instance_product_name_cn}过渡状态, 有效值为creating, starting, stopping, restarting, suspending, resuming, terminating, recovering, resetting。 +

* creating： 创建中, 由 pending 状态变成 running 状态 
* starting： 启动中, 由 stopped 状态变成 running 状态 
* stopping： 关闭中, 由 running 状态变成 stopped 状态 
* restarting： 重启中 
* suspending： 欠费暂停中, 由 running/stopped 状态变成 suspended 状态 
* resuming： 恢复中, 由 suspended 状态变成 running 状态 
* terminating： 删除中, 由 running/stopped/suspended 状态变成 terminated 状态 
* recovering： 恢复中, 由 terminated 状态变成 running 状态 
* resetting： 操作系统重置中

| create_time
| TimeStamp
| {instance_product_name_cn}创建时间, 为UTC时间, 格式可参见 link:http://www.w3.org/TR/NOTE-datetime[ISO8601].

| status_time
| TimeStamp
| {instance_product_name_cn}最近一次状态变更时间, 为UTC时间, 格式可参见 link:http://www.w3.org/TR/NOTE-datetime[ISO8601].

| image
| Dict
a| {instance_product_name_cn}的模板信息。 +

* processor_type： 支持的处理器类型, 有效值为 64bit 和 32bit 
* platform： 模板平台, 有效值为 linux 和 windows 
* os_family： 模板操作系统平台, 有效值为 windows, centos, ubuntu, debian, fedora等 
* image_size： 模板大小 
* image_name： 模板名称 
* provider： 模板提供方, 有效值为 系统模板( system ), 自有模板 ( self )

| vxnets
| Array
a| {instance_product_name_cn}加入的私有网络信息。 +

* vxnet_id： 私有网络ID 
* vxnet_name： 私有网络名称 
* vxnet_type： 私有网络类型, 1 为受管私有网络, 0 为自管私有网络。 
* nic_id： {instance_product_name_cn}加入私有网络的网卡ID 
* private_ip： {instance_product_name_cn}位于私有网络中的私有地址

| eip
| Dict
a| {instance_product_name_cn}绑定的公网 IP 信息。 +

* eip_id： 公网 IP 的 ID 
* eip_addr： IP 地址 
* bandwidth： 带宽(Mbps)

| security_group
| Dict
a| {instance_product_name_cn}的防火墙信息。 +

* security_group_id： 防火墙ID 
* is_default： 是否为默认防火墙, 1 为是, 0 为不是。 
* 只有在请求参数 verbose=1 时才会返回此信息。

| volume_ids
| Array
| {instance_product_name_cn}上挂载的硬盘ID +
只有在请求参数 verbose=1 时才会返回此信息。

| keypair_ids
| Array
| {instance_product_name_cn}上加载的SSH密钥ID +
只有在请求参数 verbose=1 时才会返回此信息。

| graphics_protocol
| String
| 图形化访问支持协议

| graphics_passwd
| String
| 图形化访问密码
|===

*Example*

Example Request

----
https://api.domain.com/iaas/?action=DescribeInstances
&instances.1=i-ogbndull
&status.1=running
&verbose=1
&COMMON_PARAMS
----
