删除一个或多个网卡

删除网卡的前提是没有{instance_product_name_cn}挂载这个网卡。请在删除网卡之前，先从{instance_product_name_cn}卸载。

*Request Parameters*

[option="header",cols="1,1,2,1"]
|===
| Parameter name | Type | Description | Required

| Nics.n
| String
| 网卡 ID
| Yes

| zone
| String
| 区域 ID，注意要小写
| Yes
|===

link:../../../introduction/parameters/[公共参数]

*Response Elements*

|===
| Name | Type | Description

| action
| String
| 响应动作

| ret_code
| Integer
| 执行成功与否，0 表示成功，其他值则为错误代码
|===

*Example*

Example Request:

----
https://api.domain.com/iaas/?action=DeleteNics
&nics.1="52:54:00:00:12:34",
&zone=testa
&COMMON_PARAMS
----

Example Response:

----
{
  "action":"DeleteNicsResponse",
  "ret_code":0
}
----
