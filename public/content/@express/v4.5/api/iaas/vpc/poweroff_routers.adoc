关闭一台或多台 VPC 网络。

include::../../../../../_components/note.adoc[]
VPC 网络只有在运行 active 状态才能被关闭，如果处于非运行状态，则返回错误信息。
include::../../../../../_components/table_footer.adoc[]

== 请求参数

|===
| 参数 | 参数类型 | 描述 | 是否必选

| routers.n
| String
| VPC 网络ID
| Yes

| zone
| String
| 区域 ID，注意要小写
| Yes
|===

link:../../../introduction/parameters/[公共参数]

== 返回数据

|===
| 参数 | 参数类型 | 描述

| action
| String
| 响应动作

| job_id
| String
| 执行任务的 Job ID

| ret_code
| Integer
| 执行成功与否，0 表示成功，其他值则为错误代码
|===

== 示例

*请求示例：*
[source]
----
https://api.domain.com/iaas/?action=PowerOffRouters
&routers.1=rtr-s5nqo8mr
&zone=testa
&COMMON_PARAMS
----

*返回示例：*
[source]
----
{
  "action":"PowerOffRoutersResponse",
  "ret_code":0,
  "job_id":"j-32c66ja8",
  "routers":[
    "rtr-s5nqo8mr"
  ]
}
----
