---
title: 检查用户会话（CheckSession）
weight: 1
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| session_type
| String
| false
| 会话类型

| sk
| String
| true
| 会话key
|===

== 返回数据
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| owner
| String
| false
| 所属用户id
|===

== 请求示例
[,javascript]
----
{
  "action": "CheckSession",
  "session_type": "web",
  "sk": "6XJVdYRNssepxiW8mMXk6cj6x9yvyIfD"
}
----

== 返回示例
[,javascript]
----
{
  "action": "CheckSessionResponse",
  "owner": "usr-dQFUel6D",
  "ret_code": 0
}
----
