title: 修改第三方APP属性（ModifyThirdAppAttributes）
weight: 7
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| client_id
| String
| true
| 客户端ID

| client_name
| String
| false
| 客户端Name

| description
| String
| false
| 第三方APP的描述信息

| logo_uri
| String
| false
| logo对应的uri地址

| logout_uri
| String
| false
| 登出对应的uri地址

| redirect_uri
| String
| false
| 重定向的uri地址

| status
| String
| false
| 

| token_issuer
| String
| false
| 
|===

== 请求示例
[,javascript]
----
{
  "client_id": "client_id",
  "client_name": "client_name",
  "description": "this is thrid app description",
  "logo_uri": "https://logo_uri.com",
  "logout_uri": "https://logout_uri.com",
  "redirect_uri": "https://redirect_uri.com",
  "status": "activated",
  "token_issuer": "sso"
}
----
