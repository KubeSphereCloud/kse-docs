---
title: 重置账户密码（ResetPassword）
weight: 31
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| passwd
| String
| true
| 密码

| user
| String
| true
| 用户
|===

== 返回数据
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| email
| String
| false
| 邮箱

| user_id
| String
| false
| 用户ID
|===

== 请求示例
[,javascript]
----
{
  "passwd": "********",
  "user": "admin"
}
----

== 返回示例
[,javascript]
----
{
  "action": "ResetPasswordResponse",
  "email": "4743******",
  "ret_code": 0,
  "user_id": "admin"
}
----
