---
title: 恢复账户（RestoreUsers）
weight: 33
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| users
| Array
| true
| 待重置的用户列表
|===

== 返回数据
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| users
| Array
| false
| 完成重置的用户列表
|===

== 请求示例
[,javascript]
----
{
  "users": [
    "admin",
    "normal_user"
  ]
}
----

== 返回示例
[,javascript]
----
{
  "action": "RestoreUsersResponse",
  "ret_code": 0,
  "users": [
    "admin",
    "normal_user"
  ]
}
----
