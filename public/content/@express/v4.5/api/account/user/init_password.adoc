---
title: 初始化账户密码（InitPassword）
weight: 22
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| is_init_passwd_first
| Integer
| true
| 首次是否需要初始化密码

| passwd
| String
| true
| 密码

| user
| String
| false
| 用户
|===

== 返回数据
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| user_id
| String
| false
| 用户ID
|===

== 请求示例
[,javascript]
----
{
  "is_init_passwd_first": 0,
  "passwd": "******",
  "user": "admin"
}
----

== 返回示例
[,javascript]
----
{
  "action": "InitPasswordResponse",
  "ret_code": 0,
  "user_id": "admin"
}
----
