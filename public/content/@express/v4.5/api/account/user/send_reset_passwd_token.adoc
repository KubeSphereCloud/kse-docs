---
title: 颁发重置密码的令牌（SendResetPasswdToken）
weight: 37
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| console
| String
| false
| 控制台ID

| email
| String
| true
| 邮箱地址

| init_password
| Integer
| false
| 初始化密码
|===

== 返回数据
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| email
| String
| false
| 邮箱地址

| left
| Integer
| false
| 

| token
| String
| false
| 

| user_id
| String
| false
| 用户ID
|===

== 请求示例
[,javascript]
----
{
  "console": "console_0",
  "email": "474******",
  "init_password": 0
}
----

== 返回示例
[,javascript]
----
{
  "action": "SendResetPasswdTokenResponse",
  "email": "474******",
  "left": 5,
  "ret_code": 0,
  "token": "token_0",
  "user_id": "admin"
}
----
