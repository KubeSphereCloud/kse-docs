:ks_navigation: **云账号**


本节介绍如何管理 KubeSphere 云账号下的订阅。

== 前提条件

* 您需要在{ks_product_both}平台具有 **platform-admin** 角色。

* {ks_product_right}平台已绑定 KubeSphere 云账号。

== 操作步骤

. 以具有 **platform-admin** 角色的用户登录{ks_product_left} Web 控制台。

. 在页面右上角点击image:/images/ks-qkcp/zh/icons/cogwheel.svg[cogwheel,18,18]进入**平台设置**页面。

+

include::../../../../../_ks_components/oper-navigate.adoc[]

. 点击**更多操作**，选择**订阅管理**。

. 在**我的订阅**页面，查看{ks_product_both}及各组件的订阅状态、订阅计划、订阅时长、到期时间等。
+
--
* 在列表上方点击搜索框并设置搜索条件，可按名称、订阅状态搜索您的订阅。

include::../../../../../_ks_components/oper-refreshListData.adoc[]

include::../../../../../_ks_components/oper-customizeColumns.adoc[]
--

. 在订阅右侧点击image:/images/ks-qkcp/zh/icons/more.svg[more,18,18]，选择**查看订单**，查看订单的详细信息。
// +
// --
// 对于快到期的订阅，可选择**续费**。

// 对于已到期的订阅，可选择**订阅**重新购买。

// 若要升级订阅方案，点击**升级**。
// --
