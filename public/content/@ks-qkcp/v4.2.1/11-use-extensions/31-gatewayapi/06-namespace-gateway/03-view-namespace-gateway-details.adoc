本节介绍如何查看项目网关的详情。

== 前提条件

* KubeSphere 平台需要安装并启用 *Gateway API 网关* 扩展组件。
* 您需要加入一个项目并在项目中具有 *项目 Gateway API 网关查看* 权限。

== 操作步骤

. 以具有 *项目 Gateway API 网关查看* 权限的用户登录 KubeSphere Web 控制台并进入您的项目。

. 在左侧导航栏选择 *Gateway API 网关 > 网关*。

. 在 *项目网关* 页签，表格显示该项目的网关，点击网关名称进入详情页。

. 在 *属性* 区域查看网关的属性。
+
[%header,cols="1,3"]
|===
| 参数 | 描述

| 状态
| 网关的状态：可用、不可用、未知。未知通常表示其绑定的网关实现不存在或不可用；不可用通常由配置异常导致。

| 网关实现
| 网关绑定的网关实现。

| 访问地址
| 网关的外部访问地址。

| 创建者
| 创建该网关的用户。

| 创建时间
| 网关的创建时间。

|===

. 在 *监听器* 页签下，查看该网关所有监听器的信息。

. 在 *应用路由* 页签下，按类型查看绑定到该网关的所有路由。
