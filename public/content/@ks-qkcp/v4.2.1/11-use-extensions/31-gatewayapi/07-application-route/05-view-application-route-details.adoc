本节介绍如何删除 Gateway API 应用路由的信息。

## 前提条件

您需要加入一个项目并在项目中具有 Gateway API 应用路由的管理权限。

## 操作步骤

1. 以具有 Gateway API 应用路由的管理权限的用户登录 KubeSphere Web 控制台并进入您的企业空间。

2. 在左侧导航栏选择 **服务与网格 > Gateway API 应用路由**。

3. 根据您需要删除的路由类型，点击相应的页签。切换项目查找不同项目的路由。

[%header,cols="1,3"]
|===
| 参数 | 描述

| 名称
| 路由名称。

| 域名
| 配置的域名限制。

| 网关
| 路由所使用的网关。

| 创建时间
| 路由的创建时间。

|===


* 找到您需要删除的路由条目，在最右侧一列，选择**删除**，然后点击**确定**。
