本节介绍如何查看企业空间网关实现的信息。

## 前提条件

- KubeSphere 平台需要安装并启用 **Gateway API 网关**扩展组件。

- 您需要加入一个企业空间并在企业空间中具有 **企业空间 Gateway API 网关查看** 权限。

## 操作步骤

1. 以具有 **企业空间 Gateway API 网关查看** 权限的用户登录 KubeSphere Web 控制台并进入您的企业空间。

2. 在左侧导航栏选择 **Gateway API 网关 > 网关实现**。

3. 在**企业空间网关实现**页签，查看企业空间网关实现的信息。

[%header,cols="1,4"]
|===
| 参数 | 描述

| 状态
| 网关实现的状态：正常、处理中、异常。

| 访问模式
a|
网关实现服务的外部访问模式：

* *NodePort*：将节点端口映射到服务端口，可通过节点 IP 和端口访问服务。
* *LoadBalancer*：在 NodePort 基础上分配外部 IP，并绑定外部负载均衡器。
+
负载均衡器监听节点端口，从而通过外部 IP 地址访问服务。
+
该模式依赖负载均衡器插件和基础设施支持，并需要配置相关注解。

| 节点端口
| 网关实现服务的 TCP 和 TCP/TLS 节点端口，仅在访问模式为 NodePort 时显示。

| 网关
| 绑定到该网关实现的网关数量。

| 创建时间
| 网关实现的启用时间。

|===
