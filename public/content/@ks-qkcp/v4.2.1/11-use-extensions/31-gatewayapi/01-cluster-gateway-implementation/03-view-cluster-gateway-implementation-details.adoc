本节介绍如何查看集群网关实现的详情。

== 前提条件

* KubeSphere 平台需要安装并启用 *Gateway API 网关* 扩展组件。
* 您需要加入一个集群并在集群中具有 *cluster-admin* 权限。

== 操作步骤

. 以具有 *cluster-admin* 权限的用户登录 KubeSphere Web 控制台并进入您的集群。

. 在左侧导航栏选择 *Gateway API 网关 > 网关实现*。

. 在 *集群网关实现* 页签，点击网关实现名称进入详情页。

. 在 *属性* 区域查看网关实现的属性。
+
[%header,cols="1,3"]
|===
| 参数 | 描述

| 状态
| 网关的状态：正常、处理中、异常。

| 控制器
| 控制器类型。

| 创建时间
| 网关的创建时间。

|===

. 在 *网关* 页签下，查看绑定到该网关实现的网关实例。

. 在 *监控* 页签下，查看请求数据的相关指标。
+
*说明：如需使用该监控功能，KubeSphere 需启用 **WizTelemetry 监控** 扩展组件。*
+
* *概览* 区域提供以下监控信息：
+
[%header,cols="1,4"]
|===
| 参数 | 描述

| 总请求数量
| 指定时间范围内接收的请求总数。

| 请求数量
| 指定时间范围内响应状态码为 4XX 或 5XX 的请求数量。点击下拉列表可分别查看 4XX 和 5XX 请求数量。

| 成功请求
| 指定时间范围内成功请求的百分比。

| 平均延迟
| 指定时间范围内所有请求的平均延迟。

| P50 延迟
| 指定时间范围内所有请求延迟的中位数。

| P95 延迟
| 指定时间范围内所有请求的 95% 分位延迟，即 95% 的请求延迟小于或等于该数值。

|===
+
* 下方以折线图显示当前选定容器组的相关指标随时间的变化趋势。

. 在 *资源状态* 页签下，查看该网关实现的负载状态，包括副本数、容器组和服务端口等。

. 在 *日志* 页签下，查看该网关实现的负载容器日志，可通过右上角按钮执行实时刷新的启动或停止、手动刷新和下载操作。
+
*说明：如需长期存储网关实现的日志或使用日志搜索功能，KubeSphere 需安装并启用 **WizTelemetry 日志** 扩展组件。*

. 在 *元数据* 页签下，查看网关实现服务的注解。在服务以 LoadBalancer 方式暴露时，通常为与负载均衡器相关的注解信息。
