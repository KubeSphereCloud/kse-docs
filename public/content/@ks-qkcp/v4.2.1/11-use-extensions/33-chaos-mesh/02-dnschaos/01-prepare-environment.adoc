本节介绍如何创建测试部署，用于后续验证 DNS 故障注入效果。

*创建测试部署*

[source,yaml]
----
apiVersion: apps/v1
kind: Deployment
metadata:
name: nginx-dns
namespace: default
spec:
replicas: 1
selector:
 matchLabels:
   app: nginx-dns
template:
 metadata:
   labels:
     app: nginx-dns
 spec:
   containers:
     - name: nginx
       image: nginx:stable-perl
       ports:
         - containerPort: 80
----


// note
[.admon.note,cols="a"]
|===
|说明

|
如需在 Pod 内执行 DNS 查询验证，可在容器中安装 `dnsutils` 工具包。
|===
