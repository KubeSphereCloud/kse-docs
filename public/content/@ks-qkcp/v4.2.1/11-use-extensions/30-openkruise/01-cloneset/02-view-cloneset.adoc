本节介绍如何查看 CloneSet 列表和详情。

== 查看 CloneSet 列表

. 登录 KubeSphere Web 控制台，进入目标集群，在左侧导航栏选择**工作负载 > CloneSet**。
. 在页面左上角的**项目**下拉列表中选择目标项目。

CloneSet 列表提供以下信息：

[%header,cols="1a,4a"]
|===
|参数 |描述

|名称
|CloneSet 的名称。

|状态
|CloneSet 当前的状态。
* **运行中**：所有容器组均已正常运行。
* **更新中**：正在执行滚动更新或原地升级。
* **已停止**：副本数为 0。

|项目
|CloneSet 所属的项目。

|更新时间
|CloneSet 最近一次更新的时间。
|===

* 在列表上方搜索框中输入关键字，可按名称搜索 CloneSet。
* 在**状态**下拉列表中可按状态筛选 CloneSet。
* 在列表右上角点击刷新图标可刷新列表，点击自定义列图标可定制显示的列。

== 查看 CloneSet 详情

. 登录 KubeSphere Web 控制台，进入目标集群，在左侧导航栏选择**工作负载 > CloneSet**。
. 在列表中点击目标 CloneSet 的名称，进入详情页面。

详情页包含以下页签：

* **属性**：基本信息，包括名称、别名、项目、创建时间、副本数量及镜像版本。
* **容器组**：该 CloneSet 下所有容器组的运行状态、所在节点及重启次数。
* **元数据**：标签和注解信息。
* **事件**：相关 Kubernetes 事件记录。
