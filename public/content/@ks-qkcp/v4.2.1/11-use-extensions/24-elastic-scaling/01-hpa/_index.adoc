Metrics Server 是一个可扩展、高效的容器资源度量源，为 Kubernetes 内置的自动扩展管道提供服务。

Metrics Server 支持基于 CPU/内存的容器水平伸缩（HPA）。系统将目标工作负载中所有 Pod 的平均 CPU/内存用量，与在 HPA 中配置的 CPU/内存指标的目标值进行比较，从而自动调整 Pod 副本的数量。

安装 Metrics Server 扩展组件后，集群和企业空间左侧导航栏的**弹性伸缩**菜单下将显⽰**容器水平伸缩**选项。
