本节介绍如何查看升级任务列表。一个升级任务可包含一个或多个网关实例的升级。

== 前提条件

* KubeSphere 平台需要安装并启用 **KubeSphere 网关**扩展组件。

* {empty}

== 操作步骤

include::../../../../_custom/clusterManagement/clusterManagement-oper-openClusterPage.adoc[]

+

include::../../../../../_ks_components/oper-navigate.adoc[]

+

. 在**升级任务**页签，查看集群中所有升级任务的信息。
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|名称
|自动生成的升级任务名称。

|状态
|任务的状态：等待中、运行中、已完成、失败。

|上次运行时间
|升级任务的上次运行结束时间。
|===

* 在列表上方点击搜索框并设置搜索条件，可按名称、状态搜索升级任务。
