本节介绍如何查看快速集成资源列表。

== 前提条件

* 您具有快速集成资源的查看权限。
* KubeSphere 平台已安装并启用快速集成扩展组件。

== 操作步骤

. 登录 KubeSphere Web 控制台，在扩展坞点击 *快速集成*。

快速集成列表提供以下信息：

[%header,cols="1,4"]
|===
|参数 |描述

|名称
|快速集成资源的名称及描述。

|菜单位置
|集成资源挂载的菜单层级，显示为 *集群* 或 *企业空间*。

|创建时间
|快速集成资源的创建时间。
|===

* 在列表上方点击搜索框并输入关键字，可按名称搜索快速集成资源。
* 在列表右上角点击刷新图标，可刷新列表信息。
