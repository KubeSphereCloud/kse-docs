本节介绍如何通过页面嵌入的方式将外部系统接入 KubeSphere，作为平台功能的一部分使用。

== 前提条件

* 您具有快速集成资源的管理权限。
* KubeSphere 平台已安装并启用快速集成扩展组件。

== 操作步骤

. 登录 KubeSphere Web 控制台，在扩展坞点击 *快速集成*，然后在页面右上角点击 *创建*。
. 在 *基本信息* 区域填写名称、描述、侧边栏菜单名称、图标和菜单位置。
. 点击 *添加资源集成*，选择 *外部系统页面集成*，在 *页面 URL* 中填写需要嵌入的外部系统页面地址。
. 设置该资源对应的 *二级菜单名称* 和 *路径*，然后点击 *保存*。如需继续添加资源，可重复点击 *添加资源集成*。
. 确认所有配置无误后，点击 *完成*。

刷新页面后，新创建的快速集成资源将出现在侧边栏菜单中，点击对应入口即可以内嵌页面的方式访问外部系统。
