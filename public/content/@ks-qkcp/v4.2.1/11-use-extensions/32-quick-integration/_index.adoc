快速集成是 KubeSphere 提供的扩展接入能力，支持将 Kubernetes 自定义资源（CRD）或外部系统页面以统一入口的形式集成至平台，便于统一访问和管理。

在 KubeSphere 扩展市场中安装快速集成扩展组件后，扩展坞将新增 *快速集成* 菜单。

== 适用场景

* Kubernetes 自定义资源（CRD）缺乏可视化管理界面，需要借助命令行或第三方工具操作。
* 外部系统与 KubeSphere 平台相互独立，用户需要在多个系统之间频繁切换。
* 不同团队或项目需要集成的扩展功能差异较大，难以统一纳入平台管理。

== 主要功能

* 将 Kubernetes 自定义资源（CRD）集成至平台，通过可视化界面统一管理。
* 通过页面嵌入方式接入外部系统，作为平台功能的一部分使用。
* 支持将集成资源挂载到 *集群* 或 *企业空间* 维度的侧边栏菜单。
* 支持自定义菜单名称、图标及二级菜单路径。
