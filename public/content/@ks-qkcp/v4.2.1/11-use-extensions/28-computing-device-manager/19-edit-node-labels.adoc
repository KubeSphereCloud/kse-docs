本节介绍如何编辑节点标签。

节点标签是用户在节点上设置的键值对，可用于选择节点。例如，在创建工作负载时，您可以指定容器组调度到具有特定标签的节点上。


== 前提条件

* 已部署算力节点。

* KubeSphere 平台需要安装并启用**算力设备管理**扩展组件。

+

include::../../../../_ks_components/oper-navigate.adoc[]

+

. 在节点列表中，点击需要操作的节点的名称打开节点详情页面。

. 在节点详情页面，点击**操作 > 编辑标签**。

. 在**编辑标签**对话框，设置节点标签的键和值，然后点击**确定**。
