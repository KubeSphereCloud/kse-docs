本节介绍如何编辑队列的基本信息。

您可以编辑队列的别名和描述。KubeSphere 不支持编辑已创建队列的名称。


== 前提条件

* KubeSphere 平台需要安装并启用 **Volcano** 扩展组件。

* {empty}

== 操作步骤

include::../../../../_custom/clusterManagement/clusterManagement-oper-openClusterPage.adoc[]

+

include::../../../../../_ks_components/oper-navigate.adoc[]

+

. 在需要操作的队列右侧点击image:/images/ks-qkcp/zh/icons/more.svg[more,18,18]，然后在下拉列表中选择**编辑信息**。

. 在**编辑信息**对话框，设置队列的别名和描述，然后点击**确定**。
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|别名
|队列的别名。不同队列的别名可以相同。

|描述
|队列的描述信息。描述可包含任意字符，最多包含 256 个字符。
|===
