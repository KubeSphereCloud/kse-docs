Volcano 是一个基于 Kubernetes 构建的容器批量计算平台，主要用于高性能计算场景。它扩展了 Kubernetes 的原生调度能力，为 AI 机器学习、大数据处理、科学计算等需要大规模并行计算的任务提供了高级调度策略、队列管理、异构硬件支持等关键特性，显著提升了集群资源利用率和复杂工作流的运行效率。更多信息，请参阅 link:https://volcano.sh/zh/docs/[Volcano 官方文档]。

安装 Volcano 扩展组件后，集群和企业空间的左侧导航栏将显示 **Volcano 调度**菜单。
