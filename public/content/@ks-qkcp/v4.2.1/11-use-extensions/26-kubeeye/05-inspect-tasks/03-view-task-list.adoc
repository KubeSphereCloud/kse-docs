:ks_navigation: **巡检任务**


本节介绍巡检任务列表的信息。

== 前提条件

* 您需要在 KubeSphere 平台具有 **platform-admin** 角色。

* 已创建巡检任务。


== 操作步骤


include::../../../../_custom/platformManagement/extensionManagement/extensionManagement-oper-logInAsPlatformAdmin.adoc[]

+

. 点击**组件坞 > 集群巡检**，进入集群巡检页面。

+

include::../../../../../_ks_components/oper-navigate.adoc[]

. 在巡检任务页面，点击**即时巡检任务**或**定时巡检任务**打开巡检任务列表。
+
--
巡检任务列表提供以下信息：

[%header,cols="1a,4a"]
|===
|参数 |描述

|名称/ID
|巡检任务的名称和 ID。

|执行状态
|
* 等待中：巡检任务待执行。
* 巡检中：正在执行巡检任务。
* 已完成：巡检任务执行完成。
* 失败：巡检任务执行失败。

|策略类型
|单次巡检或周期巡检。仅对定时巡检任务显示。

|巡检规则
|巡检任务使用的巡检规则数量。

|巡检范围
|被巡检的目标集群。

|最近巡检
|巡检任务最近一次的执行时间和持续时长。

|任务状态
|是否启用当前巡检任务。仅对定时巡检任务显示。
|===

* 在列表上方点击搜索框并输入关键字，可按名称/ID、执行状态搜索即时巡检任务，可按名称/ID、执行状态、策略类型和任务状态搜索定时巡检任务。
