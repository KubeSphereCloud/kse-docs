:ks_navigation: **概览**


link:../05-inspect-tasks/01-create-inspect-tasks/[创建巡检任务]并执行后，概览页面将展示巡检结果、巡检任务和巡检记录的概览数据。

== 前提条件

* 您需要在 KubeSphere 平台具有 **platform-admin** 角色。

== 操作步骤

include::../../../_custom/platformManagement/extensionManagement/extensionManagement-oper-logInAsPlatformAdmin.adoc[]

+

. 点击**组件坞 > 集群巡检**，进入集群巡检页面。

+

include::../../../../_ks_components/oper-navigate.adoc[]
// +
// image::/images/ks-qkcp/zh/v4.2.1/kubeeye-overview.png[]
+
[%header,cols="1a,4a"]
|===
|功能区 |描述

|最新巡检结果
|最近一次巡检的开始时间、巡检状态、巡检的集群、关联的巡检任务、巡检策略、巡检持续时间，以及巡检规则的风险等级。

|巡检任务状态
|所有巡检任务的状态和数量。

* 等待中：巡检任务待执行。
* 运行中：正在执行巡检任务。
* 已完成：巡检任务执行完成。
* 失败：巡检任务执行失败。

|近期巡检记录
|最近 10 条巡检记录的信息。有关更多信息，请参阅link:../07-inspect-records/03-view-record-list/[查看巡检记录列表]。
|===
