---

本节介绍如何删除已创建的巡检规则。

[.admon.attention,cols="a"]
|===
|注意

|删除巡检规则前，请确保巡检规则未被巡检任务使用，即关联巡检任务数量为 0。否则，删除巡检规则后，巡检任务将执行失败。

|===

== 操作步骤

登录集群节点，查看用于创建巡检规则的 yaml 文件，然后执行以下命令删除巡检规则。

[.admon. tip,cols="a"]
|===
|提示

|若集群节点上找不到用于创建巡检规则的 yaml 文件，也可根据巡检规则的 yaml 配置重新创建 yaml 文件。
|===

[source,bash]
----
kubectl delete -f xxx.yaml
----

[.admon.note,cols="a"]
|===
|说明

|将 `xxx.yaml` 替换为实际的规则文件名称，如 `kubeeye_nodeInfo_inspect.yaml`。
|===
