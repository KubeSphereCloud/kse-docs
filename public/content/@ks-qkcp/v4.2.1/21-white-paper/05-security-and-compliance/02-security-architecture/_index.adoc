分层说明：

1. 基础设施安全：物理主机、虚拟化层、操作系统加固。
2. 容器运行时安全：镜像安全、容器隔离、运行时防护。
3. 网络安全：安全组策略、Service Mesh 安全通信、微分段（Micro-segmentation）。
4. 身份与访问控制（IAM）：多租户隔离、RBAC、OAuth2.0、LDAP/AD 集成。
5. 应用与数据安全：API 安全、数据加密、敏感数据脱敏、密钥管理。
6. 安全运营中心（SOC）与审计：日志审计、威胁检测、漏洞响应。
