== 网络安全等级保护制度 2.0

平台对齐措施：

* 支持三级等保安全控制：身份鉴别、访问控制、安全审计、入侵防范、数据加密
* 提供多租户隔离与最小权限访问模型
* 实现集中日志审计与防篡改机制
* 提供合规测评模板与安全基线扫描工具

== 数据安全法（DSL）与个人信息保护法（PIPL）

合规要求：

* 数据分级分类管理
* 数据脱敏与最小化收集
* 数据跨境传输审查与备案

平台对齐措施：

* 数据加密（存储 + 传输）
* 敏感字段脱敏（配置化规则）
* 数据访问审计与可追溯机制
* 密钥托管与轮换策略

== CNCF 云原生安全最佳实践

平台对齐措施：

* 镜像签名与验证（cosign、Sigstore）
* 集群策略控制（OPA/Gatekeeper、Kyverno）
* 零信任通信（mTLS、SPIFFE/SPIRE）
* 运行时防护与可观测性（Falco、Prometheus）
