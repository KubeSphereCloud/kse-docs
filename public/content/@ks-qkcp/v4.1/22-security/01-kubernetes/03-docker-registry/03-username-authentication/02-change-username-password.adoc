. 生成认证文件。
+
--
[,bash]
----
htpasswd -Bbn <username> <passwd> > /etc/kubekey/registry/auth/passwd
----

image:/images/ks-qkcp/zh/security-handbook/change-username-password/1.png[]
--

. 重启 registry 服务。
+
--
[,bash]
----
systemctl restart registry
----
image:/images/ks-qkcp/zh/security-handbook/change-username-password/2.png[]
--

. 测试使用旧密码登录。
+
--
[,bash]
----
docker login dockerhub.kubekey.local -u huawei
Password: Huawei@123
----
