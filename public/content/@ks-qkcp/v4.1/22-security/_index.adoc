本安全加固⼿册旨在为 KubeSphere 平台的管理员和运维团队提供指导，帮助他们建⽴和维护⼀个安全可靠的容器管理环境。

通过遵循这些准则和最佳实践，

- 帮助用户识别容器环境中的潜在安全威胁和风险，包括 Kubernetes 安全加固，TLS 安全通信、容器安全加固、安全审计和安全认证等；
- 指导用户如何安全地配置 KubeSphere 和管理容器，确保其环境的稳定和安全，保护企业的数据和资源免受威胁；
- 帮助组织满足行业合规要求，可以降低潜在的安全⻛险。
