. 查询 kafka 相关的 crd。
+
[,bash]
----
kubectl get crd | grep kafka
----

. 查询 kafka cluster 配置文件 crd。
+
[,bash]
----
kubectl get kafkas.kafka.strimzi.io
----

. 按以下方式调整用户，样例见下图。
+
image:/images/ks-qkcp/zh/security-handbook/332-account-config/config.png[]
