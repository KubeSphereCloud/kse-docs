. 查询 kafka 相关的 crd。
+
[,bash]
----
kubectl get crd | grep kafka
----

. 查询 kakfa cluster 配置文件 crd。
+
--
[,bash]
----
kubectl get kafkas.kafka.strimzi.io -A
----

image:/images/ks-qkcp/zh/security-handbook/334-change-pw/01.png[]
--

. 编辑 kafka cluster 的配置文件。
+
--
[,bash]
----
kubectl edit kafka -n {NAMESPACE} # 根据实际情况修改 NAMESPACE，样例的 NAMESPACE 为 kafka

----

image:/images/ks-qkcp/zh/security-handbook/334-change-pw/02.png[]
--

. 找到 `org.apache.kafka.common.security.plain.PlainLoginModule required user_saas="dms@kafka"` 后修改 user_saas="test"(即 user_用户名=密码)，即可将 saas 用户的密码修改为 test。
+
image:/images/ks-qkcp/zh/security-handbook/334-change-pw/03.png[]

. 退出保存即可生效。
