---
title: "账号密码过期设置"
weight: 02
---

radondb-mysql 默认禁用密码过期设置，机机账号密码暂不支持修改，业务系统支持密码过期设置。方法如下:

[,bash]
----
kubectl exec -it sample-mysql-0 -c mysql -- bash
mysql -u <管理员账号> -h 127.0.0.1
ALTER USER '用户'@'%' PASSWORD EXPIRE INTERVAL 90 DAY;
# 查看过期时间
select host,user,password_expired,password_lifetime from mysql.user;
----
