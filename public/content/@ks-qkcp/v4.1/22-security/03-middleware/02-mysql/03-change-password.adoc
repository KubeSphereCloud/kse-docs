. 下载将使用的link:https://yunify-qingcloud-docs.pek3b.qingstor.com/package/mysqlpasstool.tar.gz[加密工具]。
+
--
使用方法：加密使用命令 E , 解密使用命令 D。示例：

[,bash]
----
./E RadonDB@123456
ct+rqR3NuQWlYd7jS2uUhXW0hLdUuQgSH0l6Pbj5

./D ct+rqR3NuQWlYd7jS2uUhXW0hLdUuQgSH0l6Pbj5
RadonDB@123456
----
--

. 采用 super 用户登录 mysql。

. 从 secret 中获取用户密码。
+
--
[,bash]
----
kubectl -n mysql  get secrets  sample8-client-secret -o jsonpath="{.data.super-client\.cnf}"|base64 -d;
----

. 复制 password 字段的内容，然后使用解密工具获取真正的密码。
+
image:/images/ks-qkcp/zh/security-handbook/323-change-pw/02.png[]

. 使用解密后的密码, 进入 mysql 客户端。
+
[,bash]
----
mysql -u super -p"8pzMfS5mX[NLbjP+Qv9APdo{"
----

. 使用如下命令直接修改密码。
+
--
[,bash]
----
ALTER USER '用户名'@'主机名' IDENTIFIED BY '新密码';
----

如果是使用 CR 方式创建的用户，首先找到 mysqlclusters.mysql.radondb.com 的 CR， 根据用户名在 secretSelector 中找到对应的 secretName，更改以 secretName 为名的 secret 中 Data 下与用户名对应的字段即可。

例如：要修改 normal-user 的用户，先找到名字为 normal-user 的 `mysqlclusters.mysql.radondb.com` 的 CR,  找到 secretSelector，可以看到 `secretName: sample-user-password`,

[,yaml]
----
 ....
  ## Specify the secret object for user.
  secretSelector:
    secretName: sample-user-password
    secretKey: normalUser
----

然后找到名为 sample-user-password 的 secret, 修改 normalUser 字段即可。如下：

[,yaml]
----
apiVersion: v1
kind: Secret
metadata:
  # Corresponding to the user's secretName
  name: sample-user-password
data:
  # The key corresponding to the user's secretKey
  # RadonDB@123
  superUser: ct+rqR3NuQWlYd7jS2uUhXW0hLdUuQgSH0l6Pbj5
  normalUser: ct+rqR3NuQWlYd7jS2uUhXW0hLdUuQgSH0l6Pbj5
----

[.admon.attention,cols="a"]
|===
|注意

|
字段数据需采用加密工具加密。
|===
