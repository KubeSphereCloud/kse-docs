某些扩展组件安装后，需要激活才能使用。本节以 link:https://kubesphere.com.cn/marketplace/extensions/lakeinsight/[LakeInsight] 组件为例，介绍如何激活扩展组件。


== 前提条件

已订阅并安装 LakeInsight。


== 操作步骤

. 获取 LakeInsight 激活码。
+
扩展组件订阅成功后，您会在 1 个工作日内收到该扩展组件的激活码，请留意您的邮件通知。如果没有及时收到，请联系 support@kubesphere.cloud。

. 点击顶部导航栏上的 **LakeInsight**，进入 LakeInsight 使用页面，首次登录后会自动跳转至 License 页面。

. 将收到的激活码粘贴到框内，点击验证，验证通过后会自动进入 LakeInsight 首页。
+
也可进入**关于**页面，查看当前权限并**更新 License**。
