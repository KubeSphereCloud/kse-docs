本节介绍如何升级扩展组件。

在订阅期内，当组件有新版本时，您可以在扩展市场或扩展中心升级组件。

== 前提条件

您需要在{ks_product_both}平台具有 platform-admin 角色。

== 在扩展市场升级组件

. 登录{ks_product_left} Web 控制台。
. 点击左上角的**扩展市场**，进入扩展市场页面。
. 进入组件详情页，点击组件图标下的image:/images/ks-qkcp/zh/icons/more.svg[more,18,18]，选择**升级**。

== 在扩展中心升级组件

. 登录{ks_product_left} Web 控制台。
. 点击**扩展中心**，进入扩展中心页面。
. 点击组件右侧的image:/images/ks-qkcp/zh/icons/more.svg[more,18,18]，选择**升级**。
