本节介绍如何禁用扩展组件。

组件安装完成后，会自动启用。您可以在扩展市场的组件详情页，或在扩展中心的组件菜单中，禁用扩展组件。

禁用后，再次启用，即可在集群、企业空间、项目中继续使用该组件。

== 前提条件

您需要在{ks_product_both}平台具有 platform-admin 角色。

== 在扩展市场禁用组件

. 登录{ks_product_left} Web 控制台。
. 点击左上角的**扩展市场**，进入扩展市场页面。
. 进入组件详情页，点击组件图标下的image:/images/ks-qkcp/zh/icons/more.svg[more,18,18]，选择**禁用**。

== 在扩展中心禁用组件

. 登录{ks_product_left} Web 控制台。
. 点击**扩展中心**，进入扩展中心页面。
. 点击组件右侧的image:/images/ks-qkcp/zh/icons/more.svg[more,18,18]，选择**禁用**。
