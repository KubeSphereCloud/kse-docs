+

+

. 在**网络隔离**页面，点击**启用**。
+
--
// Note
include::../../../../../_ks_components/admonitions/note.adoc[]

启用网络隔离后，当前项目中的容器组将不能与其他项目的容器组、其他节点主机环境以及集群外部的所有网段通信。您可以进一步添加白名单条目，以允许当前项目中的容器组与特定项目中的容器组、其他节点主机环境和集群外部网段通信。

include::../../../../../_ks_components/admonitions/admonEnd.adoc[]
--

. 启用网络隔离后，点击**内部白名单**或**外部白名单**页签。
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|内部白名单
|允许当前项目中的容器组与当前集群其他项目中容器组通信。

|外部白名单
|允许当前项目中的容器组与集群外部的特定网段通信。
|===
--

. 在**内部白名单**或**外部白名单**页签，点击**添加白名单条目**。

. 在**添加白名单条目**对话框，设置白名单条目的参数，然后点击**确定**。
+
====
* 对于内部白名单条目，请设置以下参数
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|流量方向
|白名单条目放行的流量方向。

* **出站**：当前项目到其他项目的方向。

* **入站**：其他项目到当前项目的方向。

|类型
|白名单条目匹配其他项目容器组的方式。

* **项目**：当前项目中的容器组可以与指定项目中的所有容器组通信。

* **服务**：当前项目中的容器组可以与指定服务的后端容器组通信。
|===
--

* 对于外部白名单条目，请设置以下参数：
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|流量方向
|白名单条目放行的流量方向。

* **出站**：当前项目到集群外部的方向。

* **入站**：集群外部到当前项目的方向。

|网段
|集群外部的网络地址和子网掩码。支持无类别域间路由（CIDR）。

|端口
|白名单条目放行的端口号。

* 对于出站白名单条目，此处的端口为集群外部地址的端口。

* 对于入站白名单条目，此处的端口为当前项目容器组的端口。
|===
--

白名单条目创建完成后将显示在内部白名单或外部白名单列表中。
