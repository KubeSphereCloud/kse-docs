本节介绍如何邀请用户加入项目。


== 前提条件link:[id=8c11b0e4dda04a7199d7f3861b2d3eca]

:relfileprefix: ../../../../
* 您需要加入一个项目并在项目中具有pass:a,q[{ks_permission}]权限。有关更多信息，请参阅xref:07-project-management/10-project-settings/04-project-members/_index.adoc[项目成员]和xref:07-project-management/10-project-settings/03-project-roles/02-view-a-project-role-list.adoc[项目角色]。

// include::../../../../_custom/projectManagement/projectManagement-prer-requiredPermission.adoc[]

:relfileprefix: ./



ifeval::["{file_output_type}" == "pdf"]

* 被邀请的用户需要先加入项目所属的企业空间。有关更多信息，请参阅《{ks_product_right}企业空间管理指南》的“邀请用户加入企业空间”章节。

endif::[]

ifeval::["{file_output_type}" == "html"]

* 被邀请的的用户需要先加入项目所属的企业空间。

endif::[]
