灰度发布任务详情页面提供以下信息：

[%header,cols="1a,4a"]
|===
|参数 |描述

|版本
|新旧版本的容器组副本数量、CPU 用量和内存用量。

|流量分配
|
* 对于蓝绿部署，显示当前接收全部分流量的服务版本。

* 对于金丝雀发布，显示新旧版本的流量接收比例。

* 对于流量镜像，显示旧版本接收新版本流量的相同副本。
