:relfileprefix: ../../../

== 操作步骤link:[id=a531121b44bf4b8082233debb82187a1]

include::../../../_custom/projectManagement/projectManagement-oper-openProjectPage.adoc[]

+

include::../../../../_ks_components/oper-navigate.adoc[]

. 在**自定义伸缩**页面，您可以查看到已创建的自定义伸缩任务。
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|名称
|自定义伸缩任务的名称和别名。

|状态
|伸缩任务的状态。

* 伸缩对象就绪：伸缩对象是否准备好开始伸缩。

* 伸缩器活跃：伸缩对象是否发生伸缩。

* 回退：伸缩器是否能够从源获取指标。

|伸缩对象
|自定义伸缩的对象。

|最小副本数
|设置允许的最小容器组副本数量。默认值为 0。

|最大副本数
|设置允许的最大容器组副本数量。默认值为 1。

|创建时间
|自定义伸缩任务创建的时间。
|===
--

+
include::../../../_custom/projectManagement/customScaling/customScaling-oper-openDetailsPage.adoc[]

+

* 在右侧的**资源状态**页签，您可以查看资源状态信息，伸缩对象信息以及触发器信息。

* 点击**事件**页签，您可以查看自定义伸缩任务相关的事件。
