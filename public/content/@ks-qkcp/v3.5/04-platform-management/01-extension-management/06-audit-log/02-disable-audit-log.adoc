. 在集群配置文件中，将 **spec:auditing:enabled** 参数设置为 **false**，然后保存文件。
+
--
// YAML
include::../../../../../_ks_components/code/yaml.adoc[]

spec:
  auditing:
    enabled: false

include::../../../../../_ks_components/code/codeEnd.adoc[]
--

. 在 **kubectl** 对话框，执行以下命令删除审计日志相关资源：
+
--
// Bash
include::../../../../../_ks_components/code/bash.adoc[]

helm uninstall kube-auditing -n kubesphere-logging-system

include::../../../../../_ks_components/code/codeEnd.adoc[]

// Bash
include::../../../../../_ks_components/code/bash.adoc[]

kubectl delete crd rules.auditing.kubesphere.io

include::../../../../../_ks_components/code/codeEnd.adoc[]

// Bash
include::../../../../../_ks_components/code/bash.adoc[]

kubectl delete crd webhooks.auditing.kubesphere.io
