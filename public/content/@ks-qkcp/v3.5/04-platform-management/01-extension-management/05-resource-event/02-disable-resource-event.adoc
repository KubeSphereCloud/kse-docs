. 在集群配置文件中，将 **spec:events:enabled** 参数设置为 **false**，然后保存文件。
+
--
// YAML
include::../../../../../_ks_components/code/yaml.adoc[]

spec:
  events:
    enabled: false

include::../../../../../_ks_components/code/codeEnd.adoc[]
--

. 在 **kubectl** 对话框，执行以下命令删除资源事件相关资源：
+
--
// Bash
include::../../../../../_ks_components/code/bash.adoc[]

helm delete ks-events -n kubesphere-logging-system
