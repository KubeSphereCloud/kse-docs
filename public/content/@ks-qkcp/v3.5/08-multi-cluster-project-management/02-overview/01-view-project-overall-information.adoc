**概览**页面提供以下信息：

[%header,cols="1a,4a"]
|===
|参数 |描述

|企业空间
|多集群项目所属的企业空间。

|创建者
|创建多集群项目的用户。

|资源状态
|多集群项目中已创建的应用资源和已使用的物理资源。在右上角点击下拉列表可选择多集群项目所在的某一集群和数据的时间范围。

* **应用资源**：多集群项目中已创建的部署、有状态副本集、持久卷声明、服务和应用路由的数量。

* **物理资源**：多集群项目的 CPU 和内存使用量。

|资源用量 Top 5
|多集群项目中特定资源用量最高的 5 个工作负载。点击下拉列表可选择多集群项目所在的某一集群，并按 CPU、内存、出站流量或入站流量排序。
|===
