== 前提条件link:[id=6ef3adcbc8b147ff9ab521c30516ee30]

include::../../../../_custom/multi-clusterProjectManagement/projectManagement-prer-requiredPermission.adoc[]


== 操作步骤link:[id=b952b37689a54d4997e364eb4e2a2977]

include::../../../../_custom/multi-clusterProjectManagement/projectManagement-oper-openProjectPage.adoc[]

+

include::../../../../../_ks_components/oper-navigate.adoc[]

+

. 在**基本信息**页面，点击**管理**，然后在下拉列表中选择**编辑注解**。

. 在弹出的对话框中，设置注解的键值对，然后点击**确定**。注解添加完成后将显示在项目基本信息页面。
+
--
* 点击**添加**可以为当前项目添加一个或多个注解。

* 点击 image:/images/ks-qkcp/zh/icons/trash-light.svg[trash-light,18,18] 可删除当前待添加的注解。
