本节介绍如何编辑项目基本信息。

您可以编辑项目的别名和描述。{ks_product_right}不支持编辑已创建项目的名称。


== 前提条件link:[id=35dbf8b4cec7497fb90f54eb067ab59c]

include::../../../../_custom/multi-clusterProjectManagement/projectManagement-oper-openProjectPage.adoc[]

+

include::../../../../../_ks_components/oper-navigate.adoc[]

+

. 在**基本信息**页面，点击**管理**，然后在下拉列表中选择**编辑信息**。

. 在**编辑信息**对话框，设置项目的别名和描述，然后点击**确定**。
