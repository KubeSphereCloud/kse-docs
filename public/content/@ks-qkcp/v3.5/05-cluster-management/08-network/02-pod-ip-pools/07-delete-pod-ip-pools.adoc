== 删除单个容器组 IP 池link:[id=affe9b77253a4c67bd7d0bb0924c714c]

include::../../../../_custom/clusterManagement/clusterManagement-oper-openClusterPage.adoc[]

+

include::../../../../../_ks_components/oper-navigate.adoc[]

+

. 在需要删除的容器组 IP 池右侧点击 image:/images/ks-qkcp/zh/icons/more.svg[more,18,18]，然后在下拉列表中选择**删除**。

. 在**删除容器组 IP 池**对话框，输入容器组 IP 池的名称，然后点击**确定**。


== 批量删除容器组 IP 池link:[id=1beba403dcff4e10bd4deeb00c8d079f]

include::../../../../_custom/clusterManagement/clusterManagement-oper-openClusterPage.adoc[]

+

include::../../../../../_ks_components/oper-navigate.adoc[]

+

. 选择需要删除的容器组 IP 池左侧的复选框，然后在容器组 IP 池列表上方点击**删除**。

. 在**批量删除容器组 IP 池**对话框，输入容器组 IP 池的名称，然后点击**确定**。
