== 操作步骤 link:[id=f3bd07ab0bd445f896cf07c560011511]

include::../../../_custom/clusterManagement/clusterManagement-oper-openClusterPage.adoc[]

+

include::../../../_custom/clusterManagement/nodes/nodes-oper-openListPage.adoc[]

+

. 在需要操作的节点右侧点击 image:/images/ks-qkcp/zh/icons/more.svg["more",18,18]，然后在下拉列表中选择**停止调度**。操作完成后节点状态将变为**不可调度**。

. 在状态为**不可调度**的节点右侧点击image:/images/ks-qkcp/zh/icons/more.svg["more",18,18]，然后在下拉列表中选择**启用调度**可以允许容器组调度到该节点。

// Note
include::../../../../_ks_components/admonitions/note.adoc[]

节点状态被设置为**不可调度**后，新创建的容器组将不再调度到该节点，节点上原本运行的容器组不受影响。
