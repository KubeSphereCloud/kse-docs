本节介绍如何添加日志接收器。

{ks_product_right}支持使用 Elasticsearch、Kafka 和 Fluentd 作为日志接收器。

ifeval::["{file_output_type}" == "html"]

* {ks_product_right}默认安装 Elasticsearch 并提供日志分析工具，您可以在{ks_product_left} Web 控制台使用日志分析工具查询 Elasticsearch 收集的日志。

endif::[]

ifeval::["{file_output_type}" == "pdf"]

* {ks_product_right}默认安装 Elasticsearch 并提供日志分析工具，您可以在{ks_product_left} Web 控制台使用日志分析工具查询 Elasticsearch 收集的日志。有关更多信息，请参阅link:../../../13-toolbox/[工具箱]章节。

endif::[]

* {ks_product_right}默认未安装 Kafka 和 Fluentd，并且目前未提供针对 Kafka 和 Fluentd 的日志分析工具。{ks_product_right}仅提供与 Kafka 和 Fluentd 对接的功能。如需使用 Kafka 或 Fluentd 作为日志接收器，您需要参阅 link:https://kafka.apache.org/documentation/[Kafka 官方文档]或 link:https://docs.fluentd.org/[Fluentd 官方文档]安装 Kafka 或 Fluentd。
