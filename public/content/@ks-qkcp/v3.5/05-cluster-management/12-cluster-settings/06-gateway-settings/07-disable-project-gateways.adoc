== 禁用单个项目网关link:[id=f339ebb13f394185b26736229c7cbf79]

include::../../../../_custom/clusterManagement/clusterManagement-oper-openClusterPage.adoc[]

+

include::../../../../../_ks_components/oper-navigate.adoc[]

+

. 在**网关设置**页面点击**项目网关**，在需要编辑的项目网关右侧点击 image:/images/ks-qkcp/zh/icons/more.svg[more,18,18]，然后在下拉列表中选择**禁用**。

. 在**禁用项目网关**对话框，输入项目网关的名称，然后点击**确定**。


== 批量禁用项目网关link:[id=5e58398f456449e09cfc9888fb0f22bd]

include::../../../../_custom/clusterManagement/clusterManagement-oper-openClusterPage.adoc[]

+

include::../../../../../_ks_components/oper-navigate.adoc[]

+

. 在**网关设置**页面点击**项目网关**，选择需要禁用的项目网关左侧的复选框，然后在列表上方点击**禁用**。

. 在**批量禁用项目网关**对话框，输入项目网关的名称，然后点击**确定**。
