您可以使用{ks_product_left} DevOps 系统在 Kubernetes 集群上部署和管理 CI/CD 任务以及相关的工作负载。本章演示如何在 DevOps 项目中进行管理和操作，包括运行流水线、创建凭证和集成工具等等。

您安装 DevOps 组件时，会自动部署 Jenkins。您可以在{ks_product_both}中像以前一样通过 Jenkinsfile 构建流水线，保持一致的用户体验。此外，{ks_product_right}还提供图形编辑面板，可以将整个流程可视化，为您直观地呈现流水线在每个阶段的运行状态。

== 产品版本

本文档适用于{ks_product_left} v3.5.0 版本。

== 读者对象

本文档主要适用于以下读者：

* {ks_product_right} DevOps 项目管理员

* {ks_product_right} DevOps 项目普通用户

* 交付工程师

* 运维工程师

* 售后工程师
