本节介绍如何快速创建 RadonDB Redis Cluster 应用并访问 Redis Cluster。


== 前提条件link:[id=795bc783279c47608cf863e5608ce0b5]

include::../../../_custom/radondb/radondb-prer-requiredPermission.adoc[]

* 您需要获取 RadonDB Redis Cluster 应用授权并获得平台管理权限以访问 kubectl。有关更多信息，请联系{ks_product_both}技术支持。



== 操作步骤link:[id=4bb56515464d4895b64b6e7b36b14275]


=== 安装 RadonDB Redis Cluster 应用link:[id=9ad0510dc19b49d8a0a4e4e5cdc6a95b]

include::../../../_custom/projectManagement/projectManagement-oper-openProjectPage.adoc[]

+

include::../../../../_ks_components/oper-navigate.adoc[]

+

include::../../../_custom/radondb/radondb-oper-clickRadondbAppsTab.adoc[]

+

. 在页面右侧选择**安装 > RadonDB Redis Cluster**。
+

include::../../../_custom/radondb/radondb-install-app.adoc[]

. 在**基本信息**页签，输入应用名称，如 radondb-test，并选择应用的安装位置，然后点击**下一步**。

. 在**应用设置**页签，设置 RadonDB Redis Cluster 的参数，然后点击**安装**。
+
--
安装后的 RadonDB Redis Cluster 应用将显示在 RadonDB 应用列表。

// Note
include::../../../../_ks_components/admonitions/note.adoc[]

有关参数的具体解释，请参阅link:../02-install-radondb-redis-cluster/[安装 RadonDB Redis Cluster]。

include::../../../../_ks_components/admonitions/admonEnd.adoc[]
--


=== 创建数据库用户link:[id=50e40272414b4f16a5bc47302a35102b]

include::../../../_custom/radondb/radondbRedisCluster/radondbRedisCluster-oper-openDetailsPage.adoc[]

+

. 在页面右侧点击**数据库用户**页签，然后点击**创建**。

. 在**创建用户**对话框，输入用户名和密码，如 test/Test@123，然后点击**确定**。用户创建完成后将显示在用户列表中。


=== 访问 RadonDB Redis Clusterlink:[id=fadc98cd06514964a377e7788d8aea5a]

include::../../../_custom/radondb/radondbRedisCluster/radondbRedisCluster-oper-openDetailsPage.adoc[]

+

. 将鼠标悬停至 Web 控制台右下角的 image:/images/ks-qkcp/zh/icons/hammer.svg[hammer,20,25]，点击 **kubectl**，在弹出的对话框执行以下命令进入 Redis 集群。
+
--
// Bash
include::../../../../_ks_components/code/bash.adoc[]

kubectl exec -it -n <namespace>   <name> -- sh

include::../../../../_ks_components/code/codeEnd.adoc[]

* <namespace>：Redis Cluster 实例所在命名空间（项目名称），如 p1。

* <name>：Redis Cluster 实例名，如 radondb-afb2u6-leader-1。


示例如下：

include::../../../../_ks_components/code/bash.adoc[]

kubectl exec -it -n p1 radondb-afb2u6-leader-1 -- sh

include::../../../../_ks_components/code/codeEnd.adoc[]
--

. 连接 Redis Cluster 集群实例。
+
--
// Bash
include::../../../../_ks_components/code/bash.adoc[]

redis-cli -h <redis_instance_address> -p <port> -c

include::../../../../_ks_components/code/codeEnd.adoc[]

* <redis_instance_address>：表示 Redis Cluster 实例的连接地址，如 10.233.85.29。可在集群详情页面的**概览**区域查看连接地址，请根据实际地址替换。

* <port>：表示 Redis Cluster 实例的端口号，默认为 6379。

* -c：连接集群节点时的必选参数。

连接示例如下：
// Bash
include::../../../../_ks_components/code/bash.adoc[]

redis-cli -h 10.233.85.29 -p 6379 -c

include::../../../../_ks_components/code/codeEnd.adoc[]
--

. 如果 Redis Cluster 实例设置了访问密码，执行以下命令完成密码验证。
+
--
// Bash
include::../../../../_ks_components/code/bash.adoc[]

AUTH <password>

include::../../../../_ks_components/code/codeEnd.adoc[]

// Note
include::../../../../_ks_components/admonitions/note.adoc[]

* 如果是使用默认账号，直接填写默认密码 radondn123@ 即可。

* 如果是新创建的账号，则填写 <user>:<password>。例如：账号为 test，密码为 Test@123，则填写 auth test:Test@123。

include::../../../../_ks_components/admonitions/admonEnd.adoc[]

密码验证成功后，会返回 **OK**。
--

. 执行以下命令查看节点信息。
+
--
// Bash
include::../../../../_ks_components/code/bash.adoc[]

cluster nodes

include::../../../../_ks_components/code/codeEnd.adoc[]
