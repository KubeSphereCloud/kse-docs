--
已安装的 RadonDB ECK 应用将显示在 RadonDB 应用列表中。

[%header,cols="1a,4a"]
|===
|参数 |描述

|名称
|RadonDB ECK 应用在{ks_product_both}平台的名称。

|状态
|RadonDB ECK 应用的运行状态。

* **创建中**：应用正在创建过程中。

* **更新中**：应用正在更新配置。

* **已完成**：应用更新完成，即将开始运行。

* **运行中**：应用运行正常。

* **失败**：应用创建过程中出现错误导致创建失败。

|应用模板
|创建 RadonDB ECK 应用所使用的应用模板名称。

|版本
|RadonDB ECK 应用的版本。

|访问地址
|RadonDB ECK 应用的访问地址。每个数据库可提供多个不同用途的访问地址。将光标悬停在访问地址列的数字上可查看读写地址和 Kibana 地址。

* 读写地址：用于从{ks_product_both}集群内部读写 Elasticsearch 数据库的地址。

* Kibana 地址：用于从{ks_product_both}集群外部读写 Elasticsearch 数据库的地址，由{ks_product_both}集群节点 IP 地址和节点端口组成。

|创建时间
|RadonDB ECK 应用的创建时间。
|===
