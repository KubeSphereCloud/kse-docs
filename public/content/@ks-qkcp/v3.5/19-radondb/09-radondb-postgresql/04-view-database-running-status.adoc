--
已安装的 RadonDB PostgreSQL 应用将显示在 RadonDB 应用列表中。

[%header,cols="1a,4a"]
|===
|参数 |描述

|名称
|RadonDB PostgreSQL 应用在{ks_product_both}平台的名称。

|状态
|RadonDB PostgreSQL 应用的运行状态。

* **创建中**：应用正在创建过程中。

* **更新中**：应用正在更新配置。

* **已完成**：应用更新完成，即将开始运行。

* **失败**：应用创建过程中出现错误导致创建失败。

* **运行中**：应用运行正常。

|应用模板
|创建 RadonDB PostgreSQL 应用所使用的应用模板名称。

|版本
|PostgreSQL 数据库的版本。

|访问地址
|PostgreSQL 数据库的访问地址。每个数据库可提供多个不同用途的访问地址。将光标悬停在访问地址列的数字上可查看 ClusterIP 读写地址和只读地址。

|创建时间
|RadonDB PostgreSQL 应用的创建时间。
|===
--

+

include::../../../_custom/radondb/radondbPostgreSQL/radondbpq-oper-openDetailsPage.adoc[]

+

. 在 RadonDB PostgreSQL 详情页面右侧的**概览**页签查看数据库的基本信息和数据库节点。
+
--
[%header,cols="3a,4a"]
|===
|参数 |描述

|基本信息
|显示 PostgreSQL 数据库运行状态、节点和副本数量、节点的 CPU 和内存资源、节点挂载的卷容量。

|节点
|显示 PostgreSQL 数据库单个节点的运行状态、IP 地址、管理类型、CPU 和内存资源、卷容量。
|===
