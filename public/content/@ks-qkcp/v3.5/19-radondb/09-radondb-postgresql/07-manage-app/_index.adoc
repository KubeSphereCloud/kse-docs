一个 RadonDB PostgreSQL 应用即一个独立运行的数据库计算资源组合，包含数据库、容器组、卷等容器资源。

本章主要介绍了如何在{ks_product_both}上管理 RadonDB PostgreSQL 应用生命周期，包括调整数据节点数量、调整资源、扩容应用卷、删除应用。
