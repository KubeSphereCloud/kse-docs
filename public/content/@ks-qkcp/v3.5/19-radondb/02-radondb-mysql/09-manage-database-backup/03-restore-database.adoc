// Note
include::../../../../../_ks_components/admonitions/note.adoc[]

* 安装成功后，会新建一个 RadonDB MySQL 集群。
* 新创建的恢复集群磁盘规格必须大于或等于原集群，以免因磁盘不足导致恢复失败。
* 集群恢复到的时间点为备份任务开始的时间。
