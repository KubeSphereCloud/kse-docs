本节介绍如何设置企业微信通知。


== 前提条件link:[id=033107abb9614ae0a819501c438ee69d]

// :relfileprefix: ../../../../
* 您需要在{ks_product_both}平台具有pass:a,q[{ks_permission}]权限。有关更多信息，请参阅link:../../../../../04-platform-management/04-access-control/02-users/_index/[用户]和link:../../../../../04-platform-management/04-access-control/03-platform-roles/_index/[平台角色]。
// include::../../../../_custom/platformManagement/platformManagement-prer-requiredPermission.adoc[]

// :relfileprefix: ./


* 您需要提前从企业微信平台获取应用访问参数。有关更多信息，请参阅link:https://developer.work.weixin.qq.com/document/path/90664[企业微信开发者文档]。


== 操作步骤link:[id=b3cfb83ba42a492ba143cd10424a0af3]

include::../../../../../_custom/platformManagement/platformManagement-oper-logIn.adoc[]

+

include::../../../../../../_ks_components/oper-selectMenu.adoc[]

+

include::../../../../../../_ks_components/oper-navigate.adoc[]

+

. 在**通知渠道**页面，点击**企业微信**页签，设置企业微信通知参数，然后点击**确定**。
+
====
* 服务器设置
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|Corp ID
|您的企业在企业微信上的 Corp ID。

|Agent ID
|企业微信应用的 Agent ID。

|Secret
|企业微信应用的 Secret。
|===
--

* 接收设置
+
--
设置接收通知的 User ID、Department ID 或 Tag ID。点击**添加**可设置多个 ID。
--

include::../../../../../_custom/platformManagement/platformSettings/notificationChannels/notificationChannels-para-notificationConditions.adoc[]

include::../../../../../_custom/platformManagement/platformSettings/notificationChannels/notificationChannels-para-sendTestInformation.adoc[]
====

. 在**企业微信**页签右上角，点击切换开关将企业微信通知功能设置为**已启用**。
