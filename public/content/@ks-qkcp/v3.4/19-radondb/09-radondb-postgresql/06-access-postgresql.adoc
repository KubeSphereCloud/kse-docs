本小节主要介绍了如何访问 RadonDB PostgreSQL 数据库。


== 前提条件link:[id=a9430d7291ad412d8502d02b1aee9ecd]

* 请确保已获取{ks_product_both}平台登录用户帐号和密码，且已获取平台操作管理权限。

* 请确保已安装 RadonDB PostgreSQL 应用，且应用运行状态正常。关于如何创建安装 RadonDB PostgreSQL，请参阅link:../03-install-radondb-postgresql/[安装 RadonDB PostgreSQL]。

* 您已经创建了 RadonDB PostgreSQL 用户。关于如何创建用户，请参阅link:../05-manage-database-users/01-create-a-database-user/[创建数据库用户]。

. 在页面左侧的**访问地址**区域，查看 ClusterIP 访问地址。

. 执行以下命令通过 psql 命令行客户端直接连接数据库：
+
--
// Bash
include::../../../../_ks_components/code/bash.adoc[]

psql -U <username> -h <ip> -p <port> -d <dbname>

include::../../../../_ks_components/code/codeEnd.adoc[]

* **username**: 用户名。

* **ip**：IP 地址或域名。此处填写步骤五中获取到的访问地址中读写地址冒号（:）前的内容，如 radondb-uonx8x-readwrite.p1。

* **port**：服务端口。此处填写步骤五中获取到的访问地址中读写地址冒号（:）后的内容，如 5432。

* **dbname**：要连接的数据库名称，如 postgres。

示例命令如下所示：

// Bash
include::../../../../_ks_components/code/bash.adoc[]


psql -U pguser -h radondb-uonx8x-readwrite.p1 -p 5432 -d postgres
