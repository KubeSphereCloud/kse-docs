本节介绍如何从{ks_product_both}应用商店安装 RadonDB Kafka。


== 前提条件link:[id=f314cc8364b545fc9c23356268c2c2d6]

include::../../../_custom/radondb/radondb-prer-requiredPermission.adoc[]

* 您需要获取 RadonDB Kafka 应用授权。有关更多信息，请联系{ks_product_both}技术支持。


== 操作步骤link:[id=aaddb043dd4840829eeda13bf32e649b]

include::../../../_custom/projectManagement/projectManagement-oper-openProjectPage.adoc[]

+

include::../../../../_ks_components/oper-navigate.adoc[]

+

include::../../../_custom/radondb/radondb-oper-clickRadondbAppsTab.adoc[]

+

. 在页面右侧选择**安装 > RadonDB Kafka**。
+

include::../../../_custom/radondb/radondb-install-app.adoc[]

. 在**基本信息**页签，设置 RadonDB Kafka 应用的基本信息和安装位置，然后点击**下一步**。
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|名称
|RadonDB Kafka 应用的名称。名称只能包含小写字母、数字和连字符（-），必须以小写字母开头并以小写字母或数字结尾，最长 32 个字符。

|版本
|RadonDB Kafka 应用的版本。

|描述
|RadonDB Kafka 应用的描述信息。描述可包含任意字符，最长 256 个字符。

|位置
|RadonDB Kafka 应用所属的企业空间、集群和项目。
|===
--

. 在**应用设置**页签，设置 Kafka 的参数，然后点击**安装**。
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|Kafka 版本
|RadonDB Kafka 应用的版本。

|资源
|分配给每个 Kafka/ZooKeeper 节点的 CPU 和内存资源。

|Kafka/ZooKeeper 节点数量
|Kafka/ZooKeeper 节点的数量。

|存储类 
|数据库使用的存储系统对应的存储类。如果下拉列表中没有符合需要的存储类，您需要联系平台管理员创建存储类。

|卷
|Kafka/ZooKeeper 节点使用的卷大小，单位为 GiB。

|===

RadonDB Kafka 安装完成后将显示在 RadonDB 应用列表中。
