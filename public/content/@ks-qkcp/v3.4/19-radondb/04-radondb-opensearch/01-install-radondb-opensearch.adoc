本节介绍如何从{ks_product_both}应用商店安装 RadonDB OpenSearch。


== 前提条件link:[id= ]

include::../../../_custom/radondb/radondb-prer-requiredPermission.adoc[]

* 您需要获取 RadonDB OpenSearch 应用授权。有关更多信息，请联系{ks_product_both}技术支持。


== 操作步骤link:[id=bcf06a62873a40d085bcf9d81e2071ab]

include::../../../_custom/projectManagement/projectManagement-oper-openProjectPage.adoc[]

+

include::../../../../_ks_components/oper-navigate.adoc[]

+

include::../../../_custom/radondb/radondb-oper-clickRadondbAppsTab.adoc[]

+

. 在页面右侧选择**安装 > RadonDB OpenSearch**。

. 在**基本信息**页签，设置 RadonDB OpenSearch 应用的基本信息和安装位置，然后点击**下一步**。
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|名称
|RadonDB OpenSearch 应用的名称。名称只能包含小写字母、数字和连字符（-），必须以小写字母开头并以小写字母或数字结尾，最长 32 个字符。

|版本
|RadonDB OpenSearch 应用的版本。

|描述
|RadonDB OpenSearch 应用的描述信息。描述可包含任意字符，最长 256 个字符。

|位置
|RadonDB OpenSearch 应用所属的企业空间、集群和项目。
|===
--

. 在**应用设置**页签，设置 OpenSearch 数据库的参数，然后点击**安装**。
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|内核版本
|OpenSearch 数据库的版本。目前支持 OpenSearch 2.3.0。

|主节点/热节点/温节点/冷节点/Dashboards 节点/Logstash 节点设置

|**资源**：分配给每个主节点/热节点/温节点/冷节点的 CPU 和内存资源。

**主节点/热节点/温节点/冷节点/Dashboards 节点/Logstash 节点数量**：OpenSearch 数据库主节点/热节点/温节点/冷节点/Dashboards 节点/Logstash 节点的数量。主节点的数量默认为 **3** 且不能修改，其他节点数量可根据需要设置。

**存储类**：节点使用的存储系统对应的存储类。默认值为 **csi-qingcloud**。如果下拉列表中没有符合需要的存储类，您需要联系平台管理员创建存储类。

**卷**：节点使用的卷大小，单位为 GiB。主节点卷大小默认为 20 GiB。Kibana 节点不能挂载卷。其他节点的卷大小可根据需要设置。
|===
--

. 点击**高级设置**，您可以设置用户名和密码，也可以跳过这一步。

. 点击**安装**。安装成功的 RadonDB OpenSearch 将显示在 RadonDB 应用列表中。
