本节介绍如何访问 RadonDB Redis Sentinel。


== 前提条件link:[id=8e4d8a334be741f7b67f7ddbc305cbe6]

* 请确保已获取{ks_product_both}平台登录用户帐号和密码，且已获取平台操作管理权限。

* 请确保已安装 RadonDB Redis Sentinel 应用，且服务状态正常。

* 拥有可以访问 Redis Sentinel 集群的 kubectl 客户端。


== 操作步骤link:[id=b6850cfc7c044dfb9482a08f345dea7d]


=== 查看节点 IP 地址link:[id=00d8e3dc890c436192fed44737e09f64]

+

include::../../../../_ks_components/oper-navigate.adoc[]

include::../../../_custom/radondb/radondb-oper-clickRadondbAppsTab.adoc[]

include::../../../_custom/radondb/radondbRedisSentinel/radondbRedisSentinel-oper-openDetailsPage.adoc[]

+

. 将鼠标悬停至 Web 控制台右下角的 image:/images/ks-qkcp/zh/icons/hammer.svg[hammer,20,25]，点击 **kubectl**，在弹出的对话框执行以下命令进入 Redis 集群。
+
--
// Bash
include::../../../../_ks_components/code/bash.adoc[]

kubectl exec -it -n <namespace>   <name> -- sh

include::../../../../_ks_components/code/codeEnd.adoc[]

* <namespace>: Redis Sentinel 实例所在名称空间（项目名称），如 p1。

* <name>: Redis Sentinel 实例名称，如 radondb-mfncrc-sentinel-2。
--

. 连接 Redis Sentinel 集群实例。
+
--
// Bash
include::../../../../_ks_components/code/bash.adoc[]

redis-cli -h <redis_instance_address> -p <port>

include::../../../../_ks_components/code/codeEnd.adoc[]

* <redis_instance_address>：表示 Redis Sentinel 实例的连接地址，如 10.233.116.42。可在集群详情页面的**概览**区域查看连接地址，请根据实际地址替换。

* <port>：表示 Redis Sentinel 实例的端口号，默认为 6379。

连接示例如下：

// Bash
include::../../../../_ks_components/code/bash.adoc[]

redis-cli -h 10.233.116.42 -p 6379

include::../../../../_ks_components/code/codeEnd.adoc[]

* <namespace>: Redis Sentinel 实例所在名称空间（项目名称），如 p1。

* <name>: Redis Sentinel 实例名称，如 radondb-mfncrc-sentinel-2。
--

. 如果 Redis Sentinel 实例设置了访问密码，执行以下命令完成密码验证。
+
--
// Bash
include::../../../../_ks_components/code/bash.adoc[]

AUTH <password>

include::../../../../_ks_components/code/codeEnd.adoc[]

// Note
include::../../../../_ks_components/admonitions/note.adoc[]

* 如果是使用默认账号，直接填写默认密码 radondn123@ 即可。

* 如果是新创建的账号，则填写 <user>:<password>。例如：账号为 test，密码为 Test@123，则填写 auth test:Test@123。

include::../../../../_ks_components/admonitions/admonEnd.adoc[]

密码验证成功后，会返回 **OK**。
--

role
