本节介绍如何删除 RadonDB Redis Sentinel 应用。

// Warning
include::../../../../_ks_components/admonitions/warning.adoc[]

此操作将删除 RadonDB Redis Sentinel 数据库和数据库中的所有数据，请谨慎执行此操作。

include::../../../../_ks_components/admonitions/admonEnd.adoc[]


== 前提条件link:[id=e051ed6a9200490187964976a2fb2585]

include::../../../_custom/radondb/radondb-prer-requiredPermission.adoc[]

include::../../../_custom/radondb/radondb-prer-enableDmp.adoc[]

* 您需要获取 RadonDB Redis Sentinel 应用授权。有关更多信息，请联系{ks_product_both}技术支持。

* 为避免数据丢失，请提前备份数据。


== 操作步骤link:[id=4a61f49f02b54149a2f96bb74367267d]

include::../../../_custom/projectManagement/projectManagement-oper-openProjectPage.adoc[]

+

include::../../../../_ks_components/oper-navigate.adoc[]

+

include::../../../_custom/radondb/radondb-oper-clickRadondbAppsTab.adoc[]

+

. 在 RadonDB Redis Sentinel 应用右侧点击 image:/images/ks-qkcp/zh/icons/more.svg[more,18,18]，然后在下拉列表中选择**删除**。

. 在**删除应用**对话框，输入 RadonDB Redis Sentinel 应用的名称，然后点击**确定**。
