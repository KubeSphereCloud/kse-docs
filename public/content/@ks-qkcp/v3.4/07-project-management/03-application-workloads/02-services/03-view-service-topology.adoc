本节介绍如何查看服务拓扑。

{ks_product_right}提供服务拓扑功能，以图形化显示每个项目中服务的互相调用关系。


== 前提条件link:[id=ead59a15936a49c58287ab7ba0904420]


:relfileprefix: ../../../../
* 您需要加入一个项目并在项目中具有pass:a,q[{ks_permission}]权限。有关更多信息，请参阅xref:07-project-management/10-project-settings/04-project-members/_index.adoc[项目成员]和xref:07-project-management/10-project-settings/03-project-roles/02-view-a-project-role-list.adoc[项目角色]。

// include::../../../../_custom/projectManagement/projectManagement-prer-requiredPermission.adoc[]

:relfileprefix: ./


ifeval::["{file_output_type}" == "html"]

* {ks_product_right}平台需要启用服务拓扑扩展组件。

endif::[]

ifeval::["{file_output_type}" == "pdf"]

* {ks_product_right}平台需要启用服务拓扑扩展组件。有关更多信息，请参阅《{ks_product_right}平台管理指南》的“扩展组件管理”章节。

endif::[]


== 操作步骤link:[id=5a6cc88de1bc4a9284fa52b506936ca9]

include::../../../../_custom/projectManagement/projectManagement-oper-openProjectPage.adoc[]

+

include::../../../../../_ks_components/oper-navigate.adoc[]

. 在**服务**页面，点击**服务拓扑**。
+
--
**服务拓扑**页面显示项目中的所有服务，如果服务之间存在数据通信，页面还将以带箭头的连线标示服务之间的连接关系。

* 点击每个服务可以查看服务的运行状态。

* 点击 image:/images/ks-qkcp/zh/icons/start-light.svg[start-light,18,18]/image:/images/ks-qkcp/zh/icons/pause-light.svg[pause-light,18,18] 可启用/禁用自动数据刷新。

* 点击 image:/images/ks-qkcp/zh/icons/refresh-light.svg[refresh-light,18,18] 可手动刷新数据。

* 点击 image:/images/ks-qkcp/zh/icons/maximize.svg[maximize,18,18]/image:/images/ks-qkcp/zh/icons/minimize.svg[minimize,18,18] 可最大化/最小化服务拓扑图。
