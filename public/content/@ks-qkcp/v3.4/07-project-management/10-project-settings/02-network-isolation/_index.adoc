本节介绍如何为项目设置网络隔离。

网络隔离功能用于控制项目中容器组的出站和入站流量。您可以使用网络隔离功能实现以下目的：

* 只允许当前项目的容器组访问特定的其他容器组或网段。

* 只允许当前项目的容器组被特定的其他容器组或网段访问。
