本节介绍如何删除边缘节点。


== 前提条件link:[id=a41ec7aecd9247418def2cb2a1870177]

您需要获取{ks_product_both}集群主节点和边缘节点的 SSH 登录信息。


== 操作步骤link:[id=346d2d53962c40e59cb6c473e2247bdc]

. 登录边缘节点并执行以下命令删除边缘节点中的配置：
+
--
// Bash
include::../../../../_ks_components/code/bash.adoc[]

 ./keadm reset

include::../../../../_ks_components/code/codeEnd.adoc[]

// Bash
include::../../../../_ks_components/code/bash.adoc[]

apt remove mosquitto

include::../../../../_ks_components/code/codeEnd.adoc[]

// Bash
include::../../../../_ks_components/code/bash.adoc[]

rm -rf /var/lib/kubeedge /var/lib/edged /etc/kubeedge/ca /etc/kubeedge/certs

include::../../../../_ks_components/code/codeEnd.adoc[]
--

. 登录{ks_product_both}集群主节点并执行以下命令删除边缘节点。
+
--
// Bash
include::../../../../_ks_components/code/bash.adoc[]

kubectl delete node <edgenode-name>

include::../../../../_ks_components/code/codeEnd.adoc[]
--

. 在{ks_product_both}集群主节点执行以下命令检查当前的节点：
+
--
// Bash
include::../../../../_ks_components/code/bash.adoc[]

kubectl get node

include::../../../../_ks_components/code/codeEnd.adoc[]

如果命令回显中没有显示需要删除的边缘节点，则表明边缘节点删除成功。
--
