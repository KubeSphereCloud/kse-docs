== 网络要求

* 请确保 **/etc/resolv.conf** 中的 DNS 地址可用，否则，可能会导致集群中的 DNS 出现问题。

* 如果您的网络配置使用防火墙规则或安全组，请务必确保基础设施组件可以通过特定端口相互通信。建议您关闭防火墙。

* 支持的 CNI 插件：Calico 和 Flannel。其他插件也适用（例如 Cilium 和 Kube-OVN 等），但请注意它们未经充分测试。
