== WizTelemetry 可观测平台 v1.3.0

重要更新如下：

- 新增扩展组件 `Grafana Alloy for WizTelemetry`，不仅支持收集指标、日志和链路追踪等可观测信号，它还提供了许多功能，帮助您在复杂的环境中快速查找和处理数据。其中包括自定义组件、GitOps 兼容性、集群支持、安全性和调试实用程序。

- 扩展组件名称中的 `WhizardTelemetry` 都改为 `WizTelemetry`.

- `WizTelemetry 审计`、`WizTelemetry 事件`、`WizTelemetry 日志`、`WizTelemetry 通知` 新增 OpenSearch ISM 索引管理配置。OpenSearch Curator 功能已被废弃，并将在未来版本中删除。建议采用 ISM（Index State Management）功能进行索引管理。

== DevOps

=== DevOps v1.1.2

==== **优化**

- Jenkins 控制台默认登录方式修改为本地数据库，简化登录流程

- 修复添加 Gitlab 仓库失败的问题
