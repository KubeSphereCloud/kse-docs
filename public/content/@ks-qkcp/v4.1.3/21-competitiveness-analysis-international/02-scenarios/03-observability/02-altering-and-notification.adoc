== KubeSphere 企业版
在全面且易用的监控功能基础上，{ks_product_right}还提供强大的告警功能。系统内置多条常用告警规则，并针对这些规则提供相应的推荐解决方案，同时用户可以根据自身业务需求灵活配置告警规则。此外，{ks_product_right}还支持丰富的通知渠道，包括飞书、钉钉、企业微信、短信、邮件、Slack、Webhook 等主流方式（基于青云科技主导的开源项目 Notification Manager）。

== OpenShift
需通过 OpenShift CLI 自定义告警规则，支持 Slack、Email、Webhook、PagerDuty 渠道通知。

== Rancher
内置部分告警规则，支持页面配置告警规则，支持 Slack、Email、Webhook、PagerDuty、Microsoft Teams、钉钉、企业微信等渠道通知。
