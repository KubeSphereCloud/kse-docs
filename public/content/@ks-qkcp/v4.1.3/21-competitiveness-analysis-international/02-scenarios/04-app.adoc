== KubeSphere 企业版
支持 Helm 应用商店，提供应用全生命周期管理，包括应用上传、信息编辑、审核和上架管理。借助 ISV、Reviewer 等多角色协同，实现完整的应用生命周期管理流程。

== OpenShift
支持 operator hub 与 Helm 形式应用仓库，提供商业软件服务购买引流入口，对开发者需要从 Git 导入代码库来创建应用程序，支持 UI 页面上传与生命周期管理。

== Rancher
仅支持商店仓库的 URL 管理，不支持开发者在 UI 界面上传应用与管理应用的生命周期。
