== KubeSphere 企业版

{ks_product_right}相比较 DaoCloud 与灵雀云，虽然产品起步时间较晚，但是其产品化程度和安装使用的易用性上相比较 DaoCloud 与灵雀云更高。其产品设计思路是依托自主研发的 LuBan 分布式可扩展开放架构，{ks_product_right}在 KubeSphere 开源版的基础上进一步优化，全面覆盖多云集群管理、微服务治理、云边协同、全栈可观测、DevOps、企业应用分发以及数据基础设施管理等核心业务场景。

== DaoCloud

DaoCloud（道客云）是一家专注于企业级云原生解决方案的提供商，其核心产品 DaoCloud Enterprise（DCE），涵盖容器、网络、存储、监控等核心组件，具备高可用性和稳定性，支持多种云环境部署。DaoCloud 在云原生领域技术实力突出，开源贡献排名全球领先，其解决方案广泛应用于金融、能源、制造等多个行业，服务上千家企业，尤其在边缘计算领域表现优异。然而，DaoCloud 的解决方案更偏向于对云原生技术有较高需求的企业，且部分高级功能可能需要通过商业订阅解锁。

== 灵雀云

image:/images/ks-qkcp/zh/competitiveness-analysis/灵雀云.png[]

灵雀云是一家专注于企业级容器技术的云原生平台提供商。其核心产品基于 Kubernetes ，提供平台化基础设施、应用交付、自动化运维管理和多租户支持等功能，支持多种资源自动扩缩容。灵雀云在金融、制造、能源等行业广泛应用，助力企业数字化转型。然而，其产品更偏向于对容器化和微服务有较高需求的企业，部分高级功能可能需要通过商业订阅解锁。
