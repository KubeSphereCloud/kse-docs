== KubeSphere 企业版

自{ks_product_left} v4.0 起，KubeSphere 引入了扩展机制并推出全新的 LuBan 架构，该架构基于 Kubernetes 构建，采用微内核 + 扩展组件模式：系统核心（KubeSphere Core）仅保留运行所必需的基础功能，而各独立业务模块则以热插拔的扩展组件形式存在。所有功能组件及第三方组件均可无缝集成到控制台中并独立维护版本，用户可在系统运行时动态安装、卸载、启用或禁用扩展组件，从而实现即插即用的灵活云原生操作系统。


== DaoCloud

DaoCloud 的技术架构以云原生为核心，支持多版本 Kubernetes 集群的统一管理，兼容多云和混合云环境。其提供微服务治理、可观测性、云原生 CI/CD 流水线、云边协同等功能，同时支持信创生态，融合 AI 与大数据技术，助力企业实现数字化转型和应用现代化。


== 灵雀云

灵雀云的技术架构以 Kubernetes 为核心的全栈云原生平台，支持混合云和多云管理，提供微服务治理、DevOps 支持、应用管理与监控、数据中间件服务、安全治理等功能，同时支持服务网格和开源生态，助力企业高效构建和管理现代化应用，加速数字化转型。
