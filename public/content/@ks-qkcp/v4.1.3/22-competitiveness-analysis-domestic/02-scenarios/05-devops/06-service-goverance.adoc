== KubeSphere 企业版
{ks_product_right}内置 SpringCloud 扩展组件，集成 Nacos 作为注册中心和配置中心，支持通过 UI 管理微服务。其 Kubernetes 管理平台为 SpringCloud 应用的部署、管理和监控提供全面支持，同时支持蓝绿部署、金丝雀发布、流量镜像，无需学习 Istio。此外，KubeSphere 内置微服务流量拓扑图，支持调整细粒度流量治理策略，提供可视化链路追踪，支持限流与熔断配置，无需手动设置。


== DaoCloud
DaoCloud 的 Kubernetes 管理平台虽不直接集成 Spring Cloud，但可为 Spring Cloud 应用的部署、管理和监控提供有力支持。同时支持蓝绿部署、金丝雀发布、A/B 测试，具备微服务南北流量管控功能，并集成 OpenTelemetry 和 Jaeger，支持可视化查看链路数据。

== 灵雀云
灵雀云的 Kubernetes 管理平台虽不直接集成 Spring Cloud，但可为 Spring Cloud 应用的部署、管理和监控提供有力支持。其功能包括集群可视化大屏监控，蓝绿部署、金丝雀发布、A/B 测试，集成 Istio 网关和虚拟服务配置以实现流量负载均衡、超时、重试等策略，以及集成 Jaeger 支持通过 TraceID 快速定位调用链路。
