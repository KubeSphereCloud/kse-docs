== KubeSphere 企业版
{ks_product_right}支持任意原生 Kubernetes 及基于 Kubernetes 的容器管理平台，提供跨集群统一应用分发与调度，支持多集群 CI/CD 流水线及多维度“企业空间”租户管理；同时深度集成 KubeEdge 并支持 K3s，实现应用分发到边缘节点，并提供统一监控与日志管理。

== DaoCloud
DaoCloud 兼容多版本 Kubernetes 集群，支持云上云下统一管理，通过 Web 界面可一键创建适配物理机和虚拟机的企业级集群。其支持 Kubernetes 版本及系统组件批量升级，简化运维流程；内置集群容灾与备份能力，保障集群高可用；提供开放式 API，支持原生 Kubernetes OpenAPI，同时实现对所有边缘节点和集群信息的统一管控。

== 灵雀云
灵雀云支持多集群管理，支持节点维护管理、资源配额管理及联邦集群创建。其支持多主架构服务网格，通过添加集群即可实现多集群纳管，各集群配置一致但相互独立。此外，灵雀云支持边缘 AI 业务部署迭代，容器可通过 Device Plugin 插件访问边缘设备的串口、网口、VGA 等端口，并支持统一管控所有边缘节点。
