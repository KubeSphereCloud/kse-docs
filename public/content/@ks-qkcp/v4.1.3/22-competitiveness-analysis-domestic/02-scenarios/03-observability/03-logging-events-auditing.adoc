== KubeSphere 企业版

{ks_product_right}提供统一管理入口——工具箱，安装后无需复杂配置即可使用。用户可以通过工具箱统一检索和查看日志、事件、审计等信息，并支持从平台、集群、租户等多角度进行管理与查询，同时展示日志数量趋势，日志上下文还可关联到业务应用详情，便于问题定位。针对无法标准化输出的业务日志，系统支持日志落盘并自动转换为标准输出格式；同时，针对用户自建的日志基础设施，平台也可外接 Elasticsearch、Kafka、Fluentd 等系统。

== DaoCloud

DaoCloud 支持采集容器、应用及第三方服务的日志，实现日志与链路追踪、监控指标的联动分析；支持集群级事件查询，并基于历史事件库提供故障修复建议；记录用户对集群、应用、配置的增删改查操作，生成符合等保、GDPR 等标准的审计报表。

== 灵雀云

灵雀云支持日志采集、存储、全文搜索与统计分析，兼容 Elasticsearch、Logstash、Fluentd、Kibana 等主流日志系统，可查看平台所有资源事件及用户操作信息。
