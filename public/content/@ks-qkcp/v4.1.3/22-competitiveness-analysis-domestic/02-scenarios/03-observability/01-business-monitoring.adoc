== KubeSphere 企业版
{ks_product_right}监控功能支持从租户、集群和平台等多个视角切入，灵活满足不同行政层级下的资源管理需求，用户可根据自身架构自由组合监控策略。平台深度集成了各类监控面板于控制台，实现跨集群、节点、项目和应用多维度的可观测性 Dashboard，无需繁琐配置即可优雅展示监控数据，避免了使用 Grafana 的高门槛。此外，针对特殊业务需求，{ks_product_right}还提供默认自定义监控面板，用户可轻松设置关键指标，并可直接导入现成的 Grafana 模板。

== DaoCloud

DaoCloud 支持从基础设施、容器到应用的多层级监控，集成 Grafana 预置仪表盘，并支持自定义面板。

== 灵雀云

灵雀云支持集群、节点、应用、Pod 等多维度监控，集成 Prometheus 和 Grafana，支持自定义面板。
