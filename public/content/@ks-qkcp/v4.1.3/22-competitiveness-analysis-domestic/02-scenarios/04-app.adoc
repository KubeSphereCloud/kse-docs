== KubeSphere 企业版
{ks_product_right}支持 Helm 应用商店，提供应用全生命周期管理，包括应用上传、信息编辑、审核和上架管理。借助 ISV、Reviewer 等多角色协同，实现完整的应用生命周期管理流程。

== DaoCloud
DaoCloud 支持云原生应用、Helm、OAM 等多技术栈，解耦底层 Kubernetes，实现一键式部署；提供界面化手动与自动扩缩容功能，支持自定义策略以灵活应对业务流量波动；支持集中管理多集群应用负载，涵盖查看、更新、回滚、升级等全生命周期操作。

== 灵雀云
灵雀云集成 Helm 和 Operator，支持配置应用仓库，兼容多种开发语言和框架，提供代码管理、构建、测试等一体化开发环境。同时支持一键部署应用，自动完成资源配置和调度，并具备自动故障恢复及高可用部署能力。
