== 前提条件link:[id=ed9f53748b54483ba6ab87fc72874b7e]

:relfileprefix: ../../../

* 您需要加入一个项目并在项目中具有pass:a,q[{ks_permission}]权限。有关更多信息，请参阅xref:07-project-management/10-project-settings/04-project-members/_index.adoc[项目成员]和xref:07-project-management/10-project-settings/03-project-roles/02-view-a-project-role-list.adoc[项目角色]。

== 编辑自定义伸缩信息link:[id=0b89add4c3664816ad1686b4b2dd767b]

include::../../../_custom/projectManagement/projectManagement-oper-openProjectPage.adoc[]

+

include::../../../../_ks_components/oper-navigate.adoc[]

. 在**自定义伸缩**页面，点击目标自定义伸缩任务后的 image:/images/ks-qkcp/zh/icons/more.svg[more,18,18]，然后在下拉列表中选择**编辑信息**。

* 在弹出的**编辑信息** 对话框，修改别名和描述信息，点击**确定**。


== 编辑伸缩设置link:[id=9f9c8a436b914c19b2fb098ee68fc1ee]

include::../../../_custom/projectManagement/projectManagement-oper-openProjectPage.adoc[]

+

include::../../../../_ks_components/oper-navigate.adoc[]

. 在**自定义伸缩**页面，点击目标自定义伸缩任务后的 image:/images/ks-qkcp/zh/icons/more.svg[more,18,18]，然后在下拉列表中选择**编辑伸缩设置**。

* 在弹出的**编辑自定义伸缩** 对话框，修改自定义伸缩参数，点击**确定**。

// Note
include::../../../../_ks_components/admonitions/note.adoc[]

关于参数的详细解释，请参阅link:../01-enable-custom-scaling/#16484ea0d8f649608f84416995476ff9[启用自定义伸缩]。

include::../../../../_ks_components/admonitions/admonEnd.adoc[]



== 删除自定义伸缩link:[id=197b0074dc594d48a0dddee57826faa9]

include::../../../_custom/projectManagement/projectManagement-oper-openProjectPage.adoc[]

+

include::../../../../_ks_components/oper-navigate.adoc[]

. 在**自定义伸缩**页面，点击目标自定义伸缩任务后的 image:/images/ks-qkcp/zh/icons/more.svg[more,18,18]，然后在下拉列表中选择**删除**。

* 在弹出的**删除自定义伸缩** 对话框，输入自定义伸缩名称，点击**确定**。
