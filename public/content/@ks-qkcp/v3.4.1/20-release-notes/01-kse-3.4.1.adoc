== DevOps

=== 优化增强

- 支持在流水线创建和编辑页面复用已创建的代码仓库。
- 优化流水线运行日志详情页面布局。
- 优化 devops-jenkins JVM 内存配置。
- 增加流水线运行回放功能的文字提示。
- 流水线中增加 email extension 插件支持。

=== 问题修复

- 修复已被删除的分支流水线仍然能够被触发的问题。
- 修复关联不存在项目的 ArgoCD 列表页报错。
- 将持续部署的 ArgoCD 加入系统项目中。
- 修复删除持续部署应用时未自动删除部署资源的问题。
- 修复下载多分支流水线制品失败的问题。

== 可观测性

=== 优化增强

- 优化节点告警信息，去掉不必要的 pod 和 namespace 标签。
- 优化告警通知：同一项目下，如果没有 warning/error/critical 级别的告警，不再为 info 级别的告警发送通知。
- 增强内置的 KubePersistentVlumeErrors 规则，产生的告警消息将包含与 PV 关联的 PVC 信息。
- 优化多集群 PrometheusRule 同步逻辑，避免主集群在成员集群解绑后规则同步状态未清理问题。
- alerting 及 observability 模块增强 httpClient 配置。
- 支持设置导出日志的最大数量。
- 支持通过告警状态查询通知历史。
- 升级 Whizard 版本至 v0.7.0。
- Whizard 组件支持 HTTPS。

=== 问题修复

- 修复 ETCD 监控配置问题，ETCD 未启用安全设置时不再配置 kube-etcd-client-certs 到 Prometheus。
- 修复 Infohibitor 规则。
- 修复 node_timex_sync_status 未采集问题。
- 修复部分内置告警规则使用的指标不存在的问题。

== 应用商店

=== 问题修复

- 修复应用商店中部分应用无法安装的问题。


== 多租户和多集群

=== 问题修复

- 修复企业空间授权集群存在重复条目的问题。

== 网络

=== 问题修复

- 修复 ippool block 字段验证。


== 安全

=== 优化增强

- 支持 Pod Security Standards 配置自动注入。

== 验证和授权

=== 问题修复

- 修复配置文件重载后 IdentityProvider 冲突的问题。
- 修复 enableMultiLogin 配置不生效的问题。

== DMP
=== OpenSearch
==== 优化增强

- 增加 JVM 自适应，若未定义，设置 JVM 为容器内存的一半。
- 增加救援模式功能，允许用户在集群为非 green 状态下重启集群。
- 支持通过配置对接第三方身份认证 OIDC。
- 提升最大磁盘规格为 15T。

==== 问题修复

- 当存储类型不支持扩容时，前端提示用户无法做扩容操作。
- 修复 pod IP 变更后，监控数据不全的问题。

=== Redis Cluster 和 Redis Sentinel
==== 优化增强

- 增加 maxmemory 参数自适应，若未定义，设置为容器内存的 95%。
- 支持调整日志级别，并优化日志内容。
- 提升安全性，针对密码做加密。

==== 问题修复

- 修复升级后，集群状态在 "更新中" 和 "运行中" 反复切换的问题。
- 修复修改密码后无法获取监控数据的问题。

== 用户体验

- 统一工作负载和服务的删除逻辑，删除时同步删除关联资源。
- 工作空间列表去掉已删除的集群标签。
- 修复了 Harbor 镜像根据 project 和 repo 名称查询的问题。
- 修复自定义伸缩（hap）里的开始时间和结束时间的 cron 校验错误问题。
- 修复项目配额限制不能更改问题。
- 修复镜像服务地址设置为 **https://docker.io**，在搜索镜像时，接口报错。
- 新增搜索镜像时列出标签列表并确认选择标签。
- 废弃重复的集群标签功能（将在 4.0 版本中移除）。
- 支持最小化安装时禁用 minio 插件（在禁用 minio 的时候，还需要禁用 devops 和 openpitrix 插件，否则禁用不生效）。

=== API 更新

- 新增 checkDevopsName 和 checkPipelineName 两个 API 用于校验 DevOps 项目和流水线名称是否重复。
- 增加验证 PromQL 合法性的 API。
- 废弃集群状态中的 Configz 字段（将在下个大版本中移除）。

== 其他

- 支持红帽 8.x 版本。
- 支持成员集群禁用 ks-console 控制台。
- 支持自定义单集群名称。
