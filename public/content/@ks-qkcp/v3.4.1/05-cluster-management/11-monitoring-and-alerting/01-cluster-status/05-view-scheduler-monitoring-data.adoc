本节介绍如何查看调度器状态。调度器根据容器组的资源需求、节点可用资源和其他约束条件决定容器组运行在哪些节点上。

. 在**集群状态**页面，点击**调度器监控**页签查看调度器的运行状态。
+
====
* **调度器监控**页签提供以下信息：
+
--
[%header,cols="1a,4a"]
|===
|指标 |描述

|调度次数
|指定时间范围内的容器组调度成功、错误和失败累计次数。

|调度速率
|指定时间范围内的容器组每秒调度成功、错误和失败次数。

|调度延迟
|指定时间范围内容器组调度延迟的 99% 分位数、90% 分位数、中位数和平均值。
|===
--
