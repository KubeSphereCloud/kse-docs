本节介绍如何访问 RadonDB Redis Cluster。


== 前提条件link:[id=ad8cf9e45dbf4b299824eba1227b8319]

* 请确保已获取{ks_product_both}平台登录用户帐号和密码，且已获取平台操作管理权限。

* 请确保已安装 RadonDB Redis Cluster 应用，且服务状态正常。


== 操作步骤link:[id=1a526a444cef4c1f806e0881cf76542f]


include::../../../_custom/projectManagement/projectManagement-oper-openProjectPage.adoc[]

+

include::../../../../_ks_components/oper-navigate.adoc[]

+

include::../../../_custom/radondb/radondb-oper-clickRadondbAppsTab.adoc[]

+

include::../../../_custom/radondb/radondbRedisCluster/radondbRedisCluster-oper-openDetailsPage.adoc[]

+

. 将鼠标悬停至 Web 控制台右下角的 image:/images/ks-qkcp/zh/icons/hammer.svg[hammer,20,25]，点击 **kubectl**，在弹出的对话框执行以下命令进入 Redis 集群。
+
--
// Bash
include::../../../../_ks_components/code/bash.adoc[]

kubectl exec -it -n <namespace>   <name>-leader-0 -- sh

include::../../../../_ks_components/code/codeEnd.adoc[]

* <namespace>：Redis Cluster 实例所在命名空间（即项目名称）。

* <name>：Redis Cluster 实例名称。
--

. 连接 Redis Cluster 集群实例。
+
--
// Bash
include::../../../../_ks_components/code/bash.adoc[]

redis-cli -h <redis_instance_address> -p <port> -c

include::../../../../_ks_components/code/codeEnd.adoc[]

* <redis_instance_address>：表示 Redis Cluster 实例的连接地址，如 10.233.85.29。可在集群详情页面的**概览**区域查看连接地址，请根据实际地址替换。

* <port>：表示 Redis Cluster 实例的端口号，默认为 6379。

* -c：连接集群节点时的必选参数。

连接示例如下：
// Bash
include::../../../../_ks_components/code/bash.adoc[]

redis-cli -h 10.233.85.29 -p 6379 -c

include::../../../../_ks_components/code/codeEnd.adoc[]
--

. 如果 Redis Cluster 实例设置了访问密码，执行以下命令完成密码验证。
+
--
// Bash
include::../../../../_ks_components/code/bash.adoc[]

AUTH <password>

include::../../../../_ks_components/code/codeEnd.adoc[]

// Note
include::../../../../_ks_components/admonitions/note.adoc[]

* 如果是使用默认账号，直接填写默认密码 radondn123@ 即可。

* 如果是新创建的账号，则填写 <user>:<password>。例如：账号为 test，密码为 Test@123，则填写 auth test:Test@123。

include::../../../../_ks_components/admonitions/admonEnd.adoc[]

密码验证成功后，会返回 **OK**。
--

. 执行以下命令查看节点信息。
+
--
// Bash
include::../../../../_ks_components/code/bash.adoc[]

cluster nodes

include::../../../../_ks_components/code/codeEnd.adoc[]
--
