+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|规则名称
|告警规则的名称。规则名称可包含任意字符，最长 63 个字符。

|触发条件
|触发告警指标、持续时间和告警级别。

|概要
|告警消息的内容概要。消息概要可包含任意字符，最长 63 个字符。

|详情
|告警消息的详细描述。消息详情可包含任意字符，最长 256 个字符。
|===
--
