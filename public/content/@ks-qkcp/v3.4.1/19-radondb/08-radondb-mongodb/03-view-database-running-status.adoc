--
已安装的 RadonDB MongoDB 应用将显示在 RadonDB 应用列表中。

[%header,cols="1a,4a"]
|===
|参数 |描述

|名称
|RadonDB MongoDB 应用在{ks_product_both}平台的名称。

|状态
|RadonDB MongoDB 应用的运行状态。

* **创建中**：应用正在创建过程中。

* **更新中**：应用正在更新配置。

* **已完成**：应用更新完成，即将开始运行。

* **失败**：应用创建过程中出现错误导致创建失败。

* **运行中**：应用运行正常。

|应用模板
|创建 RadonDB MongoDB 应用所使用的应用模板名称。

|版本
|MongoDB 数据库的版本。

|访问地址
|MongoDB 数据库的访问地址。

|创建时间
|RadonDB MongoDB 应用的创建时间。
|===
--

+

include::../../../_custom/radondb/radondbMongoDB/radondbmongodb-oper-openDetailsPage.adoc[]

+

. 在 RadonDB MongoDB 详情页面右侧的**概览**页签查看数据库的基本信息和数据库节点。
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|基本信息
|显示 MongoDB 数据库运行状态、节点和副本数量、节点的 CPU 和内存资源、节点挂载的卷容量。

|节点
|显示 MongoDB 数据库的所有节点及其创建时间、运行状态、容器组 IP 地址、节点类型、CPU 和内存资源、卷容量。
|===
