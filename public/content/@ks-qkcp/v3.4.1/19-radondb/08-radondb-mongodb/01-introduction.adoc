== 什么是 RadonDB MongoDBlink:[id=6752b700268247309f75e227f803c34a]

RadonDB MongoDB 是一款基于 MongoDB 副本集技术的分布式文档数据库。支持 MongoDB 应用部署和全生命周期管理。

== 功能特性link:[id=d9299b2a44fb47aca9963ce03773c11e]

* 一键安装部署
+
--
提供一键部署，开箱即用服务，免去复杂的手动安装、部署。
--

* 弹性扩容伸缩
+
--
提供对数据库节点数量、资源规格、卷容量的弹性扩容。
--

* 实时监控告警
+
--
提供应用节点资源和服务的监控指标，支持可视化监控应用状态。
--

* 配置参数管理
+
--
开放管理和查看数据库配置参数，提高数据库运维效率。
--
