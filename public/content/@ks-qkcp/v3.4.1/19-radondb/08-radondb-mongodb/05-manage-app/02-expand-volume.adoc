RadonDB MongoDB 创建成功后，可在{ks_product_both}应用管理页面扩展卷。

本节介绍如何扩展挂载到 RadonDB MongoDB 节点的卷。

// note
include::../../../../../_ks_components/admonitions/note.adoc[]

* 目前不支持对使用{ks_product_both}集群本地存储（存储类为 local）创建的卷进行扩展。

* 目前仅支持增加卷容量，不支持减少卷容量。

* 扩容卷可能会造成业务短暂的中断，请在业务低峰时进行。

include::../../../../../_ks_components/admonitions/admonEnd.adoc[]
