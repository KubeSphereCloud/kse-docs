--
已安装的 RadonDB OpenSearch 应用将显示在 RadonDB 应用列表中。

[%header,cols="1a,4a"]
|===
|参数 |描述

|名称
|RadonDB OpenSearch 应用在{ks_product_both}平台的名称。

|状态
|RadonDB OpenSearch 应用的运行状态。

* **创建中**：应用正在创建过程中。

* **更新中**：应用正在更新配置。

* **已完成**：应用更新完成，即将开始运行。

* **失败**：应用创建过程中出现错误导致创建失败。

* **运行中**：应用运行正常。

|应用模板
|创建 RadonDB OpenSearch 应用所使用的应用模板名称。

|版本
|OpenSearch 数据库的版本。

|访问地址
|OpenSearch 数据库的访问地址。每个数据库可提供多个不同用途的访问地址。将光标悬停在访问地址列的数字上可查看 ClusterIP 读写地址和控制面板地址。

|创建时间
|RadonDB OpenSearch 应用的创建时间。
|===
--

+

include::../../../_custom/radondb/radondbOpenSearch/radondbopensearch-oper-openDetailsPage.adoc[]

+

. 在 RadonDB OpenSearch 详情页面右侧的**概览**页签查看数据库的基本信息和数据库节点。
+
--
[%header,cols="3a,4a"]
|===
|参数 |描述

|基本信息
|显示 OpenSearch 数据库的运行状态以及主节点、热节点、温节点、冷节点、Dashboards 节点和 Logstash 节点的数量。

|节点
|显示容器组名称、状态、IP地址、节点数量、节点的 CPU 和内存资源、每个数据库节点挂载的卷容量。
|===
