== 什么是 RadonDB ECKlink:[id=17d0ba4610324a9db19cea46bbf490f2]

RadonDB ECK 是一款为用户提供实时日志处理、全文搜索和数据分析等能力的检索分析服务，基于 Elastic Cloud on Kubernetes 构建，扩展了基本的容器编排功能，支持在容器组织和管理 Elastic Stack。

== 功能特性link:[id=1e295f80a284432c8a160afcce3c0b12]

* 一键安装部署
+
--
支持一键安装部署，开箱即用服务，免去复杂的手动安装、部署。
--

* 弹性伸缩扩容
+
--
支持节点横向和纵向扩容，包括对节点数量、资源规格、卷容量的弹性扩容。
--

* 监控告警管理
+
--
提供应用节点服务的监控指标，支持可视化监控应用状态。并支持设置告警规则，当监控项满足告警规则时，系统将生成告警。
--

* 配置参数管理
+
--
开放管理和查看配置参数，提高运维效率。
--

* 可视化管理工具
+
--
可视化管理工具 Kibana，为 Elasticsearch 提供了强大的可视化界面，方便用户通过浏览器使用 Elasticsearch。
--
