--
[%header,cols="1a,4a"]
|===
|参数 |描述

|名称
|RadonDB Kafka 应用在{ks_product_both}平台的名称。

|状态
|RadonDB Kafka 应用的运行状态。

* **创建中**：应用正在创建过程中。

* **更新中**：应用正在更新配置。

* **已完成**：应用更新完成，即将开始运行。

* **失败**：应用创建过程中出现错误导致创建失败。

* **运行中**：应用运行正常。

|应用模板
|创建 RadonDB Kafka 应用所使用的应用模板名称。

|版本
|Kafka 的版本。

|访问地址
|Kafka 的访问地址。每个数据库可提供多个不同用途的访问地址。将光标悬停在访问地址列的数字上可查看 NodePort 的读写地址。

|创建时间
|RadonDB Kafka 应用的创建时间。
|===
--

+

include::../../../_custom/radondb/radondbKafka/radondbkafka-oper-openDetailsPage.adoc[]

+

. 在页面左侧区域，您可以查看 Kafka 应用属性和访问地址，下载 CA 证书，并查看 CA 密码和证书有效期。

. 在详情页面右侧的**概览**页签查看数据库的基本信息和数据库节点。
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|基本信息
|显示 Kafka 的运行状态、Kafka 和 ZooKeeper 节点数量、每个 Kafka 和 ZooKeeper 节点的 CPU 和内存资源以及挂载的卷容量。

|节点
|显示 Kafka 的所有节点及其创建时间、运行状态、容器组 IP 地址、CPU 和内存资源、卷容量。
|===
--

. 点击节点最右侧的监控图标，您可以查看单个节点的监控数据，包括网络处理器空闲率、磁盘写入速率、磁盘读取速率、内存用量、CPU 用量、每秒接收消息数量、JVM 内存用量、JVM 线程数量和可用磁盘空间。

**所有 Kafka 监控项**

[%header,cols="3,5"]
|===
|监控项 |说明

|在线中介（Broker）数量
|在线的 Kafka Broker 数量。


|ISR 少于下限分区数量
|当前 ISR 低于下限的所有 Partition 总量。


|离线分区数量
|离线的分区数量。

|每秒流入字节数（B/s）
|每秒流入的字节数，单位为 B/s。


|每秒流出字节数（B/s）
|每秒流出的字节数，单位为 B/s。

|每秒流入消息数
|每秒流入的消息数量。

|每秒客户端请求数
|每秒的客户端请求数量。

|===
