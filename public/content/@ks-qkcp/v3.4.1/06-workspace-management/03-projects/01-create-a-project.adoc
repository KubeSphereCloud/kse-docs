本节介绍如何创建项目或多集群项目。


== 前提条件link:[id=46d6dfc7f89242c1960a1e340723274a]

:relfileprefix: ../../../

* 您需要加入一个企业空间并在企业空间中具有pass:a,q[{ks_permission}]权限。有关更多信息，请参阅xref:06-workspace-management/06-workspace-settings/03-workspace-members/_index.adoc[企业空间成员]、xref:06-workspace-management/06-workspace-settings/04-workspace-roles/_index.adoc[企业空间角色]和xref:06-workspace-management/06-workspace-settings/05-departments/_index.adoc[部门]。

// include::../../../../_custom/workspaceManagement/workspaceManagement-prer-requiredPermission.adoc[]

:relfileprefix: ./


ifeval::["{file_output_type}" == "html"]

* 如需创建多集群项目，您需要启用多集群管理功能。有关更多信息，请参阅link:../../../04-platform-management/01-extension-management/[启用可插拔组件]。

endif::[]

ifeval::["{file_output_type}" == "pdf"]

* 如需创建多集群项目，您需要启用多集群管理功能。有关更多信息，请参阅《{ks_product_right}平台管理指南》的“扩展组件管理”章节。

endif::[]
