[%header,cols="1a,4a"]
|===
|参数 |描述

|用户名
|用户的名称。用户名只能包含小写字母、数字、连字符（-）和句点（.），必须以小写字母或数字开头及结尾，最长 32 个字符。

|密码
|用户的密码。密码必须包含数字、大写字母和小写字母，长度为 8 至 64 个字符。

include::../../../../_custom/platformManagement/accessControl/users/users-para-description.adoc[]
|===
