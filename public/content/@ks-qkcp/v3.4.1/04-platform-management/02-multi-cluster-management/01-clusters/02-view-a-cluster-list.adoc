* 集群列表提供以下信息：
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|名称、标签和描述
|集群的名称、标签和描述信息。

|节点数量
|集群中服务器节点的数量。

|Kubernetes 版本
|集群中运行的 Kubernetes 版本。

|提供商
|集群环境的提供商。

|创建时间
|集群加入{ks_product_left} Web 控制台的时间。
|===
--

* 在列表上方点击搜索框并设置搜索条件，可按名称、类型或标签搜索集群。
