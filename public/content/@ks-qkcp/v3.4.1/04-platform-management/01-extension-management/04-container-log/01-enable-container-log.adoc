* 将 **spec:logging:enabled** 参数设置为 **true**。
+
--
// YAML
include::../../../../../_ks_components/code/yaml.adoc[]

spec:
  logging:
    enabled: true

include::../../../../../_ks_components/code/codeEnd.adoc[]
--

include::../../../../_custom/platformManagement/extensionManagement/extensionManagement-oper-setExternalElasticsearch.adoc[]

* 如需设置系统自动清理容器日志，将 **spec:common:es:logMaxAge** 参数设置为容器日志的保存期限（单位为天），并删除 **status:es** 参数的全部内容。
+
--
// YAML
include::../../../../../_ks_components/code/yaml.adoc[]

spec:
  common:
    es:
      logMaxAge: <retention period>

include::../../../../../_ks_components/code/codeEnd.adoc[]

include::../../../../_custom/platformManagement/extensionManagement/extensionManagement-oper-deleteStatusEs.adoc[]
