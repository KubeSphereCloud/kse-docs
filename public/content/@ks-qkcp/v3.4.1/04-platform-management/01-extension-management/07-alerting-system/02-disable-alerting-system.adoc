. 在集群配置文件中，将 **spec:alerting:enabled** 参数设置为 **false**，然后保存文件。
+
--
// YAML
include::../../../../../_ks_components/code/yaml.adoc[]

spec:
  alerting:
    enabled: false

include::../../../../../_ks_components/code/codeEnd.adoc[]
--

. 在 **kubectl** 对话框，执行以下命令删除告警系统相关资源：
+
--
// Bash
include::../../../../../_ks_components/code/bash.adoc[]

kubectl -n kubesphere-monitoring-system delete thanosruler kubesphere
