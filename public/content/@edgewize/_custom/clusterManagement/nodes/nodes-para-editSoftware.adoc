* 若软件尚未安装，点击**安装**，安装容器引擎和边缘软件。

* 点击image:/images/ks-qkcp/zh/icons/trash-light.svg[trash,18,18]，可卸载容器引擎和边缘软件。

* 点击image:/images/ks-qkcp/zh/icons/refresh-light.svg[refresh,18,18]，可重启容器引擎和边缘软件。

* 点击滑块，可禁用或启用容器引擎和边缘软件。

* 点击image:/images/ks-qkcp/zh/icons/chevron-down.svg[chevron-down,18,18]，可展开并编辑配置文件。
