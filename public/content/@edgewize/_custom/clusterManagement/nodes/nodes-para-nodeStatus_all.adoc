边缘节点的运行状态。

* **注册中**：正在边缘节点上部署和启动一个 `edgewize-ota-agent` 服务。

* **注册失败**：边缘节点上的 `edgewize-ota-agent` 服务启动失败。

* **未纳管**：边缘节点注册成功，但节点未加入 Kubernetes 边缘集群。

* **运行中**：边缘节点已被 Kubernetes 边缘集群纳管，且运行正常。

* **告警**：边缘节点已被 Kubernetes 边缘集群纳管，但运行异常。您可以在节点详情页面的**任务执行信息**页签查看节点的告警信息。

* **无法调度**：节点被用户手动设置为不可调度。如果节点处在此状态，容器组将不能被调度到该节点。

* **离线**：在未纳管或运行状态下，长时间未收到边缘节点发送的消息。

* **删除中**：正在删除边缘节点。
