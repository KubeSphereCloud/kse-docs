|===
|术语 | 说明
|Meta 集群（MetaCluster）	
|Kubernetes 元集群。

|KubeSphere 企业版集群（KseCluster）	
|运行在 MetaCluster 集群中的 Kubernetes 集群（基于vcluster安装），包含 KubeSphere 企业版扩展组件、EdgeWize 控制面等。

|边缘集群（EdgeCluster）	
|运行在 MetaCluster 集群中的 Kubernetes 集群，包含 CloudCore 等边缘组件。

|===
