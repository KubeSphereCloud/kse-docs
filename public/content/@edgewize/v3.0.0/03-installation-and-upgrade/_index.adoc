本节介绍如何如何安装、卸载 EdgeWize 边缘计算平台。

EdgeWize 边缘计算平台运行在{ks_product_both}上，因此您需要先安装定制版的{ks_product_left}，才能安装 EdgeWize。


ifeval::["{file_output_type}" == "pdf"]

== 产品版本

本文档适用于 EdgeWize v3.0.0 版本。

== 读者对象

本文档主要适用于以下读者：

* EdgeWize 用户

* 交付工程师

* 运维工程师

* 售后工程师
