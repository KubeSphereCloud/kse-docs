本节介绍如何管理应用。

// 在{ks_product_both}平台，应用特指由一个或多个工作负载、服务、应用路由等资源组成的业务程序。

EdgeWize 边缘计算平台目前支持创建基于模板的应用，即通过已有的应用模板创建应用。创建应用所使用的应用模板可以是已发布到应用商店的应用模板、或创建到企业空间的应用模板。

[.admon.note,cols="a"]
|===
|说明

|
EdgeWize 边缘计算平台中，应用实际为一个类型为部署的工作负载。
|===
