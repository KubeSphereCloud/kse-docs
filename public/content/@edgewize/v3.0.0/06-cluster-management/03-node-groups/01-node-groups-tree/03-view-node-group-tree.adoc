:ks_permission: **节点组管理**
:ks_menu: **节点组 > 节点组目录**


本节介绍如何查看节点组目录。


== 前提条件

* {empty}
include::../../../../_custom/clusterManagement/clusterManagement-prer-requiredPermission.adoc[]

* 已创建一个节点组。


== 操作步骤

include::../../../../_custom/clusterManagement/clusterManagement-oper-openClusterPage.adoc[]

+

. 在左侧导航栏选择pass:a,q[{ks_menu}]。
+
--
页面左侧显示节点组目录，右侧显示特定节点组的概览信息。

节点组目录显示：节点组的名称和别名，各节点组中边缘节点的数量，以及节点组之间的层级关系。

* 在列表上方点击搜索框并输入关键字，可按名称搜索节点组。

include::../../../../../_ks_components/oper-refreshListData.adoc[]

* 在列表右上角点击image:/images/edgewize/zh/icons/expand.svg[,18,18]可展开节点组目录。点击image:/images/edgewize/zh/icons/fold.svg[,18,18]可折叠节点组目录。

* 在列表右上角点击image:/images/edgewize/zh/icons/addButton.svg[,18,18]创建节点组。

* 点击列表右侧的上下箭头，调整节点组的顺序。

* 点击节点组名称，页面右侧将展示该节点组的信息。
--
