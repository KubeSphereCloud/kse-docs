EdgeWize 边缘计算平台是一款面向云边协同场景的边缘容器应用管理平台，它在 KubeSphere LuBan 云原生可扩展开放架构之上，整合并优化边缘计算场景下的关键功能模块，为企业客户提供可落地、可迭代、场景融合的综合边缘计算解决方案，并快速构建一个更开放、更易用、更智能的云边协同基础设施平台，助力客户实现数字业务的边缘化转型。

== 产品特性

**1. 海量设备的稳定纳管**

- 虚拟控制面实现大规模设备管理
- 批量设备快速接入，支持预置配置
- 多协议适配，兼容各类边缘设备
- 弹性扩展，支持百万级设备接入

**2. 海量设备的高效运维**

- 统一监控告警平台，实时掌握设备状态
- 一键式快速升级和补丁修复
- 集中日志管理，便于问题定位
- 远程 SSH 访问，支持批量操作
- 自动化运维脚本，减少人工干预

**3. 海量设备的多级管理**

- 设备分级管理，支持多级架构
- 细粒度权限控制，满足复杂组织结构
- 权限下放，支持多层级自主管理
- 灵活的设备接入策略，适应不同场景需求
- 设备生命周期管理，从接入到退役全程可控

**4. 海量设备的应用管理**

- 应用商店提供丰富的行业解决方案
- 一键部署，加速数字化转型进程
- 镜像加速服务，提高应用分发效率
- 应用版本管理，支持灰度发布和回滚
- 跨设备应用编排，实现复杂业务场景

**5. 全方位安全保障**

- 统一证书管理，确保通信安全
- 细粒度操作审计，追踪所有管理行为
- 应用日志实时采集和分析
- 双向证书加密，保护数据传输安全
- 设备认证和授权机制，防止未授权接入
- 数据加密存储，保护敏感信息

**6. 智能化能力提升**

- 云边协同计算，优化资源利用
- AI模型自动优化部署，适配边缘环境
- 智能调度算法，平衡负载与性能
- 预测性维护，提前发现潜在问题
- 边缘智能决策，降低网络依赖
- 自适应网络优化，提高通信质量

**7. 简化运维流程**

- 直观的可视化管理界面，降低技术门槛
- 预置运维模板，快速应对常见场景
- 智能故障诊断，辅助非专业人员排障
- 自动化运维任务，减少人工干预
- 统一的运维知识库，促进经验共享

**8. 业务连续性保障**

- 边缘节点自治，保证离线情况下的持续运行
- 数据本地缓存和同步机制，确保数据一致性
- 多级容灾设计，提高系统可靠性
- 智能负载均衡，优化资源利用
- 自动故障转移，最小化服务中断

== 关键能力

**1. 云边协同基础设施**

构建智能化的云边协同基础设施，通过云端统一纳管和调度，边缘自治运行的分层架构，实现云资源和海量异构边缘资源的无缝协同。支持按需弹性伸缩，根据业务需求智能调度边缘资源，高效应对大规模异构的边缘场景。

**2. 异构资源统一抽象和编排**

面向异构硬件，基于云原生容器技术提供统一的资源抽象层，屏蔽底层硬件差异，向上层统一暴露计算、网络、存储等资源。使用 Kubernetes 等编排技术，实现跨云边缘的统一资源编排、调度和生命周期管理，最大化利用异构边缘资源。

**3. 分层架构管理**

支持平台、工作组、项目以及集群、节点组、节点的多级分层架构，可从资源以及应用纬度进行统一管控和调度，实现规模化的分布式边缘资源管理。通过细粒度的权限控制，可实现多租户、多层级的资源授权和隔离，保证不同组的资源安全互不干扰。

**4. 应用生命周期管理**

云端一键分发应用到大规模边缘节点，到边缘端运维管理，再到元数据的实时同步，提供全生命周期的一致性云边协同应用交付和管理能力，保障业务的高可用性和连续性。

**5. 统⼀身份认证和权限管控**

与 Kubesphere 统一身份认证服务兼容，可实现身份统一和权限管控以及与 DevOps 等服务的融合。支持细粒度的多租户、多角色授权，可靠的权限管控和审计溯源，实现云边资源的安全可信使用。

**6. 统⼀可观测平台**

基于云原生技术栈，构建包括监控、日志、审计等在内的统一可观测平台，提供全方位的可视化监控和智能分析诊断能力，实现跨云边资源的智能运维，提升运维效率，保障业务的高可用性。
