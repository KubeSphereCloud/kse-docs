EdgeWize 边缘计算平台适用于以下场景。

**1. 智能化应用的深度边云协同**

在数字化转型的浪潮中，边缘计算正与人工智能等多个技术领域深度融合，重塑各行各业。基于 EdgeWize 边缘计算平台，用户可在以更轻量化的方式，将大量的 AI 模型、3D 渲染工具、数据分析工具、安全防护软件、中间件等应用从云端部署到边缘侧，从而满足企业在智能监控与分析、工业视觉与检测、物联网传感数据采集、实时渲染与计算等方面的业务诉求。

**2. 边缘云资源的一站式高效管理**

边缘资源呈现出高度分散、层级复杂等特点，亟待更高效地进行资源管理、降低企业运维成本。EdgeWize 创新边缘资源管理方式，通过可视化的一站式管理平面，可一键纳管位于多云、多集群、分布式云等上的边缘侧异构、多级的计算资源，极大避免因边缘资源类型与数量增加导致的管控复杂度高、调度与配置困难、监测分散耗时等问题，进一步增智提效。

**3. 高并发、低延时的边端站点业务**

边缘计算的优势在于可通过本地计算能力，快速为边缘站点业务提供决策支撑，但存在资源受限、本地自治性与网络差等问题。EdgeWize 通过智能化能力的提升，强化边缘决策、自适应网络优化，提高边缘的网络自主性与质量。同时通过数据本地缓存与同步、多级容灾、智能负载均衡、自动故障转移等机制，能有效应对边缘临时并发场景，保障业务高度连续。
