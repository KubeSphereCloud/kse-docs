:ks_permission: **边缘企业空间管理**


本节介绍如何创建边缘企业空间。


== 前提条件

您需要在{ks_product_both}平台具有pass:a,q[{ks_permission}]权限。

== 操作步骤

. 以具有pass:a,q[{ks_permission}]权限的用户登录{ks_product_left} Web 控制台。
. 点击**企业空间管理**。
. 在页面点击**创建**。
. 在**创建企业空间**页面，设置企业空间的基本信息，然后点击**下一步**。
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|名称
|企业空间的名称。名称只能包含小写字母、数字和连字符（-），必须以小写字母或数字开头和结尾，最长 63 个字符。

|别名
|企业空间的别名。别名可包含任意字符，最长 63 个字符。

|管理员
|企业空间的管理员。

|描述
|企业空间的描述信息。描述可包含任意字符，最长 256 个字符。

|===
--

. 在**集群设置**页签，不选择任何集群，然后点击**确定**。
+
[.admon.attention,cols="a"]
|===
|注意

|
不可选择任何集群。即使选中边缘集群，仍将创建为普通的企业空间。
|===

. 在企业空间列表，点击刚创建的企业空间的名称，进入企业空间资源管理页面。

. 点击**边缘管理 > 边缘企业空间**，点击**开启**。

. 在**开启边缘企业空间转换**对话框，选择一个边缘集群，将当前普通企业空间转变为边缘企业空间。
+
[.admon.attention,cols="a"]
|===
|注意

|转换后，边缘企业空间无法再转变为普通企业空间，同时仅可以授权一个边缘集群。

|===
