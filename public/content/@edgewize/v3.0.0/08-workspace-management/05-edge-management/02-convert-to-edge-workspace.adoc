本节介绍如何将普通企业空间转换为边缘企业空间。

[.admon.attention,cols="a"]
|===
|注意

|
* 开启后，当前企业空间转变为边缘企业空间，且无法再转变为普通企业空间。

* 边缘企业空间仅可以授权一个边缘集群，若已授权多个集群，其他集群会被移除。
|===


== 前提条件

// include::../../../_custom/workspaceManagement/workspaceManagement-prer-requiredPermission.adoc[]
您需要加入一个企业空间并在企业空间中具有pass:a,q[{ks_permission}]权限。有关更多信息，请参阅“企业空间角色”和“企业空间成员”。


== 操作步骤

// include::../../../_custom/workspaceManagement/workspaceManagement-oper-openWorkspacePage.adoc[]
. 以具有pass:a,q[{ks_permission}]权限的用户登录{ks_product_left} Web 控制台并进入一个普通企业空间。

+

include::../../../../_ks_components/oper-navigate.adoc[]

. 点击**开启**，选择一个边缘集群，点击**确定**。
+
若边缘企业空间页面显示被授权的边缘集群信息，即表明当前企业空间已成功转换为边缘企业空间。
