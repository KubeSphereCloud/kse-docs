本节介绍如何移除边缘集群。

[.admon.warning,cols="a"]
|===
|警告

|
* 边缘集群被删除后，边缘集群中原有的资源不会被自动清除。
* 边缘集群被删除后，边缘集群中原有的多集群配置数据不会被自动清除，卸载 KubeSphere 或删除关联资源时可能会导致用户数据丢失。
|===

== 前提条件

您需要在{ks_product_both}平台具有 **platform-admin** 角色。

== 操作步骤

. 进入**边缘集群管理**页面。

. 在需要操作的集群右侧点击image:/images/ks-qkcp/zh/icons/more.svg[more,18,18]，然后在下拉列表中选择**删除集群**。

. 在**移除集群**对话框，确认需要移除的集群的信息，将滑块拖动到右侧，输入需要移除的集群的名称，根据需要选择或取消选择**删除该边缘集群的所有消息通知配置**，然后点击**移除**。
+
--
[.admon.note,cols="a"]
|===
|说明

|
在多集群系统中，host 集群上的通知渠道、静默规则、通知语言等配置数据会自动同步到边缘集群。如果移除集群时未选择**删除该边缘集群的所有消息通知配置**，集群被移除后仍将按照原有的配置发送通知。
|===
