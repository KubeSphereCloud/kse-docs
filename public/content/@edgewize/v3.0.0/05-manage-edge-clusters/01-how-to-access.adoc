本节介绍如何通过{ks_product_left} Web 控制台进入边缘集群管理页面。

== 前提条件

您需要在{ks_product_both}平台具有 **platform-admin** 角色。

== 操作步骤

. 以具有 **platform-admin** 角色的用户登录{ks_product_left} Web 控制台。

. 点击顶部导航栏上的image:/images/edgewize/zh/icons/grid.svg[grid,18,18]，然后点击 **edgewize-io** 图标进入**边缘集群管理**页面，即可开始使用 EdgeWize。

. 如果已有边缘集群，在边缘集群管理页面点击一个集群的名称进入边缘集群。
