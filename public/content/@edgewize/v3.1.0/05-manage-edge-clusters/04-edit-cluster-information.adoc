本节介绍编辑集群信息。

边缘集群创建后，仅支持编辑集群的别名和描述。

== 前提条件

您需要在{ks_product_both}平台具有 **platform-admin** 角色。

== 操作步骤

. 进入**边缘集群管理**页面。

. 在需要操作的集群右侧点击image:/images/ks-qkcp/zh/icons/more.svg[more,18,18]，然后在下拉列表中选择**编辑信息**。

. 在**编辑信息**对话框，设置集群的别名和描述，然后点击**确定**。
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|别名
|别名只能包含中文、字母、数字和连字符（-），不得以连字符（-）开头或结尾，最长 63 个字符。

|描述
|集群的描述信息。描述可包含任意字符，最长 256 个字符。
|===
