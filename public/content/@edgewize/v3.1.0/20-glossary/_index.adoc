|===
|术语 | 说明

|元集群（MetaCluster）
|一个实体的 Kubernetes 集群。作为基础设施层，用于部署和运行其他集群组件。

|KubeSphere 企业版集群（KseCluster）	
|运行在 MetaCluster 上的 KubeSphere 集群。部署了{ks_product_both}及其扩展组件。

|边缘集群（Edge-Cluster）	
|运行在 MetaCluster 上，通过 vcluster 部署的虚拟集群，或者独立的物理集群。部署了 CloudCore 等边缘计算相关组件，用于边缘节点的管理和控制。
|===
