本节介绍如何删除消息路由。

// Warning
include::../../../../../_ks_components/admonitions/warning.adoc[]

消息路由删除后不能恢复，请谨慎执行此操作。

include::../../../../../_ks_components/admonitions/admonEnd.adoc[]


== 前提条件

// include::../../../../_custom/projectManagement/projectManagement-prer-requiredPermission.adoc[]
您需要加入一个边缘项目并在项目中具有 **project-admin** 角色。


== 删除单个消息路由

// include::../../../../_custom/projectManagement/projectManagement-oper-openProjectPage.adoc[]
. 以具有 **project-admin** 角色的用户登录{ks_product_left} Web 控制台并进入您的边缘项目。

+

include::../../../../../_ks_components/oper-navigate.adoc[]

+

. 在需要删除的消息路由右侧点击image:/images/ks-qkcp/zh/icons/more.svg[more,18,18]，然后在下拉列表中选择**删除**。

. 在**删除消息路由**对话框，输入消息路由的名称，然后点击**确定**。


== 批量删除消息路由

// include::../../../../_custom/projectManagement/projectManagement-oper-openProjectPage.adoc[]
. 以具有 **project-admin** 角色的用户登录{ks_product_left} Web 控制台并进入您的边缘项目。

+

include::../../../../../_ks_components/oper-navigate.adoc[]

+

. 勾选需要删除的消息路由左侧的复选框，然后在消息路由列表上方点击**删除**。

. 在**批量删除消息路由**对话框，输入消息路由的名称，然后点击**确定**。
+
