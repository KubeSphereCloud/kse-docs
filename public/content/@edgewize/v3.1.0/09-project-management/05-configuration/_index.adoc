本节介绍如何管理配置资源。

边缘集群支持管理以下两类配置资源：

* 保密字典：包含 Base64 编码的键值对，用于存储密码、令牌、密钥等保密数据。

* 配置字典：包含明文键值对，用于存储普通配置数据。
