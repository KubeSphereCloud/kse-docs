**概览**页面提供以下信息：

[%header,cols="1a,4a"]
|===
|参数 |描述

|基本信息
|项目的名称、描述、所属集群、企业空间和创建者。

|项目成员和角色
|项目成员和角色数量，以及最近登录的用户。

|资源用量排行
|项目中特定资源用量最高的 5 个工作负载。点击下拉列表可选择按 CPU、内存、出站流量或入站流量排序。点击**展开**或**查看更多**查看更多排行信息。

|已创建资源数量
|显示当前项目的资源数量，如容器组、部署、任务、定时任务的数量。

|资源用量
|显示当前项目的 CPU、内存的总量和实时用量。

|项目配额
|项目中所有容器的资源预留总和的最大值，以及项目中所有容器的资源上限总和的最大值。
// 如果项目管理员未设置项目配额则不显示。

|项目告警
|显示项目的告警规则组产生的告警数量。

// 告警级别的类型包括**提醒、警告、重要和紧急**。

告警状态的类型包括：

include::../../../_custom/clusterManagement/alerts/alerts-para-alertingStatus.adoc[]

|容器组
|显示当前项目的各种类型容器组的数量。

容器组状态类型包括：

include::../../../_custom/clusterManagement/pods/pods-para-podStatus_overview.adoc[]

容器组 QoS（服务质量）类型包括：

include::../../../_custom/clusterManagement/pods/pods-para-qosTypes.adoc[]
|===
