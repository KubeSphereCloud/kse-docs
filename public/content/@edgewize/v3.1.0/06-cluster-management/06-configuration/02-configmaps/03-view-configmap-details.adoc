== 前提条件

您需要加入一个边缘集群并在集群中具有 **cluster-viewer** 角色。有关更多信息，请参阅“边缘集群角色”和“边缘集群成员”。

== 操作步骤

. 以具有 **cluster-viewer** 角色的用户登录{ks_product_left} Web 控制台并进入您的边缘集群。
