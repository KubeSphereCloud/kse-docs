本节介绍如何移除未纳管的边缘节点。

[.admon.note,cols="a"]
|===
|说明

|
如果边缘节点已经被纳管，取消纳管后才能删除节点。
|===

== 前提条件

include::../../../../_custom/clusterManagement/clusterManagement-prer-requiredPermission.adoc[]


== 删除单个边缘节点

include::../../../../_custom/clusterManagement/clusterManagement-oper-openClusterPage.adoc[]

+

include::../../../../../_ks_components/oper-navigate.adoc[]

. 在需要纳管的节点右侧点击image:/images/ks-qkcp/zh/icons/more.svg[more,18,18]，然后在下拉列表中选择**删除**。

. 在**删除节点**对话框，输入边缘节点的名称，然后点击**确定**。

== 批量删除边缘节点

include::../../../../_custom/clusterManagement/clusterManagement-oper-openClusterPage.adoc[]

+

include::../../../../../_ks_components/oper-navigate.adoc[]

. 点击**批量管理**，在上方勾选**选择全部**，然后点击**删除**。对话框会自动列出可被删除的节点。

. 选择需要操作的节点，点击**删除**。

. 在**批量删除节点**对话框，输入边缘节点的名称，然后点击**确定**。
