---
title: "容器组 IP 池"
weight: 02
---

容器组 IP 池用于为容器组分配 IP 地址。每个容器组 IP 池包含一个可在集群内部访问的私网 IP 网段。

你可以在集群中按需创建容器组 IP 池，并将其分配给项目。容器组 IP 池与项目绑定后，在项目中创建的容器组将使用从该 IP 池中分配的 IP 地址。


[.admon.attention,cols="a"]
|===
|注意

|
只有使用边缘运行时 openyurt 创建的边缘集群，边缘集群的左侧导航栏才会显示**网络 > 容器组 IP 池**菜单，其节点组管理页面的**节点组设置**下才会显示**容器组 IP 池**菜单。

使用边缘运行时 kubeedge 创建的边缘集群不会显示容器组 IP 池菜单项。
|===
