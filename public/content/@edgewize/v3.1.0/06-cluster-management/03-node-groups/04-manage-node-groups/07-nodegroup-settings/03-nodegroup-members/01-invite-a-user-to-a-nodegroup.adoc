本节介绍如何邀请用户加入节点组。


== 前提条件

* {empty}
include::../../../../../../_custom/clusterManagement/nodegroups/nodegroup-prer-requiredPermission.adoc[]

* 在{ks_product_both}平台的**用户和角色管理**中创建一个用户，并邀请该用户加入节点组所属的边缘集群，即成为边缘集群成员。
