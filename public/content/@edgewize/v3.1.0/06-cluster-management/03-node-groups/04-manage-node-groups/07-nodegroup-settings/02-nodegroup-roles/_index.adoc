本节介绍如何管理节点组角色。

节点组角色包含节点组管理或节点组查看权限。您可以将包含特定权限的角色授予节点组成员，以控制节点组成员在节点组中的权限。

节点组角色基于边缘集群角色实现。在具有节点组角色之前，用户必须先有一个边缘集群角色。

[.admon.note,cols="a"]
|===
|说明

|
在边缘集群中，节点组的创建者自动成为节点组管理员。此外，{ks_product_right}平台的 **admin** 用户默认具有所有权限。在节点组初始状态下，可由以上用户邀请其他用户加入节点组并设置用户在节点组中的角色。
|===
