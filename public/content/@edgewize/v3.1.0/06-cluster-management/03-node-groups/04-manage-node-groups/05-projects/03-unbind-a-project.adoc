== 解绑单个项目

include::../../../../../_custom/clusterManagement/nodegroups/nodegroup-oper-openNodeGroupPage.adoc[]

+

. 在左侧导航栏选择pass:a,q[{ks_menu}]。

. 在需要删除的项目右侧点击image:/images/edgewize/zh/icons/delete.svg[delete,18,18]。

. 在**删除项目**对话框，输入项目的名称，然后点击**确定**。

== 批量解绑项目

include::../../../../../_custom/clusterManagement/nodegroups/nodegroup-oper-openNodeGroupPage.adoc[]

+

. 在左侧导航栏选择pass:a,q[{ks_menu}]。

. 选择需要删除的项目左侧的复选框，然后在项目列表上方点击**删除**。

. 在**删除项目**对话框，输入项目的名称，然后点击**确定**。
