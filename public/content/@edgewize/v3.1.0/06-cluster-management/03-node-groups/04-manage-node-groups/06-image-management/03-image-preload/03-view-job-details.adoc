== 操作步骤

+

include::../../../../../../../_ks_components/oper-navigate.adoc[]

+

. 在**镜像预热**页面，点击**即时预热**或**定时预热**，然后在列表中点击一个任务的名称打开其详情页面。

+

. 在任务详情页面左侧的**属性**区域查看任务的资源属性。
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|集群
|任务所属的集群。

|状态
|
任务当前的状态。

* **运行中**：任务正在运行中。

* **成功**：任务运行成功并且已完成。

* **已暂停**：定时任务已暂停。仅对**定时预热任务**显示。

* **失败**：任务运行失败。

|定时计划
|定时任务的运行周期的 CRON 表达式。{ks_product_right}默认使用 UTC 时间，您需要根据您所在的时区进行调整。仅对**定时预热任务**显示。

|创建时间
|任务的创建时间。

|创建者
|创建任务的用户。
|===
--

. 在任务详情页面右侧的**运行记录**页签查看任务的运行记录。
+
--
用户手动重新运行即时任务，或定时任务按照定时计划运行时，系统将生成运行记录。

对于即时任务，**运行记录**页签显示以下信息：

[%header,cols="1a,4a"]
|===
|参数 |描述

|序号
|任务运行的序号，表示每条记录是任务第几次运行。

|状态
|每次运行任务的状态结果。

* **运行中**：任务正在运行中。

* **已完成**：任务运行成功并且已完成。

* **失败**：任务运行失败。

|消息
|任务运行返回的消息。

|开始时间
|任务运行开始运行的时间。

|结束时间
|任务运行结束的时间。
|===

对于定时任务，**运行记录**页签显示以下信息：

[%header,cols="1a,4a"]
|===
|参数 |描述

|任务
|定时任务创建的普通任务的名称。

|状态
|定时任务创建的普通任务的状态。

* **运行中**：任务正在运行中。

* **已完成**：任务运行成功并且已完成。

* **失败**：任务运行失败。

|开始时间
|普通任务开始运行的时间。

|结束时间
|普通任务运行结束的时间。
|===
--

. 在任务详情页面右侧点击**镜像列表**页签，查看任务中预热的镜像。

. 在任务详情页面右侧点击**预热范围**页签，查看镜像预热的目标节点。
