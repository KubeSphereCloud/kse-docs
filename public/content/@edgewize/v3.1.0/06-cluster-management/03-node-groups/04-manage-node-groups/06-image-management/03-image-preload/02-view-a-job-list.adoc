+

. 在**镜像预热**页面，点击**即时预热**或**定时预热**打开任务列表。
+
====
* 任务列表提供以下信息：
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|名称
|任务的名称和描述信息。

|状态
|
任务当前的状态。

* **运行中**：任务正在运行中。

* **成功**：任务运行成功并且已完成。

* **已暂停**：定时任务已暂停。仅对**定时预热任务**显示。

* **失败**：任务运行失败。

|定时计划
|定时任务的运行周期的 CRON 表达式。{ks_product_right}默认使用 UTC 时间，您需要根据您所在的时区进行调整。仅对**定时预热任务**显示。

|镜像数量
|任务中预热的镜像数量。

|预热范围
|镜像预热的目标节点。

|上次运行时间
|任务上一次运行的时间。仅对**即时预热任务**显示。

|创建时间
|定时任务的创建时间。仅对**定时预热任务**显示。
|===
