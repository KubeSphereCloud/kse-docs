本节介绍如何删除节点组。

[.admon.attention,cols="a"]
|===
|注意

|
* 删除节点组时，请确保节点组下没有边缘节点。若有，请先将边缘节点从该节点组中移除。
* 删除节点组后，下一级的节点组会向上移一级，比如，删除二级节点组，之前的三级节点组会自动变为二级节点组。
|===
