== 操作步骤

include::../../../../_custom/clusterManagement/clusterManagement-oper-openClusterPage.adoc[]

+

. 在左侧导航栏选择pass:a,q[{ks_menu}]。

. 在需要操作的节点组右侧点击image:/images/ks-qkcp/zh/icons/more.svg[more,18,18]，然后在下拉列表中选择**移动节点组**。

. 在**移动节点组**对话框，选择当前节点组将被移动到的目标节点组，点击**确定**。
