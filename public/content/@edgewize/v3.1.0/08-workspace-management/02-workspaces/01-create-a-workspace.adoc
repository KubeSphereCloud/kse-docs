:ks_permission: **边缘企业空间管理**


本节介绍如何创建边缘企业空间。


== 前提条件

您需要在{ks_product_both}平台具有pass:a,q[{ks_permission}]权限。

== 操作步骤

. 以具有pass:a,q[{ks_permission}]权限的用户登录{ks_product_left} Web 控制台。

. 点击顶部导航栏上的image:/images/edgewize/zh/icons/grid.svg[grid,18,18]，然后点击 **edgewize-io** 图标。

. 点击**边缘企业空间**。

. 在页面点击**创建**。

. 在**创建边缘企业空间**页面，设置企业空间的基本信息，然后点击**下一步**。
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|名称
|企业空间的名称。名称只能包含小写字母、数字和连字符（-），必须以小写字母或数字开头和结尾，最长 63 个字符。

|别名
|企业空间的别名。别名可包含任意字符，最长 63 个字符。

|管理员
|企业空间的管理员。

|描述
|企业空间的描述信息。描述可包含任意字符，最长 256 个字符。

|===
--

. 在**集群设置**页签，选择一个边缘集群，然后点击**确定**。
+
[.admon.note,cols="a"]
|===
|说明

|仅可以将一个边缘集群授权给边缘企业空间。

|===
