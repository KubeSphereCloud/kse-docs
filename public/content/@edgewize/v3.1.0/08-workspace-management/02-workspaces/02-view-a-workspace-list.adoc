:ks_permission: **边缘企业空间查看**

本节介绍如何查看企业空间列表。


== 前提条件

您需要在{ks_product_both}平台具有pass:a,q[{ks_permission}]权限。

== 操作步骤

. 以具有pass:a,q[{ks_permission}]权限的用户登录{ks_product_left} Web 控制台。

. 点击顶部导航栏上的image:/images/edgewize/zh/icons/grid.svg[grid,18,18]，然后点击 **edgewize-io** 图标。

. 点击**边缘企业空间**。
+
--
* 企业空间列表显示所有边缘企业空间及其所属的边缘集群。

* 在企业空间列表中，点击企业空间的名称可进入边缘企业空间，查看和管理边缘企业空间中的资源。
--
