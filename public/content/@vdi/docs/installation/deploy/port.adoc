title: 添加端口转发规则（可选）

---

若从外网访问桌面云平台，需要添加端口转发规则。从外网通过 Web 访问桌面云平台的地址是 `QingCloud 桌面云主机所在 VPC 网络的公网IP:8445端口转发后的端口`。

== 操作须知

软终端的网关接入端口是 ``8443``，桌面云平台 Web 访问的端口是 ``8445``，请在 QingCloud 云平台的 VPC 网络中为端口 ``8443`` 和 ``8445`` 添加端口转发规则。

== 前提条件

* 已成功创建 QingCloud 桌面云主机。
* 已安装 QingCloud 云平台，并已获取云平台的访问地址和帐号密码。

== 操作步骤

. 登录 QingCloud 云平台。
. 选择**产品与服务** > **网络服务** > **VPC 网络**，进入 VPC 网络页面。
. 点击已创建的 VPC 名称，进入 VPC 网络详情页面。
+
image::/images/vdi/installation/deploy_port_01.png[,70%]
. 选择**管理配置**页签，点击**添加规则**，弹出添加端口转发规则对话框。
+
image::/images/vdi/installation/deploy_port_02.png[,50%]
+
image::/images/vdi/installation/deploy_port_03.png[,50%]
. 对云主机的内网端口 ``8443`` 和 ``8445`` 添加端口转发规则，点击**提交**，跳转至 VPC 网络详情页面。
+
include::../../../../_components/attention.adoc[]
* 通过 Web 访问桌面云平台，需添加 8445 的端口转发规则。
* 通过软终端软件连接云桌面，需添加 8443 的端口转发规则。
* 端口转发规则中的**源端口**不能重复，否则会端口冲突。
include::../../../../_components/table_footer.adoc[]

. 端口转发规则添加完成后，点击右上方的**应用修改**，使端口转发规则生效。
