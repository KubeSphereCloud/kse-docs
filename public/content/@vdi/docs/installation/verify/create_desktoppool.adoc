---
title: 创建桌面池
weight: 50
---

QingCloud 桌面云平台通过桌面池的模式发布云桌面。桌面池是一组相同配置的桌面的组合，创建桌面池时需开启预创建开关，平台才会自动创建云桌面。云桌面创建成功后，需为云桌面绑定用户，用户才能登录软终端，查看到所能使用的云桌面。

== 前提条件

* 已登录 QingCloud 桌面云平台，并添加 QingCloud 虚拟化平台。
* 已在 QingCloud 桌面云平台中为云桌面创建规格。
* 已在 QingCloud 云平台中按照网络规划创建 VPC 网络和私有网络。

== 操作步骤

1. 进入 QingCloud 桌面云平台首页。
2. 在左侧的菜单栏中，选择 **VDI** > **桌面池**，进入桌面池页面。
3. 点击**新建桌面池**，进入**新建桌面池-基本设置**页面。
+
image::/images/vdi/installation/verify_create_desktoppool_01.png[,60%]
4. 填写桌面池基本信息，点击**下一项**，进入**新建桌面池-桌面池配置**页面。
+
image::/images/vdi/installation/verify_create_desktoppool_02.png[,60%]

5. 配置桌面池，并开启**预创建**开关，点击**下一项**，进入**新建桌面池-绑定用户组**页面。
+
include::../../../../_components/note.adoc[]
开启**预创建**开关后，系统将根据设置的**桌面池容量**，自动创建桌面池中的所有桌面。
include::../../../../_components/table_footer.adoc[]
+
image::/images/vdi/installation/verify_create_desktoppool_03.png[,60%]
6. 绑定用户组，点击**立即创建**，即可创建桌面池。
+
image::/images/vdi/installation/verify_create_desktoppool_04.png[,60%]
7. 为云桌面绑定用户。

.. 点击已创建的桌面池名称，进入桌面池详情页面。
+
image::/images/vdi/installation/verify_create_desktoppool_05.png[,60%]
.. 勾选目标云桌面，点击**更多操作**中的**绑定用户**，弹出绑定用户对话框。
+
image::/images/vdi/installation/verify_create_desktoppool_06.png[,60%]
.. 选择需关联的用户，点击**确定**，即可为此桌面绑定用户。
+
image::/images/vdi/installation/verify_create_desktoppool_07.png[,60%]
