---
title: 创建用户组及用户
weight: 30
---

用户的组织关系管理可通过用户组实现，创建用户组并在组内创建用户后，可为相同组织关系的用户分配资源及权限相同的桌面资源。

若已创建桌面池和桌面，可在创建用户组时为用户组绑定桌面池，以及在创建用户时为用户关联桌面。

== 前提条件

超级管理员已登录 QingCloud 桌面云平台。

== 创建用户组

1. 进入 QingCloud 桌面云平台首页。
2. 在左侧的菜单栏中，选择**用户** > **用户组**，进入用户组页面。
3. 点击用户组结构右侧的 **+**，进入新建用户组页面。
+
image::/images/vdi/installation/verify_create_user_01.png[,50%]
4. 设置用户组信息。
5. 点击**立即创建**，即可创建用户组。
+
image::/images/vdi/installation/verify_create_user_02.png[,50%]

== 创建用户

1. 进入 QingCloud 桌面云平台首页。
2. 在左侧的菜单栏中，选择**用户** > **用户组**，进入用户组页面。
3. 在左侧的用户组结构中，选择已创建的用户组，点击**新建用户**，进入新建用户页面。
+
image::/images/vdi/installation/verify_create_user_03.png[,50%]
4. 填写用户信息。
+
include::../../../../_components/note.adoc[]
* 若认证方式选择``使用初始默认密码``，初始默认密码是 ``password``。
* 若已创建云桌面，可选择云桌面与用户进行绑定。
include::../../../../_components/table_footer.adoc[]

5. 点击**立即创建**，完成用户创建。
