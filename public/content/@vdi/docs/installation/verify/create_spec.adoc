== 前提条件

已登录 QingCloud 桌面云平台，并在虚拟化平台中添加 QingCloud 云平台。

== 操作步骤

1. 进入 QingCloud 桌面云平台首页。
2. 在左侧的菜单栏中，选择 **VDI** > **规格管理**，进入规格管理页面。
3. 点击左侧的**新建计算规格**，进入新建规格页面。
+
image::/images/vdi/installation/verify_create_spec_01.png[,60%]
4. 填写规格信息，点击立即创建，即可创建云桌面可使用的规格。
+
image::/images/vdi/installation/verify_create_spec_02.png[,80%]
