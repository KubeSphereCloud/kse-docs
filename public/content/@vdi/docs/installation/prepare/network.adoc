== 防火墙 NAT 映射网络规划

如需要在外部网络中访问云桌面，可使用 NAT（网络地址转换模式）配置公网映射。请按照下表在防火墙端口设置转发规则。

|===
| 类型 | 用途 | 内网地址端口 | 外网地址端口

| 管理/VDI 访问 IP | 云平台Web 管理UI 地址(IP-A1) | 带内管理 IP-A1 :xxxxx | 外网地址 IP-D1:xxxxx
| 管理/VDI 访问 IP | VDI 云桌面管理平台（IP-B1）（云桌面网关/登录验证端口） | 业务网络IP-B1:8443 | 外网地址 IP-D1 :8443
| VDI 访问 IP | 终端软件更新端口（IP-B1） | 业务网络2 IP-B1:8888 | 外网地址 IP-D1:8888
| VDI 访问 IP | WebClient 端⼝（IP-B1） | 业务网络2 IP-B1:443 | 外网地址 IP-D1:18443
| 管理/VDI 访问 IP | 物理机计算节点端口（IP-A2） | 带内管理 IP-A2:5900~5950 | 外网地址 IP-D1:5900~5950
| 管理/VDI 访问 IP | 物理机计算节点端口（IP-A3） | 带内管理 IP-A3:5900~5950 | 外网地址 IP-D1:5951~6000
| 管理/VDI 访问 IP | 物理机计算节点端口（IP-A4） | 带内管理 IP-A4:5900~5950 | 外网地址 IP-D1:6001~6050
|===
