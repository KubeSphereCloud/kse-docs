在{platform_name_cn}上安装 {product_name}前，需要提前准备好所需的软件包。

== 下载软件包

从以下 FTP 路径，根据实际情况获取对应架构的桌面云系统相关软件包。

[source]
----
ftp://ins.qingcloud.com/express/桌面
----


== 软件包说明

[cols='1,3'] 
|===
| 目录/文件 | 说明 

| DesktopArmInstaller | QingCloud 桌面云平台 ARM 架构相关软件包。
| DesktopX86Installer | QingCloud 桌面云平台 X86 架构相关软件包。
| ISO | QingCloud 云桌面镜像，例如 ``Kylin 云桌面镜像 Kylin-Desktop-V10_U1-Release-Build03-20210820-GFB-arm64-V10国防版-正式版.iso``。
| xview-lander_22.09.1-ga_i386.exe	| QingCloud 桌面云软终端安装包。
| 青云LOGO.zip	| QingCloud 桌面云 LOGO 文件。
|===
