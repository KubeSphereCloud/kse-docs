{product_name}的安装依赖{platform_name_cn}，所以在安装 {product_name}之前，请先完成{platform_name_cn}的安装，确保安装完成后可正常登录云平台的 Console（云计算服务平台）和 BOSS（云计算运维管理平台）。

{platform_name_cn}的安装操作请参见《{platform_name_cn} 安装指南》，本文不做详细描述。
