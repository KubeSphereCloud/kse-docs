[cols='1a,3a']
|===
|Parameter |Description

|Select Network
|The physical network in which the subnet is created.

|VLAN ID
|The VLAN ID that corresponds to the physical subnet.

|IP Range
|The IP address range in which the physical subnet resides.

|Gateway Address
|The gateway address of the physical subnet.

|Logical Gateway
|Click **Advanced Settings** to specify whether to enable the logical gateway.

|DNS Server
|Click **Advanced Settings** to specify the IP address of the DNS server.

|Bandwidth
|Click **Advanced Settings** to set the bandwidth limit for the subnet.

|Excluded IP Addresses
|The IP address range in which the physical subnet resides. Click **Advanced Options** to exclude an IP address or a specific range of IP addresses.
|===
