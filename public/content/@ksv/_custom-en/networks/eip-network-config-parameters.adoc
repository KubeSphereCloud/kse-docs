[cols='1a,3a']
|===
|Parameter |Description

|Physical Network
|The name of the physical network used for EIPs.

|IP Range
|The IP address range used for EIPs.

|VLAN
|The VLAN ID that corresponds to EIPs.

|Gateway Address
|The gateway address used for EIPs.

|Excluded IP Range
|Click **Advanced** to exclude a single IP address or a specific range of IP addresses from the specified IP address range.
|===
