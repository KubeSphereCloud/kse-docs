The user that you use has been granted the permissions to use elastic IP addresses (EIPs). User **admin** can manage all EIPs in your cluster. Project administrators and operators can manage the EIPs created on their own and have read-only permissions on the EIPs created by other users. Project viewers can view only the EIPs created in the projects to which they belong.
