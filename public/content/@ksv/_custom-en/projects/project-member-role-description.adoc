[cols='1a,4a']
|===
|Role |Description

|Viewer
|View only all resources in the projects to which they belong.

|Operator
|Manage resources other than users and roles in the projects to which they belong.

|Administrator
|Manage all resources in the projects to which they belong.  By default, user **admin** belongs to all projects in the system and has the administrator role in all projects.
|===
