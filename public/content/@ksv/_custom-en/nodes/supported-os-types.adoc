The server node on which {ks_product} is installed must run on Linux, and the kernel version of Linux must be 4.0 or later.  We recommend that you use one of the following OSs: Ubuntu 18.04, Ubuntu 20.04, CentOS 7.9, Unity Operating System (UOS), Kylin V10, and EulerOS.
