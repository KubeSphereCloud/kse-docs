The user that you use has been granted the permissions to manage VM images. Only user **admin**, project administrators, and operators can manage VM images. Project viewers can view only VM images in the projects to which they belong. For more information about permissions, see xref:03-user-manual/05-vm-management/_index.adoc[VM management].
