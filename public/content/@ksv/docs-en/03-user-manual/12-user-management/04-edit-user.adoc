== Procedurelink:[id=ab25e5eadcc544018323103cea4c83b5]

include::../../../_custom-en/users/user-steps.adoc[]

. On the **Local Users** page, click the name of the desired user. The details page of the user appears.
. On the details page of the user, click **Edit** in the upper-left corner.
. In the dialog box that appears, modify the email and description of the user, and click **OK**.  For more information about parameters, see link:../01-create-user/[Create a user].
