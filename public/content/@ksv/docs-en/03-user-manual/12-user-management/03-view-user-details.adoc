== Procedurelink:[id=03b658bb81ab48a4a229a5436cfa1259]

include::../../../_custom-en/users/user-steps.adoc[]

. On the **Local Users** page, click the name of the user that you want to view. The details page of the user appears.
. On the left side of the details page, view basic information about the user.
+
--
[cols='1a,4a']
|===
|Parameter |Description

|Email
|The email address of the user.

The email address can be used to log in to the {ks_product} web console.

|Created by
|The time when the user was created.The creator of the user.  Only user **admin** can create users.

|Created at
|The login time.
|===
--
. On the **Projects** tab, view the projects to which the user belongs.  Each user can be added to one or more projects.
+
--
[cols='1a,4a']
|===
|Parameter |Description

|Project Name
|The projects to which the user belongs.

|Role
|The role of the user in the project.
|===
--
. On the **Login History** tab, view the login records of the user.
+
--
[cols='1a,4a']
|===
|Parameter |Description

|Time
|The login time.

|Status
|The login status.

|Source IP Address
|The IP address of the user.

|Description
|The description of the user login.
|===
