== Procedurelink:[id=d9ef18e6d03a4d75a9b47ba532a389a6]

include::../../../_custom-en/users/user-steps.adoc[]

. On the **Local Users** page, click the name of the desired user. The details page of the user appears.
. On the left side of the details page, click image:/images/ksv/en/icons/more.svg[operation-menu,13] and select **Change Password** from the drop-down list.
. In the dialog box that appears, enter and confirm a new password, and click **OK**.
