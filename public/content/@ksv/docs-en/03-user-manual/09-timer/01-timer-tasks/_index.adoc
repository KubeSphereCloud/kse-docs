Timer tasks indicate that the system can help you run apps or commands at specific intervals. {ks_product} provides the **Timer** module to help you create and understand timer tasks. You can obtain information about timer tasks, including the operations timer tasks to perform, timer policies, the number of execution times, the maximum failed attempts, and whether to skip errors if failed.
