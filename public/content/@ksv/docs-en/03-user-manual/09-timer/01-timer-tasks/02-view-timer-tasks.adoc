--
You can view all timer tasks in the project. The following table describes the parameters.

[cols="1a,4a"]
|===
|Parameter |Description

|Name
|The custom name of the timer task.

|ID
|The universally unique identifier (UUID) of the timer task in the system.

|Status
|The status of the timer task. Valid values: **Enabled** and **Disabled**.

|Operation
|The operation to perform by the timer task.  Valid values: **Start VM**, **Stop VM**, **Restart VM**, and **Create Snapshot**.

|Policy
|The policy that is used to run the timer task.  The system supports **Execute Periodically** and **Execute Once** policies.

|Success / Failure
|The number of times that the task succeeded and failed.

|Project
|The project name in which the timer task is created.
|===
--

. (Optional) Click the search box above the timer task list and specify a condition to search for timer tasks by name or ID. For example, use **Name:ab** to search for timer tasks whose name contains the string **ab**.
