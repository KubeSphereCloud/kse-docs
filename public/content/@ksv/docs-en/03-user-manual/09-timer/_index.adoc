The **Timer** module is available to all users.  User **admin** can view and manage all timer tasks and policies. The following table compares the permissions granted to non-admin users of different project roles.

[cols='1a,3a']
|===
|Project role |Permission

|Administrator
|View and manage all timer tasks and policies in the projects to which they belong.

|Operator
|View and manage all timer tasks and policies in the projects to which they belong.

|Viewer
|View only all timer tasks and policies in the projects to which they belong.
|===
