== Procedurelink:[id=76e70ce343ad4b68a3f8413aebcbc4cb]

include::../../../../_custom-en/timer/timer-policy-steps.adoc[]

. On the **Timer Policies** tab, click **Create** in the upper-right corner.
. In the dialog box that appears, configure the parameters for the timer policy. The following table describes the parameters.
+
--
[cols='1a,4a']
|===
|Parameter |Description

|Name
|The custom name of the timer policy.

include::../../../../_custom-en/networks/naming-convention.adoc[]

|Select Project
|The project name in which the timer project is created.

|Interval
|The interval at which the timer task runs. A timer task can run every day, every N days, every N hours, every week, or every month.

|Time
|The point in time at which the timer task runs.
|===
--
. Click **OK**.  The timer policy appears in the timer policy list after it is created.
