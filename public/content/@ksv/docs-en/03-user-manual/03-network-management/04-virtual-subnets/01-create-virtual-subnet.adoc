== Procedurelink:[id=ea1f56ec102c42f5ab6a03b8de5281d9]

include::../../../../_custom-en/networks/virtual-subnet-steps.adoc[]

. On the **Virtual Subnets** page, click **Create** in the upper-right corner.
. In the dialog box that appears, configure parameters for the virtual subnet. The following table describes the parameters.
+
--
[cols='1a,4a']
|===
|Parameter |Description

|Name
|The name of the virtual subnet.

include::../../../../_custom-en/networks/naming-convention.adoc[]

|VPC
|The virtual private cloud (VPC) where the virtual subnet resides.

|IP Range
|The IP address range in which the virtual subnet resides. The value is the subset of the IP address range in which the VPC resides.

|Description
|The description of the virtual subnet.

include::../../../../_custom-en/networks/description-convention.adoc[]

|Bandwidth
|Click **Advanced** to set the bandwidth limit for the subnet.

|DNS Server
|Click **Advanced** to specify the IP address of the DNS server.
|===
--
. Click **OK**.  The subnet appears in the virtual subnet list after it is created.
