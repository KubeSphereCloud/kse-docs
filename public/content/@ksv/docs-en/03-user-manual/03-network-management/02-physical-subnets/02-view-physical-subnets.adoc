--
You can view all the physical subnets in the system. The following table describes the parameters.

[cols='1a,4a']
|===
|Parameter |Description

|Name
|The name of the physical subnet.

include::../../../../_custom-en/networks/physical-network-naming-convention.adoc[]

|ID
|The UUID of the physical subnet in the system. The UUID is automatically assigned by the system.

|Status
|The status of the physical subnet.

|VLAN ID
|The VLAN ID that corresponds to the physical subnet.

|Physical Network
|The physical network in which the subnet is created.

|IP Range
|The IP address range in which the physical subnet resides.

|Gateway
|The gateway address of the physical subnet.

|Available IPs
|The number of IP addresses available in the physical subnet.

|Used IPs
|The number of IP addresses used in the physical subnet.
|===
--

. (Optional) Click the search box above the physical subnet list and set query conditions to search for physical subnets by name or ID. For example, use **Name:ab** to search for physical subnets whose name contains the string **ab**.
. (Optional) In the upper-right corner of the page, click image:/images/ksv/en/icons/refresh-light.svg[refresh,15] to refresh data.
