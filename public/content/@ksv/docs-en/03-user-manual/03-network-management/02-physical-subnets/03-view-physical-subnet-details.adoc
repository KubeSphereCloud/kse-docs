== Procedurelink:[id=a5c4910c7b694012a28c36d2c4898674]

include::../../../../_custom-en/networks/physical-subnet-steps.adoc[]

. On the **Subnets** page, click the name of the physical subnet that you want to view. The details page of the physical subnet appears.
. On the left side of the details page, view basic information about the physical subnet.
+
--
[cols='1a,4a']
|===
|Parameter |Description

|ID
|The universally unique identifier (UUID) of the physical subnet in the system. The UUID is automatically assigned by the system.

|Status
|The status of the physical subnet.

|Physical Network
|The physical network in which the subnet is created.

|VLAN ID
|The VLAN ID that corresponds to the physical subnet.

|IP Range
|The IP address range in which the physical subnet resides.

|Gateway
|The gateway address of the physical subnet.

|DNS Server
|The Domain Name System (DNS) server address that corresponds to the physical subnet.

|Bandwidth
|The maximum bandwidth of the physical subnet.

|Available IPs
|The number of IP addresses available in the physical subnet.

|Used IPs
|The number of IP addresses used in the physical subnet.

|Logical Gateway
|Specifies whether the logical gateway is enabled.

|Excluded IPs
|The IP address or IP address range that is excluded from the physical subnet.

|Created at
|The time when the physical subnet was created.

|Created by
|The username that is used to create the physical subnet.
|===
--
. On the right side of the page, view resources that are associated with the physical subnet. The following table describes the parameters.
+
--
[cols='1a,4a']
|===
|Parameter |Description

|Name
|The name of the associated resource.

|ID
|The UUID of the associated resource in the system. The UUID is automatically assigned by the system.

|Status
|The status of the associated resource.

|IP Address
|The IP address of the associated resource.
|===
