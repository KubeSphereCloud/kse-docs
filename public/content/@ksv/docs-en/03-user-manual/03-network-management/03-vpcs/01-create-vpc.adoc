== Procedurelink:[id=bd32502168064f6b91947e4bb2b070e7]

include::../../../../_custom-en/networks/vpc-steps.adoc[]

. On the **VPCs** page, click **Create** in the upper-right corner.
. In the dialog box that appears, configure parameters for the VPC.  The following table describes the parameters.
+
--
[cols='1a,4a']
|===
|Parameter |Description

|Name
|The name of the VPC.

include::../../../../_custom-en/networks/naming-convention.adoc[]

|IP Range
|The description of the VPC. The IP address range cannot be modified after the VPC is created.

|Default Virtual Subnet
|The default virtual subnet of the VPC.

|Description
|The description of the VPC.

include::../../../../_custom-en/networks/description-convention.adoc[]
|===
--
. Click **OK**.  The VPC appears in the VPC list after it is created.
