Elastic IP addresses (EIPs) are independent public IP addresses that enable Internet access.  Each public IP address is a static, valid IP address on the Internet. You can bind or unbind EIPs to or from VMs in a flexible manner. You can also configure a specific IP address range for EIPs. This way, the system will allocate EIPs within the range of IP addresses.
