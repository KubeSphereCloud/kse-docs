This topic describes how to apply for an elastic IP address (EIP) in the {ks_product_abbr_full} web console.

ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== Procedurelink:[id=a66686d07bd44f1d927b62be0d08121a]

include::../../../../_custom-en/networks/eip-steps.adoc[]

. On the **EIPs** page, click **Apply** in the upper-right corner.
. In the dialog box that appears, specify **Name** and **Description** for the EIP.  The following table describes the parameters.
+
--
[cols='1a,4a']
|===
|Parameter |Description

|Name
|The name of the EIP.

include::../../../../_custom-en/networks/naming-convention.adoc[]

|Type
|The address type of the EIP.   By default, {ks_product} uses IPv4.

|Description
|The description of the EIP.

include::../../../../_custom-en/networks/description-convention.adoc[]
|===
--
. Click **OK**. The EIP appears in the EIP list after this operation is performed.
