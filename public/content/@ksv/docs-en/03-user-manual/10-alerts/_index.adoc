The **Alerts** module helps you monitor cluster resources and ensures high reliability, availability, and performance for your {ks_product_abbr_full} cluster.  By default, alerting is enabled for your cluster when you install {ks_product}. In the {ks_product} web console, you can view node monitoring statistics, and analyze and handle alerts in real time.

You can also create alert policies and view alert messages after alerts are fired.  The **Alerts** module provides a summary of alert messages, alert policies, and details of each alert policy. You can create, delete, and edit alert policies.
