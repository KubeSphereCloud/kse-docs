== Procedurelink:[id=cb82d485d75848799f05c8ecc1e1d8f4]

include::../../../../_custom-en/vms/vm-view-steps.adoc[]

. On the **VMs** page, find the VM that you want to manage, right-click on the blank area to open the shortcuts for VM operations, and then select **Clone** from the drop-down list.
. In the dialog box that appears, configure parameters for the VM to clone.
+
--
[cols='1a,4a']
|===
|Parameter |Description

|Name
|The custom name of the VM to clone. The name can contain only lowercase letters, digits, and hyphens (-), and must start and end with a lowercase letter or a digit. The name can contain up to 16 characters in length.

|Quantity
|The number of VMs that you want to clone. You can clone up to 50 VMs at a time.

|Description
|(Optional) The description of the VM to clone. The description can be up to 128 characters in length. The character types of the description are unlimited.
|===
--
. Click **OK**. The VM appears in the VM list after it is cloned.
