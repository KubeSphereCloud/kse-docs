== Procedurelink:[id=33e179accc064a86bff540d8052913cf]

include::../../../../_custom-en/vms/vm-view-steps.adoc[]

. On the **VMs** page, click the name of a VM to view its details.
. In the upper-left corner of the details page, click **Resize**.
. In the dialog box that appears, modify the default configuration and disk size of the VM, and click **OK**.  The default configuration includes the number of CPU cores and memory size. For more information about parameters, see link:../01-create-vm/[Create a VM].
. In the **Note** dialog box, click **Restart** to make the new configuration take effect. If you click **Cancel**, the new configuration is saved.
+

// Note

include::../../../../../_ks_components/admonitions/note.adoc[]

The VM must be restarted before the new configuration takes effect.  The VM restart may take a while, during which no operations can be performed on the VM. Please select **Restart** or **Cancel** based on your business requirements.
