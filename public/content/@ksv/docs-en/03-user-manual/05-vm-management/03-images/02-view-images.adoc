. On the **System Images**, **Private Images**, or **Shared Images** tab, view the images of a specific type.  For more information about images of different types, see link:../[Images].
+
--
You can view all images in the project. The following table describes the parameters.

[cols="1a,4a"]
|===
|Parameter |Description

|Name
|The custom name of the image.

|ID
|The universally unique identifier (UUID) of the image in the system.

|Status
|The status of the image.

If an image is in an abnormal state, one of the following messages may appear:

* **Failed to create the image**
* **Failed to import the image**

|OS
|The OS type of the image.

|Version
|The OS version of the image.

|CPU
|The CPU configuration defined in the image.

|Memory
|The memory size defined in the image.

|System Disk
|The system disk size defined in the image.

|Shared by
|Specifies whether the image can be shared with other users.

|Shared
|The user who enables the sharing feature for the image. This parameter is available only on the **Shared Images** tab.
|===
--
. (Optional) Click the search box above the image list and specify query conditions to search for images by ID, name, or sharing status. For example, use **Name:ab** to search for images whose name contains the string **ab**.
. (Optional) In the upper-right corner of the page, click image:/images/ksv/en/icons/refresh-light.svg[refresh,15] to refresh data.
