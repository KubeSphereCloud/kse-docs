include::../../../../_custom-en/vms/vm-image-steps.adoc[]

. On the **System Images**, **Private Images**, or **Shared Images** tab, view the images of a specific type.  For more information about images of different types, see link:../[Images].
. Click the name of the image that you want to view. The details page of the image appears.
. On the left side of the details page, view basic information about the image.
+
--
[cols='1a,4a']
|===
|Parameter |Description

|ID
|The universally unique identifier (UUID) of the image in the system.

|Status
|The status of the image.

|OS
|The OS type of the image.

|Version
|The OS version of the image.

|Created at
|The time when the image was created.

|Created by
|The username that is used to create the image.
|===
--
. On the right-side **Overview** tab, view the default configuration of the image.
+
--
[cols='1a,4a']
|===
|Parameter |Description

|CPU
|The number of CPU cores defined in the image.

|Memory
|The memory size defined in the image.

|System Disk
|The system disk size defined in the image.
|===
--
. On the right-side **VMs** tab, view the VMs created based on the image. For more information about parameters, see link:../../01-vms/01-create-vm/[Create a VM].
