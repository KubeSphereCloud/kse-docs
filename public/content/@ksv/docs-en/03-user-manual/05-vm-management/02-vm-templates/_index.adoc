VM templates are master copies of VMs. Each template defines the specific configuration of VMs, including the OS, CPU, memory, system disk, data disk, and network.  You can use a template to create a VM or convert an existing VM to a template.  {ks_product_abbr_full} allows you to enable sharing for VM templates. After this feature is enabled, VM templates can be replicated by users in other projects.
