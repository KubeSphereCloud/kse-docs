--
You can view all VM templates in the project. The following table describes the parameters.

[cols="1a,4a"]
|===
|Parameter |Description

|Name
|The custom name of the VM template.

include::../../../../_custom-en/networks/naming-convention.adoc[]

|ID
|The universally unique identifier (UUID) of the template in the system. The UUID is automatically assigned by the system.

|OS
|The OS defined in the VM template. You can place the cursor on the OS to view the specific type and version of the OS.

|CPU
|The number of CPU cores defined in the template.

|Memory
|The memory size defined in the template.

|System Disk
|The system disk size defined in the template.

|Shared
|Specifies whether the template can be shared with other users. If this feature is enabled, users in other projects can replicate the template.

|Project
|The project name in which the template is created.

|Created at
|The time when the template was created.
|===
--

. (Optional) Click the search box above the VM template list and set query conditions to search for VM templates by name or ID.  For example, use **Name:ab** to search for VM templates whose name contains the string **ab**.
