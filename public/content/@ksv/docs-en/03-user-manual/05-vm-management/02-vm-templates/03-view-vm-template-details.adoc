. On the **Templates** page, click the name of a VM template to view its details.
. On the left side of the details page, view basic information about the VM template.
+
--
[cols='1a,4a']
|===
|Parameter |Description

|ID
|The universally unique identifier (UUID) of the template in the system. The UUID is automatically assigned by the system.

|Project
|The project name in which the template is created.

|Created at
|The time when the template was created.

|Shared
|Specifies whether the template can be shared with other users. If this feature is enabled, users in other projects can replicate the template.

|Created by
|The username that is used to create the template.
|===
--
. On the right side of the details page, view the configuration of the template.
+
--
[cols='1a,4a']
|===
|Parameter |Description

|Image
|The VM image defined in the template.

|CPU
|The number of CPU cores defined in the template.

|Memory
|The memory size defined in the template.

|System Disk
|The system disk size defined in the template.

|IP Range
|The IP address range of the VM defined in the template.

|Gateway
|The gateway address that corresponds to the VM.
|===
