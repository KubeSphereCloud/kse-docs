The **VMs** module is available to all users.  User **admin** can view and manage all VMs, VM templates, and images in all projects. The following table compares the permissions granted to non-admin users of different project roles.

[cols='1a,3a']
|===
|Project role |Permission

|Administrator
|View and manage all VMs, VM templates, and images in the projects to which they belong.

|Operator
|View and manage all VMs, VM templates, and images in the projects to which they belong.

|Viewer
|View only all VMs, VM templates, and images in the projects to which they belong.
|===
