== 新增功能link:[id=4b8b9151509840acb5a789ae83db2417]

* 新增物理网络模块。支持为虚拟机配置物理子网和弹性网络以达到更好的连通性和性能。
* 新增虚拟网络模块。支持在 VPC 网络和虚拟子网中创建虚拟机以实现安全隔离，也支持为 VPC 网络或虚拟机绑定弹性 IP 以开启公网访问。
* 新增 SSH 密钥功能。支持创建和为虚拟机绑定 SSH 密钥以安全便捷地登录 Linux 虚拟机。
* 新增安全组模块。支持创建安全组并为关联的虚拟机创建单独的入站和出站流量规则集，从而提高关联资源的安全性。
* 新增网络带宽限制功能。支持通过自定义 QoS 规则以保证网络传输带宽、降低网络传输时延，从而提高网络资源的利用率。
* 新增支持 KSV 和 KubeSphere 融合部署，实现容器和虚拟机共存共管。
* 新增支持企业级分布式存储 NeonSAN。

== 功能优化link:[id=582778c4ca7f4ab6a40dc7a8d310fbf3]

* 优化概览 UI 界面。支持通过全新的 UI 管理界面从虚拟机、磁盘、网络、物理节点、CPU、内存等维度实时查看集群中的资源情况，以更好地了解集群状态。
* 优化虚拟机管理功能。支持通过全新的 UI 管理界面实时监测和管理集群中的虚拟机状态和资源使用情况，也支持通过虚拟机、模板、镜像等方式创建虚拟机。
* 优化节点管理功能。支持通过全新的 UI 管理界面实时监测和管理集群中的物理节点状态和资源使用情况，也支持通过 Web 控制台快速添加集群节点。
* 优化虚拟机镜像功能。支持通过全新的 UI 管理界面创建和管理镜像，也支持通过 Web 控制台为镜像开启共享属性，与项目中的其他用户共享镜像。
* 优化告警和日志功能。支持通过全新的 UI 管理界面查看系统中的告警和日志，实时分析和处理告警。
