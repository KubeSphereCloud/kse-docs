== New featureslink:[id=335f5ec664454c279e13a7611927c681]

* Add support for manually setting an IP address when creating a VM.
* Add support for exposing VM services.
* Add support for viewing projects to which a user belongs and roles of the user.

[discrete]
endif::[]
== Enhancementslink:[id=ef32dc1b99e94d77ae315d363222937d]

* Add support for modifying network configuration when adjusting VM resources.
* Add support for displaying VM shortcut operation menu with a right click on the VM list page.

== Fixed issueslink:[id=35b0203735f04305996af82786689194]

* Remove the terminal connection method for VMs created using ISO images.
* Fix the issue of unable to uninstall KSV because the uninstallation is stuck.
* Fix the issue that the disk information on the list page is not updated after disk extension.
