== New featureslink:[id=d324ea519e184d2284f602034a41f238]

* Add the operation log management feature.
* Add support for deleting image files with the same name in the file server.

[discrete]
endif::[]
== Enhancementslink:[id=62eaacd9e02745119c596ecc1845c832]

* Split the Edit VM feature to Edit and Adjust Resource features.
* Optimize the shortcut operation menu in the VM details page.
* Add support for creating VMs after a maximum of 9 VMs is created when KSV is not activated, but the excess VMs will be locked.
* Add a feature to connect to VMs created from ISO images through VNC.
