== New featureslink:[id=925b002815264f96b24c20c8185fe1da]

* Add support for using multiple NICs.
* Add support for vertical scaling of VM resources.
* Add support for disk expansion.
* Add the log management feature.

== Enhancementslink:[id=e565c950941044b0a989ddc912cc23e9]

* Add the Chinese translation of the description in the user login history area.
* Optimize the icons of sorting items in ascending and descending order.
* Remove the hostPath version installation package.
* Change the overcommit ratio to 1-4 and set the default as 2.
* Change the name of master node to control plane node.
