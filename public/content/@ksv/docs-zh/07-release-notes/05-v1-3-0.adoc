== 新增功能link:[id=d324ea519e184d2284f602034a41f238]

* 新增支持操作日志管理功能。

* 新增支持删除文件服务器中的同名镜像文件。

== 功能优化link:[id=62eaacd9e02745119c596ecc1845c832]

* **编辑虚拟机**功能拆分为**编辑**和**调整资源**。

* 优化虚拟机详情页面的快捷操作菜单。

* 未激活 KSV 时，虚拟机数量达到上限 9 台后可以继续创建，但超出限制的虚拟机会被锁定。

* 使用 ISO 镜像创建的虚拟机默认使用 VNC 连接。
