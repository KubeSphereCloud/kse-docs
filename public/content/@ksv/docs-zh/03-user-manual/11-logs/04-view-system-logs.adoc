本节介绍如何使用{ks_product_abbr_full_both}Web 控制台查看系统日志列表。日志的默认保留期限为 **7** 天。

ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 前提条件link:[id=70342127712c49828331dc3eb1c56bbf]

系统 **admin** 用户可以查看集群中所有项目中的系统日志，普通项目管理员、操作员或观察员可以查看各自所属项目中的系统日志。

ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 操作步骤link:[id=8cf12168e5594a90a65f0d90bffdaf2b]

include::../../../_custom/logs/logs-steps.adoc[]

. 在**系统日志**页签，查看系统日志列表。
+
--
系统日志列表页面展示系统中所有的日志数据，并提供以下信息：
[cols='1a,4a']
|===
|参数 |描述

|组件名称
|{ks_product_right}集群中各个组件的名称。组件包括{ks_product_both}Web 控制台上的项目和后台的系统组件。

|时间
|生成日志条目的时间。

|消息
|各个组件返回的日志消息。
|===
--

. （可选）点击系统日志列表上方的搜索框并设置搜索条件，可按关键字、组件名称或时间范围搜索系统日志。例如，使用**关键字:error** 搜索包含字符串 **error** 的日志条目。

. （可选）在系统日志列表右上角点击 image:/images/ksv/zh/icons/refresh-light.svg[refresh,15px] 可刷新列表数据。
