本节介绍如何使用{ks_product_abbr_full_both}Web 控制台查看操作日志列表。日志的默认保留期限为 **7** 天。

ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 前提条件link:[id=d5f190f80cfe4e11b8acd5bb1cb05c5a]

系统 **admin** 用户可以查看集群中所有项目中的操作日志，普通项目管理员、操作员或观察员可以查看各自所属项目中的操作日志。

ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 操作步骤link:[id=e8355bb1515a48bc90f7921eb8289a36]


include::../../../_custom/logs/logs-steps.adoc[]

. 在**操作日志**页签，查看操作日志列表。
+
--
操作日志列表展示系统中所有的操作日志，并提供以下信息：

[cols='1a,4a']
|===
|参数 |描述

|时间
|生成操作日志的时间。

|操作行为
|对给定资源所执行的操作。

|状态
|服务器对浏览器请求作出的响应状态。

|项目
|被操作的资源所在的项目。

|资源类型与名称
|被操作的资源的类型与名称。

|操作者
|对资源进行操作的用户。
|===
--

. （可选）点击操作日志列表上方的搜索框并设置搜索条件，可按项目、操作行为或时间范围搜索操作日志。例如，使用**项目:default** 搜索 **default** 项目中的操作日志。

. （可选）在操作日志列表右上角点击 image:/images/ksv/zh/icons/refresh-light.svg[refresh,15] 可刷新列表数据。
