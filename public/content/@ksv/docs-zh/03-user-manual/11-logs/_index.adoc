日志包括系统日志和操作日志，系统日志收集系统中的日志数据，操作日志可以用于查看用户的操作记录。**日志**功能提供系统日志汇总列表、各系统日志的详情、操作日志汇总列表和各操作日志的详情，并支持查看实时系统日志和导出系统日志文件到本地。
