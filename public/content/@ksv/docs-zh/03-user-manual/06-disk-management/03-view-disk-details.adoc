本节介绍如何使用{ks_product_abbr_full_both}Web 控制台查看磁盘详情。

ifeval::["{ks_output_type}" == "pdf"]
[discrete]

== 操作步骤link:[id=44f3534d556743edb8fc4a569390f2f8]

include::../../../_custom/disks/disk-steps.adoc[]

. 在磁盘列表页面，点击目标磁盘名称，进入磁盘详情页面。

. 在**基本信息**区域，查看磁盘的基本信息。具体参数描述如下表所示。
+
--
[cols="1a,4a"]
|===
|参数 |描述

|ID
|磁盘在系统中的通用唯一标识符。

|状态
|磁盘当前的状态。

|容量
|创建磁盘时指定的总容量大小。

|所属项目
|磁盘所在的项目名称。

|创建时间
|磁盘创建的时间。

|创建者
|创建磁盘时使用的用户名称。
|===
--

. 在**所属虚拟机**区域，查看已挂载该磁盘的虚拟机信息。
+
--
// Note
include::../../../../_ks_components/admonitions/note.adoc[]

* 点击 image:/images/ksv/zh/icons/add-light.svg[18px,18px] 可将虚拟机挂载到当前磁盘。
* 点击虚拟机 ID 右侧的 image:/images/ksv/zh/icons/close-light.svg[18px,18px] 可将磁盘从虚拟机解绑。

include::../../../../_ks_components/admonitions/admonEnd.adoc[]
--

. 在磁盘详情页面的右侧区域，查看磁盘的快照信息。具体参数描述如下表所示。
+
--
[cols="1a,4a"]
|===
|参数 |描述

|名称
|快照的名称。

include::../../../_custom/disks/disk-snapshot-naming-convention.adoc[]

|ID
|快照在系统中的通用唯一标识符。

|描述
|快照的描述信息。

include::../../../_custom/networks/description-convention.adoc[]

|容量
|在快照创建时刻磁盘的容量大小。

|创建时间
|快照创建的时间。
|===
