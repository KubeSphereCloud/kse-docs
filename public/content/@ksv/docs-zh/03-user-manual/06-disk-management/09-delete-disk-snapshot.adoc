本节介绍如何使用{ks_product_both}Web 控制台删除磁盘快照。

删除快照时，系统会从快照列表中移除该快照并会保留虚拟机当前的状态，不会影响虚拟机本身或其他快照。

== 操作步骤link:[id=f17b7812525947eeacfba26d457ac113]

include::../../../_custom/disks/disk-steps.adoc[]

. 在磁盘列表页面，点击目标磁盘名称，进入磁盘详情页面。

. 在页面右侧的**快照**页签，找到目标磁盘快照，点击右侧的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**删除**。

. 在弹出的对话框中点击**确定**可删除快照。
