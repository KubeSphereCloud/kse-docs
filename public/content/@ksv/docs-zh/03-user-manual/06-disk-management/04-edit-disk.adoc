== 操作步骤link:[id=caa64fd9cf634f8b973523c418f104b5]

include::../../../_custom/disks/disk-steps.adoc[]

. 找到目标磁盘，点击右侧的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**编辑**。

. 在弹出的对话框中修改磁盘的**名称**和**描述**，然后点击**确定**。
