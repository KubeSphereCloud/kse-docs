本节介绍如何使用{ks_product_both}Web 控制台恢复磁盘到快照。磁盘创建后，如果您使用快照备份了磁盘，则可以根据快照恢复磁盘。

== 操作步骤link:[id=58244af7f74e4d598af15ef0ee5a479d]

include::../../../_custom/disks/disk-steps.adoc[]

. 在磁盘列表页面，点击目标磁盘名称，进入磁盘详情页面。

. 在页面右侧的**快照**页签，找到目标磁盘快照，点击右侧的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**还原磁盘**。

. 在弹出的对话框中，设置还原记录的名称和描述。
+
--
[cols="1a,4a"]
|===
|参数 |描述

|名称
|还原记录的名称。

include::../../../_custom/networks/naming-convention.adoc[]

|描述
|还原记录的描述信息。

include::../../../_custom/networks/description-convention.adoc[]
|===
--

. 点击**确定**。
