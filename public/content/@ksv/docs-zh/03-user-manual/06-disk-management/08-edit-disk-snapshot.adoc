== 操作步骤link:[id=da767e1b7d804ad2acc6409220a942d7]

include::../../../_custom/disks/disk-steps.adoc[]

. 在磁盘列表页面，点击目标磁盘名称，进入磁盘详情页面。

. 在页面右侧的**快照**页签，找到目标磁盘快照，点击右侧的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**编辑**。

. 在弹出的对话框中，设置新的快照名称和描述。
+

|快照的描述信息。

include::../../../_custom/networks/description-convention.adoc[]
|===
--

. 点击**确定**。
