== 操作步骤link:[id=b1e0eb3eb9d548e0bed1e6792e51f20a]

include::../../../_custom/disks/disk-steps.adoc[]
+
--
磁盘列表展示当前项目中的所有磁盘。具体参数描述如下表所示。

[cols="1a,4a"]
|===
|参数 |描述

|名称
|磁盘的自定义名称。

include::../../../_custom/vms/vm-naming-convention.adoc[]

|ID
|磁盘在系统中的通用唯一标识符。

|状态
|磁盘当前的状态。

|容量
|磁盘的容量大小。

|所属虚拟机
|磁盘所挂载的虚拟机的名称。

|所属项目
|磁盘所在的项目名称。

|创建者
|创建磁盘时使用的用户名称。
|===
--

. （可选）点击磁盘列表上方的搜索框并设置搜索条件，可按名称或 ID 搜索磁盘。例如，**名称:ab** 表示搜索名称包含字符串 **ab** 的磁盘。

. （可选）在磁盘列表右上角点击 image:/images/ksv/zh/icons/refresh-light.svg[refresh,15] 可刷新列表数据。
