== 操作步骤link:[id=959950457d8e4a1f96a72b300299cdcb]

include::../../../_custom/disks/disk-steps.adoc[]

. 在磁盘列表页面，点击目标磁盘名称，进入磁盘详情页面。

. 在页面右侧的**快照**页签，点击**创建**。

. 在弹出的对话框中，设置快照名称和描述。
+
--
[cols="1a,4a"]
|===
|参数 |描述

|名称
|快照的名称。

include::../../../_custom/disks/disk-snapshot-naming-convention.adoc[]

|描述
|快照的描述信息。

include::../../../_custom/networks/description-convention.adoc[]
|===
--

. 点击**确定**。快照创建后将显示在快照列表中。
