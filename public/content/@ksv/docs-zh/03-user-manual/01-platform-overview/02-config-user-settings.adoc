本节介绍如何使用{ks_product_abbr_full_both}Web 控制台修改当前用户设置。您可以通过本操作修改当前登录用户的邮箱、控制台语言和用户密码。

ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 操作步骤link:[id=1eb466b6fb1340168dbd66d2f24a784b]

. 登录{ks_product_both}Web 控制台。

. 在页面右上角点击 image:/images/ksv/zh/icons/cogwheel.svg[set-login-user,18,18]，然后选择**当前用户设置**。

. 在弹出的对话框中，选择**基本信息**和**密码**页签即可修改对应的用户邮箱、控制台语言和密码。
+
--
[cols='1a,4a']
|===
|参数 |描述

|用户名
|当前登录用户的名称。用户创建后暂不支持修改用户名称。
include::../../../_custom/security-groups/naming-convention.adoc[]

|邮箱
|当前登录用户的邮箱。

|语言
|当前控制台使用的语言。目前支持简体中文和英文两种语言类型。

|当前密码
|输入当前登录用户的密码。
include::../../../_custom/users/password-rule.adoc[]

|新密码
|输入您要设置的新用户密码。

|确认新密码
|再次输入新用户密码以确认。
|===
--

. 点击 image:/images/ksv/zh/icons/check-dark.svg[18px,18px] 保存修改。
