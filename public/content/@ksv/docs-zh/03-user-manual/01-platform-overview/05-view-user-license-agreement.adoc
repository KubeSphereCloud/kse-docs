ifeval::["{ks_output_type}" == "pdf"]
[discrete]

== 操作步骤link:[id=3ab75c6211704a7781abb1f67ddf838c]

. 登录{ks_product_both}Web 控制台。

. 在页面右上角点击 image:/images/ksv/zh/icons/cogwheel.svg[set-login-user,18,18]，然后选择**关于{ks_product_left}**。

. 在弹出的对话框中点击**《最终用户软件许可协议》**可查看{ks_product_both}用户许可协议。
