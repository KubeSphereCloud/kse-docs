本节介绍如何使用{ks_product_abbr_full_both}Web 控制台删除 SSH 密钥。为安全起见，每个用户仅能删除自己创建的 SSH 密钥。

== 删除单个 SSH 密钥link:[id=6a3653f8f5bb45839b5e29abf84808e3]

include::../../../_custom/ssh/ssh-key-steps.adoc[]

. 找到目标 SSH 密钥，点击右侧的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**删除**。

. 点击**确定**即可删除目标 SSH 密钥。
+
--
// Warning
include::../../../../_ks_components/admonitions/warning.adoc[]

删除 SSH 密钥后不可恢复，请谨慎执行此操作。

include::../../../../_ks_components/admonitions/admonEnd.adoc[]
--

== 批量删除 SSH 密钥link:[id=0ff2bebb0b2b4e408e66cd0c7ae9ffc8]

include::../../../_custom/ssh/ssh-key-steps.adoc[]

. 在 SSH 密钥列表页面，选择需要删除的 SSH 密钥名称左侧的复选框，然后在列表上方点击**删除**。

. 在弹出的对话框中点击**确定**可批量删除多个 SSH 密钥。
+
--
// Warning
include::../../../../_ks_components/admonitions/warning.adoc[]
