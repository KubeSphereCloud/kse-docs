本节介绍如何使用{ks_product_abbr_full_both}Web 控制台启动、关闭和重启虚拟机。

{ks_product_right}支持强制重启和强制关闭虚拟机操作。当虚拟机异常时，您可以重启虚拟机进行修复。当虚拟机不再使用时，您可以关闭虚拟机。

// Note
include::../../../../../_ks_components/admonitions/note.adoc[]

关闭或重启虚拟机会对集群产生一定影响，如无特殊需要（如故障修复等），请勿关闭或重启处于运行中状态的虚拟机。

include::../../../../../_ks_components/admonitions/admonEnd.adoc[]

== 操作步骤link:[id=36468a7657a448fda06260f94e523067]

include::../../../../_custom/vms/vm-view-steps.adoc[]

. 在虚拟机列表页面，在目标虚拟机的空白区域点击右键以打开虚拟机操作菜单，然后选择**关闭**、**启动**或**重启**。
+
--
// Note
include::../../../../../_ks_components/admonitions/note.adoc[]

若要强制关闭或强制重启虚拟机，在虚拟机详情页面左侧区域，点击右上角的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后选择**强制关闭**或**强制重启**。

include::../../../../../_ks_components/admonitions/admonEnd.adoc[]
--

. 在弹出的对话框中点击**确定**可启动、关闭或重启虚拟机。
+
--
// Note
include::../../../../../_ks_components/admonitions/note.adoc[]

* 虚拟机关闭、启动和重启可能需要一定时间，虚拟机状态将在操作完成后自动更新。
* 取决于虚拟机当前的状态，虚拟机可能不支持部分操作。

include::../../../../../_ks_components/admonitions/admonEnd.adoc[]
--
