== 操作步骤link:[id=629549f6b5fc40f882b9a7453e6fca87]

include::../../../../_custom/vms/vm-view-steps.adoc[]

. 在虚拟机列表页面，点击目标虚拟机名称，进入虚拟机详情页面。

. 在虚拟机详情页面左侧区域，点击右上角的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**编辑**。

. 在弹出的对话框中修改虚拟机的**名称**和**描述**，然后点击**确定**。有关更多参数信息，请参阅 link:../01-create-vm/[创建虚拟机]。
