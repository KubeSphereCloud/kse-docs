== 操作步骤link:[id=47ccb252e8474bf4a46061bac0bcc4e6]

include::../../../../_custom/vms/vm-view-steps.adoc[]

. 在虚拟机列表页面，在目标虚拟机的空白区域点击右键以打开虚拟机操作菜单，然后选择**创建快照**。

. 按照页面向导，为虚拟机快照设置名称和描述，然后点击**创建**。
