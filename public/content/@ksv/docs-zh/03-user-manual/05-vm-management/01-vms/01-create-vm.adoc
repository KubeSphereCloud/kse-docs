== 前提条件link:[id=9f46e6e14fa046b0ad2ac61dfa17e99e]

* 请确保您的{ks_product_both}集群可创建足够多的虚拟机。{ks_product_right}个人免费版支持添加最多 9 台虚拟机，超出配额限制的虚拟机将被锁定。如需添加更多虚拟机，需购买{ks_product_both}开发者版或{ks_product_both}企业版。有关更多的版本信息和配额限制，请参阅 link:../../../../01-introduction/03-editions/[版本选型]。

* 请确保您有虚拟机管理权限。仅系统 **admin** 用户、项目管理员或操作员可以对虚拟机执行管理操作，项目观察员只能查看项目中的虚拟机。

{ks_product_right}支持以下三种方式创建虚拟机：

* link:#189b0c48c87244bcbc89fe512c41c59e[基于镜像创建]

* link:#bcdafbc0126b48a29d351f7cdaf592bc[基于模板创建]

* link:#c131afce11134cf9a1b5d718941df40b[基于虚拟机创建]

== 基于镜像创建虚拟机link:[id=189b0c48c87244bcbc89fe512c41c59e]

include::../../../../_custom/vms/vm-view-steps.adoc[]

. 在虚拟机列表页面，点击**创建**，然后选择**基于镜像**，进入虚拟机创建页面。
+
include::../../../../_custom/vms/vm-create-basic-parameters.adoc[]
+
. 在**配置信息**页签的**镜像**区域，选择虚拟机使用的镜像类型。
+
--
// Note
include::../../../../../_ks_components/admonitions/note.adoc[]

* {ks_product_right}目前已支持多种主流的 Linux 发行版镜像，如 Fedora、CentOS 和 Ubuntu 等。

* 您可以根据业务需求，选择基于**系统镜像**、**自有镜像**或**共享镜像**创建虚拟机。有关各镜像类型的具体信息，请参阅 link:../../03-images/[镜像]。

include::../../../../../_ks_components/admonitions/admonEnd.adoc[]
--

. 在**配置**区域，选择虚拟机的 CPU 和内存配置。
+
--
[cols="1a,3a"]
|===
|配置名称 |描述

|默认配置 N 核 N GiB
|此处的默认配置与镜像的配置相同。

|普通级配置 2 核 4 GiB
|2 核 CPU 和 4 GiB 内存，适用于中等规模并发业务和通用数据处理。

|专业级 4 核 8 GiB
|4 核 CPU 和 8 GiB 内存，适用于企业高并发业务场景。

|自定义配置
|自定义 CPU 和内存配置。
|===
--

== 基于模板创建虚拟机link:[id=bcdafbc0126b48a29d351f7cdaf592bc]

include::../../../../_custom/vms/vm-view-steps.adoc[]

. 在虚拟机列表页面，点击**创建**，然后选择**基于模板**，进入虚拟机创建页面。
+
include::../../../../_custom/vms/vm-create-basic-parameters.adoc[]
+
. 在**基于模板**页面的**选择模板**区域，选择当前项目下的一个虚拟机模板，用于创建虚拟机。
+
--
[cols='1a,4a']
|===
|参数 |描述

|名称
|用户自定义的模板名称。

|ID
|虚拟机模板在系统中的通用唯一标识符，由系统自动分配。

|操作系统
|使用该模板创建虚拟机时的操作系统镜像。将光标移动到操作系统图标上可查看操作系统的具体类型和版本。

|CPU
|使用该模板创建时虚拟机的 CPU 核心数。

|内存
|使用该模板创建时虚拟机的内存资源配置。

|系统盘
|使用该模板创建时虚拟机的系统盘配置。
|===
--

. 在**配置**区域，选择虚拟机的 CPU 和内存配置。
+
--
[cols='1a,4a']
|===
|配置名称 |描述

|CPU
|虚拟机可使用的 CPU 核心数。

|内存
|虚拟机可使用的内存资源大小。
|===
--

== 基于虚拟机创建虚拟机link:[id=c131afce11134cf9a1b5d718941df40b]

include::../../../../_custom/vms/vm-view-steps.adoc[]

. 在虚拟机列表页面，点击**创建**，然后选择**基于虚拟机**，进入虚拟机创建页面。
+
include::../../../../_custom/vms/vm-create-basic-parameters.adoc[]
+
. 在**配置信息**页签的**虚拟机**区域，选择当前项目下的一个虚拟机，用于创建虚拟机。
+
--
[cols='1a,4a']
|===
|参数 |描述

|名称
|用户自定义的虚拟机名称。

|ID
|虚拟机在系统中的通用唯一标识符，由系统自动分配。

|操作系统
|虚拟机使用的操作系统。将光标移动到操作系统图标上可查看操作系统的具体类型和版本。

|IP 地址
|源虚拟机的 IP 地址。
|===
--
