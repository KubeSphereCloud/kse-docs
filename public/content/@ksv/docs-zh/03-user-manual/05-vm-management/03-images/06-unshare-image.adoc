== 操作步骤link:[id=632c68385cfd48ab93094a2a89e04ce8]

include::../../../../_custom/vms/vm-image-steps.adoc[]

. 在**共享镜像**页签，点击目标镜像名称右侧的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后从下拉列表中选择**取消共享**。

. 在弹出的对话框中点击**确定**。镜像共享取消后，其他项目不能再通过复制获取该镜像。已经复制的镜像不受影响。
