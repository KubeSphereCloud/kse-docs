== 操作步骤link:[id=46e258fa429c441d9e9072bc00011ad1]

include::../../../../_custom/vms/vm-image-steps.adoc[]

. 选择**系统镜像**、**自有镜像**或**共享镜像**页签，查看对应类型的镜像列表。

. 在镜像列表页面，点击目标镜像名称，进入镜像详情页面。

. 在镜像详情页面左上角点击**编辑**。
+

include::../../../../../_ks_components/admonitions/admonEnd.adoc[]
--

. 在弹出的对话框中修改镜像的**名称**和**描述**，然后点击**确定**。有关更多参数信息，请参阅 link:../01-create-image/[创建镜像]。
