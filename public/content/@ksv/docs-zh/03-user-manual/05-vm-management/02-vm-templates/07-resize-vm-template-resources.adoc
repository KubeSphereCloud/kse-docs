== 操作步骤link:[id=84ec35635d614a8bab8d5882cb7f5cc4]

include::../../../../_custom/vms/vm-template-steps.adoc[]

. 在虚拟机模板列表页面，点击目标虚拟机模板名称，进入虚拟机模板详情页面。

. 在虚拟机模板详情页面的左侧区域，点击**调整资源**。

. 在弹出的对话框中修改虚拟机模板的资源配置。有关更多参数信息，请参阅 link:../01-create-vm-template/[创建虚拟机模板]。
