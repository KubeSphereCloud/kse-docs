本节介绍如何使用{ks_product_abbr_full_both}Web 控制台编辑虚拟机模板。您可以通过编辑操作修改虚拟机模板的名称、描述和是否共享属性。如需调整虚拟机模板资源，请参阅 link:../07-resize-vm-template-resources/[调整虚拟机模板资源]。

== 在虚拟机模板列表页面编辑link:[id=0e7c253dd21242aa947f6c09f5c1f43c]

include::../../../../_custom/vms/vm-template-steps.adoc[]

. 找到目标虚拟机模板，点击右侧的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**编辑**。

. 在弹出的对话框中修改虚拟机模板的**名称**、**描述**和**是否共享**属性，然后点击**确定**。

== 在虚拟机模板详情页面编辑link:[id=f4ad9649a6ce49f6a00f11eafdcaace5]

include::../../../../_custom/vms/vm-template-steps.adoc[]

. 在虚拟机模板列表页面，点击目标虚拟机模板名称，进入虚拟机模板详情页面。

. 在虚拟机模板详情页面的左侧区域，点击右侧的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后从下拉列表中选择**编辑**。

. 在弹出的对话框中修改虚拟机模板的**名称**、**描述**和**是否共享**属性，然后点击**确定**。
