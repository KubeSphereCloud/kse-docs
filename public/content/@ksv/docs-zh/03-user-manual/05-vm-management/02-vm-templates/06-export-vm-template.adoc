== 操作步骤link:[id=1c13768f85ba4130a23baa7816aedf29]

include::../../../../_custom/vms/vm-template-steps.adoc[]

. 在虚拟机模板列表页面，点击目标虚拟机模板名称，进入虚拟机模板详情页面。

. 在虚拟机模板详情页面的左侧区域，点击image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**导出**即可将虚拟机模板保存至本地。
