== 操作步骤link:[id=0b187d9ce92b4c0b865a52eeb375e56d]

include::../../../_custom/nodes/steps-view.adoc[]

. 在节点列表页面，点击目标节点的名称，进入节点详情页面。

. 在节点详情页面的左侧区域，查看节点的基本信息和物理资源信息。
+
--
[cols='1a,4a']
|===
|参数 |描述

|状态
|节点的健康状态。

|IP
|节点的 IP 地址。同一集群内的节点可通过 IP 地址互通。

|角色
|节点在集群中的角色。{ks_product_right}提供工作节点和控制节点两种节点类型。

include::../../../_custom/nodes/node-type-description.adoc[]

|操作系统
|节点使用的操作系统类型。

include::../../../_custom/nodes/supported-os-types.adoc[]

|内核版本
|节点使用的操作系统的内核版本。

|架构
|节点使用的架构类型。

|创建时间
|节点加入集群的时间。

|CPU 型号
|节点使用的 CPU 型号。

|CPU 核数
|节点的 CPU 物理核数。CPU 总核数 = 物理 CPU 个数 x 每颗物理 CPU 的核数。

|物理盘
|节点上已插入的物理盘数量。

|物理网卡
|节点上已插入的的物理网卡数量。
|===
--

. 在节点详情页面的右侧区域，点击**概览**页签，查看节点相关的资源概览信息。
+
--
[cols='1a,4a']
|===
|区域 |描述

|用量
|节点的资源用量，包括 CPU、内存、数据盘和系统盘用量。

|已分配资源
|节点的可用资源限制和用量统计，包括 CPU 和内存资源。

|监控
|集群各类资源的用量统计信息，包括计算、网络和存储资源。

展开**监控**区域右侧的下拉框可自定义查询时间范围。{ks_product_right}支持查看最近 7 天内的资源用量情况。

|计算
|计算资源监控，包括 CPU 使用率、CPU 平均负载和内存使用率。

|网络
|网络资源监控，包括网络进出口流量监控、包转发率等。

展开**网络**区域右侧的下拉框可选择查看指定网卡对应的网络状态。

|存储
|存储资源监控，包括磁盘使用率、磁盘吞吐、IOPS、I/O 平均延迟和 I/O 队列长度。

展开**存储**区域右侧的下拉框可选择查看指定物理盘的资源用量。
|===
--

. 在节点详情页面的右侧区域，点击**虚拟机**页签，查看运行在节点上的虚拟机信息。
+
--
[cols='1a,4a']
|===
|参数 |描述

|名称
|
include::../../../_custom/nodes/name-description.adoc[]

|状态
|
include::../../../_custom/nodes/status-description.adoc[]

|操作系统
|虚拟机的操作系统类型。将光标移动到操作系统图标上可查看操作系统的具体类型和版本。

|IP 地址
|
include::../../../_custom/nodes/ip-address-description.adoc[]

|CPU 用量
|虚拟机的 CPU 用量。

|内存用量
|虚拟机的内存用量。

|系统盘
|虚拟机的系统盘容量大小。
|===
--

. 在节点详情页面的右侧区域，点击**磁盘**页签，查看已添加到节点上的物理盘信息。
+
--
[cols='1a,4a']
|===
|参数 |描述

|名称
|
include::../../../_custom/nodes/disk-name-description.adoc[]

|类型
|
include::../../../_custom/nodes/disk-type-description.adoc[]

|型号
|磁盘的型号。{ks_product_right}支持 SATA 盘、HDD 盘和 SSD 盘。

|容量
|磁盘的容量大小。

|挂载点
|磁盘的挂载点，即磁盘文件系统的入口目录。

|文件系统
|磁盘的文件系统名称。
|===
--


. 在节点详情页面的右侧区域，点击**物理网卡**页签，查看已绑定到节点上的物理网卡信息。
+
--
[cols='1a,4a']
|===
|参数 |描述

|名称
|物理网卡的名称。

|Mac 地址
|网卡的物理地址，用于在网络中唯一标示一个网卡。

|状态
|网卡的状态。

|类型
|网卡的类型。

{ks_product_right}支持物理网卡和虚拟网卡两种网卡类型。

|IP 地址
|网卡对应的 IP 地址。

|MTU
|网卡支持的最大传输单元。

|最大速率
|网卡支持的最大传输速率。

|当前速率
|网卡当前的传输速率。
|===
--

. 在节点详情页面的右侧区域，点击**告警**页签，查看节点上的告警信息。
+
--
[cols='1a,4a']
|===
|参数 |描述

|告警消息
|告警触发后自动生成的告警消息。

|告警状态
|告警的状态。包括以下状态类型：

include::../../../_custom/alerts/alert-status-description.adoc[]

|告警级别
|告警的严重程度，包括以下三个级别：

include::../../../_custom/alerts/alert-severity-levels.adoc[]

|告警策略
|用户自定义的告警策略名称。

include::../../../_custom/networks/physical-network-naming-convention.adoc[]

|告警激活时间
|触发告警的时间。
|===
--

. 在节点详情页面的右侧区域，点击**事件**页签，查看节点上的事件信息。
+
--
[cols='1a,4a']
|===
|参数 |描述

|类型
|事件的类型。

|原因
|事件发生的原因。

|发生时间
|事件发生的时间。

|来源
|发生事件的资源名称。

|消息
|事件发生后自动生成的消息。
|===
--

. （可选）点击页面右上角的**导出**可将事件信息保存至本地。
