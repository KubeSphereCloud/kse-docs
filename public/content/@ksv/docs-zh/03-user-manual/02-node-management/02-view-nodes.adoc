include::../../../_custom/nodes/steps-view.adoc[]

+
--
节点列表页面展示集群中所有的节点，并提供以下信息：

[cols="1a,4a"]
|===
|参数 |描述

|节点总数
|当前集群中的节点总数量。

|控制节点
|当前集群中的控制节点数量，负责整个集群的管理和调度。

|工作节点
|当前集群中的工作节点数量，负责承担集群中的工作负载。

|名称
|节点的名称。

在单节点模式下，节点名称由系统自动分配。在多节点模式下，节点名称由用户在安装{ks_product_both}时指定。

|IP 地址
|节点的 IP 地址。同一集群内的节点可通过 IP 地址互通。

|状态
|节点的健康状态。

|角色
|节点在集群中的角色。{ks_product_right}提供工作节点和控制节点两种节点类型。

include::../../../_custom/nodes/node-type-description.adoc[]

|CPU
|节点的 CPU 用量。

|内存
|节点的内存用量。

|系统盘
|节点的系统盘用量情况。

|已分配 CPU
|节点的可用 CPU 限制和用量统计。

|已分配内存
|节点的可用内存限制和用量统计。
|===
--

. （可选）点击节点列表上方的搜索框并设置搜索条件，可按名称或健康状态搜索节点。例如，使用**名称:ab** 搜索名称包含字符串 **ab** 的节点。

. （可选）在节点列表右上角点击 image:/images/ksv/zh/icons/refresh-light.svg[refresh,15] 可刷新列表数据。
