--
项目列表展示当前用户所属的项目，并提供以下信息：

[cols="1a,4a"]
|===
|参数 |描述

|名称
|
include::../../../_custom/projects/project-name-description.adoc[]

|ID
|
include::../../../_custom/projects/project-alias-description.adoc[]

|状态
|项目的活跃状态。

|成员
|项目所拥有的成员数量。

|虚拟机
|项目中已创建的虚拟机数量。

|CPU 用量
|项目的 CPU 用量。

|内存用量
|项目的内存用量。

|角色
|当前登录用户在项目中的角色类型。
|===
--

. （可选）点击项目列表上方的搜索框并设置搜索条件，可按名称搜索项目。例如，使用**名称:ab** 搜索名称包含字符串 **ab** 的项目。

. （可选）在项目列表右上角点击 image:/images/ksv/zh/icons/refresh-light.svg[refresh,15] 可刷新列表数据。
