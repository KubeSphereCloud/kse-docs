== 操作步骤link:[id=ab25e5eadcc544018323103cea4c83b5]

include::../../../_custom/users/user-steps.adoc[]

. 在**本地用户**页面，点击目标用户名称，进入用户详情页面。

. 在用户详情页面，点击左上角的**编辑**。

. 在弹出的对话框中修改用户的邮箱和描述信息，然后点击**确定**。有关更多参数信息，请参阅 link:../01-create-user/[创建用户]。
