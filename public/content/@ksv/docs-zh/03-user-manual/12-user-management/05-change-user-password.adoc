== 操作步骤link:[id=d9ef18e6d03a4d75a9b47ba532a389a6]

include::../../../_custom/users/user-steps.adoc[]

. 在**本地用户**页面，点击目标用户名称，进入用户详情页面。

. 在用户详情页面左上角点击 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**修改密码**。

. 在弹出的对话框输入新密码，并确认新密码，然后点击**确定**。
