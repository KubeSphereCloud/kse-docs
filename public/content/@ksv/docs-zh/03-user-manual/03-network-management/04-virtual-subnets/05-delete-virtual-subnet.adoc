== 删除单个虚拟子网link:[id=c81f34d2d6da44fdb384c815a014615e]

include::../../../../_custom/networks/virtual-subnet-steps.adoc[]

. 找到目标虚拟子网，点击右侧的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**删除**。

. 点击**确定**即可删除目标虚拟子网。
+
--

// Warning
include::../../../../../_ks_components/admonitions/warning.adoc[]

虚拟子网删除后无法恢复，请谨慎执行此操作。

include::../../../../../_ks_components/admonitions/admonEnd.adoc[]

// Note
include::../../../../../_ks_components/admonitions/note.adoc[]

请确保待删除的虚拟子网未和任何资源关联，否则虚拟子网将无法删除。

include::../../../../../_ks_components/admonitions/admonEnd.adoc[]
--

== 批量删除虚拟子网link:[id=6ed0ae36079e42dc8a348275c97ca6e9]

include::../../../../_custom/networks/virtual-subnet-steps.adoc[]

. 在虚拟子网列表页面，选择需要删除的虚拟子网名称左侧的复选框，然后在列表上方点击**删除**。

. 在弹出的对话框中点击**确定**可批量删除多个虚拟子网。
+
