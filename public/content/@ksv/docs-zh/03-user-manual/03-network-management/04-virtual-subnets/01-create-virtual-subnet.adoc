== 操作步骤link:[id=ea1f56ec102c42f5ab6a03b8de5281d9]

include::../../../../_custom/networks/virtual-subnet-steps.adoc[]

. 在虚拟子网列表页面，点击右上角的**创建**。

. 在弹出的对话框中配置虚拟子网的相关参数。具体参数描述请参见下表。
+
--
[cols='1a,4a']
|===
|参数 |描述

|名称
|虚拟子网的名称。

include::../../../../_custom/networks/naming-convention.adoc[]

|VPC
|虚拟子网所在的 VPC 网络。

|子网网段
|虚拟子网的网段范围，它是虚拟子网所在的 VPC 网段的子集。

|描述
|用户自定义的描述信息。

include::../../../../_custom/networks/description-convention.adoc[]

|带宽
|点击**高级设置**可以为子网设置带宽限制。

|域名服务器
|点击**高级设置**可以指定域名服务器的 IP 地址。
|===
--

. 点击**创建**。虚拟子网创建后将显示在虚拟子网列表中。
