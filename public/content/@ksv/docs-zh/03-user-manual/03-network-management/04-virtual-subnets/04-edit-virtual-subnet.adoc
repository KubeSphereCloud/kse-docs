本节介绍如何使用{ks_product_abbr_full_both}Web 控制台编辑虚拟子网。您可以通过编辑操作修改虚拟子网的名称和描述。

== 在虚拟子网列表页面编辑link:[id=9943bf8937674c5c97afb472577f27ec]

include::../../../../_custom/networks/virtual-subnet-steps.adoc[]

. 在虚拟子网列表页面，找到目标虚拟子网，点击右侧的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**编辑**。

. 在弹出的对话框中，修改虚拟子网的名称和描述，然后点击**确定**。

ifeval::["{ks_output_type}" == "pdf"]

== 在虚拟子网详情页面编辑link:[id=50a2d245338848279fedd260bc4f4cc1]

include::../../../../_custom/networks/virtual-subnet-steps.adoc[]

. 在虚拟子网列表页面，点击目标虚拟子网名称，进入虚拟子网详情页面。

. 在虚拟子网详情页面的左侧区域，点击左上角的**编辑**。

. 在弹出的对话框中修改虚拟子网的**名称**和**描述**，然后点击**确定**。
