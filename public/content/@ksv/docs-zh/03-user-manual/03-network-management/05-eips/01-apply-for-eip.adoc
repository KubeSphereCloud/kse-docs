本节介绍如何使用{ks_product_abbr_full_both}Web 控制台申请弹性 IP。

ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 操作步骤link:[id=a66686d07bd44f1d927b62be0d08121a]

include::../../../../_custom/networks/eip-steps.adoc[]

. 在弹性 IP 列表页面，点击右上角的**申请**。

. 在弹出的对话框中设置弹性 IP 的**名称**和**描述**。具体参数描述如下表所示。
+
--
[cols='1a,4a']
|===
|参数 |描述

|名称
|弹性 IP 的名称。

include::../../../../_custom/networks/naming-convention.adoc[]

|类型
|弹性 IP 的地址类型。 {ks_product_right}默认使用 IPv4 地址类型。

|描述
|用户自定义的描述信息。

include::../../../../_custom/networks/description-convention.adoc[]
|===
--

. 点击**确定**。弹性 IP 申请后将显示在弹性 IP 列表中。
