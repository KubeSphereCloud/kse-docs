== 前提条件link:[id=87d63641985941b0bcd48052b221ad4e]

* 请确保您有使用{ks_product_both}Web 控制台**物理资源**模块的权限。**物理资源**模块仅对系统 **admin** 用户可见。

* 请确保当前集群不存在已创建的子网，否则无法修改弹性网络。
