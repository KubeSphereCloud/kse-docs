您可以根据业务需求，将弹性 IP 和 VPC 网络或虚拟机绑定。绑定完成后，您便可以通过弹性 IP 访问对应的 VPC 网络或虚拟机。

本节介绍如何使用{ks_product_abbr_full_both}Web 控制台绑定弹性 IP。

== 操作步骤link:[id=f3ada0089408412db27f81bbffd3ac90]

include::../../../../_custom/networks/eip-steps.adoc[]

. 找到目标弹性 IP，点击右侧的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**绑定 VPC **或**绑定虚拟机**。

. 在弹出的对话框中，选择需要绑定的 VPC 或虚拟机，然后点击**确定**。
