本节介绍如何使用{ks_product_abbr_full_both}Web 控制台查看弹性 IP 列表。


ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 操作步骤link:[id=44a176bd491e44f5a0aae0235875559a]

include::../../../../_custom/networks/eip-steps.adoc[]

+
--
弹性 IP 列表展示当前项目中所有的弹性 IP。具体参数信息如下表所示。

[cols='1a,4a']
|===
|参数 |描述

|名称
|弹性 IP 的名称。

include::../../../../_custom/networks/naming-convention.adoc[]

|ID
|弹性 IP 在系统中的通用唯一标识符，由系统自动分配。

|状态
|弹性 IP 的状态。

|IP 地址
|弹性 IP 地址。

|IP 类型
|弹性 IP 的地址类型。 {ks_product_right}默认使用 IPv4 地址类型。

|绑定资源
|已和当前弹性 IP 绑定的资源。弹性 IP 可以和 VPC 或虚拟机绑定。

|创建者
|弹性 IP 的创建者。

|创建时间
|弹性 IP 的创建时间。
|===
--

. （可选）点击弹性 IP 列表上方的搜索框并设置搜索条件，可按名称或 ID 搜索弹性 IP。例如，使用**名称:ab** 搜索名称包含字符串 **ab** 的弹性 IP。

. （可选）在弹性 IP 列表右上角点击 image:/images/ksv/zh/icons/refresh-light.svg[refresh,15] 可刷新列表数据。
