== 删除单个弹性 IPlink:[id=1926356447cd4e619d8ad4a45890ccd5]

include::../../../../_custom/networks/eip-steps.adoc[]

. 找到目标弹性 IP，点击右侧的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**删除**。

. 点击**确定**即可删除目标弹性 IP。
+
--

// Warning
include::../../../../../_ks_components/admonitions/warning.adoc[]

弹性 IP 删除后无法恢复，请谨慎执行此操作。

include::../../../../../_ks_components/admonitions/admonEnd.adoc[]

// Note
include::../../../../../_ks_components/admonitions/note.adoc[]

请确保待删除的弹性 IP 未和任何资源关联，否则弹性 IP 将无法删除。

include::../../../../../_ks_components/admonitions/admonEnd.adoc[]
--

== 批量删除弹性 IPlink:[id=0810bb12bc6f407db16632f5a4bb46a9]

include::../../../../_custom/networks/eip-steps.adoc[]

. 在弹性 IP 列表页面，选择需要删除的弹性 IP 名称左侧的复选框，然后在列表上方点击**删除**。

. 在弹出的对话框中点击**确定**可批量删除多个弹性 IP。
+
