== 操作步骤link:[id=17dbeb28067045b796c4d2b59f6ed63d]

include::../../../../_custom/networks/physical-network-steps.adoc[]

. 在物理网络列表页面，点击右上角的**创建**。

. 在弹出的对话框中，配置物理网络的基本信息，然后点击**下一步**。具体参数描述如下表所示。
+
--
[cols='1a,3a']
|===
|参数 |描述

|名称
|物理网络的名称。

include::../../../../_custom/networks/physical-network-naming-convention.adoc[]

|类型

|网络的类型。支持以下网络类型：

include::../../../../_custom/networks/network-type-description.adoc[]

|描述
|用户自定义的描述信息。

include::../../../../_custom/networks/description-convention.adoc[]
|===
--

. 在**选择物理网卡**页签，选择您想要使用的物理网卡，然后点击**下一步**。
+
--
include::../../../../_custom/networks/nic-config-description.adoc[]
--

. 确认已选网卡，然后点击**确定**。网络创建完成后将显示在网络列表页面。
