== 操作步骤link:[id=695ad9f2ffd743fcbc2b4cf94b6c21f1]

include::../../../../_custom/networks/physical-network-steps.adoc[]

. 在物理网络列表页面，点击目标物理网络名称，进入物理网络详情页面。

. 在物理网络详情页面的左侧区域，查看物理网络的基本信息。
+
--
[cols='1a,4a']
|===
|参数 |描述

|ID
|物理网络在系统中的通用唯一标识符，由系统自动分配。

|状态
|物理网络当前的状态。

|类型
|物理网络的类型。

include::../../../../_custom/networks/network-type-description.adoc[]

|创建时间
|物理网络的创建时间。

|创建者
|创建物理网络时使用的用户名名称。

|===
--

. 在物理网络详情页面的右侧区域，点击**物理子网**页签，查看物理网络中的子网信息。具体参数描述如下表所示。
+
--
[cols='1a,4a']
|===
|参数 |描述

|名称
|物理子网的名称。

|ID
|物理子网在系统中的通用唯一标识符，由系统自动分配。

|物理 VLAN
|物理子网所在的 VLAN ID。

|网段
|物理子网的网段范围。

|网关
|物理子网的网关地址。

|可用 IP
|物理子网中可用的 IP 地址数量。

|已用 IP
|物理子网中已用的 IP 地址数量。
|===
--

. 在物理网络详情页面的右侧区域，点击**网卡**页签，查看物理网络的网卡信息。具体参数描述如下表所示。
+
--
[cols='1a,4a']
|===
|参数 |描述

|名称
|物理网卡的名称。

|Mac 地址
|网卡的物理地址，用于在网络中唯一标示一个网卡。


|状态
|物理网卡的状态。

|所属节点
|物理网卡所在的节点名称。

|IP 地址
|物理网卡所在的节点的 IP 地址。

|MTU
|物理网卡的最大传输单元。

|网卡速度
|物理网卡当前的传输速率。
|===
