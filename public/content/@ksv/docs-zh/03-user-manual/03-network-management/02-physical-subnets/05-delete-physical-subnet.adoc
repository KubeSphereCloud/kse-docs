本节介绍如何使用{ks_product_abbr_full_both}Web 控制台删除物理子网。

ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 前提条件link:[id=fb69bb1baa5f4c11b5bde0e05d66b8e7]

* 请确保您有使用{ks_product_both}Web 控制台物理资源模块的权限。物理资源模块仅对系统 admin 用户可见。
* 如果集群目前正在使用某个子网，则无法删除该子网。


ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 删除单个物理子网link:[id=a5d515f78db7470a8d4c0f2401e3120e]

include::../../../../_custom/networks/physical-subnet-steps.adoc[]

. 找到目标物理子网，点击右侧的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**删除**。

. 在弹出的对话框中点击**确定**可删除目标物理子网。
+
--
// Warning
include::../../../../../_ks_components/admonitions/warning.adoc[]

物理子网删除后无法恢复，请谨慎操作。

include::../../../../../_ks_components/admonitions/admonEnd.adoc[]
--

ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 批量删除物理子网link:[id=e66f1c49cba04c79a130a1e2ef42db34]

include::../../../../_custom/networks/physical-subnet-steps.adoc[]

. 在物理子网列表页面，选择需要删除的物理子网名称左侧的复选框，然后在列表上方点击**删除**。

. 在弹出的对话框中点击**确定**可批量删除多个物理子网。
+
--
// Warning
include::../../../../../_ks_components/admonitions/warning.adoc[]

物理子网删除后无法恢复，请谨慎操作。
