== 前提条件link:[id=8af32b1eabb54d208e21b1d37e763987]

* 请确保您有使用{ks_product_both}Web 控制台**物理资源**模块的权限。物理资源模块仅对系统 **admin** 用户可见。

* 请确认系统中有已创建好的物理网络。
