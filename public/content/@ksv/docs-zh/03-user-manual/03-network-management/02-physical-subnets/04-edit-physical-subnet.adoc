本节介绍如何使用{ks_product_abbr_full_both}Web 控制台编辑物理子网。您可以通过编辑操作修改物理子网的名称和描述。

ifeval::["{ks_output_type}" == "pdf"]

== 在物理子网列表页面编辑link:[id=db60c754e79e46f3923437e972b730e5]

include::../../../../_custom/networks/physical-subnet-steps.adoc[]

. 找到目标物理子网，点击右侧的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**编辑**。

. 在弹出的对话框中修改物理子网的**名称**和**描述**，然后点击**确定**。

ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 在物理子网详情页面编辑link:[id=787fcfe4cd0c4529b28769282f362d74]

include::../../../../_custom/networks/physical-subnet-steps.adoc[]

. 在物理子网列表页面，点击目标物理子网名称，进入物理子网详情页面。

. 在物理子网详情页面的左侧区域，点击左上角的**编辑**。

. 在弹出的对话框中修改物理子网的**名称**和**描述**，然后点击**确定**。
