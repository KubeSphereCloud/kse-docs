== 操作步骤link:[id=bd32502168064f6b91947e4bb2b070e7]

include::../../../../_custom/networks/vpc-steps.adoc[]

. 在 VPC 列表页面，点击右上角的**创建**。

. 在弹出的对话框中配置 VPC 相关参数。具体参数描述请参见下表。
+
--
[cols='1a,4a']
|===
|参数 |描述

|名称
|VPC 的名称。

include::../../../../_custom/networks/naming-convention.adoc[]

|网段范围
|VPC 的网段范围。VPC 创建后，网段范围不可修改。

|默认虚拟子网
|VPC 网络所在的默认虚拟子网。

|描述
|用户自定义的描述信息。

include::../../../../_custom/networks/description-convention.adoc[]
|===
--

. 点击**创建**。VPC 创建后将显示在 VPC 列表中。
