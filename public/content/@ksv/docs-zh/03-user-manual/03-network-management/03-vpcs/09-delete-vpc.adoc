== 删除单个 VPClink:[id=bfd792de828245f3a704c15d508affbe]

include::../../../../_custom/networks/vpc-steps.adoc[]

. 在 VPC 列表页面，点击目标 VPC 名称，进入 VPC 详情页面。

. 在 VPC 详情页面的左侧区域，点击右上角的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**删除**。

. 在弹出的对话框中点击**确定**可删除目标 VPC。
+
--
// Warning
include::../../../../../_ks_components/admonitions/warning.adoc[]

VPC 删除后无法恢复，创建在 VPC 中的子网也将一并删除，请谨慎执行此操作。

include::../../../../../_ks_components/admonitions/admonEnd.adoc[]

// Note
include::../../../../../_ks_components/admonitions/note.adoc[]


请确保待删除的 VPC 没有绑定任何的弹性 IP 地址，创建在 VPC 中的子网也未和任何资源关联，否则 VPC 将无法删除。

include::../../../../../_ks_components/admonitions/admonEnd.adoc[]
--

== 批量删除 VPClink:[id=59fc1f4c1ac342368d3ddfcd7c028de8]

include::../../../../_custom/networks/vpc-steps.adoc[]

. 在 VPC 列表页面，选择需要删除的 VPC 名称左侧的复选框，然后在列表上方点击**删除**。

. 在弹出的对话框中点击**确定**可批量删除多个 VPC。
+
