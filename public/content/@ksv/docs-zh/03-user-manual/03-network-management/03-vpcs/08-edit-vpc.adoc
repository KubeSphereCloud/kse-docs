本节介绍如何使用{ks_product_abbr_full_both}Web 控制台编辑 VPC 网络。您可以通过编辑操作修改 VPC 的名称和描述。

include::../../../../_custom/networks/vpc-manage-prerequisites.adoc[]

ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 在 VPC 列表页面编辑link:[id=1cf8507a4bfc433bbe0638b74e18ab6e]

include::../../../../_custom/networks/vpc-steps.adoc[]

. 找到目标 VPC，点击右侧的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**编辑**。

. 在弹出的对话框中修改 VPC 的**名称**和**描述**，然后点击**确定**。

ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 在 VPC 详情页面编辑link:[id=357c332f628144e0b4a416068fdace30]

include::../../../../_custom/networks/vpc-steps.adoc[]

. 在 VPC 列表页面，点击目标 VPC 名称，进入 VPC 详情页面。

. 在 VPC 详情页面的左侧区域，点击左上角的**编辑**。

. 在弹出的对话框中修改 VPC 的**名称**和**描述**，然后点击**确定**。
