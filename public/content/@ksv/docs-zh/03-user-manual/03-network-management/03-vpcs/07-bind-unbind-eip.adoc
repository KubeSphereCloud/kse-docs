== 在 VPC 列表页面绑定/解绑link:[id=dd1fe444f5274c46836710eb84d1a837]

include::../../../../_custom/networks/vpc-steps.adoc[]

. 找到目标 VPC，点击右侧的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**绑定弹性 IP** 或**解绑弹性 IP**。

== 在 VPC 详情页面绑定/解绑link:[id=4bdc4d8d75ce4acf96db753072e657f5]

include::../../../../_custom/networks/vpc-steps.adoc[]

. 在 VPC 列表页面，点击目标 VPC 名称，进入 VPC 详情页面。

. 在 VPC 详情页面的左侧区域，点击右上角的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**绑定弹性 IP** 或**解绑弹性 IP**。
+
