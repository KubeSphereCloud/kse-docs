[discrete]

== 操作步骤link:[id=ffe9257344d746b59e0c017800685cc4]

include::../../../../_custom/networks/vpc-steps.adoc[]

+
--
VPC 列表展示当前项目中的所有 VPC，包括默认 VPC 和用户自建 VPC。具体参数信息如下表所示。

[cols="1a,4a"]
|===
|参数 |描述

|名称
|VPC 的名称。

include::../../../../_custom/networks/naming-convention.adoc[]

|ID
|VPC 在系统中的通用唯一标识符，由系统自动分配。

|状态
|VPC 的状态。

|IP 地址范围
|VPC 的网段范围。

|绑定资源
|已和 VPC 绑定的资源。

|创建者
|VPC 的创建者。

|创建时间
|VPC 的创建时间。
|===
--

. （可选）点击 VPC 列表上方的搜索框并设置搜索条件，可按名称和 ID 搜索 VPC。例如，使用**名称:ab** 搜索名称包含字符串 **ab** 的 VPC。

. （可选）在 VPC 列表右上角点击 image:/images/ksv/zh/icons/refresh-light.svg[refresh,15] 可刷新列表数据。
