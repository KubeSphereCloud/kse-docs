本节介绍如何使用{ks_product_abbr_full_both}Web 控制台启用和停用定时任务。

ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 前提条件link:[id=bafa731fad3c47868b11a185293450b1]

系统 **admin** 用户可以启停集群中所有已创建的定时任务，普通项目管理员和操作员可以启停自身创建的定时任务，项目观察员没有启停权限。


ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 在定时任务列表页面启用或停用link:[id=ea81c0be70a44217838d3c25e1750687]

include::../../../../_custom/timer/timer-task-steps.adoc[]

. 找到目标定时任务，点击右侧的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**启用**或**停用**。

. 在弹出的对话框中，点击**确定**即可执行对应操作。

ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 在定时任务详情页面启用或停用link:[id=9de7bfa307d8436f954d35f2e4925d23]

include::../../../../_custom/timer/timer-task-steps.adoc[]

. 在定时任务列表页面，点击目标定时任务名称，进入定时任务详情页面。

. 在定时任务详情页面，点击左上角的**启用**或**停用**。

. 在弹出的对话框中，点击**确定**即可执行对应操作。
