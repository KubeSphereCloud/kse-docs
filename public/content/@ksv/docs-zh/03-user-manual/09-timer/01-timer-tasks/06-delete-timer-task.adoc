ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 前提条件link:[id=5b72b316a560454ab8449c7f41f6fd19]

系统 **admin** 用户可以删除集群中所有已创建的定时任务，普通项目管理员和操作员可以删除自身创建的定时任务，项目观察员没有删除权限。


ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 删除单个定时任务link:[id=a35b7d14eeeb4556994998fb7eba8e2e]

include::../../../../_custom/timer/timer-task-steps.adoc[]

. 在定时任务列表页面，点击目标定时任务，进入定时任务详情页面。

. 在定时任务详情页面的左侧区域，点击 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**删除**。

. 在弹出的对话框中点击**确定**可删除当前定时任务。
+
--
// Warning
include::../../../../../_ks_components/admonitions/warning.adoc[]

定时任务删除后无法恢复，请谨慎执行此操作。

include::../../../../../_ks_components/admonitions/admonEnd.adoc[]
--

ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 批量删除定时任务link:[id=0c5682eda71f470a86a9d7b7d77ea07e]

include::../../../../_custom/timer/timer-task-steps.adoc[]

. 在定时任务列表页面，选择需要删除的定时任务名称左侧的复选框，然后在列表上方点击**删除**。

. 在弹出的对话框中点击**确定**可批量删除多个定时任务。
+
--
// Warning
include::../../../../../_ks_components/admonitions/warning.adoc[]

定时任务删除后无法恢复，请谨慎执行此操作。

include::../../../../../_ks_components/admonitions/admonEnd.adoc[]
