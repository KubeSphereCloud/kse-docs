== 操作步骤link:[id=76e70ce343ad4b68a3f8413aebcbc4cb]

include::../../../../_custom/timer/timer-policy-steps.adoc[]

. 在定时策略列表页面点击**创建**。

. 在弹出的对话框中为定时策略设置以下参数：
+
--
[cols='1a,4a']
|===
|参数 |描述

|名称
|用户自定义的定时策略名称。
include::../../../../_custom/networks/naming-convention.adoc[]

|选择项目
|选择定时策略所属的项目。

|周期
|定时任务的执行周期。目前支持每天、每 N 天、每 N 小时、每周或每月执行。

|时间
|定时任务执行的具体时间点。
|===
--

. 点击**确定**。定时策略创建后将显示在定时策略列表中。
