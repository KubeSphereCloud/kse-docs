您可以使用{ks_product_abbr_full_both}Web 控制台创建、查看、使用或释放不同类型的虚拟机资源。资源类型主要分为物理资源和虚拟资源。在您使用某个资源时，{ks_product_right}提供运维工具，帮助您实时监控各类资源的使用情况。

本章介绍了{ks_product_both}相关的物理资源和虚拟资源，以及如何使用运维工具实时监控资源。
