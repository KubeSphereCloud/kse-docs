本节介绍如何使用{ks_product_abbr_full_both}Web 控制台添加安全组规则。

{ks_product_right}提供预设的安全组规则，您可以根据业务需求选择添加合适的规则。系统 **admin** 用户可以为集群中所有已创建的安全组添加规则，普通项目管理员和操作员可以为自己创建的安全组添加规则，观察员仅能查看所属项目中的安全组规则。

ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 前提条件link:[id=81f0a12a9fac4e5dbe9e56fbcb8d62e4]

请确认您已创建安全组，并且已经规划好安全组关联资源需要允许或禁止哪些出入站流量。

ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 操作步骤link:[id=df7053d52d0a45bf860757e417636513]

include::../../../_custom/security-groups/security-group-steps.adoc[]

. 在安全组列表页面，点击目标安全组名称，进入安全组详情页面。

. 在安全组详情页面的右侧区域，点击**添加规则**。

. 在弹出的对话框中，配置安全组规则相关参数。您也可以在**预设规则**区域选择添加系统内置的安全组规则。
+
--
include::../../../_custom/security-groups/security-group-rule-parameters.adoc[]
--

. 点击**确定**。安全组规则添加后将显示在安全组规则列表中。
