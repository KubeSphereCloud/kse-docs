本节介绍如何使用{ks_product_abbr_full_both}Web 控制台删除安全组规则。

系统 **admin** 用户可以删除集群中所有已创建的安全组规则，普通项目管理员和操作员可以删除自身创建的安全组规则，项目观察员没有删除权限。

== 操作步骤link:[id=fce2078b5bda4d02a4566aa60e6e0e62]

include::../../../_custom/security-groups/security-group-steps.adoc[]

. 在安全组列表页面，点击目标安全组名称，进入安全组详情页面。

. 在**上行规则**或**下行规则**页签，找到目标安全组规则，点击右侧的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**删除**。

. 点击**确定**即可删除目标安全组规则。
