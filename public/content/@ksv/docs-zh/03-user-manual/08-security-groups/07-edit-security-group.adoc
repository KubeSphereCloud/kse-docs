本节介绍如何使用{ks_product_abbr_full_both}Web 控制台编辑安全组。您可以通过编辑操作修改安全组的名称和描述。

ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 前提条件link:[id=1b4e6a8bc3b24870aed67d7211c6b644]

系统 **admin** 用户可以编辑集群中所有已创建的安全组，普通项目管理员和操作员可以编辑自身创建的安全组，项目观察员没有编辑权限。


ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 在安全组列表页面编辑link:[id=b9c9490c3be74fbd8b12e491740014e0]

include::../../../_custom/security-groups/security-group-steps.adoc[]

. 找到目标安全组，点击右侧的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**编辑**。

. 在弹出的对话框中修改安全组的**名称**和**描述**，然后点击**更新**。

ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 在安全组详情页面编辑link:[id=684495a9d3b9435287725fef39e436ec]

include::../../../_custom/security-groups/security-group-steps.adoc[]

. 在安全组列表页面，点击目标安全组名称，进入安全组详情页面。

. 在安全组详情页面的左侧区域，点击左上角的**编辑**。

. 在弹出的对话框中修改安全组的**名称**和**描述**，然后点击**更新**。
