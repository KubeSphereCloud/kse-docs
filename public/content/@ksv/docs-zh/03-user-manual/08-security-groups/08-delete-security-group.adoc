[discrete]
endif::[]
== 前提条件link:[id=8e65ded3dd7141e89fd1114556110eb0]

系统 **admin** 用户可以删除集群中所有已创建的安全组，普通项目管理员和操作员可以删除自身创建的安全组，项目观察员没有删除权限。


ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 删除单个安全组link:[id=f7a3c43c384d45eca3882f07ec785b81]

include::../../../_custom/security-groups/security-group-steps.adoc[]

. 找到目标安全组，点击右侧的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**删除**。

. 在弹出的对话框中点击**确定**可删除目标安全组。
+
--
// Warning
include::../../../../_ks_components/admonitions/warning.adoc[]

安全组删除后无法恢复，请谨慎操作。

include::../../../../_ks_components/admonitions/admonEnd.adoc[]
--

ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 批量删除安全组link:[id=e464a5458ac8407baae1021748cee4c0]

include::../../../_custom/security-groups/security-group-steps.adoc[]

. 在安全组列表页面，选择需要删除的安全组名称左侧的复选框，然后在列表上方点击**删除**。

. 在弹出的对话框中点击**确定**可批量删除多个安全组。
+
--
// Warning
include::../../../../_ks_components/admonitions/warning.adoc[]

安全组删除后无法恢复，请谨慎操作。
