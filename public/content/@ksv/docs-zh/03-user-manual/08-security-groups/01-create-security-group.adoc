== 注意事项link:[id=348ee91007ca4fc78d3f58c1a53f138b]

在初始状态下，新创建的安全组不包含任何规则。在未添加任何安全组规则之前，系统默认允许所有出站流量，拒绝所有入站流量。

== 操作步骤link:[id=194d8ac52dbc49fb873d75708e08b939]

include::../../../_custom/security-groups/security-group-steps.adoc[]

. 在安全组列表页面，点击右上角的**创建**。

. 在弹出的对话框中，配置安全组相关参数。具体参数描述请参见下表。
+
--
[cols='1a,4a']
|===
|参数 |描述

|名称
|安全组的名称。

include::../../../_custom/security-groups/naming-convention.adoc[]

|描述
|用户自定义的描述信息。

include::../../../_custom/networks/description-convention.adoc[]
|===
--

. 点击**创建**。安全组创建后将显示在安全组列表中。

ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 后续步骤link:[id=58f7e245cc8c409c97214a2f2225211c]

创建安全组后，您需要继续为安全组添加规则，允许或禁止安全组内的相关资源对公网或私网的访问。具体操作，请参阅 link:../02-add-security-group-rule/[添加安全组规则]。
