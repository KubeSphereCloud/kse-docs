本节介绍如何使用{ks_product_abbr_full_both}Web 控制台删除告警策略。

ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 前提条件link:[id=05c6ddd871b24ee5b60a56e5541926a2]

系统 **admin** 用户可以删除集群中所有已创建的告警策略，普通项目管理员和操作员可以删除自身创建的告警策略，项目观察员没有删除权限。


ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 删除单个告警策略link:[id=43d6223d48e3496c8ce7a5d846a365d3]

include::../../../_custom/alerts/alert-steps.adoc[]

. 在**告警策略**页签，找到目标告警策略，点击右侧的 image:/images/ksv/zh/icons/more.svg[operation-menu,13]，然后在下拉列表中选择**删除**。

. 在弹出的对话框中点击**确定**可删除目标告警策略。
+
--
// Warning
include::../../../../_ks_components/admonitions/warning.adoc[]

告警策略删除后无法恢复，请谨慎操作。

include::../../../../_ks_components/admonitions/admonEnd.adoc[]
--

ifeval::["{ks_output_type}" == "pdf"]
[discrete]
endif::[]
== 批量删除告警策略link:[id=9f60f396d3ec45d0ad180ac640e11bf5]

include::../../../_custom/alerts/alert-steps.adoc[]

. 在**告警策略**页签，选择需要删除的告警策略名称左侧的复选框，然后在列表上方点击**删除**。

. 在弹出的对话框中点击**确定**可批量删除多个告警策略。
+
--
// Warning
include::../../../../_ks_components/admonitions/warning.adoc[]

告警策略删除后无法恢复，请谨慎操作。
