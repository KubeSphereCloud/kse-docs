. 登录集群中的其他管理节点
. 查看操作系统信息。
+
[source,shell]
----
lsb_release -a
----
. 查看当前集群中 Zookeeper 版本信息。
+
[source,shell]
----
echo stat | nc xx.xx.xx.xx 2181
----
+
include::../../../../_components/note.adoc[]
* xx.xx.xx.xx 为当前管理节点的 IP 地址，需根据实际情况进行修改。

* 返回结果中 ``Zookeeper version`` 对应的即为版本号。
include::../../../../_components/table_footer.adoc[]

. 从以下路径获取相应的各项安装包。安装包信息需与原集群中的版本信息保持一致。
+

|===
