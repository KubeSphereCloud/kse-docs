---
title: "安装 Supervisor"
weight: 7
---

. 查看系统是否安装了 Supervisor 服务。
+
[source,shell]
----
which supervisorctl
----
+
若放回如下内容怎表示 Supervisor 服务已安装。
+
----
/usr/bin/supervisorctl
----

. 若没有安装，根据提示信息进行安装。
* Ubuntu 16.04.5 操作系统
+
[source,shell]
----
apt install -y supervisor=3.3.5-1
----
* 其他 Ubuntu 版本操作系统
+
[source,shell]
----
apt install -y supervisor
----

. 创建目录。
+
[source,shell]
----
mkdir -p /qingstor/bin /qingstor/etc/supervisord.d
chmod 0755 /qingstor/bin /qingstor/etc/supervisord.d
----

. 从其他管理节点，拷贝相应的文件和目录。
+
[source,shell]
----
scp -p <xx.xx.xx.xx>:/qingstor/bin/qs_pidproxy /qingstor/bin/qs_pidproxy
scp -p <xx.xx.xx.xx>:/usr/bin/_supervisorctl /usr/bin/_supervisorctl
scp -p <xx.xx.xx.xx>:/usr/bin/supervisorctl /usr/bin/supervisorctl
scp -p <xx.xx.xx.xx>:/etc/supervisor/supervisord.conf /etc/supervisor/supervisord.conf
scp -pr <xx.xx.xx.xx>:/qingstor/bin/supervisorctl_hooks /qingstor/bin/supervisorctl_hooks
----
+
include::../../../../../_components/note.adoc[]
<xx.xx.xx.xx> 为当前环境其他健康数据库节点的 IP 地址，需根据实际情况进行修改。
include::../../../../../_components/table_footer.adoc[]

. 启动 Supervisor 服务，并将该命令添加至 ``/etc/rc.local`` 文件中。
+
[source,shell]
----
/usr/bin/supervisord
----

. 检查 Supervisor 服务状态。
+
[source,shell]
----
systemctl status supervisor
----
