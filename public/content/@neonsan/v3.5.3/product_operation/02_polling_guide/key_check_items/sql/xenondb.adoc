== 适用范围

仅适用于 Xenondb 数据库。

== 操作步骤
. 登录 Xenondb 集群的任一节点。
. 执行如下命令，查询 xenondb 集群状态信息。
+
[,shell]
----
/opt/xenondb/bin/xenoncli cluster status
----
+

+
