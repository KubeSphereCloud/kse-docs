---
title: "安装配置 kdump"
weight: 2
---

. 逐个登录集群内的各节点。
. 执行如下命令，安装kernel crash dump。
+
[,shell]
----
sudo apt install linux-crashdump
----
+ 
出现以下弹窗，选择 ``Yes`` 即可。
+
image::/images/neonsan/ops_guide_2.png[,70%]
+
image::/images/neonsan/ops_guide_3.png[,70%]

. 执行如下命令，重新启动系统以激活crashkernel= boot参数以及开启kdump-tools配置。
+
[,shell]
----
reboot
----
. 待系统重启完成，执行如下命令查看当前节点的kdump状态。
+
[,shell]
----
kdump-config show
----
+
回显：
+

----
