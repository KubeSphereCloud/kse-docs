== 登录可视化部署页面

QingStor FirstBox 创建并拉起成功后，部署人员可根据以下信息，通过浏览器登录至 NeonSAN 可视化部署页面完成产品的配置与部署。

[,shell]
----
链接：http://xx.xx.xx.xx:1081
默认用户名：admin
默认密码：zhu88jie
----

其中，xx.xx.xx.xx 为link:../../prepare_node/create_fb[创建 FirstBox] 时指定的节点 IP 地址，需根据实际情况进行修改。
