title: "SSH 免密访问"
weight: 2
---

. 分别登录本机和远程服务器的终端，查看是否有 ~/.ssh/ 目录。
+
[source,shell]
----
cd .ssh  
---- 
+

include::../../../../../_components/note.adoc[]
* 对于 Windows 系统的本机需提前安装 Git Bash/WSL 或远程登录连接工具。
* 若无该目录，则需生成密钥，可执行如下命令生成密钥，所有参数均可为空，即一直回车确认即可:
``ssh-keygen``
* 若已存在 ～/.ssh/ 目录可直接进行步骤 2。

include::../../../../../_components/table_footer.adoc[]

. 执行如下命令，查看生成的公钥和私钥。
+
[source,shell]
----
ls ~/.ssh/
----
+

include::../../../../../_components/note.adoc[]
* 带有 .pub 后缀的为是公钥，没有 .pub 后缀的为私钥。
* 将各节点的私钥下载并保存至本机。

include::../../../../../_components/table_footer.adoc[]

. 查看 ssh 公钥，并复制。
+
[source,shell]
----
cat id_rsa.pub
----
. 登录远程服务器终端，分别执行如下命令，将步骤3中复制的本机的 ssh 公钥添加到服务器的 authorized_keys 文件中即可。
+
[source,shell]
----
cd ~/.ssh/
----
+
[source,shell]
----
vi authorized_keys
----
