== 名词解释

[options="header",cols="1,3"]
|===
|名词	|意义
|存储池
|为了资源合理利用以及管理的方便性，可将集群的存储节点划分为多个存储池，每个存储池最多支持 24 个存储节点。 目前 NeonSAN 支持 3 ~ 4096 个存储节点（Store 节点）

|neonsan-center	
|NeonSAN 管理控制服务

|neonsan-store	
|NeonSAN 存储服务

|neonsan-monitor	
|NeonSAN 告警服务

|Common 服务	
|数据库、Zookeeper 和 Portal 等

|NTP	
|Network Time Protocol，一种使计算机时间同步化的协议
|===

更多专业术语名词解释，可参考link:https://box.yunify.com/s/Xw99az7CSfTmFZP[《NeonSAN 术语手册》]。
