== 功能描述

Portal 管理平台具备 License 管理功能，主要包括：

* 查看 {product_name} License 许可证书的基本信息。
* 激活 License 。

== 操作步骤

. 登录 link:../../login[Portal 管理界面]。

. 在**总览**页面，点击用户名，选择**系统设置**，即可进入系统设置页面。
+
image::/images/neonsan/portal_system_set.png[]

. 在系统设置页面，选择 **License 管理**页签，进入 License 管理页面。在该页面可查看 License 基本信息，包括``状态``，``有效期``，``授权容量``，``集群 ID``，``集群容量``，其中``集群容量``为集群的实际容量。
+
image::/images/neonsan/portal_license_1.png[]

. 点击 License 管理页面内的**激活 License**，弹出**激活 License**窗口，根据提示输入相应的 **License 激活码**点击**确定**即可。
+
image::/images/neonsan/portal_license_2.png[,50%]
+
