== 功能描述

Portal 管理平台允许**超级管理员**角色的用户对平台内已存在的其他用户进行修改和编辑。

== 操作步骤

. 使用超级管理员账户，登录 link:../../../login[Portal 管理界面]。

. 在**总览**页面，点击用户名，选择**系统设置**，即可进入系统设置页面。
+
image::/images/neonsan/portal_system_set.png[]

. 在系统设置页面，选择**管理员**，进入用户管理页面。点击待编辑的用户所在行右侧的更多操作，选择**编辑**。
+
image::/images/neonsan/portal_edit_user_1.png[]

. 在弹出的**编辑**窗口中，支持修改``邮箱``、``用户名``和``角色``，修改完信息，点击**保存**即可。
+
image::/images/neonsan/portal_edit_user_2.png[,50%]
