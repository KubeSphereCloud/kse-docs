== 功能描述

Portal 管理平台允许**超级管理员**角色的用户对平台内已存在的其他用户的密码进行重置，**管理员**角色仅有对自身账户进行重置密码的权限。

== 操作步骤

. 登录 link:../../../login[Portal 管理界面]。

. 在**总览**页面，点击用户名，选择**系统设置**，即可进入系统设置页面。
+
image::/images/neonsan/portal_system_set.png[]

. 在系统设置页面，选择**管理员**，进入用户管理页面。点击待重置密码的用户所在行右侧的更多操作，选择**重置密码**。

. 在弹出的**重置密码**窗口中，输入新的密码，点击**重置密码**即可。
+
image::/images/neonsan/portal_reset_code_user_2.png[,50%]
