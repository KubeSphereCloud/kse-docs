---
title: 用户管理
weight: 3
---

{product_name} Portal 管理平台的用户角色包括超级管理员、管理员和观察者。不同角色的用户其平台使用权限不同。其中，针对用户管理功能，仅超级管理员支持新增、编辑、禁用以及删除用户。管理员角色用户仅支持对自己的账户进行密码重置。观察员角色用户不具备用户管理功能权限。

* 超级管理员为 {product_name} 安装部署时内置的初始用户，即 admin 用户。用于内部运维人员登录 Portal 平台和 {product_name} 集群环境。
* 管理员具备 Portal 平台内所有操作权限。
* 观察者登录 Portal 平台内后仅支持对集群状态和资源进行查看。
