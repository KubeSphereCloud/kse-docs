[cols="1,1,3a",options="header"]
|===
|英文词汇|中文/缩略语|解释说明

|license 
|许可证书
|{product_name} 的 license 证书分为试用版和商业版，证书用于记录 license 的ID、授权容量、license 申请人信息、集群配置相关描述、license 密钥等信息。license 授权证书会以邮件的形式发送给指定用户。

|license ID
|许可证 ID
|license 申请成功后，申请人可通过相应的邮件查看 license ID，全局唯一。

|cluster ID
|集群 ID
|每一个 {product_name} 集群的专属 ID 标识，全局唯一，进行 license 申请时需使用该 ID。

|capacity
|授权容量
|license 授权证书中针对当前 {product_name} 集群给予的最高可使用的存储空间，当存储空间使用量超过该值后，集群将无法创建新卷。

|status
|license 状态
|license 许可证书的状态包括：active 激活或 inactive 未激活。
|===

*应用指南* 

{product_name} 的 License 申请和使用流程详细内容可参考手册：《{product_name} 分布式存储系统 License 使用指南 》。
