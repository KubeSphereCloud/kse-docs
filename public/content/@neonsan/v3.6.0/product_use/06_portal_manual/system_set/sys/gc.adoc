== 功能描述

Portal 管理平台支持一键进行集群磁盘空间的清理，即对当前集群内的所有存储节点空间进行回收。

== 操作步骤

. 登录 link:../../../login[Portal 管理界面]。

. 在**总览**页面，点击用户名，选择**系统设置**，即可进入系统设置页面。
+
image::/images/neonsan/portal_system_set.png[]

. 在系统设置页面，选择**集群设置**，点击**清理磁盘空间**所在行右侧的**垃圾回收**按钮即可。
+
image::/images/neonsan/portal_gc_1.png[]

. 由于清理磁盘空间为异步任务，可在左侧导航栏内选择**存储服务** > **后台任务**，在后台任务列表中查看磁盘空间清理的状态和进度。
+
image::/images/neonsan/portal_gc_2.png[]
