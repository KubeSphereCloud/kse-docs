== 功能描述

Portal 管理平台为保护账户信息安全，支持各角色用户进行``二次认证``以增强账户的安全设置。

== 操作步骤

. 登录 link:../../../login[Portal 管理界面]。

. 在**总览**页面，点击用户名，选择**用户中心**，即可进入用户中心页面。

+
image::/images/neonsan/portal_2nd_certificate_1.png[]

. 在用户中心页面，选择**安全设置** > **开启认证**，在弹出的**开启二次认证**窗口中，根据提示信息逐步获取相应的**动态验证码**。
+
image::/images/neonsan/portal_2nd_certificate_2.png[,50%]

. 获取验证码后，在开启**开启二次认证**窗口中，填写**动态验证码**并点击**提交**即可。
+
image::/images/neonsan/portal_2nd_certificate_3.png[,50%]
