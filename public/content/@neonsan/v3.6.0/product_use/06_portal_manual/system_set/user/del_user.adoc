== 功能描述

Portal 管理平台允许**超级管理员**角色的用户对平台内已存在的其他用户进行删除操作。

== 操作步骤

. 使用超级管理员账户，登录 link:../../../login[Portal 管理界面]。

. 在**总览**页面，点击用户名，选择**系统设置**，即可进入系统设置页面。
+
image::/images/neonsan/portal_system_set.png[]

. 在系统设置页面，选择**管理员**，进入用户管理页面。点击待删除的用户所在行右侧的更多操作，选择**删除管理员**。
+
image::/images/neonsan/portal_del_user_1.png[]

. 在弹出的**删除管理员**窗口中，若确定无误，点击**删除**即可。
+
image::/images/neonsan/portal_del_user_2.png[,50%]
