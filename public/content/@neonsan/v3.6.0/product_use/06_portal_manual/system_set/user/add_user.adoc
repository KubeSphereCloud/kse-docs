== 功能描述

Portal 管理平台允许**超级管理员**角色的用户在平台内新增用户，添加成功的用户均可正常登录该管理平台。

== 操作步骤

. 使用超级管理员账户，登录 link:../../../login[Portal 管理界面]。

. 在**总览**页面，点击用户名，选择**系统设置**，即可进入系统设置页面。
+
image::/images/neonsan/portal_system_set.png[]

. 在系统设置页面，选择**管理员**，进入用户管理页面。在当前页面可查看到目前平台内的所有用户的基本信息，包括``用户名``，``邮箱``，``状态``，``角色``，是否开启``二次认证``以及``创建时间``。
+
image::/images/neonsan/portal_add_user_1.png[]

. 点击**添加管理员**按钮，弹出管理员设置页面。根据提示设置相应的邮箱、用户名、密码以及角色选择，点击**创建**即可。目前支持设定``管理者``或``观察者``两种角色身份。其中，观察者角色对集群状况仅有查看的权限。
+
image::/images/neonsan/portal_add_user_2.png[,50%]

. 页面跳转至用户管理界面，成功添加的用户，其信息和角色身份显示在列。
+
image::/images/neonsan/portal_add_user_3.png[]
