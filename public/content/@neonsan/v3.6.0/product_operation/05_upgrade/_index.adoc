本文档旨在介绍 NeonSAN的 升级脚本，以及如何使用该升级脚本。用户严格按照操作步骤执行，即可完成对 NeonSAN 分布式存储系统的升级。



== 适用版本

该文档适用于升级脚本 upgrade_v1.20，将 NeonSAN v3.0.0 系列版本升级至 v3.5.x 版本，或其他低版本 NeonSAN 升级至较高版本。
