---
title: "配置 NTP 服务"
weight: 6
---

. 登录新部署的管理节点，执行如下命令，安装 NTP。
+
[source,shell]
----
apt-get install ntp
----

. 修改配置文件 ``/etc/ntp.conf``， 配置 NTP。
+
[source,shell]
----
vi /etc/ntp.conf
----
+
----
...
pool 0.ubuntu.pool.ntp.org iburst
pool 1.ubuntu.pool.ntp.org iburst
pool 2.ubuntu.pool.ntp.org iburst
pool 3.ubuntu.pool.ntp.org iburst

// Use Ubuntu's ntp server as a fallback.
pool ntp.ubuntu.com
...
----
+
若以 http://ntp.org.cn/pool.php 上提供的 NTP 服务器为例。用域名进行配置时，需检查系统是否配置过了域名解析，如 ``ping cn.ntp.org.cn``，若网络不通，需将域名解析先配置正确，或者直接使用服务器 IP。
+
[source,shell]
----
vi /etc/ntp.conf
----
+
----
...
server cn.ntp.org.cn
...  
----

+
include::../../../../../_components/note.adoc[]
* 将 NTP 服务器修改为用户指定的 NTP 服务器，一般指定两个，防止 NTP 服务器故障对集群的影响。
* 如果用户没有自己的 NTP 服务器，建议选择集群中某两个节点作为 NTP 服务器。
include::../../../../../_components/table_footer.adoc[]

. 重启 NTP 服务。
+
[source,shell]
----
sudo /etc/init.d/ntp restart
----
. 查看 NTP 服务状态。返回结果中 ``Active`` 为 ``active (running)`` 即可。
+
[source,shell]
----
service ntp status
----

. 查看 NTP 服务同步状态。
+
[source,shell]
----
ntpq -np
----
