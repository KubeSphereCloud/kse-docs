---
title: "部署 NeonSAN"
weight: 6
---

. 登录 FirstBox。
+
[source,shell]
----
ssh xx.xx.xx.xx -p 6023
----

. 部署 Common 服务。
+
[,shell]
----
installer product-deploy common -I /root/ip_list
----

. 执行如下，命令查看部署任务状态。
+
[,shell]
----
installer task-list 
----
+
include::../../../../../../_components/note.adoc[]
* 返回结果中，关注 ``status`` 字段，需为 ``successful``，即证明服务部署成功，方可进行后续部署。
* 部署 Common 服务成功后，若需要更新 kernel，主机会自动重启，请等待主机重启后再继续部署下一产品，若超过 2 分钟没重启，则说明无需重启。
include::../../../../../../_components/table_footer.adoc[]

. 若部署失败，执行如下命令，获取当前部署日志，查找失败原因并解决。
+
[source,shell]
----
installer deploy-log-get -t <TASK_ID>
----
+
include::../../../../../../_components/note.adoc[]
* <TASK_ID> 需根据实际情况进行修改，可执行 ``installer task-list`` 命令进行查看。
*  若文件配置有问题，可对以下文件夹中的配置文件进行修改： +
``cd /qingstor/install/deployment/environment/group_vars/all/`` +
``cd /qingstor/install/deployment/environment/group_vars/neonsan/``
include::../../../../../../_components/table_footer.adoc[]

. 问题解决后，需执行下列命令对任务进行重试。
+
[source,shell]
----
installer task-operate <TASK_ID> --action retry
----
+
include::../../../../../../_components/note.adoc[]
* 重试再次查看任务状态，直至返回结果中 ``status`` 字段为 ``successful`` 。
* <TASK_ID> 需根据实际情况进行修改，可执行 ``installer task-list`` 命令进行查看。
include::../../../../../../_components/table_footer.adoc[]

. 执行如下命令部署 NeonSAN 服务。
+
[source,shell]
----
installer product-deploy neonsan -I /root/ip_list
----

. 执行如下，命令查看部署任务状态。
+
[,shell]
----
installer task-list 
----
+
include::../../../../../../_components/note.adoc[]
返回结果中，关注 ``status`` 字段，需为 ``successful``，即证明服务部署成功，方可进行后续部署。
include::../../../../../../_components/table_footer.adoc[]

. 部署完成后，执行如下命令确认服务是否正常。
+
[source,shell]
----
neonsan list_store && neonsan list_ssd && neonsan list_port && neonsan list_pool && neonsan list_parameter && neonsan list_rg -detail
----
+
include::../../../../../../_components/note.adoc[]
* 命令执行成功，无错误提示，即部署成功。
* 若任一命令执行失败，可登录至任一节点，进入 ``/var/log/`` 目录查看相关日志调查原因并解决后，再次执行该命令。
include::../../../../../../_components/table_footer.adoc[]
