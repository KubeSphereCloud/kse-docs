== 适用范围

仅适用于 NeonSAN v3.4.0 及以上版本。

== 操作步骤

. 登录 NeonSAN 集群任一节点。 

. 执行如下命令，获取当前集群的 License 信息，需确保当前集群的 License 已被激活，其在有效期内。
+
[,shell]
----
neonsan list_license
----
+
