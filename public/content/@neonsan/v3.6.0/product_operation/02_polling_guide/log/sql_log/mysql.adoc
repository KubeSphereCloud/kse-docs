---
title: "查看 QingsStor MySQL 日志文件"
weight: 4
---

. 逐个登录数据库集群每个节点。
. 输入如下命令，查看 MySQL 的异常日志。
+
[,shell]
----
cat /qingstor/log/mysql/neonsan-mysql-error.log 
----
. 部分错误日志关键字解释及建议，其他未列出错误可联系 NeonSAN 产研团队进行排查。
+
[options="header",cols="1,1,3"]
|===
|关键字 | 含义 |建议
|Block because not sure the the livePeers 
|MySQl 数据库无法确定存活节点 
|建议检查 QingStor MySQL 数据库状态，参考《NeonSAN 运维手册 》 ： Qingstor 融合部署 > Qingstor MySQL 数据库运维相关内容
|===
