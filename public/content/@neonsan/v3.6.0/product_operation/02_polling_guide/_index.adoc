NeonSAN 分布式存储系统由 ZooKeeper 集群，数据库集群，Prometheus 服务，Alertmanager/Alerter 服务，NeonSAN 服务组成。其中，NeonSAN 服务由 Store，Center，Monitor，Tools 组成。Monitor 服务依赖于 Prometheus 与 Alertmanager/Alerter。

日常巡检，包括对以上所有服务状态，版本信息，相关日志的查询。


== 适用版本

NeonSAN 全系列版本。

== 读者对象

本文档主要适用于以下工程师：

* 实施工程师
* 运维工程师
