## 巡检项级别说明

为更好的对各项巡检结果进行针对性的处理，本手册对其进行级别划分，共分成 3 个等级，每一级别处理形式不尽相同，具体如下：

* 1 级：针对巡检结果无需处理，仅需要对其进行观测和监控即可。
* 2 级：针对巡检结果，驻场人员可参考手册直接处理。
* 3 级：针对巡检结果，驻场人员需要上报至 NeonSAN 产研团队进行判断后，协作处理。

## 其他须知

* 集群相关的信息可以登录到任一节点执行相关命令查看。
* 每个节点自身相关的信息需要登录到该节点执行相关命令查看。
* 每个集群仅可能包含 QingStor MySQL 数据库、Galera 数据库和 Xenondb 中一种数据库，请根据实际情况选择相应的操作。
* 集群中各节点的角色信息（如 ZooKeeper 节点、数据库节点等）均可以执行如下命令进行查看。
+
[,shell]
----
ssh qingstor-firstbox "cat /etc/hosts"
----
