完成存储池配置之后，进入**安装预览**阶段。该阶段主要用于核验当前集群的安装信息，包括**版本信息**、**集群信息**以及**配置信息**。 +
此步骤是 NeonSAN 正式部署前最后一次信息核验，若**集群信息**或**配置信息**有误，可点击其旁边的**编辑**，返回相应步骤进行修改。确定所有信息无误后，点击**安装部署**。

重点检查**配置信息**中各个节点硬盘总容量是否正确。
