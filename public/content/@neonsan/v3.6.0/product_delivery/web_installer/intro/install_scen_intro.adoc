title: "安装模式介绍"
weight: 1
---


NeonSAN 是青云自主研发的企业级分布式块存储系统。用户可使用 QingCloud 提供的 QingStor Installer 包，来完成 NeonSAN 产品的安装与部署。在安装过程中需要创建 FirstBox 节点，该节点可根据配置文件里面的部署设置，通过 SSH 免密访问，将对应的安装内容，发送至待部署的各个节点上，直至安装完成。NeonSAN 作为云平台的底层存储，其安装部署模式可分为 **SANC 独立部署**和 **SANC 融合部署**。

* SANC 独立部署：NeonSAN 的各种组件均独立部署于物理服务器上。NeonSAN 存储服务与云平台的计算服务不共享物理资源。

* SANC 融合部署：NeonSAN 的部分管理组件，如 ZooKeeper、数据库等，部署在云平台的 KS 物理节点上拉起的虚拟机。NeonSAN 存储服务与云平台计算服务，共用物理资源。
