== 术语解释

[options="header",cols="1,3"]
|===
|名词	|意义
|管理节点
|部署管理服务的节点服务器，管理服务主要包括 ZooKeeper、数据库、Portal 等服务

|存储节点
|部署存储服务的节点服务器，存储服务主要包括 neonsan-center、 neonsan-store 以及 neonsan-monitor 服务

|Common 服务	
|数据库、ZooKeeper 和 Portal 等服务的统称

|NTP	
|Network Time Protocol，一种使计算机时间同步的协议
|===

更多专业术语名词解释，可参考link:https://box.yunify.com/s/Xw99az7CSfTmFZP[《NeonSAN 术语手册》]。
