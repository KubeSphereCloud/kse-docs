// 部署 NeonSAN 服务"


. 登录 FirstBox。
+
[source,shell]
----
ssh x.x.x.x -p 6023
----
+

include::../../../../_components/note.adoc[]
* x.x.x.x 需替换为 FirstBox 所在宿主机的管理 IP。
* 用户名密码为 root/zhu88jie。

include::../../../../_components/table_footer.adoc[]


. 执行如下命令部署 NeonSAN 服务。
+
[source,shell]
----
installer product-deploy neonsan -I /root/ip_list
----
+
include::../../../../_components/note.adoc[]
此步骤耗时较长，请耐心等待。
include::../../../../_components/table_footer.adoc[]

. 执行如下命令查看部署任务执行状态。
+
[source,shell]
----
installer task-show <TASK_ID>
----
+
include::../../../../_components/note.adoc[]
* <TASK_ID> 需根据实际情况进行修改，可执行 ``installer task-list`` 命令进行查看。
* 返回结果中，关注 ``status`` 字段，需为 ``successful``，即证明服务部署成功，方可进行后续部署。
include::../../../../_components/table_footer.adoc[]
	
. 若部署失败，执行如下命令，获取当前部署日志查看失败原因。
+
[source,shell]
----
installer deploy-log-get -t <TASK_ID>
----
+
include::../../../../_components/note.adoc[]
* <TASK_ID> 需根据实际情况进行修改，可执行 ``installer task-list`` 命令进行查看。
* 也可在 ``/qingstor/log/`` 目录中查看相应的部署日志：
* 使用虚拟机部署 NeonSAN 时的常见错误解决方案，详见本文：“附录 > 常见错误错误方法”。
include::../../../../_components/table_footer.adoc[]


. 问题解决后，需执行下列命令对任务进行重试。
+
[source,shell]
----
installer task-operate <TASK_ID> --action retry
----
+
include::../../../../_components/note.adoc[]
* 重试之后需再次执行步骤 3，查看任务状态，直至返回结果中 ``status`` 字段为 ``successful`` 。
* <TASK_ID> 需根据实际情况进行修改，可执行 ``installer task-list`` 命令进行查看。
include::../../../../_components/table_footer.adoc[]

. 部署完成后，请退出 FirstBox 节点，登录至 NeonSAN 集群的任一节点执行如下命令确认服务是否正常。
+
[source,shell]
----
neonsan list_store && neonsan list_ssd && neonsan list_port && neonsan list_pool && neonsan list_parameter && neonsan list_rg -detail
----
+
include::../../../../_components/note.adoc[]
* 命令执行成功，无错误提示，即部署成功。
* 若任一命令执行失败，可登录至 NeonSAN 任一节点，进入 ``/var/log/`` 目录查看相关日志调查原因并解决后，再次执行该命令：
include::../../../../_components/table_footer.adoc[]

. 对于 NeonSAN v3.4.1 及以上版本，可在任一 NeonSAN 节点，执行如下命令，创建卷，以确定 NeonSAN 试用版 License 已导入成功且激活。
+
[source,shell]
----
neonsan create_volume -volume vol/vol02 -size 10G -repcount 3
----
+
include::../../../../_components/note.adoc[]
* 命令执行成功，无错误提示即可。
* 可在任一 NeonSAN 节点上执行 ``neonsan list_license -H`` 命令，查看 License。
include::../../../../_components/table_footer.adoc[]
