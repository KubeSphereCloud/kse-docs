:ks_db_product: OpenSearch


本节介绍如何管理 RadonDB {ks_db_product} 数据库。

RadonDB {ks_db_product} 是一款灵活且可扩展的搜索和分析引擎。提供实时日志处理、全文搜索和数据分析等能力。

在{ks_product_left} Web 控制台安装“数据库管理平台”扩展组件后，link:../01-intro/02-access-dmp/[进入数据库管理平台]，即可以容器化的方式部署 {ks_db_product} 数据库，并通过图形界面对 {ks_db_product} 数据库进行管理。
