:ks_db_product: PostgreSQL

本节介绍如何访问 RadonDB {ks_db_product} 数据库。


== 前提条件

* 已获取{ks_product_both}平台登录账号和密码，且已获取平台操作管理权限。

* RadonDB {ks_db_product} 实例状态为**运行中**。

// * 已获取 {ks_db_product} 数据库用户名和密码。更多相关信息，请参阅link:../03-manage-database-users/01-create-a-database-user/[创建数据库用户]。

== 操作步骤

. 以 platform-admin 角色登录{ks_product_left} Web 控制台并进入数据库管理平台。
. 在左侧导航栏选择 **{ks_db_product}**。
. 在 {ks_db_product} 实例列表中，点击一个实例名称打开其详情页面。
. 在详情页右侧的**概览**区域，查看容器组的名称。
. 执行以下命令访问 {ks_db_product} 数据库：
+
--
// Bash
include::../../../_ks_components/code/bash.adoc[]

kubectl exec -it -n <namespace> <pod_name> -- bash

include::../../../_ks_components/code/codeEnd.adoc[]

* <namespace>：{ks_db_product} 实例所在的命名空间（即项目名称）。

* <pod_name>：{ks_db_product} 容器组的名称。

示例命令如下所示：

// Bash
include::../../../_ks_components/code/bash.adoc[]

kubectl exec -it -n test radondb-cd5s9b-autofailover-0-0 -- bash

include::../../../_ks_components/code/codeEnd.adoc[]
--
