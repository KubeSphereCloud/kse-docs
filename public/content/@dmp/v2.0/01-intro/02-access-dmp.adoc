本节介绍如何从{ks_product_left} v4.0 Web 控制台进入 RadonDB DMP 数据库管理平台，并查看其概览信息。

== 前提条件

* 以 platform-admin 角色登录到{ks_product_left} Web 控制台。更多相关信息，请参阅{ks_product_left} v4.0 使用文档的link:https://docs.kubesphere.com.cn/v4.0/05-users-and-roles/[用户和角色管理]章节。
* {ks_product_right}平台需要安装 RadonDB DMP 扩展组件。更多相关信息，请参阅{ks_product_left} v4.0 使用文档的link:https://docs.kubesphere.com.cn/v4.0/06-extension-user-guide/[扩展组件管理]章节。

== 操作步骤

. 打开 Web 浏览器，在地址栏中输入{ks_product_left} Web 控制台的 IP 地址和端口号。
. 在{ks_product_both}登录页面，输入用户信息然后点击**登录**。
. 点击顶部导航栏上的image:/images/dmp/icons/grid.svg[grid,18,18]图标，点击 **RadonDB DMP**，进入数据库管理平台。
. 在**概览**页面，查看 RadonDB DMP 提供的数据库服务、中间件服务、备份管理、告警状态等的概览信息。
