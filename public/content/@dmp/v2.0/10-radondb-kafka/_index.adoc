:ks_db_product: Kafka


本节介绍如何管理 RadonDB {ks_db_product} 服务。

RadonDB {ks_db_product} 是基于 {ks_db_product} 构建的分布式发布订阅消息系统，具有高吞吐量、低延迟、高可靠等优点，被广泛应用于网站活动追踪、日志监控、流式计算、事件采集、数据存储等应用场景。

在{ks_product_left} Web 控制台安装“数据库管理平台”扩展组件后，link:../01-intro/02-access-dmp/[进入数据库管理平台]，即可以容器化的方式部署 {ks_db_product}，并通过图形界面对 {ks_db_product} 进行管理。
