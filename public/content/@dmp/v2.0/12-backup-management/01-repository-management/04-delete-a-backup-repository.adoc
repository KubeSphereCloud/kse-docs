本节介绍如何删除 RadonDB 数据库的备份仓库。

== 前提条件

* 已创建备份仓库。

== 操作步骤

. 以 platform-admin 角色登录{ks_product_left} Web 控制台并进入数据库管理平台。
. 在左侧导航栏点击**备份管理** > **备份仓库管理**。
. 在备份仓库列表中，找到目标仓库，点击image:/images/dmp/icons/more.svg[more,18,18]，选择**删除**。
. 在弹出的**删除**对话框，输入备份仓库的名称，点击**确定**。

//note
[.admon.note,cols="a"]
|===
|备注

|

只有在备份仓库未存储数据库备份的情况下，才能删除备份仓库。

如果备份仓库已存储数据库备份，删除备份数据后，才能删除备份仓库。

|===
