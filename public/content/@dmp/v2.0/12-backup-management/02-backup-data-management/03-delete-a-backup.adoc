本节介绍如何删除数据库备份。

== 前提条件

* 已在 MySQL 或 PostgreSQL 数据库详情页的“备份管理”页签创建了备份。

== 操作步骤

. 以 platform-admin 角色登录{ks_product_left} Web 控制台并进入数据库管理平台。
. 在左侧导航栏点击**备份管理** > **备份数据管理**。
. 点击 MySQL 和 PostgreSQL 页签查看对应数据库的备份。
. 找到目标备份，点击image:/images/dmp/icons/more.svg[more,18,18]，选择**删除**。
. 在弹出的**删除**对话框，输入备份名称，点击**确定**。
