:ks_db_product: MySQL

本节介绍如何管理 RadonDB {ks_db_product} 数据库实例。

// RadonDB {ks_db_product} 是基于 {ks_db_product} 的企业级数据库产品，支持一主多从高可用架构，以及自动备份、自动扩容等管理功能。

RadonDB {ks_db_product} 是一款基于 {ks_db_product} 的企业级分布式关系型数据库，提供高可用、高性能、高扩展和低成本的数据服务。

在{ks_product_left} Web 控制台安装“数据库管理平台”扩展组件后，link:../01-intro/02-access-dmp/[进入数据库管理平台]，即可以容器化的方式部署 {ks_db_product} 数据库，并通过图形界面对 {ks_db_product} 数据库实例进行管理。
