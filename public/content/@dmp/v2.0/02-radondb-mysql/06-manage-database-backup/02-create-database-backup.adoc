:ks_db_product: MySQL

本节介绍如何创建 RadonDB {ks_db_product} 数据库备份。

// Note
include::../../../../_ks_components/admonitions/note.adoc[]

备份过程可能会导致数据库服务压力过大，请在非高峰时段执行此操作。

include::../../../../_ks_components/admonitions/admonEnd.adoc[]


== 前提条件

* RadonDB {ks_db_product} 实例状态为**运行中**。
* 您已在**运维管理 > 备份管理 > 备份仓库管理**中link:../../../12-backup-management/01-repository-management/01-create-a-backup-repository[创建备份仓库]。


== 操作步骤

. 以 platform-admin 角色登录{ks_product_left} Web 控制台并进入数据库管理平台。
. 在左侧导航栏选择 **{ks_db_product}**。
. 在 {ks_db_product} 实例列表中，点击一个实例名称打开其详情页面。
. 在页面右侧点击**备份管理**页签打开数据库备份列表。
. 点击**创建备份**。
. 在**创建备份**对话框，输入备份任务的名称，然后点击**确定**。
+
--
数据库备份将显示在**备份列表**中。
