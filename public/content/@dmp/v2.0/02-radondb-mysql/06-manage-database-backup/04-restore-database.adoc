:ks_db_product: MySQL

本节介绍如何从备份恢复 RadonDB {ks_db_product} 数据库。


== 前提条件

* RadonDB {ks_db_product} 实例状态为**运行中**。
* 您已经创建了 {ks_db_product} 数据库备份，且状态为备份成功。


== 操作步骤

. 以 platform-admin 角色登录{ks_product_left} Web 控制台并进入数据库管理平台。
. 在左侧导航栏选择 **{ks_db_product}**。
. 在 {ks_db_product} 实例列表中，点击一个实例名称打开其详情页面。
. 在页面右侧点击**备份管理**页签，找到目标备份任务，点击image:/images/dmp/icons/more.svg[more,18,18]，选择**从备份恢复**。
. 在**基本信息**页签，编辑将通过备份创建的实例名称，然后点击**下一步**。
. 在**应用设置**页签，确认将创建的实例的参数。
. 点击**确定**，将自动创建新的 RadonDB {ks_db_product} 实例。
