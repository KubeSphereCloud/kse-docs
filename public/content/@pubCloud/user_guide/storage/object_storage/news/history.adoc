      content: 新增版本管理、WORM（Write Once Read Many）功能，可以更好满足用户在金融、医疗、政务等业务场景下对数据高安全性需求，满足合规性的审核要求，也可以有效应对勒索病毒等场景。</br> - 版本管理：提供 Bucket 级别的文件多版本设置，用户开启版本管理功能后，所有针对 Object 的覆盖上传、数据删除操作，将会以历史版本形式保存下来。用户可以查看、还原各个版本的 Object，并恢复历史文件和数据。</br> - WORM：提供数据合规保留设置，用户可选择不同级别设置。针对 Bucket 设置桶级别锁，保留期内任何用户都无法删除和覆盖对象，保护数据或满足监管要求；针对 Object 设置对象版本锁，根据选择的模式受保护的对象版本将不会被覆盖或删除。此级别的锁支持通过S3 Objectlock的方式进行数据备份和存储。
