**关于低频存储数据取回的费用说明：**

* 因为低频存储取回有一定成本，所以需要收额外的取回费用。
* 所有从 Bucket 下载数据的行为都是下载。下载低频存储里的数据，其费用为：`下载费用 + 数据取回费用`。其中 `数据取回费用` 指的是 QingStor 对象存储将数据从低频存储取回到标准存储的费用。
* 内网下载数据，不收取费用。

== 免费额度

注册成功并完成认证的 QingStor 对象存储用户，青云QingCloud 为您提供一定额度的 QingStor 对象存储免费使用套餐。每个 Bucket 享有如下免费政策:

* 标准存储空间：20 GB
* 总外网下载流量：10 GB
* 总外网读请求 (GET/HEAD)：100 万次
* 总外网写请求 (PUT/POST/DELETE)：10 万次

== 欠费说明

若当月费用扣除后余额不足，则用户的所有 Bucket 将会处于 `暂停` 状态，此时用户无法使用 QingStor 对象存储的任何功能。等到用户充值足够的金额后，用户的所有 Bucket 将会自动恢复。
