本章节指导您如何修改连接云网。

== 前提条件

* 已获取管理控制台的账号和密码。
* 已创建连接云网。

== 操作步骤

. 登录管理控制台。
. 选择**产品与服务** > *网络服务* > *SD-WAN（新版）*，进入**连接云网**页面。
+
image::/images/cloud_service/sd_wan/qs_cloud_network.png[]

. 点击目标连接云网所在行的**操作**列的**修改**，弹出**修改连接云网**窗口。
+
image::/images/cloud_service/sd_wan/um_wan_modify.png[,70%]


. 修改连接云网的**名称**和**描述**信息。
. 点击**修改连接云网**，完成连接云网的名称和描述的修改。
+
若页面弹出**编辑成功**提示信息，则说明修改连接云网名称和描述成功。
