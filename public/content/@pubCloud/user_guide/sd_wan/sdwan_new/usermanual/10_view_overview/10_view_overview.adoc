用户可以通过查看概览页面了解设备状态、告警数据、带宽趋势、接入点 TOP15的带宽使用率和丢包率以及企业云网的全局 WAN 状态和连接状态。

== 设备状态

用户可以查看当前企业云网的设备状态，包含全部设备数量、未激活设备数量、在线设备数量、离线设备数量、升级中设备数量、已到期设备数量以及 7 天内到期设备的数量。

image::/images/cloud_service/sd_wan/um_overview_device_status.png[]


== 告警数据

用户可以查看高危告警的数量、待处理指标告警数量、待处理事件告警数量。

image::/images/cloud_service/sd_wan/um_overview_alarm_status.png[]


== 带宽趋势

用户可以查看当前企业云网全部接入点或者某个接入点的带宽趋势，包含私网进、私网出、公网进、公网出等。

image::/images/cloud_service/sd_wan/um_overview_bandwidth_trend.png[]

== 接入点 TOP15

用户已查看接入点 TOP15 的带宽使用率和丢包率信息。

image::/images/cloud_service/sd_wan/um_overview_access_top15.png[]


== 连接云网状态

用户可以查看企业云网的状态，包含全部状态、在线、离线状态的 WAN 口状态和连接状态。

用户可以搜索查看指定接入点的 WAN 口状态和连接状态，用户也可以筛选在线或者离线状态的光盒或者 VCPE。

image::/images/cloud_service/sd_wan/um_overview_wan_status.png[]
