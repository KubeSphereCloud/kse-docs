本章节介绍如何查看操作日志内容。

== 前提条件

* 已创建连接云网。
* 已创建接入点。

== 操作步骤

. 登录管理控制台。
. 选择**产品与服务** > *网络服务* > *SD-WAN（新版）*，进入**连接云网**页面。
+
image::/images/cloud_service/sd_wan/qs_cloud_network.png[]


. 在左侧导航栏中，点击**操作日志**，进入**操作日志**页面。
+

include::../../../../../../_components/note.adoc[]
输入**操作行为**关键字、选择**操作状态**或者选择**操作时间**，可搜索指定的相关操作日志。

include::../../../../../../_components/table_footer.adoc[]
+
image::/images/cloud_service/sd_wan/um_log_list.png[]

+
参数说明，如下表所示。
+
|===
| 参数 | 参数说明

| 操作行为
| 用户的操作行为。

| 状态
| 操作的状态。

| 操作时间
| 操作的时间。
|===
