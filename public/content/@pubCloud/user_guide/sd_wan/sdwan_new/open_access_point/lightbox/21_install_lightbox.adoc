本章节介绍如何安装光盒 CPE。

== 前提条件

已申请光盒 CPE。

== 约束限制

若使用 4G/5G SIM 卡，安装 4G/5G SIM 卡后，需要将光盒 CPE 断电重启，系统才能正常识别。

== 操作步骤

. 将光盒 CPE 上电。
. 安装天线（以安装 WIFI 天线为例说明）。
+

include::../../../../../../_components/attention.adoc[]
* 若使用 4G/5G SIM 卡，则需要安装 4G/5G 天线。4G/5G 天线及 SIM 卡安装位置，请参见link:../10_lightbox_overview/#设备说明[设备说明]。
* 安装 4G/5G SIM 卡后，需要将光盒 CPE 断电重启，系统才能正常识别。

include::../../../../../../_components/table_footer.adoc[]
+
image::/images/cloud_service/sd_wan/cpe_wifi.png[,70%]


. 将可以上网的网线或者运营商的线路插入到光盒 CPE 的 Combo 口（默认 WAN 口）。
+
您可以选择插入到 Combo0 或者 Combo1。Combo 口包含光口和电口，您可以根据需要选择使用光口或者电口。
+

include::../../../../../../_components/note.adoc[]
* Combo【光口（光纤） + 电口（网线）】，默认 WAN 口。若使用光纤口，则需要插入千兆光模块。
* 若是运营商的线路接入，需要登录设备控制台，完成静态地址或 PPPOE 拨号配置。
+
登录设备控制台的方式，请参见link:../../equipment/10_login_equipment[登录设备控制台]。

include::../../../../../../_components/table_footer.adoc[]
+
image::/images/cloud_service/sd_wan/cpe01.png[,70%]

. 按照网络规划将光盒 CPE 通过 LAN 口接入到 PC 机或者网络设备上。
+
image::/images/cloud_service/sd_wan/cpe02.png[,70%]

. 验证网络连通性。
+
通过光盒 CPE 访问目标网络的 PC 或者网络设备。
