欢迎使用{platform_shortname}云点播（Video On Demand）。青云云点播服务是基于{platform_shortname}已有公有云产品的多年积累，为有音视频应用需求的客户提供的一站式点播场景解决方案，目前{platform_shortname}云点播包括音视频存储管理、音视频转码处理、音视频加速播放等能力。

{platform_shortname}云点播服务提供以下 API 接口供您使用。

== 媒资上传

[cols="1,2"]
|===
| API | 说明

| link:../30_media_upload/10_apply/[ApplyUpload]
| 申请将音视频文件上传到媒资管理后台。

| link:../30_media_upload/20_confirm/[CommitUpload]
| 确认媒资上传状态。
|===

== 媒资管理
[cols="1,2"]
|===
| API | 说明

| link:../40_media_mgt/12_delete/[DeleteMedia]
| 删除媒资文件。

| link:../40_media_mgt/15_get_info/[GetMediaInfo]
| 根据媒资 ID 查询媒资详情。

| link:../40_media_mgt/16_get_clip_info/[GetMediaClipInfo]
| 根据媒资 ID 查询裁剪处理后的视频信息。

| link:../40_media_mgt/19_search_list/[SearchMediaList]
| 批量查询媒资信息。

| link:../40_media_mgt/20_forbid/[ForbidMedia]
| 根据媒资 ID 在媒资后台将音视频禁播。

| link:../40_media_mgt/22_no_forbid/[ReleaseMedia]
| 根据媒资 ID 在媒资后台将将禁播的音视频解禁。
|===
