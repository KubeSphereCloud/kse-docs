本文提供 Android 播放器 SDK 的接口说明。

== 播放器接口

[cols="1,2"]
|===
| API | 描述

| https://vod-sdk.pek3b.qingstor.com/doc/Android/index.html[setMediaController]
| 为播放器设置视频控制器。 +
用户可自定义自己的视频控制器，但必须继承抽象类 https://vod-sdk.pek3b.qingstor.com/doc/Android/com/qingcloud/vod/player/mediacontroller/IQCMediaController.html[IQCMediaController]。 +
如无设置，默认使用青云自定义的控制器 QCMediaController。 如果只想显示视频，不需要控制器，传Null。

| https://vod-sdk.pek3b.qingstor.com/doc/Android/index.html[playWithUrl]
| 直接传入视频地址进行播放。

| https://vod-sdk.pek3b.qingstor.com/doc/Android/index.html[playWithItem]
| 配置一个视频原子对象进行播放。

| https://vod-sdk.pek3b.qingstor.com/doc/Android/index.html[release]
| 退出播放器并销毁。
|===

== 播放基础接口
[cols="1,2"]
|===
| API | 描述

| https://vod-sdk.pek3b.qingstor.com/doc/Android/index.html[play]
| 开始播放。

| https://vod-sdk.pek3b.qingstor.com/doc/Android/index.html[stop]
| 停止播放。

| https://vod-sdk.pek3b.qingstor.com/doc/Android/index.html[pause]
| 暂停播放。

| https://vod-sdk.pek3b.qingstor.com/doc/Android/index.html[seekTo]
| 跳转到视频流指定时间点，单位毫秒。

| https://vod-sdk.pek3b.qingstor.com/doc/Android/index.html[getDuration]
| 获取总时长，单位毫秒。

| https://vod-sdk.pek3b.qingstor.com/doc/Android/index.html[getCurrentPosition]
| 获取当前播放时间，单位毫秒。

| https://vod-sdk.pek3b.qingstor.com/doc/Android/index.html[isPlaying]
| 是否正在播放。

| https://vod-sdk.pek3b.qingstor.com/doc/Android/index.html[suspend]
| 停止播放，并释放播放器资源。

| https://vod-sdk.pek3b.qingstor.com/doc/Android/index.html[setBrightness]
| 设置亮度。

| https://vod-sdk.pek3b.qingstor.com/doc/Android/index.html[enableCodec]
| 设置是否启动硬解码。

| https://vod-sdk.pek3b.qingstor.com/doc/Android/index.html[getBufferPercentage]
| 获取视频缓冲进度百分比。
|===
