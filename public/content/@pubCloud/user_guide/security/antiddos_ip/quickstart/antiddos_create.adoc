本小节为您介绍如何创建高防 IP 实例。

== 操作步骤

. 登录管理控制台。
. 在控制台导航栏中，选择**产品与服务** > *安全服务* > *高防IP*，进入**高防 IP-实例列表**页面。
+
image::/images/cloud_service/security/antiddos_ip/01.png[]

. 点击**购买高防IP**，进入实例列表配置信息页面。
+
image::/images/cloud_service/security/antiddos_ip/02.png[]

. 根据防护需要，选择对应的产品规格，点击**立即购买**。
+
参数解释如下：
+
[cols="1,3"]
|===
| 参数 | 说明

| 名称
| 高防IP实例名称

| 线路
| 目前支持电信单线和 BGP 多线两种线路。

| IP 个数
| 系统为每个订单分配一个独享高防 IP。

| 基础防护带宽
| 提供基础带宽防护能力。带宽支持范围和费用请参见link:../../billing/price/[高防 IP 计费]。

| 最大防护带宽
a| 最大防护带宽支持的配置范围如下： +

* 单线：50Gbps、100Gbps、200Gbps、300Gbps、400Gbps、500Gbps、600Gbps、700Gbps +
* BGP：20Gbps、50Gbps、100Gbps、200Gbps、300Gbps、400Gbps、500Gbps、600Gbps、700Gbps +
选购了最大防护带宽服务后，当访问流量大于基础防护带宽时，会触发最大防护带宽并产生最大防护带宽账单。（弹性计费按天后付费） 

include::../../../../../_components/note_in_table.adoc[]

* 当访问流量（正常业务流量+攻击流量）未超过基础防护带宽值时，高防 IP 提供基础防护带宽防护。 +
* 当访问流量（正常业务流量+攻击流量）超过基础防护带宽值时，但未超过最大防护带宽时，触发弹性计费，高防IP提供最大防护带宽防护。 +
* 当访问流量（正常业务流量+攻击流量）超过最大防护带宽值时，触发弹性计费，进入黑洞机制，黑洞时长15分钟。

include::../../../../../_components/footer_in_table.adoc[]

| CC 防护峰值
| 1000000QPS

| 回源带宽
| 回源带宽是高防机房清洗后回源给源站的业务流量带宽。 +
每个实例免费赠送20Mbps业务带宽，可以付费增加，最大支持300Mbps。请根据所有要接入当前实例防护的业务的整体规模估算需要的业务带宽，取入流量和出流量中的最大值。

| 自动续费
| 若创建时未勾选此项，请及时进行手动续费，否则实例到期后未续费高防 IP 资源将被系统回收。
|===

. 跳转到**确认订单**页面，信息核对无误后，点击**确认支付**。
+
image::/images/cloud_service/security/antiddos_ip/03.png[]
+
开通完成后，进入实例列表界面。高防IP需正常使用，还需要进行业务接入，详情请参见link:../antiddos_access/[业务接入]。
