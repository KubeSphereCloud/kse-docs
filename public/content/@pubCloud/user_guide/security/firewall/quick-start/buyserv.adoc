云防火墙服务只支持包年包月的购买方式。您需要选择合适的服务版本、规格、购买时长，并一次性完成预付费。

== 前提条件

* 已获取管理控制台登录账号和密码，且账号已实名认证。
* 账号余额大于 0。

== 开通云防火墙服务

若首次使用云防火墙服务，请先进行开通。

. 登录管理控制台。
. 在顶部菜单栏，选择**产品与服务** > **安全服务** > **云防火墙 CFW**，进入云防火墙服务开通页面。
. 点击**立即开通**。
+
image::/images/cloud_service/security/firewall/open.png[]

. 阅读并同意《服务协议》，点击**确定开通**。
+
等待初始化配置成功，进入云防火墙**业务系统资产**页面。

== 购买云防火墙

. 进入云防火墙控制台，在**业务系统资产**页面 ，点击**购买云防火墙**。
. 完成云防火墙服务的购买配置。
+
image::/images/cloud_service/security/firewall/bought.png[]
+
[cols="1,2"]
|===
| 参数 | 说明

| 当前版本
a| * 云防火墙服务的版本。
* 支持四种版本：专业版、高级版、企业版和旗舰版。不同的版本支持的可防护的带宽跟公网 IP 不同，您可以单击某个版本，在**版本选择**下查看当前版本的功能说明。更多信息，请参见link:../../intro/function[功能特性]。 +
* 关于不同版本的定价信息，请参见link:../../billing/price[计费指南]。

| 可防护公网 IP 数
a| 互联网边界隔离防护功能中支持添加保护的公网IP的数量，单位：个。可选范围：

* *专业版*：20~50 +
* *高级版*：50~200 +
* *企业版*：100~1024
* *旗舰版*：300~2048

| 公网流量处理能力
a| 云防火墙可以防护的公网方向的流量峰值，单位：Mbps。可选范围：  +

* *专业版*：10~50 +
* *高级版*：50~200 +
* *企业版*：100~1024
* *旗舰版*：300~2048

include::../../../../../_components/note_in_table.adoc[]

建议与您业务的公网带宽保持一致。

include::../../../../../_components/footer_in_table.adoc[]

| 购买时长
a| 服务购买时长。可以根据需要开启**到期自动续费**，需要一次性购买才能享受优惠。

include::../../../../../_components/note_in_table.adoc[]

* 购买半年享受 9 折优惠。
* 购买1年以上时长的云防火墙服务，可享受 8.5 折优惠。
* 购买 2 年以上享受 7 折优惠。
* 购买 3 年以上享受 6 折优惠。

include::../../../../../_components/footer_in_table.adoc[]

|===

. 单击**立即购买**并完成支付。
+
支付成功后，您可以在**云防火墙控制台**页面的右上角查看当前云防火墙的服务版本和剩余保护期。
