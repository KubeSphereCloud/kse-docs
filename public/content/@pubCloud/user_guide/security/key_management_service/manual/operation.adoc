== 密钥管理服务界面

登录管理控制台，在顶部导航栏点击**产品与服务**，然后点击**安全服务**中的**密钥管理服务**。

image::/images/cloud_service/security/kms/log_in.png[]

== 修改密钥

您可以修改密钥名称和描述信息。

在密钥管理服务页面，右键点击要修改的用户密钥，然后选择**修改**。

image::/images/cloud_service/security/kms/modify_key_1.png[]

image::/images/cloud_service/security/kms/modify_key_2.png[]

== 创建密钥别名

您可以为密钥增添别名。

. 在密钥管理服务页面，点击要添加别名的用户密钥，进入密钥详细信息页面。
+
image::/images/cloud_service/security/kms/key_detail.png[]

. 点击**创建**，弹出创建密钥别名窗口。
+
image::/images/cloud_service/security/kms/create_alias.png[]

. 输入密钥别名，点击**提交**。

== 删除密钥别名

您可以删除密钥别名。

include::../../../../../_components/note.adoc[]
删除密钥别名不会影响使用此密钥的云产品。如果您有业务通过 API 调用此密钥别名会造成调用失败，您可以重新为密钥添加此别名。

include::../../../../../_components/table_footer.adoc[]


. 在密钥管理服务页面，点击要删除别名的用户密钥，进入密钥详细信息页面。
+
image::/images/cloud_service/security/kms/key_detail.png[]

. 点击**解绑**，弹出删除密钥别名窗口。
//image::/images/cloud_service/security/kms/delete_alias.png[]

. 点击**确认**。

== 禁用密钥

您还可以禁用密钥。密钥被禁用后，使用该密钥的云服务器、硬盘或镜像不会马上受到影响，但当云服务器处于关机状态，然后重新启动使用密钥时，就会导致云服务器启动失败无法使用。如果服务器处于开机状态直接重启，则不会受到禁用密钥的影响，云服务器仍然可以正常使用。

在密钥管理服务页面，右键点击要禁用的用户密钥，然后选择**禁用**。

//image::/images/cloud_service/security/kms/disabel_key.png[]

== 启用密钥

对于已经禁用的密钥，您可以重新启用该密钥。

在密钥管理服务页面，右键点击要启用的用户密钥，然后选择**启用**。

//image::/images/cloud_service/security/kms/enable_key.png[]

== 计划删除密钥

计划删除密钥是指用户可以在永久删除密钥之前，设置一个等待期限。在等待期限中，该密钥将无法使用，一旦云服务器处于关机状态，然后重新启动使用密钥时，就会导致云服务器启动失败无法使用。超过等待期限，密钥将被永久删除。

include::../../../../../_components/attention.adoc[]

一旦密钥被永久删除，加密中的数据将无法解密。

include::../../../../../_components/table_footer.adoc[]


在密钥管理服务页面，右键点击计划删除的用户密钥，然后选择**计划删除**。

//image::/images/cloud_service/security/kms/delete_key.png[]

== 取消计划删除密钥

在计划删除的等待期限中，用户可以取消计划删除。密钥取消计划删除后，就可以被使用了。

在密钥管理服务页面，右键点击计划删除的用户密钥，然后选择**取消计划删除**。

//image::/images/cloud_service/security/kms/cancel_delete_key.png[]
