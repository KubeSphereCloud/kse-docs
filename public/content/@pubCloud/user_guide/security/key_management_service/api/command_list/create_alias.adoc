创建 CMK 的别名。

*Request Parameters*

|===
| Parameter name | Type | Description | Required

| key_id
| String
| 密钥 ID
| Yes

| alias_name
| String
| 密钥别名
| Yes

| description
| String
| 别名描述
| No
|===

link:../../parameters/[_公共参数_]

*Response Elements*

|===
| Name | Type | Description

| ret_code
| int
| 返回码

| key_id
| string
| 密钥ID

| description
| string
| 描述

| alias_name
| string
| 密钥别名

| name
| string
| 密钥名称
|===

*Example*

_Example Request_:
[source,url]
----
https://kms.api.qingcloud.com/?action=CreateAlias
&alias_name=
&key_id=
&COMMON_PARAMS
----

_Example Response_:
[source,json]
----
{
"key_id":"cmk-NCF91CL0",
"alias_name":"alias/test01",
"description":"",
"name":"",
"ret_code":0
}
----
