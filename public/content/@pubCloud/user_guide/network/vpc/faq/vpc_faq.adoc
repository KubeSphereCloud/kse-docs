{platform_name_cn}提供了两种网络配置模式：**受管**与**自管**。

* 受管网络是全自动化的， 您的云服务器的所有网络配置与管理都交给{platform_name_cn}系统完成。
* 自管网络是需要手动管理， 您必须自己关心云服务器的网络配置与管理，比如 IP 地址、路由等。

所以，当您使用自管网络时，需要手动配置网络。当受管网络所连接的路由器上关闭了 DHCP 功能时，也需要手动配置网络。
