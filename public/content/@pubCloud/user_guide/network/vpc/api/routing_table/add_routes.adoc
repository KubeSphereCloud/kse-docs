创建一条路由表规则。

== 请求参数

[cols="1,1,3,1"]
|===
| 参数 | 参数类型 | 描述 | 是否必选

| action
| String
| 请求动作，可选值：AddRoutes
| Yes

| routing_table
| String
| 路由表ID
| Yes

| rules
| Array
| 路由表规则信息，参考 <<rules,RulesItems>>
| Yes
|===

link:../../get_api/parameters/[_公共参数_]

[[rules]]*RulesItems*
[cols="1,1,3"]
|===
| 参数 | 参数类型 | 描述
| routing_table_rule_name
| String
| 路由表规则的名称


| network
| String
| 路由表规则目标网段


| nexthop
| String
| 路由表规则的下一跳，可以是路由器ID或NAT网关ID，也可以是IP地址

|===



== 返回数据
[cols="1,1,3"]
|===
| 参数 | 参数类型 | 描述

| action
| String
| 响应动作

| routing_table_rules
| Array
| 新创建的路由表规则的ID列表

| ret_code
| Integer
| 执行成功与否，0 表示成功，其他值则为错误代码
|===

== 示例

*请求示例：*
[source]
----
https://api.qingcloud.com/iaas/?action=AddRoutes
&routing_table=rt-w7xy0bq0
&rules=[{\"routing_table_rule_name\":\"test\",\"network\":\"192.168.10.0/24\",\"nexthop\":\"rtr-8vwgl9kv\"}]
&COMMON_PARAMS
----

*返回示例：*
[source]
----
{
	"action": "AddRoutesResponse",
	"routing_table_rules": ["r-tfzvmsqr"],
	"ret_code": 0
}
----
