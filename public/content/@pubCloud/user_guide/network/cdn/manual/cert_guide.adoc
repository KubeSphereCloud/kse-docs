== 概述

若需使用 HTTPS 协议，需要服务器绑定相应的证书。CDN 如需使用 HTTPS 协议时，用户需在此处添加相应的证书后，再开启 HTTPS 服务。

== 操作指南


. 登录管理控制台，选择**产品与服务** > *网络服务* > *CDN 服务*，进入 CDN 页面。

. 在左侧边栏，点击**证书**，进入证书列表页面。
+
image::/images/cloud_service/network/cdn/cert_list.png[]

. 点击** + 添加**，弹出**创建证书**对话框，用户根据页面提示信息，填写相关参数后，点击**提交**即可完成创建。
+
image::/images/cloud_service/network/cdn/cert_add.png[cert_add,50%]

. 对于已添加的证书，用户点击**右键** > **更多操作**来修改或删除证书。
