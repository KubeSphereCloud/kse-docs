负载均衡器提供以下 API 接口供您使用。

== 负载均衡器实例

|===
| API | 说明

| link:../loadbanlancer/create_lb/[CreateLoadBalancer]
| 创建负载均衡器。

| link:../loadbanlancer/describe_lb/[DescribeLoadBalancers]
| 查询负载均衡器。

| link:../loadbanlancer/bind_eips_to_lb/[AssociateEipsToLoadBalancer]
| 绑定公网 IP 到负载均衡器。

| link:../loadbanlancer/dissociate_eips_lb/[DissociateEipsFromLoadBalancer]
| 从负载均衡器解绑公网 IP。

| link:../loadbanlancer/loadbalancer_join_vxnet/[LoadBalancerJoinVxnet]
| 负载均衡器加入私有网络。

| link:../loadbanlancer/loadbalancer_leave_vxnet/[LoadBalancerLeaveVxnet]
| 负载均衡器离开私有网络。

| link:../loadbanlancer/modify_lb_attribute/[ModifyLoadBalancerAttributes]
| 修改负载均衡器名称及描述。

| link:../loadbanlancer/resize_loadbalancers/[ResizeLoadBalancers]
| 修改负载均衡器规格类型。

| link:../loadbanlancer/update_loadbalancers/[UpdateLoadBalancers]
| 更新负载均衡器配置。

| link:../loadbanlancer/start_loadbalancers/[StartLoadBalancers]
| 启动负载均衡器。

| link:../loadbanlancer/stop_loadbalancers/[StopLoadBalancers]
| 关闭负载均衡器。

| link:../loadbanlancer/delete_loadbalancers/[DeleteLoadBalancers]
| 删除负载均衡器。
|===

== 监听器

|===
| API | 说明

| link:../listener/add_lb_listeners/[AddLoadBalancerListeners]
| 添加负载均衡监听器。

| link:../listener/describe_loadbalancer_listeners/[DescribeLoadBalancerListeners]
| 查询负载均衡监听器列表。

| link:../listener/modify_listener_attribute/[ModifyLoadBalancerListenerAttributes]
| 修改负载均衡监听器属性。

| link:../listener/bind_certs_to_listener/[AssociateServerCertsToLBListener]
| 绑定服务器证书到负载均衡监听器。

| link:../listener/unbind_certs_from_listener/[DissociateServerCertsFromLBListener]
| 将服务器证书从负载均衡监听器解绑。

| link:../listener/delete_listeners/[DeleteLoadBalancerListeners]
| 删除负载均衡监听器。
|===
