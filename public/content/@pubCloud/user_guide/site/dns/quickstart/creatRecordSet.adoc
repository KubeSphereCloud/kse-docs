本小节主要主要介绍如何快速配置解析记录。

== 步骤一：添加域名

. 登录管理控制台。
. 选择**产品与服务** > *域名与网站* > *云解析 DNS*，进入域名列表页。
+
image::/images/cloud_service/site/dns/create_domain_1.png[DNS域名列表]

. 点击**添加**，在下方弹窗中输入您在注册商处申请的域名。
+
image::/images/cloud_service/site/dns/create_domain_2.png[添加域名,70%]
+

include::../../../../../_components/attention.adoc[]

您的域名必须真实有效且未被纳入工信部备案黑名单。

DNS 将定期同步 link:https://www.iana.org/domains/root/db[IANA 域名后缀列表]，若您发现新的域名后缀无法添加，请及时与我们联系。

include::../../../../../_components/table_footer.adoc[]

== 步骤二：创建解析记录

. 在域名列表页面，点击需要解析的域名，进入**解析记录**页面。
+
image::/images/cloud_service/site/dns/dns_parse_1.png[点击域名]

. 点击**添加记录**，显示添加记录输入框。
+
image::/images/cloud_service/site/dns/dns_parse_3.png[添加记录]
+
也可以在对应的云服务器名中右键**添加记录**。
+
image::/images/cloud_service/site/dns/right_menus_create.png[创建记录]

. 按对话框指引，填写云服务器名和记录值，选择线路、记录类型、模式和 TTL 后提交即可。
+
示例：将 www.eg.com 默认解析指向网站服务器 IP 地址 1.1.1.1 。
+
image::/images/cloud_service/site/dns/dns_A.png[添加解析信息]
+

include::../../../../../_components/note.adoc[]

QingCloud DNS 默认提供 5 条线路，包括**全网默认、中国电信、中国联通、中国移动、港澳台及海外**。若需自定义解析线路，可参考link:../../manual/dnsrecord/setresolline[配置解析线路]。
