== 操作场景

域名注册成功后，如果您需要查询域名的基本信息，您可以登录到青云控制台查看。

== 操作步骤

. 登录管理控制台，选择**产品与服务** > *域名与网站* > *域名注册*，进入域名注册页面。
+
image::/images/cloud_service/site/domain/dn_service.png[]

. 在域名列表中，找到待查询的域名，单击操作列下的**管理**，进入基本信息页面。
+
image::/images/cloud_service/site/domain/dn_list.png[]

. 在基本信息页面，可查看到域名的基本信息。
+
image::/images/cloud_service/site/domain/dn_basic_info.png[]
