== 操作场景

域名证书是域名所有者的身份证明，常用于域名备案。

== 操作步骤

. 登录管理控制台，选择**产品与服务** > *域名与网站* > *域名注册*，进入域名注册页面。
+
image::/images/cloud_service/site/domain/dn_service.png[]

. 在域名列表中，找到要打印或下载证书的域名，点击操作列下的**管理**，进入基本信息页面。
+
image::/images/cloud_service/site/domain/dn_list.png[]

. 在基本信息的域名证书中，点击**下载域名证书**。
+
image::/images/cloud_service/site/domain/download_certificate.png[]
