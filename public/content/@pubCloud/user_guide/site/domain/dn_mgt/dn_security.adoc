安全设置开启后，将禁止域名转移，可以有效的保护域名不被恶意转出。

== 操作步骤

. 登录管理控制台，选择**产品与服务** > *域名与网站* > *域名注册*，进入域名注册页面。
+
image::/images/cloud_service/site/domain/dn_service.png[]

. 在域名列表中，找到要开启安全设置的域名，点击操作列下的**管理**，进入基本信息页面。
+
image::/images/cloud_service/site/domain/dn_list.png[]

. 在基本信息的**安全设置**中，点击**开启**。
