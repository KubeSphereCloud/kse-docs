== 操作场景

注册域名后需及时进行实名认证，提交实名认证后，审核机构将根据您提交的材料进行审核。域名实名认证资料的审核时间通常为 3~5 个工作日。

本文为您介绍域名实名认证审核时长及如何查看实名认证进度状态。

== 状态说明

实名认证状态包含四种：未实名认证、审核中、实名认证失败、实名认证成功。

* 未实名认证 +
请提交实名认证资料进行认证。详情请参见域名实名认证所需资料。
* 审核中 +
域名实名认证正在审核中，审核一般需要 3~5 个工作日。
审核期间，域名会处于 Serverhold 状态，待审核通过后的 1~2 个工作日恢复 ok 状态。
* 实名认证失败 +
实名认证资料不符合要求已被审核机构驳回申请。请查看实名认证审核失败原因，并参见实名认证失败原因及解决方案修复问题后单击重新实名认证，重新提交实名认证资料。
* 实名认证成功 +
说明域名实名认证已经成功，您可以进行后续的备案和域名解析操作。

== 查询实名认证进度

信息模板实名认证结果查询：

. 登录管理控制台，选择**产品与服务** > *域名与网站* > *域名注册*，进入域名注册页面。
+
image::/images/cloud_service/site/domain/dn_service.png[]

. 在左侧域名注册导航中，点击**信息模板**。
+
image::/images/cloud_service/site/domain/dn_authen_template.png[]

. 在信息模版列表中，查看该信息模版的**认证状态**。
+
域名持有者实名认证结果查询：

. 登录管理控制台。
. 选择**产品与服务** > *域名与网站* > *域名注册*，进入域名注册页面。
+
image::/images/cloud_service/site/domain/dn_service.png[]

. 在域名列表中，查看域名的**认证状态**。
+
image::/images/cloud_service/site/domain/dn_not_authen.png[]
