== 操作场景

当信息模板的**认证状态**为``未实名认证``或者``名认证失败``时，您可以修改信息模板中的除了**用户类型**以及**域名所有者**以外的相关信息。

include::../../../../../_components/note.adoc[]

不支持修改认证状态为``已实名认证``和``审核中``的信息模板。不支持修改信息模板中的**用户类型**和**域名所有者**。 如果需要变更上述信息，需创建新的信息模板。

include::../../../../../_components/table_footer.adoc[]

== 操作步骤

. 登录管理控制台，选择**产品与服务** > *域名与网站* > *域名注册*，进入域名注册页面。
+
image::/images/cloud_service/site/domain/dn_service.png[]

. 在左侧域名注册导航中，点击**信息模板**。
+
image::/images/cloud_service/site/domain/dn_authen_template.png[]

. 在信息模板列表中，找到要修改的信息模板，点击操作列下的**修改**，进入修改信息模板页面。
. 修改联系人相关信息后，点击**提交**。
