== 操作场景

当不再使用某个信息模板时，可以删除信息模板。

include::../../../../../_components/note.adoc[]

删除操作无法恢复，请谨慎操作。已被设为默认信息模板的模板，无法删除。不管信息模板是什么**认证状态**，都可以进行删除信息模板操作。

include::../../../../../_components/table_footer.adoc[]

== 操作步骤

. 登录管理控制台，选择**产品与服务** > *域名与网站* > *域名注册*，进入域名注册页面。
+
image::/images/cloud_service/site/domain/dn_service.png[]

. 在左侧域名注册导航中，点击**信息模板**。
+
image::/images/cloud_service/site/domain/dn_authen_template.png[]

. 在信息模板列表中，找到要删除的信息模板，点击操作列下的**删除**。
. 在确认信息中，点击**确定**。
