== 证书加密算法如何选择？

相同类型算法位数越多，性能消耗越严重，但安全性越高。 ECC 算法用较低的位数即可获取与 RSA 高位数的安全性，同时性能也比较高，但是要求客户端版本较高，旧版客户端可能不支持 ECC 算法。 具体的算法选择需要结合企业实际也许需求进行权衡。

== 是否支持吊销？

支持吊销，提工单即可。

== 证书过期如何知晓？

通过 QingCloud 官方平台购买的证书以及用户自行说上传的证书，当证书即将过期，平台会分别在过期前的 30、15、7 天，通过短信及邮件给用户发送提醒。

== HTTPS 的性能损耗有多少？

性能损耗取决于多方面因素，具体以实测为准，可以使用该工具测试： https://www.httpvshttps.cn/[HTTPS VS HTTP 测速]。

== 如何检测证书的兼容性？

可选择使用测试工具进行测试，如 \https://myssl.com/。

== 如何选择一个合适的证书？

* 个人选择 DV 证书
* 企业选择 OV 或者 EV 证书
* 金融相关行业选择 EV 证书

以上建议仅供参考。
