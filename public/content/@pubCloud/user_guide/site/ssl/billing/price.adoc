== 费用计算

* GeoTrust/TrustAsia 多域名证书价格 =（多证书价格(默认含五个域名) + 额外域名价格 * 额外域名数量）* 年
* Symantec 多域名证书价格 = 单个域名价格 * 域名数量 * 年
