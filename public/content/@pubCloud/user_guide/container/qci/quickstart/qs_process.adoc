本入门教程将指导您从 Docker Hub 拉取 Nginx 镜像创建一个 {qci_shortname} 实例，并通过公网访问所部署的 Nginx 服务。

完成本操作需要以下几个步骤：

. link:../qs_prepare/[准备工作]
+
您需要完成一些准备工作，才能顺利创建 {qci_shortname} 实例。

. link:../qs_create_qci/#_步骤1创建_qci_实例[创建 {qci_shortname} 实例]
+
通过创建 QCI 实例来部署容器服务。

. link:../qs_create_qci/#_步骤2公网访问容器服务[公网访问容器服务]
+
通过 VPC 端口转发功能，使用 VPC 公网 IP 访问容器服务。
