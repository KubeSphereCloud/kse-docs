title: "功能概览"
weight: 1
---

为方便用户更好的对其云资源进行管理，青云提供以下运维工具。

* 标签
+
随着业务的增长，用户使用的资源日益增多，资源检索会成为日常的需求。使用标签功能，用户可以对云资源进行分类分组。检索时，可以根据标签过滤各种基础设施资源，便于日常管理。


* 操作日志
+
操作日志会记录最近一段时间内的所有操作历史。用户的操作可分为``本区操作``与``全局操作``。相应的，日志功能，也分为``本区操作日志``与``全局操作日志``。


* 回收站
+
用户删除的资源，包括云服务器、硬盘、备份、自有镜像等，均会在回收站中保留 2 个小时。在此期间，用户可对已删除的资源进行恢复。青云的回收站功能，支持在资源恢复时，选择计费模式。
+
回收站的资源，自删除始，2 小时后，将被彻底销毁，不可恢复。


* 定时器
+
定时器（Scheduler）功能可用来定期执行一系列任务。比如定时开关云服务器、创建备份、调整带宽等。这些任务可在指定的周期重复执行，也可仅执行一次，且每个任务支持多个资源批量操作。任务执行后，系统会发送通知至用户指定的用户列表，并保留近期执行的任务记录。
+
每种任务都可支持最多 5 个资源的批量操作，如若不够可通过多个任务将资源分组执行。定时器的资源操作是基于 API 实现。默认情况下每个用户每小时最多调用 API 的次数为 2000次。
+
目前支持的定时任务有：

** 开启云服务器
** 关闭云服务器
** 重启云服务器
** 创建备份
** 跨区复制备份
** 调整公网 IP 带宽上限
** 调整公网 IP 计费模式
** 开启 VPC 网络
** 关闭 VPC 网络
** 启动负载均衡器
** 关闭负载均衡器
** 启用防火墙规则
** 禁用防火墙规则
