本文介绍如何从云市场购买 API 类商品。

== 操作步骤

. 登录https://marketplace.qingcloud.com/[云市场]首页，找到需要购买的商品。
+
image::/images/cloud_service/appcenter/um_cloud_market.png[]
+
具体操作可参见link:../../find_app/[查找商品]。

. 点击待购买的 API 商品，进入商品详情页。
+
您可以查看该商品商品详情、API接口、价格详情、使用文档、用户案例和服务支持的相关信息。
+
image::/images/cloud_service/appcenter/um_purchase_api_1.png[]

. 根据商品实际情况，选择商品规格、套餐，并阅读和同意相关协议和说明。
. 点击**立即购买**，进入**确认订单**页面。
. 查看订单信息，确认无误后，点击**确认支付**。
+
支付成功后，点击**查看订单**，进入控制台费用中心的**订单管理**页面。可以查看到已购买商品的订单。
