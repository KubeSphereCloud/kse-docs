本文介绍如何使用在云市场购买的 API 商品。

== 前提条件

已在云市场购买 API 类商品并支付成功。具体操作请参考link:../../purchase_app/[购买商品]。

== 操作步骤

. 登录link:https://marketplace.qingcloud.com/[云市场]首页，在页面右边顶部菜单，选择**买家中心**，进入买家中心控制台。

. 在**已购买服务**页面，找到已购买的 API 商品。
+
image::/images/cloud_service/appcenter/um_buyers_have_api.png[img]

. 点击**再次购买**，可快速购买同款商品。
. 点击**套餐列表**，进入商品详情页。可查看品套餐的使用情况。
+
image::/images/cloud_service/appcenter/um_buyers_api_details.png[img]

. 点击 *API 接口*，进入 API 商品详情的接口说明页面，可查看 API 接口说明。
. 点击**使用文档**，可下载 API 接口的使用说明文档。
