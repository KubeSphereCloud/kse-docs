OpenSearch 服务目前支持的原生 OpenSearch 版本和兼容的 Elasticsearch OSS 版本如下：

|===
| OpenSearch 服务 | 原生 OpenSearch | Elasticsearch OSS

| OpenSearch 2.3.0 - v1.0.0
| 2.3.0
| 7.10.2
| OpenSearch 1.2.4 - v1.0.0
| 1.2.4
| 7.10.2
| OpenSearch 1.1.0 - v1.0.0
| 1.1.0、1.0.0
| 7.10.2、7.5.1
|===

include::../../../../../_components/note.adoc[]

若需启动 OpenSearch 项目，建议您选择支持的最新 OpenSearch 版本。

若现有域使用的是较旧的 Elasticsearch 版本，您可以选择保留该域或迁移您的数据。
