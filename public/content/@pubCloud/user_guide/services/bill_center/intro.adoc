费用中心为用户提供基于页面的云产品交易和账单管理能力，通过费用中心，您可以清晰地了解自己的订单、账单和消费明细等相关信息，您还可以进行续约、退订、发票管理等操作。

== 功能列表

* **账户总览**
+
支持查看账户余额、优惠券、近期账单、消费预估、待续约资源等信息概览，以及充值汇款、发票申请等操作。

* **账务管理**
+
[cols="1,3"] 
|===
| 功能 | 说明

| 钱包	| 支持充值及充值记录查看。
| 优惠券 | 查看有效优惠券、过期优惠券及激活优惠券。
|===

* **消费账单**
+
[cols="1,3"] 
|===
| 功能 | 说明

| 账单总览	| 支持按月查看账单概况，包括资源消费、资源抵扣及账单发票情况，并支持账单导出。

| 消费明细	| 支持查看每个月的消费明细，并支持多维度条件筛选及数据导出。

| 消费预估	| 支持根据当前的按需资源、包年包月资源以及即将产生续约操作的预留资源进行费用预估。
|===

* **订单管理**
+
[cols="1,3"] 
|===
| 功能 | 说明

| 活动订单管理	| 支持按照订单状态、产品类型及时间查询订单及查看订单详情。
| 续费管理	| 支持预留合约、包年包月产品的自动及手动续费。
|===

* **发票管理**
+
[cols="1,3"] 
|===
| 功能 | 说明

| 发票管理	| 支持申请开具发票及管理发票抬头。
| 邮寄地址管理	| 支持以添加及保存常用的合同/发票邮寄地址。
|===

* **合同管理**
+
[cols="1,3"] 
|===
| 功能 | 说明

| 合同管理	| 支持创建框架合同、充值合同和消费合同。
| 甲方信息管理	| 支持保存甲方信息，包含通讯地址、联系人、联系电话和电子邮箱等。
|===
