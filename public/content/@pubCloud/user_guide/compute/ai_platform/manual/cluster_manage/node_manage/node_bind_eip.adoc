AI 算力集群中的计算节点通过绑定公网 IP 后，可通过公网进行登录访问。

== 前提条件

* 已创建 AI 算力集群。
* AI 算力集群和公网 IP 所在区一致。

== 绑定公网 IP

. 登录管理控制台。
. 点击**产品与服务** > **计算** > **AI 算力平台**，选择左侧导航栏**集群管理**。
. 在**节点列表**页签，点击指定节点**操作**列的更多操作，选择**绑定公网 IP**。

. 在弹出的**绑定公网 IP**窗口中，选择可用的公网 IP，点击**确定**即可。若无可用的公网 IP，则可点击**前往公网 IP 页**进行申请后，再进行绑定。
+
include::../../../../../../../_components/note.adoc[]
在申请公网 IP 时，其**绑定方式**需勾选**外部绑定**。 
include::../../../../../../../_components/table_footer.adoc[]

== 解绑公网 IP

. 登录至 AI 算力平台的集群管理页面。
. 选择**节点列表**页签，点击待解绑公网 IP 节点**操作**列的更多操作，选择**解绑公网 IP**。
. 在弹出的二次确认提示窗口中，点击**确定**即可。
