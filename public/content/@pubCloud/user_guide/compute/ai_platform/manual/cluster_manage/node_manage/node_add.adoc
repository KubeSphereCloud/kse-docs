---

AI 算力集群创建完成后，支持新增计算节点。

== 限制条件

仅支持添加计算节点。

== 注意事项

* 添加节点可能会导致服务中断，建议在业务低峰时进行。
* 添加新的计算节点将会产生额外的费用。

== 操作步骤

. 登录管理控制台。

. 选择**产品与服务** > *计算* > *AI 算力平台*，在左侧导航栏选择**集群管理**。
. 在**节点列表**页签，点击**添加节点**。
+
image::/images/cloud_service/compute/ai/node_add_1.png[]

. 在弹出的**添加节点**窗口中，设置**节点数量**和**节点名称**，点击**确认添加**。
+
include::../../../../../../../_components/note.adoc[]
添加的计算节点配置与原集群内的节点配置一致。
