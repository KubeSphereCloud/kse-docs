{coreshub_name_cn}满足面向人工智能场景的资源与服务需求，为用户提供 GPU 主机、 AI 训练集群、AI 推理集群、并行文件存储、镜像仓库等 AI 专用产品，租户隔离，提供安全、可靠的云上开发与训练环境


== 功能架构

image::/images/cloud_service/compute/coreshub/func_arch_1.png[,90%]

////
* 通过对算力的特点分析与应用场景分析，实现对 GPU 算力、多存储等多元资源的统一管理，包含资源组、算力调度、任务调度、GPU Direct、NVLink、RDMA 等技术、支持从模型微调到大模型训练业务需求。

* 在计算颗粒度层面，支持显卡直通、多点并行。其中，部分数据中心 GPU 卡支持再次对显卡显存资源再切分，如 NVIDIA A 和 H 系列，NVIDIA A100 80G 可切分规格为 1/7 GPU A100 10G、1/3 GPU A100 20G、1/2 GPU A100 20G，用户可根据需求选择不同颗粒度的计算资源，系统将根据算力调度策略进行分发使用。

* 用户自服务层面，提供用户充值使用到释放资源的全流程服务。用户可以选择共享资源按任务占用卡时计算，也可购买专属资源按需、按配置包年包月计费，内置全面的计算环境镜像仓库、函数库、驱动等软件支持，全面提升算力中心的运行效率与平台能力。
////

== 产品优势

* **史诗级算力**
+
单节点 8 张 NVIDIA 数据中心卡，单机 NVSwitch 架构，结合高性能 CPU 为您提供强劲算力。

* **极高网络性能**
+
基于 Mellanox HDR InfiniBand 解决方案，支持 InfiniBand 无限带宽技术，具有极高的吞吐量和极低的延迟。

* **专用高速存储服务**
+
服务器本地 NVme 存储，支持挂载全闪并行文件存储。独享互联网出口带宽。

* **弹性扩容**
+
根据您的业务增长，可不间断业务弹性增加云上 AI 算力资源。

* **深度学习环境集成**
+
集成 CUDA、Python、Docker 等能力，提供公有镜像仓库 TensorFlow、PyTorch、Horovod 等深度学习框架，以及自定义镜像仓库。

* **高端专属服务**
+
1v1 专属管家式服务，保障您的业务持久稳定运行。
