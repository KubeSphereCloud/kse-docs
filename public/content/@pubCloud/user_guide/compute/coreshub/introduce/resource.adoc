== 资源优势

* *极致算力*
+
NVIDIA 显卡，结合高性能 CPU 为用户提供强劲算力。

* *极高网络性能*
+
基于 Mellanox HDR InfiniBand 解决方案，支持 InfiniBand 无限带宽技术，具有极高的吞吐量和极低时延。

* *完美配套服务*
+
服务器本地采用 NVMe 存储设备，支持挂载全闪并行文件存储。独享互连网出口带宽。

* *弹性扩容*
+
根据用户业务增长，可不间断业务的同时，弹性增加云上 AI 算力资源。

* *深度学习环境集成*
+
集成 k8s 调度系统。 +
公共镜像集成 CUDA、Python 等基础功能外，同时也集成 TensorFlow、PyTorch 等深度学习框架。
