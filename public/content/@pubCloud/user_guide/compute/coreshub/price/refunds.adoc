本小节主要介绍 AI 智算平台的退订说明及相关操作。

== 退订

针对按**包年/包月**计费的资源，用户可以随时删除资源，退订相关服务，但已经产生扣费的资源将会按照以下方式计算退订费用。

[,shell]
----
退订费用 = 原支付费用（含折扣） - 优惠券抵扣 - 已消耗的费用
----

include::../../../../../_components/note.adoc[]
* 若退订费用 > 0，退还费用。
* 若退订费用 \<= 0，不予退还。
include::../../../../../_components/table_footer.adoc[]

== 停止计费

针对**按需计费**的资源，用户若需要停止计费，则需删除资源。

删除资源后，按需付费的所有服务将立即释放。

**按需计费**的资源，按小时计费，按小时扣费。若删除资源时，相应资源的实际使用时长不足 1 小时，系统将会退还未使用时长的费用。

include::../../../../../_components/note.adoc[]
资源删除后不可恢复，请备份数据后，再执行删除操作。
include::../../../../../_components/table_footer.adoc[]
