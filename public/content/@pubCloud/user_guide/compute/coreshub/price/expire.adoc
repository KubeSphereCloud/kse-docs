== 到期说明

**包年/包月**的**资源**，若创建时勾选**自动续约**，则到期后可自动续约 1 个月。

若未勾选**自动续约**且未手动续约，用户将无法正常使用资源。存储资源默认保留 24 小时，若逾期未及时续费，相关数据将会被删除。

== 欠费说明

若账户的可用额度（含账户余额、代金券、优惠券等）小于待结算的账单，即被判定为账户欠费。账户欠费后，根据 AI 智算平台各计费项的计费模式，欠费后的处理策略如下所示。

* 包年包月：已预先支付了费用，不影响使用。
* 按量付费：欠费后，相关资源将无法使用。

== 费用充值

账号余额不足时，请及时充值，以保障资源的正常使用。

青云账号支持多种充值途径：支付宝、网上银行、微信支付、线下银行转账。可在link:https://console.qingcloud.com/finance/wallet/[充值页面]选择。

如果需要发票，请到link:https://console.qingcloud.com/finance/invoices/[发票管理]提出申请。

== 余额不足提醒

青云系统会定期检查用户余额和当时名下计费资源的消费预估，如果检查发现余额即将不足，会提前给用户发送通知。

通知策略是：分别在提前 30，15，7，3，1 天时发送提醒。

默认是发送给账号的注册手机号及通知邮箱，如果用户希望自定义接受者列表，可在消息中心的link:https://console.qingcloud.com/notify/receiveConfig[消息接收配置页面]中修改**财务消息**对应的消息接收人。
