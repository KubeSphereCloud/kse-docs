{coreshub_name_cn}的总览页面展示了当前平台内的所有实例、存储空间、资源组内的计算资源状态、最近任务状态等信息，可供用户**快速查看或访问**当前平台各种资源与训练任务。

== 前提条件

* 已经获取 {console_name}账户和密码。
* 平台内已创建的有各项资源、实例或任务等。

== 操作步骤

. 登录 {console_name}。
. 在顶部导航栏，点击**产品与服务** > **{coreshub_name_cn}** > **{coreshub_name_cn}**，进入其**总览**页面。
+
image::/images/cloud_service/compute/coreshub/overview_1.png[]
+
页面信息说明：
+
* ``容器实例``：展示当前平台内已有的容器实例的总数、运行状态中容器实例的数量，以及容器实例中计算资源系统盘和数据盘的使用情况。容器实例的创建可参考link:../develop_training/k8s/k8s_create[创建容器实例]相关章节内容。
* ``文件存储``：展示当前平台内并行文件存储总容量、使用容量以及可用容量的基本信息。
* ``快速访问容器实例``：展示当前平台内可快速访问的容器实例，用户可通过快速连接直接登录至相应实例的 JupyterLab 页面。
* ``我的资源组``：展示当前平台内已创建的资源组数量，所有资源组内计算节点数量信息以及 GPU 卡数信息。资源组的创建可参考link:../res_manage/res_group/group_create[创建资源组]相关章节内容。 
* ``最近任务``：展示当前平台内最近训练任务的名称/ID、状态、框架/镜像、创建时间、开始运行时间、更新时间、运行时长以及相关操作。
