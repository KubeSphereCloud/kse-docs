JupyterLab 是一个开源的 Web 应用程序，为用户提供了一个交互式的开发环境，用户在 JupyterLab 上可创建和共享包含实时代码、方程、可视化和叙述性文本的文档等。


== 界面介绍

JupyterLab 界面主要包括两大部分，即文件浏览区和工作区。

image::/images/cloud_service/compute/coreshub/develop_training_k8s_login_1.png[]

== 上传文件

. 在**文件浏览区**，双击文件夹名称，进入待上传文件的文件夹内。

. 点击文件浏览区上面的文件上传标识，在弹出的文件目录中选择待上传的文件即可。
+
image::/images/cloud_service/compute/coreshub/develop_training_jupyter_1.png[,50%]


== 文件/文件夹管理

在**文件浏览区**，右键点击指定文件夹或文件，即可进行文件管理。

image::/images/cloud_service/compute/coreshub/develop_training_jupyter_2.png[]

== 打开/访问终端

在**工作区域**，点击 **Other > Terminal** 即可新打开一个终端。JupyterLab 在关闭终端或 Notebook 选项卡后，默认不会终止运行。
