{coreshub_name_cn}集成内置了容器镜像仓库，为用户提供即开即用的**预置镜像**仓库，包括 TensorFlow、Pytorch、Jupyter 等。同时，支持用户**自定义镜像仓库**，将自己本地的计算环境上传至镜像仓库中使用。

== 预置镜像

在预置镜像列表页面，用户可直接使用预置镜像提交训练任务。这些预置镜像经过充分的验证，并提前内置了常用安装包或工具包，用户无需花费过多时间进行环境配置。目前已有预置镜像如下表。

== 自定义镜像仓库


若预置镜像无法满足实际的开发需求，用户可构建自定义镜像。{coreshub_name_cn}支持用户基于**预置镜像**或者 **Dockerfile**，自行配置环境依赖，从而构建自定义镜像，用于创建训练任务。
