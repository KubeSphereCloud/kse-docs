== 云服务器挂载硬盘类型

[cols="1,1,3", options="header"]
|===
|云服务器级别| 云服务器类型 | 支持挂载硬盘类型
|入门级
|基础型
|容量型云盘或通用型 SSD 云盘

|企业级
a|* 计算型
* 通用型
* 内存型
|增强型 SSD、容量型、企业型 SSD、通用型 SSD

|专业级
a|* GPU 云服务器
* 弹性裸金属云服务器
|增强型 SSD、容量型、通用型 SSD
|===

== 云服务器和硬盘区域（zone）

云服务器需要与硬盘所属同个可用区，如云服务器（北京 3 区-B） 硬盘也必须为北京 3 区-B；

== 副本策略

云服务器副本策略是否与硬盘副本策略一致。

例：云服务器副本策略为同数据中心多副本 -- 硬盘副本策略也需一致。
