本小节主要介绍如何屏蔽云服务器操作系统类型。

== Console 位置

在创建云服务器时，可以选择云服务器的操作系统镜像：

image::/images/ep_offline/cadmin/example_iaas_image_00.png[,70%]

== 参数路径

[cols='1,1,2']
|===
| 应用 | 配置 | 参数

| webconsole-iaas
| iaas-image
| unsupported_os_family：配置不支持的操作系统。
|===

== 操作步骤

. link:../../../manual/login[登录配置中心]。
. 在左侧导航栏，选择**配置管理**，进入配置管理页面。
. 选择 `webconsole-iaas` 应用，选择 `iaas-image` 配置。
+
image::/images/ep_offline/cadmin/example_iaas_image.png[]
. 在 iaas-image 配置所在行，点击右侧的**管理配置**，进入 iaas-image 配置页面。
+

include::../../../../../../_components/note.adoc[]
配置页面支持表单模式和文本模式，默认为表单模式（默认值可以自行定义，详细操作请参见link:../../../manual/system[系统设置]），可以通过右侧的按钮临时进行切换。

include::../../../../../../_components/table_footer.adoc[]
+
找到 **unsupported_os_family* **参数，点击**添加数组项**，添加需要屏蔽的操作系统。下图表示在 Console 页面屏蔽 redhat 操作系统。
+
image::/images/ep_offline/cadmin/example_iaas_image_01.png[]

. 配置完成后，点击**启用**，弹出的保存并启用对话框。
. 输入描述信息，点击**保存并启用**，完成操作。
+
大约一分钟后生效，登录 Console 可在创建云服务器页面看到修改效果。
