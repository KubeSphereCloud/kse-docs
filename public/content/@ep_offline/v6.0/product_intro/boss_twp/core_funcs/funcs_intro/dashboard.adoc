== 资源总览

资源总览是对资源情况的总体概览，主要体现平台现状和资源池对未来的支撑。现状体现在平台健康状况，平台资源监控，虚拟资源使用情况，最近告警记录和基础服务的异常；资源池容量主要从物理资源使用率，虚拟资源使用率角度来评估资源池的使用情况。

== 计算大屏

云计算管理平台的大屏展示可用于公司监控室，外来参观和日常运维等场景，将总体情况投放到大屏幕上展示，是对于用户物理资源和虚拟资源的可视化展示，使用户对当前资源使用情况一目了然。方便用户快速掌握重点业务的运行情况。

青云QingCloud 云计算管理平台的大屏主要由四部分组成。左侧是用户资源概况，主要统计用户对云服务器、硬盘以及公网/基础网络IP的使用情况，并根据过去三个月的使用情况预测未来三十天资源的使用趋势。中间占用最大体积的是计算节点压力概况，主要展示该系统所有计算节点中的内存、CPU、数据盘、系统盘及备份盘的使用率，并根据过去的使用情况预测未来24小时内的使用率趋势。右边展示的是计算节点运行概况和计算节点使用率资源使用率前五的计算节点。

== 物理机总览

物理机也即弹性裸金属服务器，弹性裸金属服务器总览能够让管理员对云平台上的裸金属服务器运行状况有一个宏观上的了解。主要是对弹性裸金属服务器的状态、告警、资源使用率、资源池容量等信息进行统计，并从标签维度对裸金属服务器进行统计排行。

== 用户总览

用户总览主要是从不同维度对云平台上的租户信息进行统计并排行。主要记录“活跃中”、“已禁用”租户占比；租户上已存在的子账户数量占比排行；不同区域下的租户虚拟主机用量排行；不同区域下的租户虚拟硬盘使用空间排行；不同区域下的租户物理主机使用数量排行。
