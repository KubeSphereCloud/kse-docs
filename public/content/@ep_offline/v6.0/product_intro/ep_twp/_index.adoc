本文档主要介绍青云企业云平台的整体产品及技术架构，以及平台下的基础产品及公共服务相关的功能架构及技术原理。

== 适用版本

本文档适用于企业云 v6.1 及以上版本。

== 适用读者

本文档适用于对企业云产品的基础概念、功能、基础使用流程有一定了解，想要对平台及子产品的技术实现、工作原理等方面进行更深入了解的读者，包括但不限于以下类型读者：

* 售前工程师
* 销售经理
* 市场经理
* 技术支持工程师
* 运维工程师
* 研发工程师

本文预期读者对云平台中相关基本概念及名词有一定了解，若您不具备相关知识，也可在阅读过程中查阅link:./10_reference/[相关参考信息]。
