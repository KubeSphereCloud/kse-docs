云防火墙服务是一种 SaaS 服务，通过引流模块实现南北向流量牵引，用于防护进出公网 IP 的网络攻击。整体架构如下图所示。

image::/images/ep_offline/ep_twp/cfw_02.png[]

云防火墙包含**托管云防火墙资源池**和**云平台引流模块**两部分。

* *托管云防火墙资源池*
+
选定一个云平台的管理账户（如 *admin@yunify.com#scs*）作为云防火墙资源池的虚拟资源存放租户。在这个托管云租户下，会创建系统必须的云资源，如 VPC、云防火墙实例等。云防火墙实例的云资源属于托管云租户，用户只有使用权，无资源拥有权。
+

include::../../../../../../_components/note.adoc[]
云防火墙资源池表示存放防火墙镜像的资源池，而云防火墙实例表示使用防火墙镜像创建的云服务器实例。

include::../../../../../../_components/table_footer.adoc[]

* *云平台引流模块*
+
云平台提供网络流量调度功能，基于引流模块可以将公网 IP 的流量牵引到云防火墙实例，从而实现透明无感的南北向网络防护能力。
