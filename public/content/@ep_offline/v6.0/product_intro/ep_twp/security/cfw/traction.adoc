云防火墙 CFW 基于云平台引流组件 MCN 实现流量牵引。

以外部公网访问云服务器为例：

* 未使用云防火墙时，公网 IP 入方向流量路径为：Internet > VG > VM。
* 使用云防火墙进行牵引后，公网 IP 入方向流量路径是：Internet > VG > MCN > CFW > MCN > VG > VM。其中，云防火墙实例会有两张网卡，一张网卡用于管理，例如运维管理、业务配置等，另一张网卡用于业务，即流量牵引的专有线路。如下图所示。
