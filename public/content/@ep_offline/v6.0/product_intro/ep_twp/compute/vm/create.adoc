image::/images/ep_offline/ep_twp/vm_create_process.png[]

. 前端通过 API/SDK/CLI 发起创建请求。
. 后端根据link:../place/[安置规则]查找安置服务器，明确用于创建云服务器的计算节点。
. 准备创建云服务器的所需资源，生成 XML 文件，XML 文件定义了主机的 CPU、内存、cpu_model 等配置。
+
* 准备存储空间用于存放云服务器相关文件。
* 准备创建云服务器的镜像文件。
* 准备云服务器系统盘用于存放系统盘相关文件。
* 准备云服务器的系统文件，包括主机的密码、密钥以及主机名等配置文件。
. 根据 XML 文件，使用 `virsh define xxx.xml` 命令创建云服务器实例。
. 主机创建成功后, 会执行主机系统盘/数据盘限速，以及相关网络的配置。
