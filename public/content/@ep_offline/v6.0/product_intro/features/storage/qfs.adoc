文件存储 QFS 是面向海量非结构化数据的通用数据存储平台。采用分布式架构，兼容完整的 POSIX 语义，支持标准的 SMB 、NFS、FTP 等文件访问协议，可轻松承载多种业务应用，为用户提供高可靠、易扩展、高性价比、简单易用的海量数据存储服务。

[discrete]
=== 功能介绍

* 支持租户创建、管理自己的文件系统资源。
* 支持通过 NFS 协议、SMB 协议进行访问和共享。
* 支持设置文件系统共享级别的访问权限。

[discrete]
=== 价值说明

文件存储 QFS弥补了当前 vNAS 产品在稳定性、扩展性、性能上的不足，更好的满足了客户对于分布式文件存储的需求。同时对于大型的私有云客户而言，能够在云平台上管理操作文件存储，极大的方便了不同部门对文件存储的使用，同时也降低了相应的运维成本，QFS 的租户资源隔离能力更好地保证了不同租户之间的数据逻辑安全。
