== 支持文件存储

文件存储 EPFS（Elastic Parallel File System）是基于 Lustre 协议的网络共享的文件存储服务，替代传统 NAS 的多节点支持瓶颈，为 POC 用户提供专业的并行文件系统环境。

文件存储 EPFS 采用全闪服务器，用户可以创建自己的文件存储挂载点，在创建 EHPC 集群和图形节点时可以选择挂载此共享存储，集群中的所有节点均可连接该存储，从而实现多节点间的数据共享。


== 对接 IAM

实现集群共享使用，管理员管控集群，成员使用集群计算的场景，面向高校项目组、独立研发部门等提供。
