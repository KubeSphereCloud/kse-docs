== 新增云服务监控功能

云服务监控是所有云服务产品监控数据的展示平台，可以为用户提供账号下的云服务资源列表以及对应的监控指标和告警策略。

支持在云监控 CloudSat 页面和服务自身的监控页面中查看各服务资源的监控图表，了解资源的运行情况。

云服务监控自动获取当前账号下云服务资源，现已对接的云服务有**负载均衡**和**应用服务**（其他服务正在对接中）。

* **负载均衡**：云服务监控支持负载均衡器，用户可以在云监控中查看账号下的负载均衡的监控数据，并支持以负载均衡节点、监听器和监听器后端多维度设置告警策略。
* **应用服务**：云服务监控支持应用服务，用户可以在云监控中查看账号下的集群和部分应用的监控数据，并支持多维度设置告警策略。

== 重构告警服务功能

以 CloudSat 数据为基础提供告警策略、告警记录和一键告警功能。

* 告警策略
+
告警策略为用户提供监控指标告警功能。用户可以对云服务器的监控项和云服务产品的监控项设置告警策略。当监控数据满足告警策略触发条件时，发送告警通知给用户，帮助用户在第一时间了解云服务产品的异常情况，并迅速处理，避免造成业务上损失。
+
image::/images/ep_offline/feature/cloudsat_alarm_record.png[]

* 告警记录
+
支持查看近半年的告警历史记录，帮助用户回溯和定位问题。

* 一键告警
+
提供一键开启指定云服务资源监控的告警服务，帮助用户快速建立云服务的告警体系，以便及时处理异常。
