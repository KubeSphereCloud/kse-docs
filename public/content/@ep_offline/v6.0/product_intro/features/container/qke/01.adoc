== 功能介绍

裸金属节点（BM1）相比虚拟机能带来更好的计算性能和更低的网络延迟，使 Kubernetes 集群发挥更高的优势。QKE 集群将支持使用裸金属服务器作为集群 Worker 节点（工作节点），提升集群计算性能，为用户提供更好的服务。

== 价值说明

降本增效是企业上云的主要目标，资源的利用率和性能是其中的关键因素，在虚拟机上管理 Kubernetes 集群相比在服务器上部署 Kubernetes 仍存在一定资源损耗，金融行业客户在新兴技术领域的追求实践步伐一直领先于其他行业使用者，出于对计算性能和网络低时延的要求，选择裸金属节点作为集群的工作节点承担集群的计算工作是一个可观的方案，可将裸金属节点理解为通过云平台管理的物理服务器，将需要高性能计算的业务在裸金属节点上运行，计算性能和时延表现通常情况下比虚拟机表现更加优越，并且能够实现和虚拟机业务的资源隔离，既支持了对高性能资源的追求，也保证了不同业务资源的隔离。
