---
title: MCN 作为 Border
weight: 01
---

MCN （Multi Cloud Network ）多云网络管理服务，是一个基于软件实现的SDN管理控制器。一般使用两个云平台计算节点，作为 HA 高可用主备节点，用来对 SDN 网络进行编排和管理。

MCN 实现了对 SDN 虚拟网络的编排，主要功能包括：

* 从物理网络到虚拟网络的转换
* 支持不同 VPC 之间的通信
* 对 VLAN 网络的流量进行编排，指定网络内的路由规则，实现流量牵引

企业云 v6.1 版本主要强化了 VPC 之间的通信功能，实现了 VPC Border 的功能。在项目中无需单独采购 H3C 交换机，使用云平台节点即可实现 VPC 之间通信，以及 VPC 对外的通信。
