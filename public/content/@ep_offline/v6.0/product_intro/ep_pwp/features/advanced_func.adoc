== 计算
[cols="1,3"]
|===
|功能特性|描述

|弹性裸金属
|青云企业云平台采用第三代英特尔至强可扩展处理器（代号：Ice Lake）和 DPU 架构，并通过引入智能网卡，实现存储、网络等数据处理功能，让用户可独享 CPU 计算资源，从而实现裸金属服务器的完整算力保留，以极致性能满足用户各类核心业务需求。

弹性裸金属同时具备云服务器的弹性、敏捷性、灵活性与高可用，支持按需使用，且用户的业务应用可直接访问处理器和内存，无任何虚拟化开销。

|GPU 计算
|为满足高性能计算的需求，青云企业云平台提供 NVIDIA Tesla P100 型号的 GPU，单台云服务器可配置 2 张 GPU 卡，单机单浮点峰值计算能力最多可达 20Tflops，并采用直通方式与云服务器对接，省去虚拟化带来的损耗，全面释放物理 GPU 的计算加速能力。

用户可自定义 GPU 云服务器的配置，可灵活指定 CPU、内存、GPU 类型和数量、系统盘大小等。

|弹性高性能计算
|基于企业云平台架构的 EHPC 集群是用于高性能并行计算的资源集，由云平台弹性计算实例组成，包含了调度管理集群资源的高性能调度器，以及运行业务所需的软件栈，解决超大规模的科学，工程和商业问题。

EHPC 集群具备节点动态调整能力，具有弹性伸缩的可扩展性，用户可按需选择计算实例类型，无需排期，即买即用。同时，为用户提供的高性能并行文件系统 EPFS 使用分布式并行架构，具备 GB 级吞吐、百万级 IOPS。

|===


== 存储

[cols="1,3"]
|===
|功能特性|描述

|第三方存储对接
|青云企业云平台支持与第三方存储的对接。

|对象存储
|对象存储是面向海量非结构化数据的通用数据存储平台，为用户提供安全可靠、低成本的云端存储服务，适用于保存图片、视频、镜像、日志等海量非结构化数据，能够支撑移动互联网及物联网业务应用场景，并为其提供数据的生命周期管理。

|文件存储 QFS
|提供一个无限容量的在线文件存储和访问平台，每个用户可以创建多个文件系统用以存储，可以将文件系统挂载给任意客户端使用，文件系统挂载后在客户端体现为不同的目录。

|并行文件存储 EPFS
|并行文件存储 EPFS（Elastic Parallel File System）是企业云平台提供的并行文件系统服务，用于面向超级计算服务等高性能场景。
|===


== 容器服务

[cols="1,3"]
|===
|功能特性|描述

|容器引擎
|容器引擎（QingCloud Kubernetes Engine，简称 QKE）是在青云企业云平台上构建的企业级分布式多租户的 Kubernetes 容器服务，集成了云平台的云服务器、存储、网络等资源。用户可通过容器引擎一键部署高可用、高性能的 Kubernetes 集群。

容器引擎支持可视化界面、多集群管理、自动伸缩、运维监控、CI/CD、微服务治理、应用管理、服务与网络管理等业务场景，让用户轻松高效地在云端运行 Kubernetes 容器化应用。

|Harbor 镜像仓库
|Harbor 是一个开源的镜像仓库服务，用于容器镜像、Helm Chart 等符合 OCI 标准的 Artifact 的安全托管及高效分发，能够帮助用户跨云原生计算平台（如 Kubernetes 和 Docker）一致且安全地管理 Artifact。

AppCenter 将 Harbor 制作成 App，能直接在 AppCenter 进行一键部署，并提供了一个高可用、高安全、高性能的解决方案。

|etcd
|etcd 是为分布式系统设计的分布式高可用键值存储服务，主要用于提供注册服务，配置服务等功能。应用程序可以从 etcd 中读取、写入数据，并监控数据变化。

AppCenter 将 etcd 制作成 APP，方便用户在云平台搭建 etcd 集群。用户可借助 AppCenter 提供的能力对集群进行动态扩容、备份恢复、健康检查和监控。

|===


== 安全

[cols="1,3"]
|===
|功能特性|描述

|安全资源池
|安全资源池与云平台深度融合，基于云平台的多租户和安全产品的能力，解决租户级别的安全产品服务供给问题，让云租户可以根据自身的安全合规要求申请安全产品，从而满足等保相关要求。安全资源池以资产为视角，提供不同种类的安全能力，可提供涵盖网络层、主机层、应用层、数据层的整体安全防护能力，对云上资产进行全方位防护，快速实现等保合规建设。

安全资源池的详细功能介绍，请参考link:../../appedix/function_list/#_安全服务[产品功能列表]相关内容。

|云防火墙 CFW
|云防火墙（Cloud FireWall，简称 CFW）是一款网络安全产品，用于防护进出 EIP 的网络攻击。用户开通云防火墙后 EIP 的流量将被自动引流至云防火墙。经过云防火墙的网络访问会经过拦截策略、入侵防护策略的检查，有威胁的访问会被拦截，并产生安全事件日志。用户可以自定义拦截规则，选择入侵防护的规则，可以看作是安全组功能的加强版。

|===


== 高可用

[cols="1,3"]
|===
|功能特性|描述

|Region 多机房高可用
|跨可用区容灾功能可以在物理架构上提供机房级别的容灾能力。系统将用户数据分散到同城的 2 个可用区，满足用户对关键系统、重要数据的高可用性和数据持久性的要求。当出现火灾、洪水、台风、断电、断网等灾难事件，导致某个机房不可用时，云平台仍然可以提供服务，保证业务不中断，数据不丢失。
