== GUI 管理界面

[cols="1,3"]
|===
|功能特性|描述

|云计算服务平台
|简称 Console，是青云企业云平台面向用户提供的操作使用及运营与一体的界面应用程序。用户可通过 Console 来创建并使用相应的云平台资源。

更多信息，请参考link:../../appedix/function_list/[产品功能列表]。

|云计算运维管理平台
|简称 BOSS，是青云企业云平台面向用户提供的运维管理界面程序。用户可以通过 BOSS 对云平台中的资源和服务进行统一运维，如监控、告警等。

更多信息，请参考link:../../appedix/function_list/[产品功能列表]。

|===


== 计算

[cols="1,3"]
|===
|功能特性|描述

|云服务器
|通过优化的 KVM 虚拟化技术，提供图形化控制台对虚拟服务器的全生命周期进行管理，支持创建、更新、迁移、重启和删除等操作。

|SSH 密钥
|云服务器支持 SSH 密钥登录，较普通的用户密码登录方式更为安全可靠，使得入侵者破解密钥的难度远远超出破解密码的难度。

|镜像
|青云企业云平台提供包含 Linux、Windows 等操作系统的系统镜像，也支持用户自行定义的自有镜像。使得用户方便的使用系统镜像或自有镜像创建云服务器。

|===


== 存储

[cols="1,3"]
|===
|功能特性|描述

|分布式存储
|支持 SDS 2.0、SDS 3.0。

|硬盘
|青云企业云平台为云服务器提供块存储设备，支持多种规格和类型，并可弹性扩展，可满足用户不同场景的业务需求。支持硬盘多副本与定时备份。

|文件存储 vNAS
|基于标准的 NFS 和 Samba（CIFS）网络协议实现的分布式文件存储服务，使得用户可在多个主机客户端以及不同的操作系统间进行数据共享。为用户提供高性能、强一致性的文件存储服务。支持按需收费，弹性可扩展。

|备份
|为用户提供云服务器以及云硬盘的备份恢复功能。支持全量备份与增量备份。

|===


== 网络
[cols="1,3"]
|===
|功能特性|描述

|VPC 网络
|专属私有网络 VPC 可帮助用户构建一个专属隔离的网络环境。在 VPC 网络内，用户可以自定义 IP 地址范围、创建子网，并在子网内创建云服务器、数据库、大数据等各种云资源。

|VBC 基础网络
|VBC 基础网络由企业云平台统一规划和管理，自动分配内网 IP，易用性强。

|NAT 网关
|分布式的网络地址转换服务，可以为私有网络的云服务器提供复用公网 IP 的能力。支持 SNAT 和 DNAT 功能，使多个云服务器可以共享公网 IP 访问互联网或提供互联网服务。

|负载均衡
|提供安全快捷的流量分发服务，使得来自多个公网地址的访问流量经由负载均衡器后，可以自动分配到多台云服务器上。 +
支持自动检测与隔离故障云服务器，提高业务系统的服务能力和可用性。 +
支持千万级别并发访问请求，可轻松应对大流量访问，满足业务需求。

|安全组
|通过策略配置，安全组能极大地提高一个内部网络或云服务器的安全性，并过滤不安全的服务，从而降低内部网络或云服务器的风险。

|网络访问控制
|网络 ACL 是私有网络的进出流量控制表。用户可自主创建和配置 ACL 规则，声明哪些流量可以进、出私有网络，并通过关联私有网络的出、入方向规则控制出入私有网络的数据流量。

|===


== 安全

[cols="1,3"]
|===
|功能特性|描述

|Web 应用防火墙（WAF）
|提供 Web 安全防护，为 Web 应用拦截各种 SQL 注入、XSS 跨站脚本、网站挂马等常见攻击。过滤海量恶意访问，并持续更新防护策略，从而降低网站资产和数据泄露的风险，保障 Web 应用的稳定性。

|===
