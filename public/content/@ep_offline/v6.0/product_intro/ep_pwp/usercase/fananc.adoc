== 用户概况

互联网金融品牌“钱大掌柜”。经过两年多的发展已拥有百万数量级终端客户，2015 年共计销售财富管理产品 6800 多亿元，形成了一个汇集银行理财、信托、基金、贵金属、银证转账、余额理财、金融资产交易所产品等多类型财富管理业务的综合化开放式平台。

== 业务挑战

* 业务量呈现大流量、高并发和大数据的态势。
* 随着业务产品研发周期越来越短，对于 IT 响应速度要求越来越高。
* 随着“银银平台”规模的扩大，“钱大掌柜”同样面对的生态类型也越来越广。

== 解决方案

* 通过私有云中的“资源编排”功能支撑 DevOps 模式下的敏捷开发理念，使得“钱大掌柜” APP 中新功能和新业务的上线周期可以缩短到两周，这样可以针对金融市场和客户需求变化，做到快速响应。
* 利用青云私有云 SDN 模块中的负载均衡功能支撑在业务高峰期的请求转发。
* 在“钱大掌柜”业务规模扩张的过程中，青云软件定义存储为高 IOPS、存储资源池无缝扩容和数据安全提供了保证。
