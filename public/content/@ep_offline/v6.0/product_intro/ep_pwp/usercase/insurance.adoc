== 解决方案

* 基于 QingCloud 云计算产品构建了泰康集团云平台，为集团提供了响应快速、动态灵活、可持续的云计算能力。目前规模已达 1400 台物理机，近 20000+ 虚拟机。
* 通过青云全栈云解决方案构建泰康云平台，可交付云服务器 VM、物理裸机 BM、分布式块存、分布式对象存储、软件 SDN 和各类 PaaS 服务，为泰康保险互联网保险业务战略转型目标的落地提供强有力的技术支撑。

== 客户收益

* 通过企业云平台，泰康保险对业务流程进行了优化和改造，满足了互联网业务的便捷、简单、高效的需求，从而解决传统业务系统的效能问题。
* 为未来泰康保险公有云服务的自主运维系统提供开发测试环境，加速业务产品和功能的迭代，更好地适应金融行业的发展与创新。
* 通过财险业务的全面上云，实现了从传统架构向云计算平台的平滑过渡与迁移。目前泰康保险财险业务的核心数据库也部署在青云的分布式存储集群上，增量业务可 100% 以云计算平台作为基础架构进行设计部署。
