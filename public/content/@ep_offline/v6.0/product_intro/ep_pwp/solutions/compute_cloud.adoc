== 方案简介

随着社会智能化及产业数字化的蓬勃发展，一方面要求多元算力架构具备海量数据处理分析能力，另一方面希望算力资源能够弹性高效，使用户可以以较低成本，获得更加灵活的通用算力服务。

青云科技算力云基于成熟的青云企业云平台产品，融合开源技术、开放的应用集成框架，在传统 HPC 超算平台和数据中心的基础之上，构建多元算力操作系统来满足多样化的算力交付。一方面对计算、存储、网络、安全进行统一资源管理和调度；另一方面整合应用统一交付、云计算服务模式、智能数据中心运维运营等功能，让用户可以方便、灵活、高效、低成本的使用和管理算力服务。支撑不同类型的数据计算分析、应用运行和业务处理。

按服务层面划分，青云算力云提供了四个层面的服务平台：

* 一是算力生产供应平台。包括多样性算力、标准云平台算力服务、HPC 高性能计算能力、物联网和边缘计算能力以及 GPU、AI 为主的 AI 算力能力。

* 二是数据开放共享平台。依托青云多年在数据、大数据、PaaS 平台上的经验，基于大数据工作台来支撑企业数据在生产、采集、计算上的服务要求。
* 三是智能生态建设平台。主要提供开放的应用框架，无论是传统架构应用，还是云原生应用、AI 分析类应用，或是各种行业专属应用，都可以通过应用中心的框架进行云化，成为可以被访问的 PaaS 服务或者 SaaS 服务，从而创建丰富的计算服务生态和行业应用生态。
* 四是产业创新开发平台。聚焦行业应用开发和迭代，基于低代码开发平台、DevOps 流程开发平台，以及 FaaS 等技术能力，为业务流程的重建和业务场景的迭代提供有力的工具。

青云算力云使四个平台的服务能力相互融合，实现应用、算法模型对算力的高效驱动，行业和算力的有效连接，从而更好地支持算力在工业仿真、生命科学、智能制造、医药分析、地理海洋等相关应用领域的落地和应用。

== 方案优势

* 多元算力
+
利用云计算技术实现对多种主流的 CPU、GPU、NPU、FPGA 等异构算力的统一分配和调度，并且具备将异构算力统一封装提供服务的能力；利用云计算技术为算力中心赋能，一站式提供多种算力服务：HPC 高性能计算云服务、人工智能算力服务、通用云平台服务。

* 自主可控
+
青云坚持自主研发、自主创新的发展道路，算力云核心技术与核心组件全部自主研发，具备自主的知识产权。青云与众多信创芯片、操作系统、服务器、数据库、中间件等产业上下游厂商达成深度合作，积极完成产品的兼容适配。算力云能够提供全栈信创云计算的服务能力，屏蔽底层信创硬件差异。真正实现平台研发、升级、维护全过程的安全可控，确保应用负载持续、安全、稳定的运行。

* 创新云服务模式
+
青云算力云为传统的超算服务融入了云计算对 IT 资源管理的先进理念，重新构建了 HPC/EHPC 服务，实现安全的多租户管理。用户可自助创建登录节点，调用平台提供的超大规模物理计算资源，根据使用需求，弹性定制用户专属的资源并自行安装应用软件，从而实现高性能服务的高度灵活性，让新的超算服务具备 HPC 的超高性能，以及云计算的安全，弹性和可扩展性等性能。

* 以行业应用为核心，统一交付
+
青云算力云提供了一套用于快速开发、部署、运营云应用的框架，这个框架支持多元异构算力平台，解决传统 HPC 应用程序安装、升级以及安装包之间的依赖性等问题，让系统的部署和运维变得更加高效，可以在高性能计算场景中广泛使用和推广。同时，算力云中所构建的应用可以在各种商业云环境以及当下最热门的云原生容器云中广泛使用，从而促进建立完整、多元与开放的应用生态。

* IOT 物联网技术打造高效能智慧化智算中心
+
青云算力云依托 IOT 设备对数据中心海量设备进行数字化管理，实现数据中心的智慧化赋能。一方面摆脱传统人工运维模式下的低效管理模式，另一方面对设备能耗进行精细化的人工智能管理，可以促进节能减排、优化 PUE，是实现 IT 行业履行“双碳政策”的有效举措。
