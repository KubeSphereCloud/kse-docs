根据产品功能，企业云平台 License 分为 PaaS License 和 IaaS License。

* PaaS License：按 vCPU 配额、应用集群数量授权。可以精确控制每一个应用所使用的 vCPU 配额，同时还可以控制应用的集群数量。
* IaaS License：按计算节点数量授权。可以控制云平台中计算节点（Hyper节点）数量。

根据 License 有效期，企业云平台 License 分为试用版 License、POC License 和生产环境 License。
[cols='1,2,1']
|===
| 分类	| 有效期	| 启用时间

| 试用版 License
a| 有效期为 3 个月。
