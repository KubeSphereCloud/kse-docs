从物理角度，企业云平台由多种角色的节点构成。

image::/images/ep_offline/installation/intro_architecture_01.jpg[,60%]

* 虚拟网关节点（Vgateway，VG）：VG 相当于一个 VXLAN 路由器，提供了对内部网络的基本安全保护，以及负载均衡器集群的流量转发控制。
+
在 VG 节点上运行的服务包括：
+
[cols="1,2,2"]
|===
| 服务 | 描述 | 部署说明

| vgateway
| （可选）虚拟网关服务。
| 若没有 VG 节点，不用部署该服务。

| proxy
| Web Console接入代理。
| 若没有 VG 节点，该服务部署在 KS 节点。

| dnsmaster
| DNS 服务，为云平台提供内网域名服务。
| 若没有 VG 节点，该服务部署在 KS 节点。
|===

* 管理节点（Kernel Service，KS）：提供企业云平台的基础管理功能，包括内部数据库、Web 界面、API 接口、租户信息管理、计费管理、BOSS 系统、VM 镜像库、VM 备份服务等。
+
在 KS 上运行的服务包括：
+
[cols="1,3"]
|===
| 服务 | 描述

| dnsmaster	 a| DNS 服务，用于提供内网域名服务节点。

include::../../../../../_components/note_in_table.adoc[]
若有 VG 节点，该服务部署在 VG 节点。

include::../../../../../_components/footer_in_table.adoc[]

| zoocassa	a| zoocassa 为一组服务的统称，包含如下 4 个服务：

* Zookeeper 服务：用于节点的集群管理。
* Cassandra 数据库：用于存放监控数据。
* Redis 数据库：用于计费服务缓存。
* Memcache 数据库：用于前端数据缓存。
| pgserver	| PostgreSQL 数据库，存放云平台管理信息。
| proxy	a| Web Console 接入代理。

include::../../../../../_components/note_in_table.adoc[]
若有 VG 节点，该服务部署在 VG 节点。

include::../../../../../_components/footer_in_table.adoc[]

| webservice	| WebConsole 服务。
| snapshot	a| 数据备份服务。

include::../../../../../_components/note_in_table.adoc[]
若有 VBR 节点，该服务部署在 VBR 节点。

include::../../../../../_components/footer_in_table.adoc[]

| vbr	a| 虚拟备份恢复节点。

include::../../../../../_components/note_in_table.adoc[]
若有 VBR 节点，该服务部署在 VBR 节点。

| seed	| 镜像模板服务节点。
| pgpool	| 数据库连接池。
| boss	| 云计算运维管理平台。
| cronus（influxdb）	| 监控组件数据库。
| warehouse	| 使用计费功能，需要配置该服务。
| hpc	| （可选）超性能计算服务。
| smc	| （可选）迁移中心。比如将虚拟机从 VMware 平台迁移到企业云平台。
| cloudsat	| 云监控服务。
|===
+
如一个标准的 4 节点 KS+VBR  服务，每个节点上包含的服务为：
+
image::/images/ep_offline/installation/intro_architecture_02.jpg[,60%]

* 备份节点（VBR）：VBR 服务器主要用于数据备份，如同一个虚拟带库。可以按需要增加 VBR 节点。
+
在 VBR 节点上运行的服务包括：
+
[cols="1,1,2"]
|===
| 服务 | 描述 | 部署说明

| snapshot	| 数据备份服务	| 若没有专用的备份节点服务器，该服务部署在 KS 节点。
| vbr	| 虚拟备份恢复节点	| 若没有专用的备份节点服务器，该服务部署在 KS 节点。
|===

* 计算节点（Hyper）：为企业云平台提供计算、网络、存储资源，可随业务规模横向扩展。计算节点上需要运行 hyper 服务。
* 对象存储：包括为对象存储服务提供网关代理入口服务的对象存储网关以及对象存储节点。
* 存储节点：根据需要，如果有海量冷数据需要低价存储介质的保存，如 SATA，可以增加专门的存储节点。
