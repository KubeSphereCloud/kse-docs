企业云平台分为基础功能和增值功能。

* 基础功能：云平台基础软件包中默认包含的功能，仅由 IaaS License 控制。
* 增值功能：云平台基础软件包中不包含的功能，需要单独进行安装配置、且由单独的 License 进行控制。

== 功能清单

售卖模式说明如下：

* 默认：基础功能中默认包含，且 Console 中菜单默认打开。
* 可选：基础功能中默认包含，但 Console 中菜单默认不打开，需要单独进行安装配置，不需要额外的 License。
* 增值：需要单独进行安装配置、且由单独的 License 进行控制。
