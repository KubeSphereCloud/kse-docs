企业云平台提供如下图形化操作界面：

* 云计算运维管理平台（BOSS）
+
云计算运维管理平台（Business & Operation Support System，简称 BOSS）是企业云平台的统一运维管理门户，除了为运维人员提供统一的基础运维功能之外，还对云平台中的物理资源、虚拟资源和服务进行统一管理。

* 云计算服务平台（Console）
+
云计算服务平台为云用户提供计算、网络、存储、安全、数据库与缓存、大数据服务、消息队列与中间件、容器服务、运维与监控等服务。

* 配置中心
+
配置中心用于统一集中化管理 Console 和 BOSS 的配置，同时提供可视化操作界面，降低配置维护难度，提升修改配置的效率。
