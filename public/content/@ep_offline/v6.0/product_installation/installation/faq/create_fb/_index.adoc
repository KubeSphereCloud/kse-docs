== 问题一

* **问题现象**
+
创建 FirstBox 时提示“The interface [xxx] you provide is not matched to address [xxx]”。
+
image::/images/ep_offline/installation/faq_fb_01.png[]

* **问题原因**
+
网络模式为 VLAN 模式时，bootstrap_allinone.sh 脚本的 `-m` 参数需指定规划的 VLAN ID。

* **解决方法**
+
--
. 登录已上传企业云平台基础软件安装包的 KS 节点。
. 执行命令 `ip r` ，查看当前 KS 节点的 VLAN ID。
. 按正确命令重新创建 FirstBox。
+
示例：如下以 VLAN ID 为 vlan48 为例。
+
----
/root/qingcloud-installer/bootstrap/bootstrap_allinone.sh -m vlan48 -a 192.168.201.190 -f
----
--

== 问题二

* **问题现象**
+
创建 FirstBox 时提示输入 root 密码。
+
image::/images/ep_offline/installation/faq_fb_05.png[]

* **问题原因**
+
此问题中，FirstBox 的地址 192.168.80.10 是交换机的地址，IP 冲突了。
+
此时即使手动输入密码成功创建了 FirstBox，后续登录 FirstBox 时仍然会有问题。

* **解决方法**
+
VLAN 模式下，建议云平台独享一个 C 段。如果条件不允许，请提前和客户规划好 IP 地址，避免冲突。
