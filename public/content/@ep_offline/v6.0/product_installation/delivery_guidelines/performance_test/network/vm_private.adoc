|===
| 验收目的 | 验证 VM 基于私有网络间的性能。
| 预置条件 | * 企业云平台已部署完成。

* 内置 `ubuntu14.04.3` 操作系统镜像。


| 操作步骤 | . 通过浏览器登录企业云平台的 Web Console。

. 在 Web Console 相应页面创建 VPC 网络 VPC-1。
. 在 Web Console 相应页面创建私有网络 Private-1。
. 将私有网络 Private-1 加入 VPC-1。
. 创建两台 `ubuntu14.04.3` 的云服务器 VM-1，VM-2，分别加入该私有网络 Private-1。
. 防火墙下行放开 `5000` 端口。
. 使用 SSH 工具登录 VM-1，执行命令行启动 Iperf Server 端。
+
[source,shell]
----
iperf -s -p 5000
----

. 使用 SSH 工具登录 VM-2，执行命令行启动 Iperf Client 端。
+
[source,shell]
----
iperf -c <server端 IP> -p 5000 -f M -i 1
----

. 分别观察 VM-1 与 VM-2 上的返回结果。
