| 验收目的 | 在系统不限速的前提下，验证 SANC 超高性能型云服务器的性能。
| 预置条件 | * {platform_name_cn}已部署完成，且当前运行状态正常，所有 Hyper 节点状态均正常。
* 已创建好测试用户 User-1 并分配适当资源配额。
* 清除系统默认设置限速参数。
+
[source,ini]
----
  iotune:
    read_bytes_sec_mass_vol: 0
    read_bytes_sec_vos_vol: 0
    read_iops_sec_flash_vol: 0
    read_iops_sec_vos_vol: 0
    total_iops_sec: 0
    total_iops_sec_vos_vol: 0
    write_bytes_sec: 0
    write_bytes_sec_flash_vol: 0
    write_bytes_sec_mass_vol: 0
    write_bytes_sec_vos_vol: 0
    write_iops_sec_flash_vol: 0
    write_iops_sec_vos_vol: 0
----

| 操作步骤 | 

. 使用 User-1 的账户密码登录{platform_name_cn}的 Web Console。
. 创建一台操作系统为 `ubuntu14.04.3` 的 SANC 超高性能型云服务器 VM-1，并加入基础网络。
. 创建通用型 SSD 云盘 Disk-1，并挂载至云服务器 VM-1。
. 使用 SSH 工具登录云服务器 VM-1，针对 Disk-1 创建分区，并格式化为 `ext4` 文件系统，挂载至 `/mnt` 目录。
. 在 VM-1 上执行如下命令行，开启 FIO 性能测试，并记录测试结果。
+
4K 随机读
+
[source,ini]
----
fio -name=test_1 -filename=/mnt/ -bs=4k -direct=1 -ioengine=libaio -group_reporting --eta-newline=1 -runtime=600 -time_based -rw=randread -iodepth=32 -numjobs=2
----
+
4K 随机写
+
[source,ini]
----
fio -name=test_1 -filename=/mnt/ -bs=4k -direct=1 -ioengine=libaio -group_reporting --eta-newline=1 -runtime=600 -time_based -rw=randwrite -iodepth=32 -numjobs=2
----
+
128K 顺序读
+
[source,ini]
----
fio -name=test_10 -filename=/mnt/ -bs=128k -direct=1 -ioengine=libaio -group_reporting --eta-newline=1 -runtime=600 -time_based -rw=read -iodepth=8 -numjobs=1
----
+
128K 顺序写
+
[source,ini]
----
fio -name=test_10 -filename=/mnt/ -bs=128k -direct=1 -ioengine=libaio -group_reporting --eta-newline=1 -runtime=600 -time_based -rw=write-iodepth=8 -numjobs=1
----


| 预期结果 | 4k 随机写性能区间在 40 ~ 70K IOPS。
