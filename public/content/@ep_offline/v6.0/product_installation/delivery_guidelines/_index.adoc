新安装部署的企业云平台，需经过一系列的环境确认与检查操作，以确保环境安装正确，且性能达到指定要求后，再交付给用户使用。

本文档将详细说明，如何对一个新部署的企业云平台进行交付前的测试验证。本文档适用于企业云平台 v6.1 版本。

== 读者对象

本文档主要适用于以下读者：

* 实施工程师
* 运维工程师
* 技术支持工程师
