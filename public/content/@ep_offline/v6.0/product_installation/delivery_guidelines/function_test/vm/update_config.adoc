---
title: 调整云服务器配置
weight: 3
---


[cols='^1h,4a']
|===
| 验收目的 | 验证云平台支持对已创建的云服务器调整 CPU 与内存规格。
| 预置条件 | * {platform_name_cn}已部署完成，且当前运行状态正常，所有 Hyper 节点状态均正常。
* 已创建好测试用户 User-1 并分配适当资源配额。
* 已成功创建云服务器 VM-1，其配置为 `2核4G`；云服务器 VM-2，其配置为 `2核4G`。

| 操作步骤 | 
. 使用 User-1 的账户密码登录{platform_name_cn}的 Web Console。
. 对云服务器 VM-1，对其进行更改配置操作，调整其配置为 `4核8G`。
. 针对云服务器 VM-2，先将其关机，再对其进行更改配置操作，调整其配置为 `4核8G` 后，再开机。
. Web Console 上确认 VM-1 与 VM-2 的配置。

 


| 预期结果 | . VM-1 不支持更改配置操作，配置为 `2核4G`。
. VM-2 更改配置操作成功，开机、关机操作成功，配置成功调整为 `4核8G`。


| 验证结果 | 
| 备注 | 云服务器需在关机状态下调整 CPU 与内存规格。
|===
