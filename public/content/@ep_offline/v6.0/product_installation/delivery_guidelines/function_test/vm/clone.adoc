| 验收目的 | 验证云服务器的克隆功能。
| 预置条件 | * {platform_name_cn}已部署完成，且当前运行状态正常，所有 Hyper 节点状态均正常。
* 已创建好测试用户 User-1 并分配适当资源配额。
* 已成功创建云服务器 VM-1。

| 操作步骤 | 
. 使用 User-1 的账户密码登录{platform_name_cn}的 Web Console。
. 对云服务器 VM-1 进行克隆云服务器操作。
. 在 Web Console 上观察克隆云服务器的操作是否成功。
. 通过 Web Console 登录新克隆的云服务器。

| 预期结果 | . 克隆云服务器操作成功，在云服务器列表页面可查看新克隆的云服务器。
. 新克隆的云服务器成功分配到 IP，并能成功登录。
