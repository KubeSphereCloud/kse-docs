---
title: 扩容云硬盘
weight: 2
---


[cols='^1h,4a']
|===
| 验收目的 | {platform_name_cn}的云硬盘扩容。
| 预置条件 | * {platform_name_cn}已部署完成，且当前运行状态正常，所有 Hyper 节点状态均正常。
* 已创建好测试用户 User-1 并分配适当资源配额。

| 操作步骤 | 
. 使用 User-1 的账户密码登录{platform_name_cn}的 Web Console。
. 创建云硬盘 Disk-1，容量为 50G。
. 对云硬盘 Disk-1 进行 `扩容` 操作，容量调整为 100G。
. Web Console 上确认云硬盘 Disk-1 的容量为 100G。



| 预期结果 | 未挂载至云服务器的云硬盘扩容操作成功。

| 验证结果 | 
| 备注 | 云硬盘没有挂载至云服务器的情况下才能扩容。
|===
