弹性高性能计算 EHPC（Elastic High Performance Computer，以下简称 EHPC）是基于青云QingCloud 云基础设施，为用户提供基于公有云、私有云、混合云等多种产品形态，面向科研机构、学校教育、工业生产、行业计算等用户提供弹性灵活、快捷高效、安全可靠的超算服务。适用于生命科学、CAE 仿真、海洋气象、影视渲染、石油勘探、深度学习、测绘地理等高性能计算场景。

EHPC 集群所使用的并行文件存储 EPFS（Elastic Parallel File System，以下简称 EPFS） 是基于 Lustre 协议的网络共享文件存储服务，可实现多节点间的数据共享。EHPC 的用户可以提前创建自己的文件存储挂载点，在创建高性能计算集群和图形节点时，选择挂载此共享存储，集群中的所有节点均可连接该文件存储，从而实现多节点间的数据共享。

因此，EHPC 服务的安装部署主要包括 EHPC 应用的安装以及 EPFS 服务的安装。其中，EPFS 服务为 Lustre 文件系统的管理服务，为了方便用户能够便捷的在浏览器页面实现文件的上传与下载操作，部署 EPFS 时需安装 MinIO 服务。 

另外，EHPC 服务的安装部署依赖于企业云平台的搭建和 AppCenter 应用的支持，用户需提前安装部署完成企业云平台以及 AppCenter 服务。一般来说，安装部署企业云平台的过程中会默认自动安装 AppCenter 服务。
