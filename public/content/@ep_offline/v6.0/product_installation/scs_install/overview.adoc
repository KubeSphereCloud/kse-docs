安全产品（security control system，SCS）包含两个产品：云防火墙（CFW）和安全资源池（SINTG）。

* 云防火墙产品包含 scs、srp、logw_cfw、cfw 四个服务。
* 安全资源池产品包含 scs、srp、sintg 三个服务。

include::../../../../_components/note.adoc[]
用户可以根据实际需求，只安装其中一个产品，或两个产品一起安装。
