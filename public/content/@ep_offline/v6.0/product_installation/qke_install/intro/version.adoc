== QKE 版本特性

[cols='1,3']
|===
| QKE 版本 | 更新记录

| v3.0.0
a|

1. 新增裸金属节点作为集群工作节点，支持添加、删除操作，提供更强算力。
2. 支持对接云平台 IAM，提升集群组织级账号权限管理能力。
3. 支持使用 U10000 作为后端存储，并提供页面展示及管理能力，满足多读多写场景要求。
4. 修复了一些问题，提升集群稳定性与资源管理能力。

| v2.3.1
a|

1. 新增集群节点资源预留。
2. 重构了可视化控制台的访问功能，支持集群内部访问和外部访问两种方式。
3. 其他问题和修复，提升产品安全性和资源管理。
|===
