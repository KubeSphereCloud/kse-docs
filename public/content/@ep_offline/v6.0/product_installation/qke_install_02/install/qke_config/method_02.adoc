== 操作步骤

1. 登录一个 Webservice 节点。
2. 执行以下命令，进入 /pitrix/lib/pitrix-webconsole-paas/configs 目录。
+
[source]
----
cd /pitrix/lib/pitrix-webconsole-paas/configs
----
3. 执行以下命令，新增 local_config.yaml 文件。
+
[source]
----
touch local_config.yaml
----
+
内容参照如下 local_config.yaml 示例，主要修改 zone_groups 和 volume_class 相关参数。
+
[cols='1,3'] 
|===
| 参数 | 说明 

| zone_groups a| 

* abroad：海外 region。若没有海外 region，可以删除此参数。
* domestic：国内 region。
| volume_class 相关参数 a| default_volume_class 为默认的磁盘类型。

* 如果 NeonSAN 环境是纯 SSD 环境，则配置 volume_class 5。
* 如果 NeonSAN 环境为混插型硬盘，则配置 volume_class 6。
|===
4. 执行以下命令，重启 apache。
+
[source]
----
service apache2 restart
----
