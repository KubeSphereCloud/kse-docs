---
title: 删除Console（DeleteConsole）
weight: 3
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| console
| String
| true
| Console的ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| console_id
| String
| false
| Console的ID
|===

== 请求示例
[,javascript]
----
{
  "action": "DeleteConsole",
  "console": "zzzz"
}
----

== 返回示例
[,javascript]
----
{
  "action": "DeleteConsoleResponse",
  "console_id": "zzzz",
  "ret_code": 0
}
----
