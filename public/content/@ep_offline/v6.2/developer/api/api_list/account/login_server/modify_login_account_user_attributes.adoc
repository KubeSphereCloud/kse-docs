---
title: 修改第三方登录账号关联用户属性（ModifyLoginAccountUserAttributes）
weight: 12
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| login_account_id
| String
| true
| 登录账号ID

| status
| String
| false
| 状态

| user_id
| String
| true
| 用户ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| login_account_id
| String
| false
| 登录账户ID

| user_id
| String
| false
| 用户ID
|===

== 请求示例
[,javascript]
----
{
  "login_account_id": "login_account_id_1",
  "status": "activated",
  "user_id": "user_id_1"
}
----

== 返回示例
[,javascript]
----
{
  "action": "ModifyLoginAccountUserAttributesResponse",
  "login_account_id": "login_account_id_1",
  "ret_code": 0,
  "user_id": "user_id_1"
}
----
