---
title: 恢复子用户（RestoreSubUsers）
weight: 29
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| users
| Array
| true
| 子用户ID的列表
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| users
| Array
| false
| 子用户ID的列表
|===

== 请求示例
[,javascript]
----
{
  "action": "RestoreSubUsers",
  "users": [
    "usr-xqm5ieFD"
  ]
}
----

== 返回示例
[,javascript]
----
{
  "action": "RestoreSubUsersResponse",
  "ret_code": 0,
  "users": [
    "usr-xqm5ieFD"
  ]
}
----
