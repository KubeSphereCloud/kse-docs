---
title: 确认邮箱验证码（ConfirmEmailVeriCode）
weight: 8
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| user
| String
| true
| 用户

| veri_code
| String
| true
| 验证码
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| email
| String
| false
| 邮箱

| user_id
| String
| false
| 用户ID
|===

== 请求示例
[,javascript]
----
{
  "user": "admin",
  "veri_code": "veri_code_0"
}
----

== 返回示例
[,javascript]
----
{
  "action": "ConfirmEmailVeriCodeResponse",
  "email": "4743*****",
  "ret_code": 0,
  "user_id": "admin"
}
----
