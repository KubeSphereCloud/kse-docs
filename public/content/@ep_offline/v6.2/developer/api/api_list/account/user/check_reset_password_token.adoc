---
title: 检查重置密码的凭证（CheckResetPasswordToken）
weight: 6
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| token
| String
| true
| 
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| user_id
| String
| false
| 用户ID
|===

== 请求示例
[,javascript]
----
{
  "token": "token_0"
}
----

== 返回示例
[,javascript]
----
{
  "action": "CheckResetPasswordTokenResponse",
  "ret_code": 0,
  "user_id": "user_id"
}
----
