---
title: 修改用户IAM域名（ModifyIAMDomain）
weight: 22
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| iam_domain
| String
| false
| iam域名

| user_id
| String
| true
| 用户ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| iam_domain
| String
| false
| iam域名

| user_id
| String
| false
| 用户ID
|===

== 请求示例
[,javascript]
----
{
  "iam_domain": "iam_domain_0",
  "user_id": "user_id"
}
----

== 返回示例
[,javascript]
----
{
  "action": "ModifyIAMDomainResponse",
  "iam_domain": "iam_domain_0",
  "ret_code": 0,
  "user_id": "user_id"
}
----
