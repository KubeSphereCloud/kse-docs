---
title: 删除用户会话（DeleteSession）
weight: 4
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| sk
| String
| true
| 会话key
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| sk
| String
| false
| 会话key
|===

== 请求示例
[,javascript]
----
{
  "action": "DeleteSession",
  "sk": "6XJVdYRNssepxiW8mMXk6cj6x9yvyIfD"
}
----

== 返回示例
[,javascript]
----
{
  "action": "DeleteSessionResponse",
  "ret_code": 0,
  "sk": "6XJVdYRNssepxiW8mMXk6cj6x9yvyIfD"
}
----
