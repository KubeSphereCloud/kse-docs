---
title: 登录服务器认证（LoginServerAuthorize）
weight: 10
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| code
| String
| false
| 授权码，OAUTH2类型的登录服务器认证时使用

| login_ip
| String
| false
| 登录IP

| login_server_id
| String
| true
| 登录服务器ID

| passwd
| String
| false
| 密码，LDAP类型的登录服务器认证时使用

| user
| String
| false
| 用户，LDAP类型的登录服务器认证时使用
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| sk
| String
| false
| 登录服务器认证成功后返回的session_key

| user
| Object
| false
| 认证成功的用户信息

| user.lang
| String
| false
| 云平台用户语言

| user.user_id
| String
| false
| 云平台用户id
|===

== 请求示例
[,javascript]
----
{
  "code": "1",
  "login_ip": "127.0.0.1",
  "login_server_id": "login_server_id_1",
  "passwd": "**********",
  "user": "admin"
}
----

== 返回示例
[,javascript]
----
{
  "action": "LoginServerAuthorizeResponse",
  "ret_code": 0,
  "sk": "********************",
  "user": {
    "lang": "ZH_CN",
    "user_id": "admin"
  }
}
----
