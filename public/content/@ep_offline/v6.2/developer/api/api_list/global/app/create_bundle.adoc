== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| bundle_description
| String
| true
| bundle

| bundle_id
| String
| false
| bundle包ID

| bundle_name
| String
| true
| bundle
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| bundle_description
| String
| false
| bundle

| bundle_id
| String
| false
| bundle包ID

| bundle_name
| String
| false
| bundle
|===

== 请求示例
[,javascript]
----
{
  "bundle_description": "青云出品",
  "bundle_id": "bun-qingcloud",
  "bundle_name": "青云出品"
}
----

== 返回示例
[,javascript]
----
{
  "action": "CreateBundleResponse",
  "bundle_description": "青云出品",
  "bundle_id": "bun-qingcloud",
  "bundle_name": "青云出品",
  "ret_code": 0
}
----
