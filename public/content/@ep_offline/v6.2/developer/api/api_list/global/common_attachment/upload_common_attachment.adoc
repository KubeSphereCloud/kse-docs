title: 上传一般附件（UploadCommonAttachment）
weight: 4
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| attachment_content
| String
| true
| 附件内容

| attachment_type
| String
| true
| 附件类型

| category
| String
| false
| 目录

| filename
| String
| false
| 文件名称

| filext
| String
| true
| 文件格式

| owner
| String
| false
| 拥有者

| resource_id
| String
| false
| 资源ID

| resource_type
| String
| true
| 资源类型

| sub_category
| String
| false
| 子目录
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| attachment_id
| String
| false
| 附件id

| url
| String
| false
| 附件链接
|===

== 请求示例
[,javascript]
----
{
  "attachment_content": "*",
  "attachment_type": "archive",
  "filext": "zip",
  "resource_type": "app"
}
----

== 返回示例
[,javascript]
----
{
  "action": "UploadCommonAttachment",
  "attachment_id": "ca-6n9q88u9",
  "ret_code": 0
}
----
