---
title: 创建消息设置（CreateNotifierSettings）
weight: 1
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| console_id
| String
| true
| 控制台

| smtp
| Json
| false
| smtp邮件设置
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| console_id
| String
| false
| 控制台
|===

== 请求示例
[,javascript]
----
{
  "action": "CreateNotifierSettings",
  "console_id": "devopscloud",
  "smtp": "{\"smtp_server\":\"mail.qin***oud.com\",\"smtp_port\":587,\"ssl_on\":false,\"login_user\":\"notiuser\",\"login_passwd\":\"*******\",\"starttls\":false,\"notify_senders\":[\"no*****@qingcloud.com\"],\"notify_timeout\":15,\"base_email_template\":\"\"}"
}
----

== 返回示例
[,javascript]
----
{
  "action": "CreateNotifierSettingsResponse",
  "console_id": "devopscloud",
  "ret_code": 0
}
----
