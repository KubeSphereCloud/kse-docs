---
title: 删除消息设置（DeleteNotifierSettings）
weight: 2
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| console_id
| String
| true
| 控制台
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| console_id
| String
| false
| 控制台
|===

== 请求示例
[,javascript]
----
{
  "action": "DeleteNotifierSettings",
  "console_id": "devopscloud"
}
----

== 返回示例
[,javascript]
----
{
  "action": "DeleteNotifierSettingsResponse",
  "console_id": "devopscloud",
  "ret_code": 0
}
----
