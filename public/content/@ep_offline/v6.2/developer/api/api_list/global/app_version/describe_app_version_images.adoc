---
title: 查询应用版本依赖的镜像（DescribeAppVersionImages）
weight: 11
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| app_id
| String
| false
| 应用ID

| image_id
| String
| false
| 镜像ID

| limit
| Integer
| false
| 个数

| offset
| Integer
| false
| 偏移

| reverse
| Integer
| false
| 是否倒序

| sort_key
| String
| false
| 排序字段

| user_id
| String
| false
| 用户ID

| version_id
| String
| false
| 版本ID

| zone_id
| String
| false
| 区域
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| app_version_image_set
| Array
| false
| 镜像信息列表

| app_version_image_set[].app_id
| String
| false
| 应用ID

| app_version_image_set[].app_name
| String
| false
| 应用名称

| app_version_image_set[].app_version_name
| String
| false
| 应用版本名称ID

| app_version_image_set[].create_time
| String
| false
| 创建时间

| app_version_image_set[].image_id
| String
| false
| 镜像ID

| app_version_image_set[].user_id
| String
| false
| 所有者

| app_version_image_set[].version_id
| String
| false
| 版本ID

| app_version_image_set[].zone_id
| String
| false
| 区域

| total_count
| Integer
| false
| 总数量
|===

[,javascript]

{

}
