== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| promotion_ids
| Array
| true
| 促销
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| result
| Object
| false
| 查询结果合集

| result.TODO
| String
| false
| 待定义
|===

== 请求示例
[,javascript]
----
{
  "promotion_ids": [
    "prom-trial-0001",
    "prom-trial-0002"
  ]
}
----

== 返回示例
[,javascript]
----
{
  "action": "DescribeLeftEverydayPromotionQuotaResponse",
  "result": {
    "prom-trial-0001": 10,
    "prom-trial-0002": 0
  },
  "ret_code": 0
}
----
