---
title: 从回收站恢复资源（RecoverRecycleResources）
weight: 4
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| owner
| String
| false
| 回收资源的所有者

| resources
| Array
| false
| 通过资源列表过滤查询

| zone
| String
| true
| 请求区ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| job_ids
| Array
| false
| 

| resource_ids
| Array
| false
| 
|===

== 请求示例
[,javascript]
----
{
  "action": "RecoverRecycleResources",
  "owner": "usr-ei6RFhNN",
  "resources": [
    "i-3dxf6id6"
  ],
  "user_id": "usr-ei6RFhNN",
  "zone": "pek3"
}
----

== 返回示例
[,javascript]
----
{
  "action": "RecoverRecycleResourcesResponse",
  "job_ids": [
    "j-8mgj8gzk52e"
  ],
  "resource_ids": [
    "i-3dxf6id6"
  ],
  "ret_code": 0,
  "trace_id": "I1yAwOLJ"
}
----
