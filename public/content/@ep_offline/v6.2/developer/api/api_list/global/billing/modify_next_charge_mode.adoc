---
title: 修改下个周期计费模式（ModifyNextChargeMode）
weight: 19
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| auto_renew
| Integer
| false
| 否，仅弹性转包月时需要

| console
| String
| true
| 控制台ID

| immediate
| Integer
| true
| 是否立即转，0:当前周期到期后转，1:立即转

| is_confirm
| Integer
| true
| 是否确认要转换

| months
| Integer
| false
| 转成几个月周期的,

| next_charge_mode
| String
| true
| 目标计费周期，包月转弹性为elastic,

| next_price_info
| String
| false
| 

| refund_policy
| String
| false
| private

| resource
| String
| true
| 资源ID

| root_user
| String
| true
| 主账号ID

| user
| String
| true
| 用户ID

| zone
| String
| true
| 区域ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| failed
| Array
| false
| 失败未处理的资源ID

| success
| Array
| false
| 成功处理的资源ID
|===

== 请求示例
[,javascript]
----
{
  "action": "ModifyNextChargeMode",
  "auto_renew": 0,
  "console": "qingcloud",
  "immediate": 1,
  "months": 1,
  "next_charge_mode": "monthly",
  "resource": "i-ob7ft6fe",
  "root_user": "usr-w9XhbDCL",
  "user": "usr-w9XhbDCL",
  "zone": "sh1a"
}
----

== 返回示例
[,javascript]
----
{
  "action": "ModifyNextChargeModeResponse",
  "failed": [],
  "ret_code": 0,
  "success": [
    "i-ob7ft6fe"
  ]
}
----
