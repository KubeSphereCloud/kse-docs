---
title: 查询集群增加节点费用（查询集群扩容费用ResizeClusterFee也使用该接口）（AddClusterNodesFee）
weight: 2
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| cluster
| String
| true
| 集群ID

| node_count
| String
| true
| 节点数量

| node_role
| String
| true
| 节点角色

| zone
| String
| true
| 可用区ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| resize_fee
| Array
| false
| 扩容费用

| resize_fee[].charge_mode
| String
| false
| 计费模式

| resize_fee[].duration
| String
| false
| 计费间隔

| resize_fee[].node_role
| Array
| false
| 节点角色

| resize_fee[].price_diff
| String
| false
| 差价

| ret_code
| Integer
| false
| 返回状态码
|===

== 请求示例
[,javascript]
----
{
  "action": "AddClusterNodesFee",
  "cluster": "cl-57iza1dg",
  "node_count": 1,
  "node_role": "readinstance",
  "owner": "admin",
  "zone": "staging"
}
----

== 返回示例
[,javascript]
----
{
  "action": "AddClusterNodesFeeResponse",
  "resize_fee": [
    {
      "charge_mode": "elastic",
      "duration": "1h",
      "node_role": [
        "readinstance",
        "maininstance",
        "slave"
      ],
      "price_diff": "3.4990"
    }
  ],
  "ret_code": 0,
  "trace_id": "t5LUROJy"
}
----
