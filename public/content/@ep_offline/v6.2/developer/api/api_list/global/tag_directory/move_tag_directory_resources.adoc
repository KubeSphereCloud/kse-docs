---
title: 移动资源到指定目录（MoveTagDirectoryResources）
weight: 6
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| directory_id
| String
| false
| 资源目录id

| owner
| String
| false
| 资源所有者

| resource_type
| String
| false
| 资源类型

| resources
| Array
| false
| 资源id列表

| zone_id
| String
| false
| 可用区id
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| directory_id
| String
| false
| 资源目录id

| resources
| Array
| false
| 资源id列表
|===

== 请求示例
[,javascript]
----
{
  "action": "MoveTagDirectoryResources",
  "directory_id": "dir-un9bilqw",
  "owner": "usr-ncKckFrR",
  "resource_type": "instance",
  "resources": [
    "i-wlhdth15"
  ],
  "zone": "pek3a"
}
----

== 返回示例
[,javascript]
----
{
  "action": "MoveTagDirectoryResourcesResponse",
  "directory_id": "dir-un9bilqw",
  "resources": [
    "i-wlhdth15"
  ],
  "ret_code": 0
}
----
