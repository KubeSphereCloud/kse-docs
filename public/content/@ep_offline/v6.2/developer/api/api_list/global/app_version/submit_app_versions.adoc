---
title: 应用版本提交审核（SubmitAppVersions）
weight: 23
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| version_ids
| Array
| true
| 应用版本
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| version_ids
| Array
| false
| 应用版本
|===

== 请求示例
[,javascript]
----
{
  "version_ids": [
    "appv-xxxxxx"
  ]
}
----

== 返回示例
[,javascript]
----
{
  "action": "SubmitAppVersionsResponse",
  "ret_code": 0,
  "version_ids": [
    "appv-xxxxxx"
  ]
}
----
