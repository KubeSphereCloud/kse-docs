---
title: 解绑资源通知列表（DetachResourceNotificationList）
weight: 6
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| resources
| Array
| true
| 资源id列表

| zone
| String
| true
| 资源所在zone
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| resources
| Array
| false
| 资源id列表
|===

== 请求示例
[,javascript]
----
{
  "action": "DetachResourceNotificationList",
  "resources": [
    "cl-xxxxxxxx"
  ],
  "zone": "test"
}
----

== 返回示例
[,javascript]
----
{
  "action": "DetachResourceNotificationListResponse",
  "resources": [
    "cl-xxxxxxxx"
  ],
  "ret_code": 0
}
----
