title: 激活资源的租赁信息（ActivateResource）

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| resource
| String
| true
| 资源ID

| start_time
| String
| true
| 资源开始收费的时间

| user
| String
| true
| 

| zone
| String
| true
| 
|===

== 返回数据

此接口仅包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。

== 请求示例
[,javascript]
----
{
  "resource": "wacc-ybzrzt03",
  "start_time": "2023-04-28T11:47:38Z",
  "user": "usr-NFe7VCe8",
  "zone": "pek3a"
}
----

== 返回示例
[,javascript]
----
{
  "action": "ActivateResourceResponse",
  "ret_code": 0
}
----
