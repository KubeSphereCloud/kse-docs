== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| user
| String
| true
| 用户编号
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| action
| String
| false
| 操作名称

| data
| Object
| false
| 数据

| data.TODO
| String
| false
| 待定义

| ret_code
| Integer
| false
| 操作结果

| total_count
| Integer
| false
| 总数
|===

== 请求示例
[,javascript]
----
{
  "user": "usr-xxx"
}
----

== 返回示例
[,javascript]
----
{
  "action": "GetUserResourcesSummaryResponsse",
  "data": {
    "instance": {
      "all_count": 25,
      "console_id": "pek",
      "resource_type": "instance",
      "user_id": "usr-xxx",
      "zones": {
        "jinna1": 13,
        "pek3": 12
      }
    }
  },
  "ret_code": 0,
  "total_count": 100
}
----
