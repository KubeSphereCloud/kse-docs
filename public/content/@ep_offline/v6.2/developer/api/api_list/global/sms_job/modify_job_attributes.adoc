---
title: 修改短信任务（ModifySMSJobAttributes）
weight: 3
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| sms_job
| String
| false
| 短信任务id

| status
| String
| true
| 状态
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| sms_job_id
| String
| false
| 短信任务id
|===

== 请求示例
[,javascript]
----
{
  "action": "ModifySMSJobAttributes",
  "sms_job": "sms-xxxxxxxx",
  "status": "pending"
}
----

== 返回示例
[,javascript]
----
{
  "action": "ModifySMSJobAttributesResponse",
  "ret_code": 0,
  "sms_job_id": "sms-xxxxxxxx"
}
----
