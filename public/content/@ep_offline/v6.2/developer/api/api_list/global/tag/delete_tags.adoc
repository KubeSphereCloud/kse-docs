---
title: 删除标签（DeleteTags）
weight: 3
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| scope
| String
| false
| 范围

| system
| Boolean
| false
| 是否是系统标签

| tags
| Array
| true
| 标签id集合

| zone
| String
| false
| 区域ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| tags
| Array
| false
| true
|===

== 请求示例
[,javascript]
----
{
  "action": "DeleteTags",
  "scope": "zone",
  "system": false,
  "tags": [
    "[tag-zyre1eoh]"
  ]
}
----

== 返回示例
[,javascript]
----
{
  "action": "DeleteTagsResponse",
  "ret_code": 0,
  "tags": [
    "[tag-zyre1eoh]"
  ]
}
----
