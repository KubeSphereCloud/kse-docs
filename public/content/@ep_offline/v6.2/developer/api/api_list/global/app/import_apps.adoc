== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| apps_info
| String
| true
| 导入的应用id以及相应版本id

| resource_path
| String
| true
| app路径

| zone
| String
| true
| 可用区
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| log_id
| String
| false
| 日志id
|===

== 请求示例
[,javascript]
----
{
  "apps_info": "{\"app-tg3lbp0a\":[\"appv-39rsbbce\",\"appv-70gegwmp\"]}",
  "resource_path": "root@sr01n27:/pitrix/appcenter_migration/migrate_appcenter2_zookeeper",
  "zone": "testing1a"
}
----

== 返回示例
[,javascript]
----
{
  "action": "ImportAppsResponse",
  "log_id": "exl-tugmrii9",
  "ret_code": 0
}
----
