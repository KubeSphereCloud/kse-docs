== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| auto_leasing
| Integer
| false
| private

| charge_type
| String
| false
| private

| fee
| Integer
| true
| private

| operator
| String
| false
| private

| remarks
| String
| false
| private

| sub_type
| String
| false
| private

| trade
| String
| false
| private

| user
| String
| true
| private
|===

== 返回数据

此接口仅包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。



== 请求示例
[,javascript]
----
{
  "action": "Recharge",
  "fee": 100,
  "user": "usr-xxxxxx"
}
----

== 返回示例
[,javascript]
----
{
  "action": "RechargeResponse",
  "ret_code": 0,
  "user_id": "usr-xxxxxx"
}
----
