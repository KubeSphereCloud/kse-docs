---
title: 将资源从回收站清理掉（CleanRecycleResources）
weight: 2
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| check_mode
| Int
| false
| 清理模式:0仅列表内资源/1全部

| resources
| Array
| false
| 需要清理的资源列表

| zone
| String
| true
| 请求区ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| msg
| String
| false
| 
|===

== 请求示例
[,javascript]
----
{
  "action": "CleanRecycleResources",
  "owner": "usr-ei6RFhNN",
  "resources": [
    "i-fxkq9dvw"
  ],
  "zone": "pek3"
}
----

== 返回示例
[,javascript]
----
{
  "action": "CeaseRecycleResourcesResponse",
  "msg": "clean success;",
  "ret_code": 0,
  "trace_id": "nkoAgya4"
}
----
