---
title: 初始化用户余额信息（CreateUserBalance）
weight: 4
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| user
| String
| true
| 用户ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| user
| String
| false
| 用户ID
|===

== 请求示例
[,javascript]
----
{
  "user_id": "usr-3zHdjYkY"
}
----

== 返回示例
[,javascript]
----
{
  "action": "CreateUserBalanceResponse",
  "ret_code": 0,
  "user_id": "usr-3zHdjYkY"
}
----
