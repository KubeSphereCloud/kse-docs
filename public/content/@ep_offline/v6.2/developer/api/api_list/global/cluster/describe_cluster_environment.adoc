---
title: 查询集群环境变量（DescribeClusterEnvironment）
weight: 3
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| app_id
| String
| true
| 集群ID

| zone
| String
| true
| 可用区ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| config
| Array
| false
| 环境配置

| config[].description
| String
| false
| 参数描述

| config[].key
| String
| false
| 环境参数键

| config[].properties
| Array
| false
| 参数属性

| config[].properties[].changeable
| String
| false
| 是否可被修改

| config[].properties[].description
| String
| false
| 属性描述

| config[].properties[].key
| String
| false
| 参数键

| config[].properties[].range
| String
| false
| 可选范围

| config[].properties[].required
| String
| false
| 是否必填

| config[].properties[].type
| String
| false
| 属性

| config[].type
| String
| false
| 环境参数类型

| ret_code
| String
| false
| 返回状态码
|===

== 请求示例
[,javascript]
----
{
  "action": "DescribeClusterEnvironment",
  "cluster_id": "cl-2kogt3ly",
  "zone": "pek3"
}
----
