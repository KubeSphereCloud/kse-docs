---
title: 验证通知列表项（VerifyNotificationItem）
weight: 6
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| notification_item_content
| String
| true
| 需要验证的联系方式

| owner
| String
| false
| 所属用户id

| verification_code
| String
| true
| 验证码
|===

== 返回数据

此接口仅包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。

== 请求示例
[,javascript]
----
{
  "action": "VerifyNotificationItem",
  "notification_item_content": "86-133****8888",
  "owner": "usr-xxxxxxxx",
  "verification_code": "123456"
}
----

== 返回示例
[,javascript]
----
{
  "action": "VerifyNotificationItemResponse",
  "ret_code": 0
}
----
