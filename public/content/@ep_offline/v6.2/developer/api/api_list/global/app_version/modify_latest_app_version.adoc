---
title: 设置应用系列最新版本（ModifyLatestAppVersion）
weight: 18
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| app_id
| String
| true
| 应用ID

| edition_id
| String
| true
| 系列ID

| version_id
| String
| true
| 应用版本ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| version_id
| String
| false
| 应用版本ID
|===

== 请求示例
[,javascript]
----
{
  "app_id": "app-xxxxxx",
  "edition_id": "default",
  "version_id": "appv-xxxxxx"
}
----

== 返回示例
[,javascript]
----
{
  "action": "ModifyLatestAppVersionResponse",
  "ret_code": 0,
  "version_id": "appv-xxxxxx"
}
----
