---
title: 标记用户站内信所有消息已读（ReadAllNotificationCenterUserPosts）
weight: 3
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| post_type
| Array
| false
| 通知类型
|===

== 返回数据

此接口仅包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。

== 请求示例
[,javascript]
----
{
  "action": "ReadAllNotificationCenterUserPosts",
  "post_type": [
    "finance",
    "failures",
    "service",
    "products",
    "activity",
    "security",
    "appcenter",
    "cloud_market"
  ]
}
----

== 返回示例
[,javascript]
----
{
  "action": "ReadAllNotificationCenterUserPostsResponse",
  "ret_code": 0
}
----
