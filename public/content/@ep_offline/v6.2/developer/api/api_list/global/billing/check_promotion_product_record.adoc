---
title: 获取资源的促销记录信息（CheckPromotionProductRecord）
weight: 2
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| desired_promotion_type
| String
| false
| 促销类型

| resource_id
| String
| true
| 资源ID
|===

== 返回数据

此接口仅包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。

== 请求示例
[,javascript]
----
{
  "desired_promotion_type": "new_comer",
  "resource_id": "i-0te3yics"
}
----

== 返回示例
[,javascript]
----
{
  "action": "CheckPromotionProductRecordResponse",
  "ret_code": 0
}
----
