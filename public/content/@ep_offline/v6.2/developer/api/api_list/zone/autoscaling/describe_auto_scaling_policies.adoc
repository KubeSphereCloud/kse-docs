title: 查询自动伸缩策略（DescribeAutoScalingPolicies）
weight: 7

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| autoscaling_policies
| Array
| false
| 自动伸缩策略ID

| owner
| String
| true
| 所有者

| project_id
| String
| false
| 项目ID

| verbose
| Integer
| true
| 是否可见

| zone
| String
| true
| 区域
|===

== 请求示例
[,javascript]
----

    "asp-7w1cta60"
  ],
  "owner": "admin",
  "verbose": 1,
  "zone": "testing"
}
