---
title: 集群绑定安全组（ApplySecurityGroupToCluster）
weight: 1
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| cluster
| String
| true
| 集群ID

| security_group
| String
| true
| 安全组ID

| zone
| String
| true
| 可用区
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| cluster
| String
| false
| 集群ID

| job_id
| String
| false
| 任务ID

| ret_code
| String
| false
| 返回状态码

| security_group
| String
| false
| 安全组ID
|===

== 请求示例
[,javascript]
----
{
  "action": "ApplySecurityGroupToCluster",
  "cluster": "cl-6c20jvk9",
  "security_group": "sg-w5ei3cjp",
  "zone": "testing"
}
----

== 返回示例
[,javascript]
----
{
  "action": "ApplySecurityGroupToClusterResponse",
  "cluster": "cl-6c20jvk9",
  "job_id": "j-jjq063uyixs",
  "ret_code": 0,
  "security_group": "sg-w5ei3cjp",
  "trace_id": "z5NAoOHe"
}
----
