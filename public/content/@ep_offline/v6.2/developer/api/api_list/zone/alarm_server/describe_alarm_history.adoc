title: 获取告警历史信息（DescribeAlarmHistory）
weight: 1

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| action
| String
| true
| action名称

| alarm
| String
| false
| 告警id

| alarm_policy_category
| String
| false
| 告警策略类型

| from_fg
| Boolean
| false
| 是否来自fg的请求

| history_type
| Array
| false
| 告警历史类型

| limit
| Integer
| true
| 返回数据条数

| offset
| Integer
| true
| 分页起始索引

| owner
| String
| true
| 用户id

| verbose
| String
| false
| 是否远程获取告警历史

| zone
| String
| true
| zone区域
|===

== 请求示例
[,javascript]
----
{
  "action": "DescribeAlarmHistory",
  "alarm": "alm-wcvojma9",
  "alarm_policy_category": "threshold",
  "history_type": [
    "status_change",
    "trigger_action",
    "config_update"
  ],
  "limit": 100,
  "offset": 0,
  "owner": "usr-rrPKHDpJ",
  "verbose": 1,
  "zone": "staging"
}
----
