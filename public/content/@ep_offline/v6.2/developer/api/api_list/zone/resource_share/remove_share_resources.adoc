---
title: 移除共享资源（RemoveShareResources）
weight: 9
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| owner
| String
| false
| 资源的所有者

| resource_share_id
| String
| false
| 共享单元id

| resources
| Array
| false
| 需要移除的共享资源

| zone
| String
| true
| 请求区ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| message
| String
| false
| 提示信息
|===

== 请求示例
[,javascript]
----
{
  "action": "RemoveShareResources",
  "owner": "admin",
  "resource_share_id": "rsg-cd89eiuy",
  "resources": [
    "vxnet-9ll660z"
  ],
  "zone": "testing"
}
----

== 返回示例
[,javascript]
----
{
  "action": "RemoveShareResourcesResponse",
  "message": true,
  "ret_code": 0,
  "trace_id": "7s1c6tgE"
}
----
