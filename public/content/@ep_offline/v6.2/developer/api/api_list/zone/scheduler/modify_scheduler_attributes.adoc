---
title: 修改定时器属性（ModifySchedulerAttributes）
weight: 11
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| hhmm
| String
| false
| 时分

| owner
| String
| false
| 资源的所有者

| repeat
| Integer
| false
| 重复次数

| scheduler
| String
| true
| 定时器id

| scheduler_name
| String
| false
| 定时器名称

| ymd
| String
| false
| 年月日

| zone
| String
| true
| 请求区ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| scheduler_id
| String
| false
| 定时器id
|===

== 请求示例
[,javascript]
----
{
  "action": "ModifySchedulerAttributes",
  "hhmm": "00:50",
  "owner": "usr-BnnisHWR",
  "repeat": "0",
  "scheduler": "schd-0g60lk75",
  "scheduler_name": "fdsfdfdsfddsfdsf",
  "ymd": "2023-07-05",
  "zone": "testing"
}
----

== 返回示例
[,javascript]
----
{
  "action": "ModifySchedulerAttributesResponse",
  "ret_code": 0,
  "scheduler_id": "schd-0g60lk75",
  "trace_id": "yUp3wciF"
}
----
