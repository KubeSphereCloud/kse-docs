== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| action
| String
| true
| action名称

| alarm_policy_category
| String
| false
| 告警策略类型

| alarms
| Array
| false
| 告警id数组

| limit
| Integer
| true
| 返回数据条数

| offset
| Integer
| true
| 分页起始索引

| owner
| String
| true
| 用户id

| policy_usage
| String
| false
| 告警策略用途

| resource_id
| String
| false
| 资源id，以英文逗号分隔

| resource_type
| String
| false
| 资源类型

| reverse
| Integer
| false
| 是否逆序

| sort_key
| String
| false
| 排序字段名称

| status
| Array
| false
| 告警状态

| verbose
| String
| false
| 是否返回告警关联资源信息

| zone
| String
| true
| zone区域
|===

== 请求示例
[,javascript]
----
{
  "action": "DescribeAlarms",
  "limit": 10,
  "offset": 0,
  "owner": "usr-jdxUgG5s",
  "policy_usage": "alarm",
  "reverse": 1,
  "sort_key": "update_time",
  "status": [
    "alarm"
  ],
  "zone": "testing"
}
----
