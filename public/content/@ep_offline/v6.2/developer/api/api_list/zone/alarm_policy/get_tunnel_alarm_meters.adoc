---
title: 获取隧道告警指标（GetTunnelAlarmMeters）
weight: 15
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| meters
| Array
| true
| 指标

| owner
| String
| false
| 用户

| resource
| String
| true
| 资源id

| zone
| String
| false
| 区域
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| meters
| Array
| false
| 指标id数组

| resource_id
| String
| false
| 资源id
|===

== 请求示例
[,javascript]
----
{
  "action": "GetTunnelAlarmMeters",
  "meters": "rtr-fszu8ige||qing",
  "owner": "usr-ywtcdYqC",
  "resource": "rtr-e99ue5c1",
  "zone": "pek3"
}
----

== 返回示例
[,javascript]
----
{
  "action": "GetTunnelAlarmMetersResponse",
  "meters": [
    "gre|rtr-fszu8ige|stat"
  ],
  "ret_code": 0
}
----
