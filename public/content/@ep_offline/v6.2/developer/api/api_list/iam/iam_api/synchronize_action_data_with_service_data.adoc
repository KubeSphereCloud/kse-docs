---
title: 同步IAM纳管服务属性到纳管API（SynchronizeIAMActionDataWithServiceData）
weight: 22
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| NOT_NEED
| String
| false
| 无需参数
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| updated_actions
| Array
| false
| 
|===

== 请求示例
[,javascript]
----
{
  "action": "SynchronizeIAMActionDataWithServiceData"
}
----

== 返回示例
[,javascript]
----
{
  "action": "SynchronizeIAMActionDataWithServiceDataResponse",
  "ret_code": 0,
  "updated_actions": [
    "DescribeInstances"
  ]
}
----
