---
title: 删除IAM用户组（DeleteIAMGroups）
weight: 4
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| group_ids
| Array
| true
| IAM用户组ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| group_ids
| Array
| false
| IAM用户组ID
|===

== 请求示例
[,javascript]
----
{
  "action": "DeleteIAMGroups",
  "group_ids": [
    "iamg-7u5rtv8s"
  ]
}
----

== 返回示例
[,javascript]
----
{
  "action": "DeleteIAMGroupsResponse",
  "group_ids": [
    "iamg-7u5rtv8s"
  ],
  "ret_code": 0
}
----
