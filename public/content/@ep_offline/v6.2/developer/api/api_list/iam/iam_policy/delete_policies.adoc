---
title: 删除IAM策略（DeleteIAMPolicies）
weight: 3
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| audit_log
| String
| false
| 审计日志

| policies
| Array
| true
| IAM策略ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| controller
| String
| false
| 控制器类型

| policies
| Array
| false
| IAM策略ID列表
|===

== 请求示例
[,javascript]
----
{
  "action": "DeleteIAMPolicies",
  "policies": [
    "iamp-m8r7qhwn"
  ]
}
----

== 返回示例
[,javascript]
----
{
  "action": "DeleteIAMPoliciesResponse",
  "controller": "self",
  "policies": [
    "iamp-m8r7qhwn"
  ],
  "ret_code": 0
}
----
