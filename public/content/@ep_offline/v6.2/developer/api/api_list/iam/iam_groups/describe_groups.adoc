---
title: 查询IAM用户组（DescribeIAMGroups）
weight: 6
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| group_ids
| Array
| false
| 组ID

| group_name
| String
| false
| 组名

| limit
| Integer
| false
| 偏移量

| offset
| Integer
| false
| 起始坐标

| root_user
| String
| false
| 根账号ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| group_set
| Array
| false
| 所有者

| group_set[].console_id
| String
| false
| 控制台ID

| group_set[].create_time
| String
| false
| 创建时间

| group_set[].description
| String
| false
| 描述

| group_set[].group_id
| String
| false
| 组ID

| group_set[].group_name
| String
| false
| 组名

| group_set[].owner
| String
| false
| 所有者

| group_set[].qrn
| String
| false
| 资源唯一标识

| group_set[].root_user_id
| String
| false
| 主账户ID

| total_count
| Integer
| false
| 返回结果总数
|===

== 请求示例
[,javascript]
----
{
  "action": "DescribeIAMGroups",
  "limit": 20,
  "offset": 0
}
----
