---
title: 查询信任策略（DescribeIAMTrustPolicy）
weight: 8
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| resource
| String
| true
| 资源ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| policy_set
| Array
| false
| 受信任策略
|===

== 请求示例
[,javascript]
----
{
  "action": "DescribeIAMTrustPolicy",
  "resource": "usr-dy7mtlyu"
}
----

== 返回示例
[,javascript]
----
{
  "action": "DescribeIAMTrustPolicyResponse",
  "policy_set": "[iamp-dy7mtlyu]",
  "ret_code": 0
}
----
