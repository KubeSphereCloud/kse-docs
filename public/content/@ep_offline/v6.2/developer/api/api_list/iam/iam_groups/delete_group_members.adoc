---
title: 删除IAM用户组成员（DeleteIAMGroupMembers）
weight: 3
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| group_id
| String
| false
| IAM用户组ID

| user_ids
| Array
| false
| 用户ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| group
| String
| false
| IAM用户组ID

| user_id
| Array
| false
| IAM用户组成员ID列表
|===

== 请求示例
[,javascript]
----
{
  "action": "DeleteIAMGroupMembers",
  "group_id": "iamg-6sysj1u7",
  "user_id": [
    "usr-g8ZyBo6l"
  ]
}
----

== 返回示例
[,javascript]
----
{
  "action": "DeleteIAMGroupMembersResponse",
  "group": "iamg-6sysj1u7",
  "ret_code": 0,
  "user_id": [
    "usr-g8ZyBo6l"
  ]
}
----
