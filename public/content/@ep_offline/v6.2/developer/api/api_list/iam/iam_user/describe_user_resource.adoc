---
title: 查询IAM用户创建的资源（DescribeIAMUserResource）
weight: 6
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| limit
| Integer
| false
| 偏移量

| offset
| Integer
| false
| 起始坐标

| resource_type
| String
| false
| 资源类型

| resources
| Array
| false
| 资源ID列表

| reverse
| Integer
| false
| 是否降序查询

| search_word
| String
| false
| 搜索关键字

| sort_key
| String
| false
| 排序使用的列

| users
| Array
| false
| IAM用户ID列表

| zone_id
| String
| false
| Zone
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| iam_user_resource_set
| Object
| false
| 资源列表

| iam_user_resource_set.create_time
| String
| false
| 绑定时间

| iam_user_resource_set.iam_user_id
| String
| false
| IAM用户ID

| iam_user_resource_set.resource_id
| String
| false
| 资源ID

| iam_user_resource_set.resource_type
| String
| false
| 资源类型

| iam_user_resource_set.status
| String
| false
| 状态

| iam_user_resource_set.zone
| String
| false
| 可用区
|===

== 请求示例
[,javascript]
----
{
  "action": "DescribeIAMUserResource",
  "users": [
    "iamu-k0i2m98t"
  ]
}
----

== 返回示例
[,javascript]
----
{
  "action": "DescribeIAMUserResourceResponse",
  "iam_user_resource_set": [
    {
      "create_time": "2023-08-01T08:02:07Z",
      "iam_user_id": "iamu-k0i2m98t",
      "resource_id": "i-esdaf97w",
      "resource_type": "instance",
      "status": "active",
      "zone": "testing"
    }
  ],
  "ret_code": 0
}
----
