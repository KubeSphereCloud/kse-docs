---
title: 添加IAM纳管API信息（AddIAMApiAction）
weight: 1
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| action_groups
| String
| false
| 分组

| action_name
| String
| false
| API

| audit_log
| String
| false
| 审计日志

| keys
| String
| false
| 接口参数

| optional_resource_keys
| String
| false
| 可选资源接口参数名

| optional_resources
| String
| false
| 可选资源列表

| required_resource_keys
| String
| false
| 必选资源接口参数名

| required_resources
| String
| false
| 必选资源列表

| service_name
| String
| true
| 服务名（英文）

| translated_name
| String
| false
| Action名称（中文）
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| action_group
| String
| false
| API分组

| attributes
| Object
| false
| 属性

| attributes.actionGroups
| String
| false
| 操作码分组

| attributes.optionalResources
| String
| false
| 可选参数

| attributes.requiredResources
| String
| false
| 必选参数

| audit_log
| String
| false
| 审计日志

| create_time
| String
| false
| 创建时间

| gateway
| String
| false
| 网关

| id
| String
| false
| 操作码ID

| keys
| Object
| false
| 必选可选参数

| keys.optional
| String
| false
| 可选参数

| keys.required
| String
| false
| 必选参数

| permission
| String
| false
| 默认权限

| service_name
| String
| false
| 服务名

| status
| String
| false
| 状态

| translated_name
| String
| false
| 服务名（中文）

| type
| String
| false
| 类型（服务，接口）

| visible
| String
| false
| 是否可见
|===

== 请求示例
[,javascript]
----
{
  "action": "AddIAMApiAction",
  "action_groups": [
    "Other"
  ],
  "action_name": "testecm",
  "service_name": "ecm"
}
----
