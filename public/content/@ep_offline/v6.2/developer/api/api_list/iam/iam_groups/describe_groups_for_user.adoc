---
title: 查询账号拥有的IAM用户组（DescribeIAMGroupsForUser）
weight: 7
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| limit
| Integer
| false
| 偏移量

| offset
| Integer
| false
| 起始坐标

| reverse
| Integer
| false
| 是否降序查询

| sort_key
| String
| false
| 排序使用的列

| user_id
| String
| false
| 账号ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| group_set
| Array
| false
| 

| group_set[].console_id
| String
| false
| 控制台ID

| group_set[].create_time
| String
| false
| 创建时间

| group_set[].group_id
| String
| false
| IAM用户组ID

| group_set[].group_name
| String
| false
| IAM用户组名称

| group_set[].owner
| String
| false
| 所有者ID

| group_set[].qrn
| String
| false
| IAM用户组资源唯一标识

| group_set[].root_user_id
| String
| false
| 主账户ID
|===

== 请求示例
[,javascript]
----
{
  "action": "DescribeIAMGroupsForUser",
  "limit": 20,
  "offset": 0,
  "user_id": "admin"
}
----
