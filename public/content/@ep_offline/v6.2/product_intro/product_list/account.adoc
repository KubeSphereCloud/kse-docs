[cols='1,1,3a']
|===
| 产品  | 交付模式  | 产品简介


| API 密钥| 默认 | 支持用户通过发送 API 指令来访问我们的服务。
| 账户登录 | 默认 | 支持邮箱+密码、手机号+密码、手机号+验证码三种登录方式。其中手机号作为用户名登录，要求{product_name}对接指定短信服务商（百悟或漫道）。
| 账户注册| 默认 | 支持邮箱或手机号注册，其中其中手机号作为用户名注册，要求{product_name}对接指定短信服务商（百悟或漫道）。
| 账户实名认证| 可选 |提供两种认证模式，支付宝快速认证（使用支付宝活体验证和天眼查进行认证）和普通认证。认证对象包括个人实名认证、企业实名认证。
| 账户安全设置| 默认 |包括修改密码、账号的安全设置、二次认证、登录历史以及应用授权。
| 二次认证| 默认 |可开启二次认证，二次认证采用谷歌的 Antherticator，开启成功之后需要每次输入动态口令。
| 登录历史| 默认 |支持用户查看近 100 次登录历史。
| 子账户| 默认 |子账户作为独立的子租户，子账户的也拥有配额、IAM 用户、资源、计费等能力。主账号能够新增、删除、禁用启用、修改子账号，同时还支持调整子账号以下能力。

* 配额

支持单独限制和共享主账号的配置

* 计费

支持独立计费和使用主账号计费

* 支持快速切换子账号身份

| 账户组| 默认 |支持对账号进行分组管理。


|===
