企业空间作为可扩展的组件部署于云平台的 Global 服务器中，主要包括企业空间组件、企业空间操作界面、云平台操作界面以及云平台各资源组件。

* 企业空间组件：其中企业空间的业务组件实现了各类核心功能，并提供了 RPC 服务供企业空间 API 服务调用。企业空间 API 服务对外为企业空间操作界面提供 HTTP 接口，对内为云平台各类接入到企业空间的资源组件提供了资源管理的接口。
* 企业空间操作界面：通过企业空间 API 的接口，用户可以在企业空间操作界面管理其内的各项业务。
* 云平台各资源组件：云平台其他对接了企业空间的业务，通过相应的 API 实现基于企业空间的资源筛选功能。
* 云平台操作界面：用户在云平台其他界面操作对接了企业空间资源时，可以实现根据所属企业空间的身份和策略来管理相应资源。
