企业空间的软件架构主要包括：企业空间 Web 控制台、核心业务、云平台服务依赖以及底层组件四个部分。

* 企业空间 Web 控制台：即企业空间的前端管理界面，用户可以通过此界面管理企业空间的各项业务。
* 核心业务：企业空间管理的各项核心业务实现。主要包括组织管理、成员管理、角色管理、资源管理、流程管理以及策略管理业务。
* 云平台的服务依赖：企业空间管理所依赖的云平台服务，主要包括云平台的账户服务、IAM 服务、配额服务以及消息服务，基于这些服务进一步完善企业空间管理的能力。
* 底层组件：企业空间管理所需的基础设施，包括数据库、缓存以及存储服务等。
