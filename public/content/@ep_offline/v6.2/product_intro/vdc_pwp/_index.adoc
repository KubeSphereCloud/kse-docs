企业空间是企业云平台面向客户提供的，对大量资源进行多级管理的功能模块。

本文档主要描述了企业空间的产品介绍和定位、功能架构、软件及部署架构以及应用场景等，以帮助用户全面了解企业空间的功能原理，从而更好的使用企业空间管理。

== 读者对象

本文档适用于以下读者：

* 售前工程师
* 销售经理
* 产品经理
* 产品架构师
* 运维工程师

== 名词解释

[options="header",cols="1,2"]
|===
|名词 |解释
|BOSS 运维平台
|云计算运维管理平台，简称 BOSS，是企业云平台面向用户提供的运维管理界面程序。

|Console 工作台
|云计算服务平台（Console）是企业云平台的云服务控制台，为云用户提供计算、网络、存储、安全、数据库与缓存、大数据服务、消息队列与中间件、容器服务、运维与监控等服务。

|===
