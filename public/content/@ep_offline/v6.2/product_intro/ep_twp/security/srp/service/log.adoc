日志审计帮助用户对各类日志进行全面的标准化处理，并进行全维度、跨设备、细粒度的关联分析，以图表的形式展现在线服务的业务访问情况。通过对访问记录的深度分析，发掘潜在威胁，起到追根索源的目的，并且记录服务器返回的内容，便于取证式分析。

*组网说明*：

* 安装 Agent 的服务器为宿主机。如果宿主机在基础网络，日志审计实例加入与宿主机相同的基础网络；如果宿主机在 VPC 网络，日志审计实例加入相应的 VPC网络，以实现日志审计实例和宿主机之间的正常通信。
* 通过 Agent 上报宿主机的系统日志到日志审计实例中，实现实时监视操作行为及攻击信息。
