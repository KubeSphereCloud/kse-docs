主机安全集成了丰富的系统防护与加固、网络防护与加固等功能，具备业界领先的勒索专防专杀、网络隔离预防、补丁修复等主机安全能力，帮您快速发现网站潜在安全隐患。通过在业务主机安装 Agent 执行主机安全防护策略，平台管理员可通过安全资源池管理界面对所有主机资产的安全情况进行统一管理。

*组网说明*：

* 主机安全实例在创建时会被创建在基础网络中。
* 组网对受防护主机所在的网络没有太大要求。一般情况下，同一个区域内，主机安全实例（EDR）与其他云服务器天然互通，不需要主机安全实例加入特定的网络中。
* 给防护主机安装主机安全的 Agent，Agent 会自动连接主机安全实例，并进行数据上传，实现对主机的监控。
