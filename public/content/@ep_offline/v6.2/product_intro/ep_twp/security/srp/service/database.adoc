数据库审计产品可以帮助用户实现对进出核心数据库的访问流量进行数据报文字段级的解析操作，还原操作细节，并给出详尽的操作返回结果。通过安装 Agent 采集用户对数据库的访问行为，并以可视化的方式将所有的访问都呈现在管理者的面前，让数据库不再处于不可知、不可控的情况，数据威胁将被迅速发现和响应。

*组网说明*：

* 安装 Agent 的服务器为宿主机。如果宿主机在基础网络，数据库审计实例加入与宿主机相同的基础网络；如果宿主机在 VPC 网络，数据库审计实例加入相应的 VPC 网络，以实现数据库审计实例和数据库服务器之间的正常通信。
* 访问数据库的数据通过 Agent 采集并上传到数据库审计实例中，达到对数据库系统的安全防护。
