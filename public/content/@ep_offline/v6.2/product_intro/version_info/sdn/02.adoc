== 使用新镜像版本

升级负载均衡节点实例的镜像，支持以下更多功能：

* HTTPS 加密支持 TLS 1.3 算法
* 支持 HTTP2 + Gzip 方式使用，支持对 HTTP2 报文的压缩
* 多进程改成多线程，提升计算资源使用效率
* 采用新版 HTTP 协议标准，HTTP Header 小写处理

本次升级将更新负载均衡镜像升级逻辑，重启、迁移、重建负载均衡实例不再自动升级镜像版本，用户需要手动操作升级，避免无意的升级导致非预期的程序行为。

== 负载均衡高可用优化

* 增加常驻进程对后端状态进行检查，当常驻进程发现后端节点掉线时将发出告警，并重新连接后端服务。
* LB 节点内部服务告警。保持实时对后端进程探活，当节点内部服务宕机时，将主动发起重启操作。

== 负载均衡监控告警优化

* 负载均衡发布新的监控标签页，实现节点更多维度的性能监控。
* 负载均衡嵌套 CloudSat 监控，通过节点内的监控服务统一将数据发送到 CloudSat 服务，进行统一的采集、存储、分析。用户可以在云监控中查看账号下的负载均衡的监控数据。
* 监控指标增加了节点 CPU、内存、以及关联的 EIP 流量、数据包速率等监控项，并提供节点、监听器、后端等多维度的告警功能。
