云计算运维管理平台（Business & Operation Support System），简称 BOSS，是{product_name}面向用户提供的运维管理界面程序。

== 集中管理

BOSS 为用户提供了统一运维门户和统一的基础运维功能，对{product_name}中的物理资源、虚拟资源以及应用服务等进行统一管理。其功能包括硬件资源监控、服务状态监控、资源状态监控、日志查看、告警等。

== 自动运维

同时，BOSS 支持软件机器人自动运维和故障无害的运维技术，能极大提高用户的运维效率和管理水平。

== 兼容性强

BOSS 从服务的角度出发，运行在{product_name}操作系统与监控系统之上，关注如何将所有的资源封装为可度量的服务，并让最终用户以最便捷灵活的形式按需使用这些服务。

同时，BOSS 统支持通过 API 将信息提交到统一告警平台，满足管理要求。良好的接口开放性，使得 BOSS 能方便快捷的支持功能扩展。

== 简单易用

用户可通过 BOSS 简单方便的查看物理机、虚拟机的运行情况；管理物理机、虚拟机等物理设施以及虚拟设施的状态；查看告警日志等。BOSS 为用户提供全面的运维管理能力。

{product_name}可借助 BOSS，对其业务、运维、配置集成管控与分析，在系统频繁变更与上线的同时，提升运维服务水平。
