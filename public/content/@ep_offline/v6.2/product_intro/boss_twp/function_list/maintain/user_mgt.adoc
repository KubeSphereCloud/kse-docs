[cols="1,1,3a"]
|===
|类别|功能特性  |描述

.4+^.^|云用户管理 | 云用户列表| * 列表展示{product_name}上注册的所有云用户，可供管理员查看与检索。
* 列表信息包括云用户的状态、名称、账户类型、角色、权限以及创建时间。
* 支持导出云用户列表，方便管理员查看分析云用户数据。

| 云用户详情 a| * 支持查看具体用户关联的资源信息与配额信息。
* 提供账户安全信息以及登录历史查看入口。

| 云用户管理 a| * 支持创建、删除云用户，支持批量修改云用户配额。
* 支持云用户基本信息修改、角色权限修改、付费类型修改、密码重置、修改登录邮箱等操作。
* 支持禁用、启用、删除云用户。
* 支持云用户的资源回收、资源转移等操作。

| 云用户认证 a| * 支持对云用户进行个人认证或企业认证。
* 展示当前{product_name}中用户认证的统计数据，列表显示云用户认证记录。

.2+^.^|第三方用户管理 | 第三方用户列表 a|* 列表展示{product_name}上注册的所有第三方用户，可供管理员查看与检索。
* 列表信息包括第三方用户的状态、第三方账号、用户邮箱及 ID、登录服务器、登录的服务器类型以及用户创建的时间。


| 云用户绑定 a|* 支持与云用户的绑定与解绑操作。
* 支持查看绑定的云用户详情以及对绑定的云用户进行管理操作。



.2+^.^|控制台登录 | 创建控制台登录服务器 | 创建控制台登录服务器，与用户已有的服务器进行对接，使得用户可以通过已有账号登录管理控制台。

* 支持服务器类型有 AD 服务器与 LDAP 服务器。

| 登录服务器列表 a|* 列表展示{product_name}上已创建的所有登录服务器，可供管理员查看与检索。
* 列表信息包括登录服务器的状态、服务器地址、服务器端口、是否使用 SSL 以及服务器创建的时间。


|API 密钥  | API 密钥列表| * 列表展示{product_name}上已创建的所有 API 密钥，可供管理员查看与检索。
* 列表信息包括 API 密钥的状态、名称、创建 API 的云用户邮箱及 ID、设置的白名单以及创建时间。
* 与创建该密钥的云用户关联，可查看相应的云用户信息。

.2+^.^|配额  | 配额详情|按区域与产品类型，展示默认配额。

| 配额管理| 支持管理员按需求、区域以及产品类型，调整资源配额。


|===
