== 报表中心

云计算管理平台的报表生成是分散在各个功能模块中的，在具有“导出”按钮的功能页面都可以去产生报表。云计算管理平台的报表生成是异步的，报表生成任务在“报表管理中心”中查看其执行状态，同时可以下载生成的报表。云计算管理平台会定期生成包括系统使用量报告、客户消费报告等统计报告。

== 告警中心

云计算管理平台的告警中心包括告警记录、告警策略和告警屏蔽。告警记录是所有发生过告警的记录，可以在列表中对各个状态进行查看，也可以对告警记录进行处理；告警策略是对告警项进行规则设置，可以分为告警规则和通知规则，在该功能下可对平台收到的告警日志条目进行告警级别的定制。通知规则可通过压缩解决消息量过大问题，并将告警通过通知组决定使用对应通知组进行消息推送；创建屏蔽告警规则完成后，告警屏蔽立即生效，并在压制时间范围内，对应节点出现的所有告警信息都不再出发告警和生成告警记录。已有的告警记录不受影响。
