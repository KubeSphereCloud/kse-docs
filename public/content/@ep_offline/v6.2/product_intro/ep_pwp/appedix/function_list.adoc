== GUI 管理界面

[cols="1,3"]
|===
|功能特性|描述

|云计算服务平台
|云计算服务平台（简称 Console）是青云企业云平台面向用户提供的操作使用及运营与一体的界面应用程序。云计算服务平台提供计算、存储、网络等多种云资源，并配备大数据、数据库、中间件、容器平台、AI 等增值服务套件。

用户可通过 Console 自主创建云资源进行使用，并可对云资源进行统一的运维和运营。支持租户自助式的对个人资源进行管理和运维，同时也保证各个租户之间的资源隔离。

云计算服务平台从“自运营”的角度出发，提供人性化交互管理界面，摒弃 B 端“轻交互”的概念，设计贴合个人用户。租户可以通过账户管理、项目管理、IAM 鉴权、计量&计费管理、监控等功能自主管理资源。

|云计算运维管理平台
a|云计算运维管理平台（Business & Operation Support System，简称 BOSS）是青云企业云平台面向用户提供的运维管理界面程序。BOSS 提供了统一运维门户和基础运维功能，对云平台中的物理资源、虚拟资源和服务进行统一管理，包括硬件资源监控、服务状态监控、资源状态监控、日志查看、告警等功能。支持软件机器人自动运维和故障无害的运维技术，提高运维效率。具备良好的接口开放性，支持对系统的功能扩展。

BOSS 平台主要以管理、监控和告警三个方向对云平台资源进行管理和运维。

* 管理：以“上帝视角”查看私有云平台，以超级管理员的身份对云资源进行操作，对管理节点、计算节点、SDN 等终端用户无感知的设施进行运维操作。BOSS 还支持接入更多的青云管理工具，例如：BI2.0、V2V 迁移工具等。
* 监控：BOSS 从物理资源和虚拟资源两个角度对私有云的运行数据进行收集，提供云平台的资源使用情况，帮助运维人员了解云资源的增长趋势，并帮助决策集群扩容的时机。
* 告警：BOSS 通过 API 获取集群中所有告警的信息，通过分类、存储、聚合以判断告警是否需要发起通知，支持对通知对象和通知频率进行定制，且支持邮件、短信、webhook 推送等通知方式。

|===

== 计算

[cols="1,3"]
|===
|功能特性|描述

|云服务器
|企业云平台提供多种类型云服务器：如弹性裸金属云服务器、GPU 云服务器、弹性高性能计算等。

|弹性裸金属
|青云企业云平台采用第三代英特尔至强可扩展处理器（代号：Ice Lake）和 DPU 架构，并通过引入智能网卡，实现存储、网络等数据处理功能，让用户可独享 CPU 计算资源，从而实现裸金属服务器的完整算力保留，以极致性能满足用户各类核心业务需求。

弹性裸金属同时具备云服务器的弹性、敏捷性、灵活性与高可用，支持按需使用，且用户的业务应用可直接访问处理器和内存，无任何虚拟化开销。


|GPU 计算
|为满足高性能计算的需求，青云企业云平台提供 NVIDIA Tesla P100 型号的 GPU，单台云服务器可配置 2 张 GPU卡，单机单浮点峰值计算能力最多可达 20Tflops，并采用直通方式与云服务器对接，省去虚拟化带来的损耗，全面释放物理 GPU 的计算加速能力。

用户可自定义 GPU 云服务器的配置，可灵活指定 CPU、内存、GPU 类型和数量、系统盘大小等。

|弹性高性能计算
|基于企业云平台架构的 EHPC 集群是用于高性能并行计算的资源集，由云平台弹性计算实例组成，包含了调度管理集群资源的高性能调度器，以及运行业务所需的软件栈，解决超大规模的科学，工程和商业问题。

EHPC 集群具备节点动态调整能力，具有弹性伸缩的可扩展性，用户可按需选择计算实例类型，无需排期，即买即用。

|镜像
|青云企业云平台提供包含 Linux、Windows 等操作系统的系统镜像，也支持用户自行定义的自有镜像。使得用户方便的使用系统镜像或自有镜像创建云服务器。

|SSH 密钥
|云服务器支持 SSH 密钥登录，较普通的用户密码登录方式更为安全可靠，使得入侵者破解密钥的难度远远超出破解密码的难度。

|安置策略组
a|安置策略组提供了云服务器分组功能，提供集中和分散两种类型：

* 集中：安置策略组内的云服务器趋向于部署/迁移到相同的物理节点上。
* 分散：安置策略组内的云服务器趋向于部署/迁移到不同的物理节点上。

|===

== 存储服务

[cols="1,3"]
|===
|功能特性|描述

|硬盘
|青云企业云平台为云服务器提供块存储设备，支持多种规格和类型，并可弹性扩展，可满足用户不同场景的业务需求。支持硬盘多副本与定时备份。

|备份
|为用户提供云服务器以及云硬盘的备份恢复功能。支持全量备份与增量备份。

|文件存储 vNAS
|企业云平台为用户提供的 NAS 服务是基于标准的 NFS 和 Samba（CIFS）网络协议实现的数据传输。通过创建一个 NAS 服务器，用户可以在多个云服务器客户端以及不同的操作系统间进行数据共享。同时提供权限组和账号配置功能，便于用户对云服务器客户端进行访问控制和管理配置。

|文件存储 QFS
|提供一个无限容量的在线文件存储和访问平台，每个用户可以创建多个文件系统用以存储，可以将文件系统挂载给任意客户端使用，文件系统挂载后在客户端体现为不同的目录。

|并行文件存储 EPFS
|并行文件存储 EPFS（Elastic Parallel File System）是企业云平台提供的并行文件系统服务，用于面向超级计算服务等高性能场景。目前，并行文件存储 EPFS 仅在高性能计算专区部署。用户可根据需求，在该区域选择开通该服务并创建 EPFS。

|对象存储
|对象存储是面向海量非结构化数据的通用数据存储平台，为用户提供安全可靠、低成本的云端存储服务。适用于保存图片、视频、镜像、日志等海量非结构化数据，能够支撑移动互联网及物联网业务应用场景，并为其提供数据的生命周期管理。

|第三方存储对接
|青云企业云平台支持与第三方存储的对接。

== 网络
[cols="1,3"]
|===
|功能特性|描述

|VPC 网络
|专属私有网络 VPC 可帮助用户构建一个专属隔离的网络环境。在 VPC 网络内，用户可以自定义 IP 地址范围、创建子网，并在子网内创建云服务器、数据库、大数据等各种云资源。

|VBC 基础网络
|VBC 基础网络由企业云平台统一规划和管理，自动分配内网 IP，易用性强。

|私有网络
|私有网络用于云服务器之间的互联，类似于物理交换机（L2Switch）组成的局域网，是组成 VPC 网络的基础网络模块，用来连接不同的云服务器。


|NAT 网关
|分布式的网络地址转换服务，可以为私有网络的云服务器提供复用公网 IP 的能力。支持 SNAT 和 DNAT 功能，使多个云服务器可以共享公网 IP 访问互联网或提供互联网服务。

|弹性公网 EIP
|青云企业云平台提供独立的公网 IP 资源，包括公网 IP 地址及公网出口带宽服务。公网 IP 地址是在互联网上的合法静态 IP 地址。青云企业云平台提供的公网 IP 地址与账户而非特定的资源关联，因此可以将申请到的公网 IP 地址与任意云服务器（包括私有网络内的云服务器）、路由器、负载均衡器绑定，并且可以解绑再分配到其他资源上。

|负载均衡
|负载均衡提供安全快捷的流量分发服务，使得来自多个公网地址的访问流量经由负载均衡器后，可以自动分配到多台云服务器上。负载均衡支持自动检测与隔离故障云服务器，提高业务系统的服务能力和可用性。同时支持千万级别并发访问请求，可轻松应对大流量访问，满足业务需求。

|SDN 网卡直通
|网卡（Nic）是一种虚拟网卡，可以分配到云服务器，一台云服务器最多可以绑定 64 张网卡。SDN 网卡直通可以摆脱 Overlay 性能损耗，支持独立配置防火墙和直通 LB 服务，IP 地址由底层统一提供，方便配置管理。

|边界路由器
|边界路由器是用于连接不同网络的路由器，青云企业云平台的边界路由器可以连接多种不同的网络场景：一是连接不同 VPC 内部的虚拟私有网络，实现更大规模的云服务器组网；二是连接虚拟私有网络和物理网络，实现虚拟机和物理机混合组网。

|安全组
|通过策略配置，安全组能极大地提高一个内部网络或云服务器的安全性，并过滤不安全的服务，从而降低内部网络或云服务器的风险。

|网络访问控制
|网络 ACL 是私有网络的进出流量控制表。用户可自主创建和配置 ACL 规则，声明哪些流量可以进、出私有网络，并通过关联私有网络的出、入方向规则控制出入私有网络的数据流量。

|===

== 安全服务

[cols="1,3a"]
|===
|功能特性|描述

|Web 应用防火墙（WAF）
|Web 应用防火墙提供 Web 安全防护，为 Web 应用拦截各种 SQL 注入、XSS 跨站脚本、网站挂马等常见攻击。过滤海量恶意访问，并持续更新防护策略，从而降低网站资产和数据泄露的风险，保障 Web 应用的稳定性。

|云防火墙 CFW
|云防火墙（Cloud FireWall）是一款网络安全产品，用于防护进出 EIP 的网络攻击。用户开通云防火墙后，进出 EIP 的流量将被自动引流至云防火墙。经过云防火墙的网络访问会被云防火墙的拦截策略、入侵防护策略进行检查，有威胁的访问会被拦截，并产生安全事件日志。用户可以自定义拦截规则，选择入侵防护的规则，可以看作是安全组功能的加强版。

|安全资源池
|青云推出安全资源池方案，帮助解决租户级别的安全产品服务供给问题，让每个云租户可以根据自身的安全合规要求申请安全产品满足等保相关要求。安全资源池包括以下组件：

* 防火墙
+
防火墙提供应用层防火墙、入侵防护、防病毒、反 APT、VPN、智能带宽管理、多出口链路负载均衡、内容过滤、URL 过滤等多重安全功能，实现对用户、应用和内容的深入分析，为用户提供高性能、可视化、精准有效的应用层一体化安全防护。
* Web 应用防火墙
+
用于防护 SQL 注入、XSS、命令注入等各类 Web 攻击和威胁。通过结合机器学习、智能语义分析等优势技术，为用户的网站系统构建一道安全的屏障。
* 漏洞扫描
+
是一款融合安全厂商在信息安全漏洞挖掘、渗透测试技术研究和漏洞检查方法的最佳实践基础上，集主机安全扫描、网站安全扫描、数据库安全扫描、弱口令发现和基线配置核查于一身的检测类安全产品，通过提前发现系统的安全隐患从而帮助用户提高网络安全防护性能和抗破坏能力。漏洞扫描产品仅需与用户业务系统网络打通，即可对系统存在的漏洞和风险进行检测。
* 主机安全
+
提供主机系统防护与加固、主机网络防护与加固等功能，具备业界领先的勒索病毒专防专杀、东西向流量微隔离、补丁修复、外设管控、文件审计、违规外联检测与阻断等主机安全能力，全面提升终端安全防护水平。
* 堡垒机
+
为用户提供云主机资产的运维审计功能，产品覆盖 SSH、RDP、VNC、Telnet、FTP/SFTP 等多种协议，实现事前授权、事中监察、事后审计等完整的运维闭环。
* 日志审计
+
通过对网络设备、安全设备、主机和应用系统日志进行全面的标准化处理，并进行全维度、跨设备、细粒度的关联分析，透过事件的表象真实地还原事件背后的信息，从而协助用户全面审计信
息系统整体安全状况。
* 数据库审计
+
支持对进出核心数据库的访问流量进行数据报文字段级的解析操作，完全还原操作细节，输出详尽的操作返回结果。以可视化的方式将所有访问都呈现在管理者的面前，数据库不再处于不可知、不可控的情况，数据威胁将被迅速发现和响应。

== 容器服务

[cols="1,3"]
|===
|功能特性|描述

|容器引擎
|容器引擎（QingCloud Kubernetes Engine，简称 QKE）是在青云企业云平台上构建的企业级分布式多租户的 Kubernetes 容器服务，集成了云平台的云服务器、存储、网络等资源。用户可通过容器引擎一键部署高可用、高性能的 Kubernetes 集群。

容器引擎支持可视化界面、多集群管理、自动伸缩、运维监控、CI/CD、微服务治理、应用管理、服务与网络管理等业务场景，让用户轻松高效地在云端运行 Kubernetes 容器化应用。

|Harbor 镜像仓库
|Harbor 是一个开源的镜像仓库服务，用于容器镜像、Helm Chart 等符合 OCI 标准的 Artifact 的安全托管及高效分发，能够帮助用户跨云原生计算平台（如 Kubernetes 和 Docker）一致且安全地管理 Artifact。

AppCenter 将 Harbor 制作成 App，能直接在 AppCenter 进行一键部署，并提供了一个高可用、高安全、高性能的解决方案。

|etcd
|etcd 是为分布式系统设计的分布式高可用键值存储服务，主要用于提供注册服务，配置服务等功能。应用程序可以从 etcd 中读取、写入数据，并监控数据变化。

AppCenter 将 etcd 制作成 APP，方便用户在云平台搭建 etcd 集群。用户可借助 AppCenter 提供的能力对集群进行动态扩容、备份恢复、健康检查和监控。

|===

== 运维与管理

[cols="1,3"]
|===
|功能特性|描述

|资源编排
|资源编排（Resources Orchestration）是对资源的抽象建模和实例生成、维护的一种管理方式。用于生成一组彼此关联，即有拓扑关系的 IaaS/PaaS 资源组合的模板。用户可以应用这一模版，将模板描述的组合关系编排出一套实体资源，实现资源的自动化部署和运维。

|自动伸缩

|运维工具
|青云企业云平台提供以下运维工具：定时器、操作日志、标签、回收站、监控告警等。通过运维工具可满足日常的运维工作。


|服务器迁移中心
|提供 V2V 迁移服务，通过服务器迁移中心可将其它虚拟化平台的云主机系统及数据完整的迁移至青云企业云平台。

|VMware 纳管
|Vmware 纳管，即 VMwarevSphere 纳管，通过 VMWare 提供的公开 API 接口完成 VMwarevSphere 纳管，无缝集成 VMwarevSphere 虚拟化管理平台部分功能，实现云平台统一管理虚拟化平台。
