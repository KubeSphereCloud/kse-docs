自定义监控允许用户根据自己的业务场景建立特有的可观测性标准。

== 功能特性

* 支持自定义监控指标和监控形式。
* 支持将常用监控工具集成至云端，如用于 Kubernetes 监控的事实标准工具 Prometheus。
* 为各类云原生应用提供云上业务支撑，以及细粒度全监控功能，以提前检测出任何可能对业务造成影响的问题。

== 主要组件

监控引擎组件：

* *Prometheus*
* *Prometheus Operator*
