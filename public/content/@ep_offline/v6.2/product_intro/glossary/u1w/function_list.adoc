[cols="1a,3a"]
|===
|术语 |说明

|QingStor U10000 |青云QingCloud 非结构化存储产品名。包括对象存储与文件存储能力。
|Region |地域，表示数据中心所在的物理位置。
|Zone |QingStor U10000 的每个地域由多个相互隔离的机房组成。我们称之为可用区。
|Global Zone |部署有 Global 服务的 Zone。
|Service |对象存储的顶层命令空间。在该命名空间下，每一个用户可以创建多个 Bucket。
|Endpoint |访问域名。
|Object | 是对象存储中储存的基本数据单元，也是用户操作的基本数据单元，又称作为对象、文件。
|Object Key |对象名称
|Bucket |用户创建的存储桶。用于存储用户上传的各种对象。

|===
