== 配置可视化

配置中心提供可视化的操作界面，且已完成模型化的配置文件支持通过表单模式进行编辑，可以更清晰地知道参数的含义，降低配置查找、编辑难度。

== 支持配置校验

配置中心提供**模型**功能，可以在模型中定义配置文件的语义结构（例如配置文件中每个参数的名称、类型等），再将模型应用到具体的配置上。在修改配置内容时系统就可以根据**模型**定义的语义规则验证配置内容正确性，从而减少因格式问题导致的生产事故。

== 配置版本管理

通过对配置进行集中管理，解决文件管理的难题，并且可以对配置的历史版本进行管理，支持随时将配置回退到任一历史版本。
