== 上传安装包


. 任选一台 KS 节点作为 FirstBox 的宿主机。并将序号为 1 的压缩包上传至 `/root` 目录下。

. 登录至该物理服务器，执行如下命令行，解压已上传的安装包。包名需根据实际情况替换。
+
[source,shell]
----
tar -zxvf /root/qingcloud-installer_6.1.x-20221230_XXXX_amd64.tar.gz -C /root/
----
