---
title: 配置主机名
weight: 4
---




操作系统成功安装完成后，用户需逐一登录各物理服务器，修改主机名。

== 命名要求

为方便管理与后期运维，{product_name}要求主机名称中应包含待部署环境的 Zone ID、逻辑机柜号以及逻辑机位号。

[source,shell]
----
<Zone ID><逻辑机柜号><逻辑机位号>

----

== 命名示例

待部署环境的 Zone ID 为 `testcloud1`，当前主机所在的逻辑机柜号为 `r01`，逻辑机位号为 `n00`，对应的主机名如下。

[source,txt]
----
testcloud1r01n00

----

== 操作步骤

. 逐一登录待部署环境的各物理服务器。

. 执行以下操作，按照前文的命名要求，修改各主机名。
* 方式一
+
使用命令行进行修改，其中 `<hostname>` 为待使用的主机名。
+
[source,shell]
----
hostnamectl set-hostname <hostname>

----
* 方式二
+
直接修改配置文件 `/etc/hostname`，写入待使用的主机名。
+
[source,shell]
----
vi /etc/hostname
----
