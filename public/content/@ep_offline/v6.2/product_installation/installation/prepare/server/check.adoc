== 重启服务器

** 方式一：通过服务器管理页面进行重启。我们推荐该方式。
** 方式二：登录各个物理服务器，执行 `reboot` 命令重启。
** 方式三：直接对各物理服务器进行下电再上电操作。


== 确认硬件配置
** 各物理服务器的主机名称配置正确。
** 各物理服务器的网络配置信息，如网卡 IP 和子网掩码等均配置正确。
** 各物理服务器上硬盘盘符、容量、型号识别等信息均正确。
** 各物理服务器之间网络互通，执行 `ping` 操作，返回网络连接正常。
** 各 VG 或 KS 节点时间配置正确。
