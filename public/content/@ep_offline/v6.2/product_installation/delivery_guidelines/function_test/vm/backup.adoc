| 验收目的 | 验证云服务器的备份功能。
| 预置条件 | * {product_name}已部署完成，且当前运行状态正常，所有 Hyper 节点状态均正常。
* 已创建好测试用户 User-1 并分配适当资源配额。
* 已成功创建云服务器 VM-1。

| 操作步骤 | 
. 使用 User-1 的账户密码登录{product_name}的 Web Console。
. 通过 Web Console 登录云服务器 VM-1。
. 在云服务器 VM-1 内的 `/root` 目录下创建文件 `test-1`。
. 在 Web Console 上对云服务器 VM-1 执行 `创建备份` 操作。
. 在 Web Console 页面观察 `创建备份` 的操作结果。
. 删除云服务器 VM-1 内的 `/root` 目录下的文件 `test-1`。
. Web Console 的备份列表页面，选择刚创建的备份链进行 `回滚`。
. 确认云服务器 VM-1 内的 `/root` 目录下的文件 `test-1` 成功恢复。


| 预期结果 | . 基于云服务器创建备份执行成功。
. 基于备份恢复云服务器成功。
