|===
| 验收目的 | 创建私有网络并挂载至 VPC。
| 预置条件 | * {product_name}已部署完成，且当前运行状态正常，所有 Hyper 节点状态均正常。
* 已创建好测试用户 User-1 并分配适当资源配额。
* 已成功创建 VPC 网络 VPC-1。

| 操作步骤 | 
. 使用 User-1 的账户密码登录{product_name}的 Web Console。
. 在 Web Console 相应页面创建私有网络 Private-1。
. 将私有网络 Private-1 加入 VPC-1。
. 创建云服务器 VM-1，VM-2，并将 VM-1 与 VM-2 加入 Private-1 网络。
. 在云服务器 VM-1 内 `ping` 云服务器 VM-2。
. 在云服务器 VM-2 内 `ping` 云服务器 VM-1。


| 预期结果 | 私有网络成功创建，并能直接使用。
