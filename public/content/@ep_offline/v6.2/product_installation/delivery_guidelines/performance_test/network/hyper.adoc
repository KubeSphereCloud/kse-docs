| 验收目的 | Hyper 间性能测试。
| 预置条件 | * {product_name}已部署完成。

* 任选两台 Hyper 节点：Hyper-A，Hyper-B。
* Hyper-A 与 Hyper-B 节点上均安装有 Iperf 工具。


| 操作步骤 | . 使用 SSH 工具逐一登录 Hyper-A，Hyper-B。
. 在 Hyper-A 上执行命令行启动 Iperf Server 端。
+
[source,shell]
----
iperf -s -p 5000
----

. 在 Hyper-B 上执行命令行启动 Iperf Client 端。
+
[source,shell]
----
iperf -c < server端 IP > -p 5000 -f M -i 1
----
