== 术语说明

[cols="1a,4a"]
|===
|术语 |说明 

| PaaS License | 按 vCPU 配额、应用集群数量授权。可以精确控制每一个应用所使用的 vCPU 配额，同时还可以控制应用的集群数量。
| IaaS License | 按计算节点数量授权。可以控制{product_name}中计算节点的数量。

|===
