title: 应用包下载
weight: 2
---

公有云导出的 AppCenter 应用镜像文件统一保存在内部link:ins.qingcloud.com[ FTP 服务器]的 `/pitrix` 目录下。根据镜像文件包所包含的应用版本，目前可将其分为 `全量包` 与 `推荐版本包` 两种类型。

为确保 FTP 服务器上的镜像包包含公有云上最新的 AppCenter 应用数据，FTP 目录每天会进行自动更新同步，无需人工介入。

// //更新周期如下。

// * `全量包` 是包含所有 AppCenter 应用以及应用的所有版本，更新时间为 `每天早上 10点`。
// * `推荐版本包` 是根据link:version_set[推荐版本设置]里面的配置，有选择性的导出的 AppCenter 应用以及其版本，更新时间为 `每天中午 12点`。


* `全量包` 是包含所有 AppCenter 应用以及应用的所有版本，包含推荐的与不被推荐的版本，更新时间为 `每天早上 10点`。

* `推荐版本包` 是根据link:version_set[推荐版本设置]里面的配置，有选择性的导出的 AppCenter 应用以及其版本，更新时间为 `每天中午 12点`。

include::../../../../../_components/note.adoc[]

* 若{product_name}需要的应用版本不在 FTP 服务器上，实施人员可联系 AppCenter 值班同事手动导出目标应用的镜像文件并上传至 FTP 服务器。

* 推荐的版本是经过 AppCenter 平台确认过，该版本与指定的 IaaS 平台可兼容，建议用户根据 IaaS 平台的版本下载安装被推荐的应用包。
