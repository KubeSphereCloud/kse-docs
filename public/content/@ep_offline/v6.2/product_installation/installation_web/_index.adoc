| BOSS | 云计算运维管理平台，是{product_name}的统一运维管理门户，除了为运维人员提供统一的基础运维功能之外，还对云平台中的物理资源、虚拟资源和服务进行统一管理。

| Hyper 节点 | Hyper 节点又称之为计算节点，为{product_name}提供计算、网络、存储等云资源。
