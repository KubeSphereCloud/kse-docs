弹性高性能计算 EHPC 服务的安装部署依赖于企业云平台的搭建和 AppCenter 服务的支持，因此，用户需提前安装部署完成企业云平台以及 AppCenter 服务。一般来说，安装部署企业云平台的过程中默认自动安装 AppCenter 服务。

本文旨在介绍在 v6.1（20221230）版本的企业云平台（IaaS）上安装弹性高性能计算 EHPC 的方法。

== 版本要求

* 青云企业云平台 v6.1（20221230）
* AppCenter 2.0
* 弹性高性能计算 EHPC v1.4.8

== 读者对象

本文档主要适用于以下读者：

* 实施工程师
* 运维工程师
* 技术支持工程师

== 名词解释
[options="header",cols="1,3"]
|===
|名词|解释说明
|EHPC 
|弹性高性能计算 Elastic High Performance Computer 的英文简称。

|EPFS
|弹性并行文件存储 Elastic Parallel File System 的英文简称。

|Lustre
|一种平行分布式文件系统，EPFS 是基于该协议的网络共享文件存储服务。

|MinIO
|一种开源的对象存储，为 EHPC 提供便捷的文件上传与下载功能。

|Console
|云平台的云计算服务平台。

|BOSS
|云平台的云计算运维管理平台。

|配置中心
|云平台用于统一集中化管理 Console 和 BOSS 的配置，同时提供可视化操作界面，降低配置维护难度，提升修改配置的效率。

|===
