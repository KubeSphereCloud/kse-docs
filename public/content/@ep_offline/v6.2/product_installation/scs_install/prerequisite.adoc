* 已完成企业云平台的安装。详细操作请参考《企业云平台 安装指南》。
* 准备专属 hyper（仅安装 CFW 时需要）
+
当需要安装云防火墙（CFW）时，需准备专属的两台 hyper 节点，需满足如下要求：

** Hyper 为非 local 模式。
** hyper 节点需配置 plg 后缀为 ks、scs 或 srp 的安置组，请提前创建，例如 plg-00000ks、plg-00000scs 或 plg-00000srp。三者其一即可，且不能配置其他安置组。创建并配置安置组的详细操作请参见《企业云平台 安装指南》或 BOSS 联机文档。
