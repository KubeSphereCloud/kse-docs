该虚拟机用于部署安恒云。

== 虚拟机要求

[cols='1,3'] 
|===
| 参数	| 说明

| 镜像	| scs/ahvcmp*.lz4
| 规格	| 8 核 16G，数据盘可不配。
| 安全组	| 与 srp 服务同处一个安全组。
| 基础网络	| 与 srp 服务同处一个内网。
| 区域	| 多 region 场景，每个 region 都需要部署一个虚拟机。
|===

== 操作步骤

1. 使用系统自动创建的子账号（例如默认的 `scs` 账号），登录云平台 Console。
2. 选择**产品与服务** > **计算** > **云服务器**，进入云服务器页面。
3. 点击**创建云服务器**，进入创建云服务器页面。
