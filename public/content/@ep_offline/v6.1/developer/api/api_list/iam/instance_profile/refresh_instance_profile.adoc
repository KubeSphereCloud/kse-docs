---
title: 刷新主机IAM配置（RefreshIAMInstanceProfile）
weight: 2
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| instances
| Array
| true
| 主机ID

| interval
| Integer
| true
| 间隔

| limit
| Integer
| true
| 数量限制

| zone
| String
| true
| Zone
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| total_count
| Integer
| false
| 
|===

== 请求示例
[,javascript]
----
{
  "action": "RefreshIAMInstanceProfile",
  "instances": [
    "i-pb3n66x1"
  ],
  "zone": "testing"
}
----

== 返回示例
[,javascript]
----
{
  "action": "RefreshIAMInstanceProfileResponse",
  "ret_code": 0,
  "total_count": 1
}
----
