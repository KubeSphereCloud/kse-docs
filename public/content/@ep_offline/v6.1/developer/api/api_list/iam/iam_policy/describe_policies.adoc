== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| base_type
| String
| false
| 类型

| console
| String
| false
| ConsoleID

| controller
| Array
| false
| 控制器类型

| has_draft
| String
| false
| 是否存在草稿

| is_draft
| String
| false
| 是否草稿

| limit
| Integer
| false
| 偏移量

| offset
| Integer
| false
| 起始坐标

| owner
| String
| false
| 所有者

| policies
| Array
| false
| IAM策略ID列表

| policy_name
| String
| false
| 策略名称

| reverse
| Integer
| false
| 是否降序查询

| root_user
| String
| false
| 根用户ID

| search_word
| String
| false
| 搜索关键字

| sort_key
| String
| false
| 排序使用的列

| status
| Array
| false
| 状态

| tag
| String
| false
| 标签
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。

| 参数名 | 类型 | 是否必须 | 描述

| policy_set

| false
| IAM策略详情列表

| policy_set[].console_id

| false

| policy_set[].controller

| policy_set[].create_time

| false
| 创建时间

| policy_set[].effect
| String
| false
| 允许/拒绝

| policy_set[].has_draft
| String
| false
| 是否存在草稿

| policy_set[].is_default
| String
| false
| 是否默认

| policy_set[].is_draft
| String
| false
| 是否草稿

| policy_set[].owner
| String
| false
| 所有者

| policy_set[].policy_id
| String
| false
| IAM策略ID

| policy_set[].policy_name
| String
| false
| IAM策略名称

| policy_set[].qrn

== 请求示例
[,javascript]
----
{
  "action": "DescribeIAMPolicies",
  "limit": 20,
  "offset": 0,
  "owner": "usr-ZiEgnuY1",
  "reverse": 1,
  "sort_key": "create_time",
  "status": [
    "active"
  ]
}
----
