---
title: 创建IAM用户登录会话,获取SK登录凭证（CreateSessionForIAMUser）
weight: 3
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| ignore_pwd
| Integer
| false
| 忽略密码，需超级管理员

| login_ip
| String
| false
| 登录IP

| passwd
| String
| false
| 密码

| session_type
| String
| false
| 登录终端类型

| user
| String
| false
| IAM用户ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| sk
| String
| false
| 登录凭证

| user
| Object
| false
| IAM用户详情

| user.iam_user_id
| String
| false
| IAM用户ID

| user.lang
| String
| false
| 用户语言设置

| user.user_id
| String
| false
| 所有者ID
|===

== 请求示例
[,javascript]
----
{
  "action": "CreateSessionForIAMUser",
  "ignore_pwd": 1,
  "user": "iamtest@wangcw"
}
----

== 返回示例
[,javascript]
----
{
  "action": "CreateSessionForIAMUserResponse",
  "ret_code": 0,
  "sk": "3lPMG8VJ510TAMmxtWS2eS5gdMtUBELh",
  "user": {
    "iam_user_id": "iamu-ssqasbx8",
    "lang": "zh-cn",
    "user_id": "usr-ZiEgnuY1"
  }
}
----
