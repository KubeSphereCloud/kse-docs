---
title: 复制IAM策略（DuplicateIAMPolicy）
weight: 10
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| policy
| String
| true
| IAM策略ID

| policy_name
| String
| true
| IAM策略名称
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| controller
| String
| false
| 控制器类型

| policy_id
| String
| false
| IAM策略ID
|===

== 请求示例
[,javascript]
----
{
  "action": "DuplicateIAMPolicy",
  "policy": "iamp-dy7mtlyu",
  "policy_name": "copy1"
}
----

== 返回示例
[,javascript]
----
{
  "action": "DuplicateIAMPolicyResponse",
  "controller": "self",
  "policy_id": "iamp-m8r7qhwn",
  "ret_code": 0
}
----
