---
title: 删除IAM身份（DeleteIAMRoles）
weight: 3
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| roles
| Array
| true
| IAM身份ID列表
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| roles
| Array
| false
| IAM身份ID
|===

== 请求示例
[,javascript]
----
{
  "action": "DeleteIAMRoles",
  "roles": [
    "iamr-xv3xwskt"
  ]
}
----

== 返回示例
[,javascript]
----
{
  "action": "DeleteIAMRolesResponse",
  "ret_code": 0,
  "roles": [
    "iamr-xv3xwskt"
  ]
}
----
