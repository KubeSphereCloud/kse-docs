---
title: 查询IAM纳管服务与资源的映射关系（DescribeIAMServiceResourceMapping）
weight: 17
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| NO_NEED
| String
| false
| 无请求参数
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| resources
| Object
| false
| IAM纳管服务与资源的映射关系

| resources.project
| Array
| false
| 项目
|===

== 请求示例
[,javascript]
----
{
  "action": "DescribeIAMServiceResourceMapping"
}
----

== 返回示例
[,javascript]
----
{
  "action": "DescribeIAMServiceResourceMappingResponse",
  "resources": {
    "project": [
      "project_id",
      "owner"
    ]
  },
  "ret_code": 0
}
----
