---
title: 替换资源的IAM身份（ReplaceIAMRoleOfResource）
weight: 14
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| resource
| String
| false
| 资源ID

| role
| String
| true
| IAM身份ID

| zone
| String
| true
| ZONE
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| resource
| String
| false
| 资源ID

| role
| String
| false
| IAM身份ID
|===

== 请求示例
[,javascript]
----
{
  "action": "ReplaceIAMRoleOfResource",
  "resource": "i-07cdxc0x",
  "role": "iamr-xj92xa37"
}
----

== 返回示例
[,javascript]
----
{
  "action": "ReplaceIAMRoleOfResourceResponse",
  "resource": "i-07cdxc0x",
  "ret_code": 0,
  "role": "iamr-xj92xa37"
}
----
