---
title: 查询IAM纳管API标签信息（DescribeIAMApiActionTags）
weight: 11
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| apis
| Array
| true
| API
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| tags
| Object
| false
| 标签

| tags.api
| String
| false
| API操作

| tags.owner
| String
| false
| 归属的产品服务

| tags.tag
| String
| false
| 标签
|===

== 请求示例
[,javascript]
----
{
  "action": "DescribeIAMApiActionTags",
  "apis": [
    "DescribeInstances"
  ]
}
----

== 返回示例
[,javascript]
----
{
  "action": "DescribeIAMApiActionTagsResponse",
  "ret_code": 0,
  "tags": {
    "DescribeInstances": [
      {
        "api": "DescribeInstances",
        "owner": "appCenter",
        "tag": "instance"
      }
    ]
  }
}
----
