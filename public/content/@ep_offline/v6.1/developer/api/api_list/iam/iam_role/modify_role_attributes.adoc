---
title: 修改IAM身份信息（ModifyIAMRoleAttributes）
weight: 13
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| description
| String
| true
| 描述

| duration_seconds
| Integer
| false
| ID_TOKEN有效/刷新时间

| role
| String
| true
| IAM身份ID

| role_name
| String
| true
| IAM身份名称
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| role_id
| String
| false
| IAM身份ID
|===

== 请求示例
[,javascript]
----
{
  "action": "ModifyIAMRoleAttributes",
  "role": "iamr-xv3xwskt",
  "role_name": "testrole2"
}
----

== 返回示例
[,javascript]
----
{
  "action": "ModifyIAMRoleAttributesResponse",
  "ret_code": 0,
  "role_id": "iamr-xv3xwskt"
}
----
