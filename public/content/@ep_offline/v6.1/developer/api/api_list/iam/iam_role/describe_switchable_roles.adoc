---
title: 查询关联IAM身份（DescribeSwitchableRoles）
weight: 8
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| owner
| String
| true
| 身份归属账号ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| roles
| Object
| false
| IAM身份

| roles.description
| String
| false
| 描述信息

| roles.email
| String
| false
| 邮箱

| roles.gravatar_email
| String
| false
| 头像邮箱

| roles.gravatar_url
| String
| false
| 头像url

| roles.owner
| String
| false
| IAM身份所有者

| roles.qrn
| String
| false
| IAM身份资源唯一标识

| roles.role_id
| String
| false
| IAM身份ID

| roles.role_name
| String
| false
| IAM身份名称

| roles.user_name
| String
| false
| 用户名

| roles.user_type
| Integer
| false
| 用户类型：0-主账号,1-子账号
|===

== 请求示例
[,javascript]
----
{
  "action": "ModifyIAMRoleAssociationAttributes",
  "owner": "usr-MQBpcyCV"
}
----

== 返回示例

{
  "action": "ModifyIAMRoleAssociationAttributesResponse",
  "ret_code": 0,
  "roles": {
    "description": "",

    "role_id": "iamr-fcdddkxg",

    "user_type": 0
  }
