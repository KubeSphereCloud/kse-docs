---
title: 取消关联IAM策略（DetachIAMPolicy）
weight: 9
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| entity_urns
| Array
| true
| IAM用户或身份ID的QRN

| policies
| Array
| true
| IAM策略ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| entity_urns
| Array
| false
| IAM用户或身份ID的QRN

| policies
| Array
| false
| IAM策略ID
|===

== 请求示例
[,javascript]
----
{
  "action": "DetachIAMPolicy",
  "entity_urns": [
    "qrn:qingcloud:iam::usr-ZiEgnuY1:role/iamr-yd9lhi1a"
  ],
  "policies": [
    "iamp-m8r7qhwn"
  ]
}
----

== 返回示例
[,javascript]
----
{
  "action": "DetachIAMPolicyResponse",
  "entity_urns": [
    "qrn:qingcloud:iam::usr-ZiEgnuY1:role/iamr-yd9lhi1a"
  ],
  "policies": [
    "iamp-m8r7qhwn"
  ],
  "ret_code": 0
}
----
