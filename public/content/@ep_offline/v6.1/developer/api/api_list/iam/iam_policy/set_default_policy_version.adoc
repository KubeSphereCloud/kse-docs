---
title: 应用草稿为默认版本（SetDefaultIAMPolicyVersion）
weight: 12
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| policy
| String
| true
| IAM策略ID

| policy_version
| Integer
| true
| 策略版本号
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| policy
| String
| false
| IAM策略ID

| version
| Integer
| false
| 策略版本号
|===

== 请求示例
[,javascript]
----
{
  "action": "SetDefaultIAMPolicyVersion",
  "policy": "iamp-dy7mtlyu",
  "policy_version": 1
}
----

== 返回示例
[,javascript]
----
{
  "action": "SetDefaultIAMPolicyVersionResponse",
  "policy": "iamp-dy7mtlyu",
  "ret_code": 0,
  "version": 1
}
----
