---
title: 驳回版本审核（RejectAppVersion）
weight: 21
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| message
| String
| true
| 审核意见

| operator_email
| String
| true
| 审核人邮箱

| operator_name
| String
| true
| 审核人

| review_role
| String
| true
| 审核渠道

| version_id
| String
| true
| 应用版本ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| version_id
| String
| false
| 应用版本ID
|===

== 请求示例
[,javascript]
----
{
  "message": "审核驳回",
  "operator_email": "xxx@test.com",
  "operator_name": "管理员1",
  "review_role": "dev",
  "version_id": "appv-xxxxxx"
}
----

== 返回示例
[,javascript]
----
{
  "action": "RejectAppVersionResponse",
  "ret_code": 0,
  "version_id": "appv-xxxxxx"
}
----
