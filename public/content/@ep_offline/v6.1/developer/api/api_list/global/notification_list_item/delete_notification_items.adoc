---
title: 删除通知接收方式（DeleteNotificationItems）
weight: 2
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| notification_items
| Array
| true
| 
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| notification_items
| Array
| false
| 接收方式id
|===

== 请求示例
[,javascript]
----
{
  "action": "DeleteNotificationItems",
  "notification_items": [
    "ni-xxxxxxxx"
  ]
}
----

== 返回示例
[,javascript]
----
{
  "action": "DeleteNotificationItemsResponse",
  "notification_items": [
    "ni-xxxxxxxx"
  ],
  "ret_code": 0
}
----
