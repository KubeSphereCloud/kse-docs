== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| app
| Array
| false
| 应用

| app_name
| String
| false
| 应用名称

| app_type
| Array
| false
| 查询应用类型

| display_columns
| Array
| false
| 返回字段

| exclude_apps
| Array
| false
| 排除应用列表

| filter_monitor
| Integer
| false
| 是否只查询自定义监控应用

| limit
| Integer
| false
| 个数

| offset
| Integer
| false
| 偏移

| owner
| String
| false
| 应用所有者

| reverse
| Integer
| false
| 是否倒序

| shared
| Integer
| false
| 是否共享

| sort_key
| String
| false
| 排序字段

| status
| Array
| false
| 状态

| tags
| Array
| false
| 标签

| verbose
| Integer
| false
| 冗余字段级别

| visibility
| String
| false
| 可见状态

| with_hide
| Integer
| false
| 是否查询隐藏应用

| without_third
| Integer
| false
| 是否只查询青云出品应用
|===
