---
title: 解除用户关联应用（DeleteUserApps）
weight: 4
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| app
| Array
| true
| 应用id

| owner
| String
| false
| 待解绑用户id
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| ret_code
| Integer
| false
| 状态码
|===

== 请求示例
[,javascript]
----
{
  "app": [
    "app-owxi28r5"
  ],
  "owner": "admin"
}
----

== 返回示例
[,javascript]
----
{
  "action": "DeleteUserAppsResponse",
  "ret_code": 0
}
----
