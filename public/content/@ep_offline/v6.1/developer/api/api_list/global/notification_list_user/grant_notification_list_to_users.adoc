---
title: 授权通知列表给其他用户（GrantNotificationListToUsers）
weight: 2
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| notification_list
| String
| true
| 通知列表id

| users
| Array
| true
| 用户id
|===

== 返回数据

此接口仅包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。

== 请求示例
[,javascript]
----
{
  "action": "GrantNotificationListToUsers",
  "notification_list": "nl-xxxxxxxx",
  "users": [
    "usr-xxxxxxxx"
  ]
}
----

== 返回示例
[,javascript]
----
{
  "action": "GrantNotificationListToUsersResponse",
  "ret_code": 0
}
----
