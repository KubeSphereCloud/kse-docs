---
title: 获取邮件模板（DescribeEmailTemplates）
weight: 4
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述


|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| email_templates
| Array
| false
| 模板集合

| total_count
| Integer
| false
| 查询结果总数
|===

== 请求示例
[,javascript]
----
{
  "action": "DescribeEmailTemplates"
}
----

== 返回示例
[,javascript]
----
{
  "action": "DescribeEmailTemplatesResponse",
  "email_templates": [
    "coupon-remind.html",
    "coupon.html"
  ],
  "ret_code": 0,
  "total_count": 2
}
----
