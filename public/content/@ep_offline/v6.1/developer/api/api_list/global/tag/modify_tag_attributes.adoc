---
title: 修改标签属性（ModifyTagAttributes）
weight: 6
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| color
| String
| false
| 标签颜色

| description
| String
| false
| 标签描述

| scope
| String
| false
| 范围

| system
| Boolean
| false
| 是否是系统标签

| tag
| String
| true
| 标签id

| tag_key
| String
| false
| 标签key

| tag_name
| String
| true
| 标签名称

| zone
| String
| false
| 区域ID
|===

== 返回数据

此接口仅包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。

== 请求示例
[,javascript]
----
{
  "action": "ModifyTagAttributes",
  "color": "default",
  "description": "",
  "scope": "zone",
  "system": false,
  "tag": "tag-7gvvxz5z",
  "tag_key": "",
  "tag_name": "lzl3"
}
----

== 返回示例
[,javascript]
----
{
  "action": "ModifyTagAttributesResponse",
  "ret_code": 0
}
----
