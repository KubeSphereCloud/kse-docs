weight: 1

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| apps
| Array
| true
| 应用id

| bundle
| String
| true
| bundle包ID

| rank
| Integer
| false
| 排序(默认1)
|===

== 返回数据

此接口仅包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。

== 请求示例
[,javascript]
----
{
  "apps": [
    "app-xxxxxx"
  ],
  "bundle": "bun-qingcloud",
  "rank": 1
}
----

== 返回示例
[,javascript]
----
{
  "action": "AddAppsToBundleResponse",
  "ret_code": 0
}
----
