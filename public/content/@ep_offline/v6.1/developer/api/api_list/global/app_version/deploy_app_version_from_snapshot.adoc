== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| app_id
| String
| true
| 应用ID

| auto_renew
| Integer
| false
| 自动续费

| charge_mode
| String
| false
| 计费模式

| conf
| String
| true
| 集群详细配置

| debug
| Integer
| false
| 开发者模式

| months
| Integer
| false
| 续费时长（月）

| multi_deploy_zones
| Array
| false
| 多可用区域部署

| order_id
| String
| false
| 订单ID

| snapshot_id
| String
| true
| 备份ID

| version_id
| String
| true
| 应用版本ID

| zone
| String
| true
| 可用区ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| app_id
| String
| false
| 应用ID

| app_version
| String
| false
| 应用版本

| cluster_id
| String
| false
| 集群ID

| cluster_name
| String
| false
| 集群名称

| job_id
| String
| false
| 任务ID

| node_count
| String
| false
| 节点数量

| node_ids
| Array
| false
| 节点id

| ret_code
| Integer
| false
| 返回状态码

| vxnet_id
| String
| false
| 私有网络ID
|===
