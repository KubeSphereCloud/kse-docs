---
title: 恢复用户欠费资源。用户充值后也会促发此接口。（ResumeResources）
weight: 22
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| user
| String
| false
| 用户ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| user_id
| String
| false
| 用户ID
|===

== 请求示例
[,javascript]
----
{
  "action": "ResumeResources",
  "user": "usr-jj8r3F2k"
}
----

== 返回示例
[,javascript]
----
{
  "action": "ResumeResourcesResponse",
  "ret_code": 0,
  "user_id": "usr-jj8r3F2k"
}
----
