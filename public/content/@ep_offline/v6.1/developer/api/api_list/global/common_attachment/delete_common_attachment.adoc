---
title: 删除一般附件（DeleteCommonAttachment）
weight: 1
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| attachment_ids
| Array
| true
| 附件id列表
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| message
| String
| false
| 操作结果
|===

== 请求示例
[,javascript]
----
{
  "attachment_ids": [
    "ca-6n9q88u9"
  ]
}
----

== 返回示例
[,javascript]
----
{
  "action": "DeleteCommonAttachment",
  "message": "ok",
  "ret_code": 0
}
----
