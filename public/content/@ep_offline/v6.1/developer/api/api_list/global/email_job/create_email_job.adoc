---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| activated
| Integer
| false
| 是否已认证

| console_id
| String
| false
| 控制台

| content
| String
| false
| 邮件内容

| end_balance
| String
| false
| 最大余额

| role
| Array
| false
| 角色

| search_word
| String
| false
| 模糊查询

| sms_content
| String
| false
| 短信内容

| start_balance
| String
| false
| 最小余额

| status
| Array
| false
| 状态

| template
| String
| true
| 模板

| title
| String
| false
| 邮件标题

| user_type
| Array
| false
| 用户类型

| users
| Array
| false
| 用户id
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| email_job_id
| String
| false
| 邮件任务id

| total_count
| Integer
| false
| 结果总数
|===

== 请求示例
[,javascript]
----
{
  "action": "CreateEmailJob",
  "activated": 1,
  "console": [],
  "content": "email test",
  "role": [],
  "search_word": "",
  "status": [],
  "template": "",
  "title": "email-title",
  "user_type": null,
  "users": [
    "usr-xxxxxxxx"
  ]
}
----

== 返回示例
[,javascript]
----
{
  "action": "CreateEmailJobResponse",
  "email_job_id": "ejob-xxxxxxxx",
  "ret_code": 0,
  "total_count": 1
}
----
