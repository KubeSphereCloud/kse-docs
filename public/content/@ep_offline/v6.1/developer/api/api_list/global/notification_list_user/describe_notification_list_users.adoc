---
title: 获取用户通知列表（DescribeNotificationListUsers）
weight: 1
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| limit
| Integer
| false
| 查询数量

| notification_list
| String
| true
| 用户通知列表id

| offset
| Integer
| false
| 查询偏移量

| reverse
| Integer
| false
| 是否倒序
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| notification_list_user_set
| Array
| false
| 用户通知列表集合

| notification_list_user_set[].console_id
| String
| false
| 控制台

| notification_list_user_set[].create_time
| String
| false
| 创建时间

| notification_list_user_set[].notification_list_id
| String
| false
| 用户通知列表id

| notification_list_user_set[].root_user_id
| String
| false
| 跟用户id

| notification_list_user_set[].user
| Object
| false
| 用户信息

| notification_list_user_set[].user.email
| String
| false
| 邮箱

| notification_list_user_set[].user.user_id
| String
| false
| 用户id

| total_count
| Integer
| false
| 查询结果总数
|===

== 请求示例
[,javascript]
----
{
  "action": "DescribeNotificationListUsers",
  "notification_lists": "nl-xxxxxxxx"
}
----
