---
title: 修改资源目录属性（ModifyTagDirectoryAttributes）
weight: 5
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| directory_id
| String
| false
| 资源目录id
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| directory_id
| Array
| false
| 资源目录id列表
|===

== 请求示例
[,javascript]
----
{
  "action": "ModifyTagDirectoryAttributes",
  "directory_id": "dir-yrdghz3u",
  "directory_name": "研发部"
}
----

== 返回示例
[,javascript]
----
{
  "action": "ModifyTagDirectoryAttributesResponse",
  "directory_id": "dir-yrdghz3u",
  "ret_code": 0
}
----
