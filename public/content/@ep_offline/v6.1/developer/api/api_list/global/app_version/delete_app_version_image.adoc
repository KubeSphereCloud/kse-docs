---
title: 删除应用版本镜像（DeleteAppVersionImage）
weight: 3
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| image_id
| String
| true
| 镜像ID

| version_id
| String
| true
| 版本ID

| zone_id
| String
| true
| 区域
|===

== 返回数据

此接口仅包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。

== 请求示例
[,javascript]
----
{
  "image_id": "img-xxxxxx",
  "version_id": "appv-xxxxxx",
  "zone_id": "testing"
}
----

== 返回示例
[,javascript]
----
{
  "action": "DeleteAppVersionImageResponse"
}
----
