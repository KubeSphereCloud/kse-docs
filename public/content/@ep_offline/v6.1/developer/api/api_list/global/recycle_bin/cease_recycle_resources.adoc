---
title: 彻底销毁资源（CeaseRecycleResources）
weight: 1
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| destroy_mode
| Int
| false
| 销毁模式:1自动0手动

| owner
| String
| false
| 回收资源的所有者

| resources
| Array
| false
| 需要销毁的资源列表

| zone
| String
| true
| 请求区ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| job_ids
| Array
| false
| 

| resource_ids
| Array
| false
| 
|===

== 请求示例
[,javascript]
----
{
  "action": "CeaseRecycleResources",
  "owner": "usr-ei6RFhNN",
  "resources": [
    "i-fxkq9dvw"
  ],
  "user_id": "usr-ei6RFhNN",
  "zone": "pek3"
}
----
