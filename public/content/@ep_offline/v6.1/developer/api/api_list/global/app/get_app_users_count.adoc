---
title: 查询用户安装应用统计信息（GetAppUsersCount）
weight: 16
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| apps
| Array
| true
| app_id数组
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| app_users
| Object
| false
| app安装使用量

| app_users.app_id
| Integer
| false
| 查询参数中app对应的安装总量

| total_count
| Integer
| false
| 查询结果总数
|===

== 请求示例
[,javascript]
----
{
  "apps": [
    "app-xxxxx1",
    "app-xxxx2"
  ]
}
----

== 返回示例
[,javascript]
----
{
  "action": "GetAppUsersCountResponse",
  "app_users": {
    "app_id": 1
  },
  "ret_code": 0,
  "total_count": 1
}
----
