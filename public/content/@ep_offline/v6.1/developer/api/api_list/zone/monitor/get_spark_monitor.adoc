---
title: 获取监控数据（GetSparkMonitor）
weight: 25
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| action
| String
| false
| 方法

| end_time
| String
| false
| 结束时间

| meters
| Array
| false
| 指标

| owner
| String
| false
| 用户Id

| resource
| String
| false
| 资源Id

| start_time
| String
| false
| 开始时间

| step
| String
| false
| 粒度

| zone
| String
| false
| 区域
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| action
| String
| false
| 方法

| meter_set
| Object
| false
| 监控详情

| meter_set.data_set
| Object
| false
| 监控数据

| meter_set.meter_id
| String
| false
| 指标Id

| resource_id
| String
| false
| 资源Id

| ret_code
| Integer
| false
| 返回码

| trace_id
| String
| false
| 追踪Id
|===

== 请求示例
[,javascript]
----
{
  "action": "GetSparkMonitor",
  "end_time": "2023-06-07T05:05:00Z",
  "meters": [
    "healthy"
  ],
  "resource": "cl-av5lr3nr",
  "start_time": "2023-06-07T05:00:00Z",
  "step": "5m"
}
----

== 返回示例
[,javascript]
----
{
  "action": "GetSparkMonitorResponse",
  "meter_set": [
    {
      "data_set": [],
      "meter_id": "healthy"
    }
  ],
  "resource_id": "cl-av5lr3nr",
  "ret_code": 0,
  "trace_id": "0py4178J"
}
----
