---
title: 恢复删除的告警策略（RecoverAlarmPolicies）
weight: 19
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| alarm_policies
| Array
| true
| 策略id数组

| owner
| String
| false
| 用户

| zone
| String
| false
| 区域
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| alarm_policies
| Array
| false
| 策略id数组
|===

== 请求示例
[,javascript]
----
{
  "action": "RecoverAlarmPolicies",
  "alarm_policies": [
    "alp-yqugoish"
  ],
  "owner": "usr-rrPKHDpJ",
  "zone": "staging"
}
----

== 返回示例
[,javascript]
----
{
  "action": "RecoverAlarmPoliciesResponse",
  "alarm_policies": [
    "alp-yqugoish"
  ],
  "ret_code": 0
}
----
