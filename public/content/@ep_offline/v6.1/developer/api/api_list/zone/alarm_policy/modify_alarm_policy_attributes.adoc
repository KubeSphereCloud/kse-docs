---
title: 修改告警策略属性（ModifyAlarmPolicyAttributes）
weight: 17
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| alarm_policy
| String
| true
| 策略id

| alarm_policy_name
| String
| true
| 策略名称

| description
| String
| true
| 策略描述

| owner
| String
| false
| 用户

| zone
| String
| false
| 区域
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| alarm_policy_id
| String
| false
| 告警策略id
|===

== 请求示例
[,javascript]
----
{
  "action": "ModifyAlarmPolicyAttributes",
  "alarm_policy": "alp-yqugoish",
  "alarm_policy_name": "test2222",
  "description": "1111",
  "owner": "usr-rrPKHDpJ",
  "zone": "staging"
}
----

== 返回示例
[,javascript]
----
{
  "action": "ModifyAlarmPolicyAttributesResponse",
  "alarm_policy_id": "alp-yqugoish",
  "ret_code": 0
}
----
