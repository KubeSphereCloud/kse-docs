== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===

| action
| String
| false
| 方法

| limit
| Integer
| false
| 每页数量

| offset
| Integer
| false
| 页数

| false

| false

| false
| 标签

| Integer
| false
| 详情

| zone
| String
| false
| 区域
|===
