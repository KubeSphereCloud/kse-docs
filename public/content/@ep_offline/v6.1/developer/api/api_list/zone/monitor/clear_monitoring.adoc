---
title: 清理监控数据（ClearMonitoring）
weight: 2
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| action
| String
| false
| 方法

| owner
| String
| false
| 用户Id

| target_host
| Array
| false
| 目标主机

| topics
| String
| false
| 主题Id

| zone
| String
| false
| 区域
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| action
| String
| false
| 方法

| ret_code
| Integer
| false
| 返回码

| topic_letters
| Object
| false
| 主题详情

| topic_letters.ret_code
| String
| false
| 返回码

| topic_letters.target_host
| String
| false
| 目标主机

| topic_letters.topic_id
| String
| false
| 主题Id

| trace_id
| String
| false
| 追踪Id
|===

== 请求示例
[,javascript]
----
{
  "action": "ClearMonitoring",
  "owner": "usr-rrPKHDpJ",
  "resource": "cl-av5lr3nr",
  "target_host": "sr01n17",
  "topics": "cluster"
}
----

== 返回示例
[,javascript]
----
{
  "action": "ClearMonitoringResponse",
  "ret_code": 0,
  "topic_letters": [
    {
      "ret_code": "0",
      "target_host": "sr01n17",
      "topic_id": "cluster"
    }
  ],
  "trace_id": "0py4178J"
}
----
