---
title: 添加定时器任务（AddSchedulerTasks）
weight: 2
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| owner
| String
| false
| 资源的所有者

| scheduler
| String
| true
| 定时器id

| scheduler_task_name
| String
| true
| 定时器任务名称

| scheduler_tasks
| Array
| true
| 定时器任务列表

| task_params
| Object
| true
| 任务参数

| task_params.resources
| Array
| true
| 资源列表

| task_type
| String
| true
| 任务类型

| zone
| String
| true
| 请求区ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| scheduler_tasks
| Array
| false
| 
|===

== 请求示例
[,javascript]
----
{
  "action": "AddSchedulerTasks",
  "owner": "usr-BnnisHWR",
  "scheduler": "schd-0g60lk75",
  "scheduler_tasks": [
    {
      "scheduler_task_name": "fdsfds",
      "task_params": {
        "resources": [
          "i-g1l5c926"
        ]
      },
      "task_type": "start_instances"
    }
  ],
  "zone": "testing"
}
----

== 返回示例
[,javascript]
----
{
  "action": "AddSchedulerTasksResponse",
  "ret_code": 0,
  "scheduler_tasks": [
    "st-6051s6cn"
  ],
  "trace_id": "Ye2OVhJ4"
}
----
