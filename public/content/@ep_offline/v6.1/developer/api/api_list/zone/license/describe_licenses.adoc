== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| license_id
| String
| true
| 证书ID

| license_type
| String
| true
| 证书类型

| limit
| Integer
| true
| 查询个数

| offset
| Integer
| true
| 偏移量

| product_id
| String
| true
| 产品id

| product_name
| String
| true
| 产品名称

| reverse
| Integer
| true
| 查询结果是否反转

| status
| String
| true
| 状态

| sub_product_id
| String
| true
| 子产品id

| zone
| String
| true
| 区域
|===
