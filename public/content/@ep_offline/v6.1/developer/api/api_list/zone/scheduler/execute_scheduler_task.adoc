---
title: 执行定时器任务（ExecuteSchedulerTask）
weight: 9
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| owner
| String
| false
| 资源的所有者

| scheduler_task
| String
| true
| 定时器任务ID

| zone
| String
| true
| 请求区ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| scheduler_task_id
| String
| false
| 定时器任务ID
|===

== 请求示例
[,javascript]
----
{
  "action": "ExecuteSchedulerTask",
  "owner": "usr-BnnisHWR",
  "scheduler_task": "st-vy8dwhzo",
  "zone": "testing"
}
----

== 返回示例
[,javascript]
----
{
  "action": "ExecuteSchedulerTaskResponse",
  "ret_code": 0,
  "scheduler_task_id": "st-vy8dwhzo",
  "trace_id": "gb6GucEJ"
}
----
