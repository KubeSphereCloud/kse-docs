---
title: 检查账户锁（CheckUserLock）
weight: 2
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| api_action
| String
| false
| api行为

| lock_type
| String
| true
| 账户锁类型

| user
| String
| true
| 用户id
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| is_locked
| Boolean
| false
| 是否锁定

| item_type
| String
| false
| 

| lock_type
| String
| false
| 账户锁类型

| user_id
| String
| false
| 用户id
|===

== 请求示例
[,javascript]
----
{
  "api_action": "GetSecretAccessKey",
  "lock_type": "api",
  "user": "admin"
}
----

== 返回示例
[,javascript]
----
{
  "action": "CheckUserLockResponse",
  "is_locked": true,
  "item_type": "user_lock",
  "lock_type": "api",
  "ret_code": 0,
  "user_id": "admin"
}
----
