---
title: 查询用户账户配额剩余（GetAccountQuotaLeft）
weight: 2
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| resource_types
| Array
| false
| 查询的资源类型

| user
| String
| false
| 用户id
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| account_quota_left_set
| Array
| false
| 配额剩余详情

| account_quota_left_set[].left
| String
| false
| 剩余数量

| account_quota_left_set[].resource_type
| String
| false
| 资源类型

| user_id
| String
| false
| 用户id
|===

== 请求示例
[,javascript]
----
{
  "action": "GetAccountQuotaLeft",
  "user": "usr-KmJmFe0V"
}
----
