---
title: 获取用户偏好（DescribeUserPreference）
weight: 1
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| users
| Array
| true
| 用户id列表
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| user_preference_set
| Array
| false
| 用户偏好列表

| user_preference_set[].indep_quota
| String
| false
| 用户独立配额区域

| user_preference_set[].user_id
| String
| false
| 用户id
|===

== 请求示例
[,javascript]
----
{
  "users": "admin"
}
----

== 返回示例
[,javascript]
----
{
  "action": "DescribeUserPreferenceResponse",
  "ret_code": 0,
  "total_count": 0,
  "user_preference_set": [
    {
      "indep_quota": "testing",
      "user_id": "admin"
    }
  ]
}
----
