---
title: 删除用户公共附件（DeleteUserCommonAttachment）
weight: 1
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| attachment
| String
| true
| 附件id列表

| owner
| String
| true
| 附件所有者的用户id
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| attachment_id
| String
| false
| 附件id
|===

== 请求示例
[,javascript]
----
{
  "action": "DeleteUserCommonAttachment",
  "attachment": "usca-ebkch62k",
  "owner": "usr-ddyfbbTV"
}
----

== 返回示例
[,javascript]
----
{
  "action": "DeleteUserCommonAttachmentResponse",
  "attachment_id": "usca-ebkch62k",
  "ret_code": 0
}
----
