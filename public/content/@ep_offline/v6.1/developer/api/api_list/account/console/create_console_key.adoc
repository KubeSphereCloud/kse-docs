---
title: 创建控制台密钥（CreateConsoleKey）
weight: 2
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| console_key_name
| String
| false
| 控制台密钥的名称

| description
| String
| false
| 控制台密钥的描述

| owner
| String
| false
| 创建控制台密钥的所属用户ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| console_key_id
| String
| false
| 控制台密钥id

| secret_console_key
| String
| false
| 控制台密钥的私钥
|===

== 请求示例
[,javascript]
----
{
  "action": "CreateConsoleKey",
  "console_key_name": "zzzz",
  "description": "aaaaaaa",
  "owner": "admin"
}
----

== 返回示例
[,javascript]
----
{
  "action": "CreateConsoleKeyResponse",
  "console_key_id": "SEVDSHXGUNTCOPAITNNE",
  "ret_code": 0,
  "secret_console_key": "MtxVMeHNQpUbMM8swnuDT3nMy7x2z0piyY1k0QYE"
}
----
