---
title: 查询第三方登录账号关联用户（DescribeLoginAccountUsers）
weight: 7
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| console
| String
| false
| 

| limit
| Integer
| false
| 查询条数

| login_accounts
| Array
| false
| 登录账号列表

| offset
| Integer
| false
| 查询偏移量

| search_word
| String
| false
| 查询关键字

| status
| Array
| false
| 账户状态列表

| users
| Array
| false
| 用户列表
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| login_account_user_set
| Array
| false
| 查询账户用户结果

| total_count
| Array
| false
| 查询总数
|===

== 请求示例
[,javascript]
----
{
  "console": "console",
  "limit": 20,
  "login_accounts": [
    "login_account_1",
    "login_account_2"
  ],
  "offset": 10,
  "search_word": "admin",
  "status": [
    "activated"
  ],
  "users": [
    "user_1",
    "user_2"
  ]
}
----

== 返回示例
[,javascript]
----
{
  "action": "DescribeLoginAccountUsersResponse",
  "login_account_user_set": [],
  "ret_code": 0,
  "total_count": 0
}
----
