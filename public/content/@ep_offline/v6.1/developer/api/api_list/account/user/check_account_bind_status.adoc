---
title: 检查账户绑定状态（CheckAccountBindStatus）
weight: 4
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| account_id
| String
| false
| 账户ID

| user_id
| String
| false
| 用户ID
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| code
| String
| false
| 
|===

== 请求示例
[,javascript]
----
{
  "account_id": "account_id",
  "user_id": "user_id"
}
----

== 返回示例
[,javascript]
----
{
  "action": "CheckAccountBindStatusResponse",
  "code": 0,
  "ret_code": 0
}
----
