---
title: 授予用户区域访问权限（GrantZoneAccess）
weight: 19
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| quota_times
| Integer
| false
| 

| user
| String
| true
| 用户

| zones
| Array
| false
| 区域列表
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| user_id
| String
| false
| 用户ID
|===

== 请求示例
[,javascript]
----
{
  "quota_times": 1,
  "user": "admin",
  "zones": [
    "pk3"
  ]
}
----

== 返回示例
[,javascript]
----
{
  "action": "GrantZoneAccessResponse",
  "ret_code": 0,
  "user_id": "admin"
}
----
