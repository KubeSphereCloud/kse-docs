title: 确认邮箱（ConfirmEmail）
weight: 7
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| send_welcome_email
| Integer
| false
| 是否发送欢迎邮件

| user
| String
| true
| 用户

| verification_code
| String
| true
| 验证码
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| email
| String
| false
| 邮箱

| user_id
| String
| false
| 用户ID
|===

== 请求示例
[,javascript]
----
{
  "send_welcome_email": 0,
  "user": "admin",
  "verification_code": "4sa52sdh"
}
----

== 返回示例
[,javascript]
----
{
  "action": "ConfirmEmailResponse",
  "email": "473******",
  "ret_code": 0,
  "user_id": "admin"
}
----
