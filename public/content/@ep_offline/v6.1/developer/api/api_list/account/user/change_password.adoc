---
title: 更换登录密码（ChangePassword）
weight: 3
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| newpasswd
| String
| true
| 新密码

| oldpasswd
| String
| true
| 旧密码

| user
| String
| false
| 用户
|===

== 返回数据
此接口除以下返回参数外，还包含公共响应头。关于公共响应头的更多信息，请参见link:../../../../overview/#_api_返回结构[前文内容]。
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| user_id
| String
| false
| 用户ID
|===

== 请求示例
[,javascript]
----
{
  "newpasswd": "******",
  "oldpasswd": "******",
  "user": "user"
}
----

== 返回示例
[,javascript]
----
{
  "action": "ChangePasswordResponse",
  "ret_code": 0,
  "user_id": "admin"
}
----
