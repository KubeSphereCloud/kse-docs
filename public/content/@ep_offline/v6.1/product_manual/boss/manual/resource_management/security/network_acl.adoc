== 背景信息

网络访问控制（ACL）是私有网络（VXNet）的进出流量控制表，用于控制整个私有网络（VXNet）的进出流量策略和配置隧道/VPN/NAT 的 ACL 过滤规则，以完成 VPC 的网络流量在通过隧道/VPN/NAT 与其他网络互通时的访问控制，方便用户实现精细化的安全规则设置。

== 场景描述

该任务指导管理员进入 *安全：网络访问控制* 页面，查看云环境中所有规则子集的列表信息，以及修改规则子集为全局规则子集。

== 功能描述

网络访问控制列表展示了当前云环境中所有的访问控制名称及 ID、应用修改状态、用户邮箱及 ID、创建时间、状态更新时间信息。

== 前提条件

已获取 *云计算运维管理平台* 的账号和密码。

== 操作步骤

. 登录 {platform_name_boss}。
. 选择 *资源管理* > *安全* > *网络访问控制*，查看网络访问控制列表和详情。
+
image::/images/boss/manual/resource_mgt/network_acl_operation.png[]

. 点击网络访问控制的名称，在详情页面可查看网络访问控制详细信息以及上行规则和下行规则列表信息。
+
image::/images/boss/manual/resource_mgt/network_acl_info.png[]
