== 背景信息

用户借助 “IP/端口集合” 功能把具有相同特征的一组 IP 或者一组端口设置成为 “IP/端口集合”，并且在安全组规则中进行添加，实现批量管理功能。

== 场景描述

该任务指导管理员进入 *安全：IP/端口集合* 页面，查看云平台 IP/端口集合列表信息。

== 功能描述

*安全：IP/端口集合* 页面集中展示了云平台中所有 IP/端口集合的名称及 ID、类型、安全组名称及 ID、创建时间信息。

== 前提条件

已获取 *云计算运维管理平台* 的账号和密码。

== 操作步骤

. 登录 {platform_name_boss}。
. 选择 *资源管理* > *安全* > *IP/端口集合*，进入 *安全：IP/端口集合* 页面，可查看 IP/端口集合列表和详情。
+
image::/images/boss/manual/resource_mgt/security_group_ipsets_operation.png[]

. 点击 IP/端口集合的名称，进入 IP/端口集合详情页面，可查看详细信息。

=== 导出 IP / 端口集合列表

. 在 *安全: IP/端口集合* 页面左上方选择区域或可用区，在右上角点击 *导出*。
. 点击报表管理中心。
+
进入 *报表管理中心* 页面。

. 在模块为 *IP 端口集合* 所在行点击对应文件名称。
+
IP/端口集合列表信息保存到本地。
