**地址**：``\http://<虚拟机IP地址>/console``

**账号**：管理控制台默认账户为 ``superadmin``；密码可以自定义设置，或者使用默认的 ``Zhu88jie@``。

== 本地 PC 和虚拟机网络互通

如果本地 PC 和安恒云虚拟机网络互通，请在浏览器地址栏输入 ``\http://虚拟机IP地址/console`` 进行访问。

== 本地 PC 和安恒云虚拟机的网络不通

如果本地 PC 和安恒云虚拟机的网络不通，本地 PC 无法直接访问安恒云管理控制台。

可以通过如下三种方式，配置后进行访问。

* **方式一：绑定公网 IP**
+
如果云平台支持公网，可以为安恒云虚拟机绑定一个EIP，然后通过EIP进行访问，即在浏览器地址栏输入 `\http://虚拟机EIP地址/console` 进行访问。

* **方式二：配置隧道代理**
+
. 在本地 PC，执行以下命令开启隧道代理。
+
[source]
----
sudo ssh -i ~/.ssh/id_rsa -g -f -N -L {PC Port}:${虚拟机IP}:443 root@${FB节点IP} -p 22
----
+
默认访问端口为 `443`。如果本地 PC 的 443 端口已被占用，可以修改为其他未被占用的端口号，例如 2234。
+
示例：
+
[source]
----
sudo ssh -i ~/.ssh/id_rsa -g -f -N -L 443:10.30.1.8:443 root@172.31.60.190 -p 22
----
+
. 配置隧道代理后，在浏览器地址栏输入 ``\http://127.0.0.1:port/console`` 进行访问。
+
如果使用默认访问端口 443，访问时无需输入端口号。

* **方式三：创建 Windows 虚拟机**
+
在安恒云虚拟机同一网段再创建一台 Windows 系统的虚拟机，然后通过该虚拟机进行访问。
+
. 使用系统自动创建的子账号（例如默认的 `scs` 账号），登录云平台 Console。
. 选择**产品与服务** > **计算** > **云服务器**，进入云服务器页面。
. 点击创建云服务器，进入创建云服务器页面。
+
** 选择规格：任意规格。
** 选择镜像：选择任意 Windows 镜像。
** 选择基础网络：与 srp 服务保持一致。
** 选择安全组：与 srp 服务保持一致。
. 登录创建的 Windows 虚拟机，在虚拟机中浏览器地址栏输入 ``\http://虚拟机 IP 地址/console`` 进行访问。
