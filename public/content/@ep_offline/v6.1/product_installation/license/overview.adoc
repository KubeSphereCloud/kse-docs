根据 License 的有效期，{product_name} License 分为试用版 License、POC License 和生产环境 License。

用户需要关注 License 的有效期，为了确保业务的连续性，请在 License 到期前，重新申请并注册 License。

== 试用版 License

{product_name}基础软件部署完成后，系统自动注册 IaaS & PaaS 的试用版 License，用户可以直接试用软件功能。从软件部署后开始计时，有效期为 3 个月。

== POC License

用于 POC。相关人员可分别为 IaaS 与 PaaS 申请 License，申请 License 时，指定期限即为其有效期。

==  生产环境 License

用户可分别为 IaaS 与 PaaS 申请 License，申请 License 时，指定期限即为其有效期。
