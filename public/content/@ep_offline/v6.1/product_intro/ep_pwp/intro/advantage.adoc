== 弹性灵活，业务有保障

支持基于负载均衡构建自动伸缩的应用集群，实现真正的实时弹性扩缩容，保障业务高峰期海量客户的正常访问。
采用分布式实时资源调度，保证 IT 资源的合理分配和最高使用率，减少因资源分配而导致的故障。

== 资源丰富，满足各类需求

提供 IaaS、PaaS、SaaS 及 AI 服务应用，可以在云平台各层不断增加服务，具有很强的扩展性。提供全栈的云计算服务能力，从虚拟机、LVM 到容器，满足各种场景下的云计算能力需求。

提供一站式高性能计算服务，以及稳定、高性价比的计算资源。计算节点之间以 P2P 点对点式同步备份，最大可支持高达数十万节点的计算集群，既保障了业务数据的安全，也为业务扩容保留广阔的空间。

== 智慧运营，洞悉业务营收

具有完整的青云公有云运营、运维能力，帮助客户实现可自运营的云平台。基于公有云背景，在计量计费、应用运营、运维管理等多个层面进行企业级增强，提供公有云的体验，私有化的部署，实现了计量计费和运营能力的落地，将 IT 投入转化为业务营收。

== 快速部署，开放兼容

提供自动化、可快速部署、可管理、安全高效的开发测试环境，还能够与企业当前已有的系统无缝集成，支持与各类异构技术生态和应用生态的对接，保护企业现有 IT 资产，为软件产品大规模快速迭代提供捷径，满足市场创新需求。
