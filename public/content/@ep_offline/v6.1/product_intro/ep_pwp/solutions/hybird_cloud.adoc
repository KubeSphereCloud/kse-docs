== 方案简介

混合云是当前企业应对数字化转型挑战的重要工具策略，其架构能够同时兼顾敏捷性和安全性，提升资源的扩展性和使用效率，降低业务应用在全生命周期中的复杂性，有效控制综合 IT 成本，提升企业的综合竞争实力。

青云混合云提供的是一个具有“全栈”、“全态”、“全域”能力的全维云平台，客户可以根据自身的需要和发展阶段，自由的选择、组合相应的功能模块。青云希望通过混合云帮助用户更好的规划符合自己业务需求的数字化演进战略，完成传统应用的现代化改造，新旧体系的融合，构建全面敏捷开放的现代化 IT 体系：

* 依托青云公、私、托、混一体化架构，以及新一代智能网络能力，构建数据可自由迁移、业务可按需安置的狭义混合云。
* 通过 KubeSphere 容器平台，完成混合云从面向资源到面向业务应用服务的转型，实现业务的自由迁移和无缝混合部署。
* 通过边缘计算及 IoT 平台，进入物联空间，将云延展至广域空间，构建覆盖全域的混合云架构。

== 方案优势

* 统一架构，极致体验
+
青云企业云平台无论在公有云或者私有云市场，都经过了大规模商业实践的检验，并经过多年不断的产品技术迭代，已经全面覆盖了 IaaS、PaaS 和应用管理，是一个完备、可靠、高品质的的全栈云平台。同时，青云企业云平台以完全一致的技术架构交付公有云、私有云、托管云、行业云等各种形态的云服务和云平台产品，不同形态之间具备统一的 API、功能组件、服务能力、操作方式，及运维管理体系，为用户提供贯通一致的云平台使用体验。

* 云网一体，深度协同
+
将自主开发和运营的 SDN 智能网络服务与各形态的云平台进行了深度整合优化，只需几分钟即可构建起专属的企业级广域网络，实现云平台间 VPC 级别的安全可靠的网络互通，并可随云平台一起实现智能动态伸缩调节，应对不断变化的业务需求。云网的深度一体化融合，为构建混合云提供强大的网络能力保障。

* 无缝互联，数据流动
+
通过青云分布式软件存储的远程数据同步能力，并依托 SDN 智能网络实现的网络连接保障，可以实现数据的跨平台备份与迁移，同时提供高可靠的数据安全保障，为混合云实现业务灾备系统奠定了基础。

* 运维管理，统一平面
+
青云通过统一多云管理平台，对接青云及各类第三方云平台，将运维管理能力进行汇总，为用户建立起统一的运维管理平面。通过这一平面，用户可以对混合云各个组件进行统一的运维操作，同时也可建立起全局视角，对混合云的整体资源进行更有效的管理与调度。

* 从“资源为核心”走向“业务应用服务为核心”
+
基于青云容器平台能够便捷的构建混合多云应用服务，屏蔽多云状态下基础设施层的差异性，支持各类应用的统一分发与管理，助力应用全面云化转型。
