== 方案简介

随着云计算的普及，数据库云化成为企业在数字化转型过程中的必然选择。云数据库能充分利用云计算“资源池化”的优势，在成本、可用性、易用性、扩展性和并行处理方面较传统数据库有很大优势。云数据库即开即用，用户可以根据自身的业务情况弹性开支、灵活调整。无需从头采购基础软硬件，无需考虑专业人员部署，节省了人力物力，消除了人员、硬件、软件的重复配置，让软、硬件升级变得更加容易。随着云时代分布式应用、大数据、容器、AI等技术架构的发展，也为数据库云化的功能及效果带来更强的支撑和驱动力。

青云企业云平台提供多种类型云数据库，包括：关系型数据库、分布式数据库、时序数据库、分析型数据库、数据库管理平台等企业级云数据库产品和服务，已为中国上千家企业及社区用户提供了成熟的企业级服务。丰富的数据库类型对应不同业务，统一具备青云云数据库的共有能力，并持续完善增强：

* 支持多种算力平台：可基于标准云服务平台和云原生容器平台提供数据库服务，服务能力完全一致。
* 快速构建成熟可用的数据库服务能力：自动部署、水平扩展、自动化运维、统一监控。
* 服务能力多样化：支持多种类型 RDS 数据库服务、支持多种分布式数据库、分布式时序数据库、缓存数据库。
* 丰富的安全性：支持云数据库同城多活、异地灾备架构，支持与对象存储整合实现数据库服务的任意时间点恢复。
* 全面兼容信创：云数据库支持基于信创平台构建数据库服务能力，全面兼容信创平台。

同时青云联合国内顶尖的数据库服务领域的合作伙伴，可以为用户提供端到端的数据库相关服务，包括便不限于数据库方案设计和实施、迁移、数据库性能优化，以及安全、运维、紧急救援等多类型服务。

== 方案优势

* 屏蔽底层复杂度，专注业务开发
+
青云企业云数据库服务平台，通过将数据库能力服务化、自动化，将有效帮助企业用户专注于业务层能力构建，无需关注数据库底层技术架构细节如高可用、管理和运维等复杂问题，帮助企业用户大幅提升业务价值。
* 更全面的数据库服务能力
+
青云企业云平台可提供主流的 RDS 服务（MySQL、PGSQL...）外，还支持主流的缓存数据库服务、多种 NOSQL
数据库、分布式数据库、时序数据库，基于完全同一套框架能力，多服务间充份整合可联动，支持丰富的数据流动特性，兼顾企业多种业务的复杂数据库服务需求。

* 与通用云计算平台和云原生平台深度整合，场景能力更深入
+
青云云数据库平台与青云企业云平台和青云云原生 KubeSphere 两大算力平台深度整合，支持丰富的企业级特性，并充分利用标准云计算平台和云原生平台实现跨云平台移植、秒级水平伸缩、高资源利用率与隔离、运维效率高、持续部署与测试等特性，以更轻量、更灵活、更开放、更标准平台来支撑企业业务的发展和需求。

* 开源和云原生生态
+
青云云数据库采取持续的开源策略，维护云原生社区生态，保证产品的开放性、通用性、成长性，具备持续稳定的生命力。
