堡垒机为用户提供云服务器资产的运维审计功能，产品覆盖 SSH、RDP、VNC、Telnet、FTP/SFTP 等多种协议，支持通过浏览器 Web 页面和本地 C/S 客户端工具的方式访问云服务器，从而实现事前授权、事中监察、事后审计等完整的运维闭环。仅需与用户业务系统网络打通，用户便可使用堡垒机对云服务器等资产进行运维。

*组网说明*：

* 堡垒机安全实例通过 SSH、Telnet、FTP、RDP、VNC、Oracle、DB2、MySQL、SQL Server 等运维协议控制与它同网段的云服务器。
* 运维用户只能通过访问堡垒机安全实例，间接访问堡垒机安全实例下面的云服务器，以实现审计运维用户的运维操作。
* 堡垒机安全实例可以控制基础网络、VPC 网络下的云服务器，只需堡垒机安全实例加入与需要保护的云服务所在网络。
