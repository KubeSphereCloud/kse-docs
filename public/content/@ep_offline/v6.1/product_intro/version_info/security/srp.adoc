== 防火墙组件

支持 EIP 一键引流，通过一键开启/关闭防护，实现流量牵引目的，简化繁琐的路由配置步骤，降低技术门槛，减少引流失败机率，提升用户体验。

image::/images/ep_offline/feature/security_01.png[]

== 数据库审计

支持自动安装/卸载 Agent 功能，针对青云镜像的资产，可通过自动安装 Agent 实现资产纳管的目的。简化繁琐的操作步骤，提升用户体验。

image::/images/ep_offline/feature/security_02.png[]

== 主机安全

* 支持一键读取资产信息
+
通过**同步资产**功能，自动读取用户该区域下所有的云服务器资产，避免资产遗漏的问题。

* 支持自动安装/卸载 Agent 功能
+
针对青云镜像的资产，可通过自动安装 Agent 实现资产纳管的目的。简化繁琐的操作步骤，提升用户体验。

image::/images/ep_offline/feature/security_03.png[]

== 操作日志

新增操作日志功能，可以记录用户操作行为，包括操作时间、状态、实例 ID、实例类型、操作行为字段。

当出现安全实例购买失败、资源不足、部署失败、激活失败、单点登录失败等情况，可快速定位问题，提高解决问题的效率。

image::/images/ep_offline/feature/security_04.png[]
