---
title: 上线 VMware 纳管服务
weight: 20
---

通过 VMWare 提供的公开 API 接口完成 VMware vSphere 纳管，无缝集成 VMware vSphere 虚拟化管理平台部分功能，实现对企业云平台、VMware vSphere 虚拟化管理平台的统一管理。

* 管理 VMware vSphere 平台
+
支持在云平台上添加、查看、修改、删除 Vmware vSphere 平台。
+
image::/images/ep_offline/feature/vmware_vsphere_list.png[]


* 管理 VMware 虚拟机
+
支持在云平台上对已纳管的 Vmware vSphere 平台中的虚拟机进行管理，包括打开/关闭虚拟机电源、关闭/重启虚拟机操作系统、查看虚拟机基本信息、查看虚拟机监控、修改虚拟机等。
+
image::/images/ep_offline/feature/vmware_vm_list.png[]
