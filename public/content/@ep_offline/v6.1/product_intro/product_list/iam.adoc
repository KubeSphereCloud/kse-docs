title: "IAM 访问鉴权管理"
weight: 9

[cols='1,1,3']
|===
| 产品  | 交付模式  | 产品简介



| 身份管理 | 可选 | 提供临时身份管理，支持为其他账号以及产品资源提供临时身份，支持设置关联策略，从而实现权限控制。
当拥有临时身份时，账号可通过切换身份来管理租户内的资源。

用户可通过 API 或下载相应语言的 SDK 来代入身份访问资源。

| 策略管理| 可选 | 通过 IAM 策略定义一系列服务或资源的访问权限范围。IAM 策略可附加在身份或用户上，当用户拥有该策略，则代表该身份具备此策略所定义的权限。

支持策略的自定义，包括勾选策略对应的访问资源 API 以及适应的资源范围。

| 用户管理| 可选 |一个租户空间下可以创建多个用户，资源归属于租户，用户可根据权限使用资源。用户管理提供用户的新增、设置密码、设置访问方式、关联策略等能力。
| 操作日志| 可选 |提供该租户下对应的操作日志查看和筛选。

|===
