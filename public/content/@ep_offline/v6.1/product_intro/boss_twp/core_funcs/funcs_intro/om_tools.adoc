== V2V 迁移

V2V 迁移工具提供静态迁移能力，可将其他云平台（Vmware、Hyper-v、OpenStack等）的虚拟主机及数据盘通过 WEB 上传的方式迁移到青云云平台。

image::/images/ep_offline/boss_twp/30_40.png[]

== 在线命令

为了简化对物理主机的管理，青云提供了远程执行检查命令的功能。管理员只需要在云计算管理平台管理界面就可以对指定物理主机执行选择的命令，并得到相应结果。管理员不需要输入密码，方便管理员通过后台进行运维操作。

image::/images/ep_offline/boss_twp/30_41.png[]

== 操作记录

操作日志主要展示区域或可用区中执行的操作和任务，以及罗列云平台中各用户执行的任务列表和详情。

== 定时任务

定时任务能够实现按周期执行或者或者指定时间执行依次指定任务，现在版本支持“生成资源使用量报告“和”批量迁移主机“这 2 个任务类型。管理员可以设置一次性执行或者定期重复执行。
