基础设施管理综合展现和管理云平台的物理资源和云平台级的关键服务。

== 云服务器

青云 IaaS 平台所交付的云服务器服务，是云计算的核心业务，也是用户最为关心的计算的核心。云计算管理平台对云平台的云服务器进行管理、资源监控和分配。

image::/images/ep_offline/boss_twp/30_20.png[]


== 计算节点

计算节点是平台用于分配虚拟主机等资源的节点，包含 CPU 、内存和硬盘的物理计算资源，一个计算节点需占用超融合机箱设备的一个节点，一个计算节点上可运行多个虚拟主机。这个模块可以对负责计算的物理节点进行监控，列表支持监控和普通列表两种视图。

== 管理节点

管理节点是承载云平台核心服务的物理节点和虚拟机节点。

== 基础服务

基础服务展示维护云环境正常对外服务的各个服务进程运行状况，包括管理节点和计算节点等服务进程。

image::/images/ep_offline/boss_twp/30_21.png[]

== 基础组件

基础组件是系统底层IAAS平台依赖的基础第三方组件，这些组件的正常与否也直接决定了云平台的整体是否能正常运行，平台底层主要依赖PostgreSQL，Pgpool新增的内容， Redis， Cassandra， Zookeeper， Memcached 等组件。

image::/images/ep_offline/boss_twp/30_22.png[]

== 安置组

青云云计算管理平台的安置组根据硬件类型用于云平台划分资源调度策略，根据不同类型的资源调度请求下发到不同安置组的计算节点，管理员可以根据实际资源池划分需求，配置相应的安置组规则，不同安置组之间不会触发自动迁移。

image::/images/ep_offline/boss_twp/30_23.png[]
