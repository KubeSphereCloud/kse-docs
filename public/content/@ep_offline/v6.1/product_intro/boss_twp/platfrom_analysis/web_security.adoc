---
title: "平台 Web 安全策略"
weight: 15
---

云计算管理平台经过多个版本的迭代，已经基本完善了各种安全策略，可满足客户等保三级认证的需求。

[discrete]
=== session 策略

* 支持用户配置 session 有效期。
* 支持每次请求时刷新 session 过期时间。
* 支持在浏览器关闭时过期 session 以退出登录并限制同一账号同时登录 session 的数量以实现 BOSS 平台的 Web 安全。

[discrete]
=== 密码策略

* 系统采用强密码规则，支持配置密码过期时间，过期后系统自动禁用账号。
* 支持配置在过期前发送提醒以增加用户账号的安全性。

[discrete]
=== 登录保护策略

系统支持配置统一账号在限定时间内连续登录失败多次会被临时锁定账号一段时间，以避免账户被暴力破解盗用。

[discrete]
=== 双因素验证

* 支持 TOTP（Time-Based One-Time Password Algorithm：基于时间的一次性口令）双因素验证和配置，强制用户在指定时间内开启双因素认证否则禁号。
* 提升了双因素验证码的有效期，在保证安全性的同时，也可避免用户因设备时间与 CST（Central Standard Time：）时间不同步导致的体验问题。

[discrete]
=== 账号保护策略

支持配置在连续长时间不登录后自动禁用该账号。

[discrete]
=== 审查日志转储策略

支持审查日志定时转储。
