== 前提条件
已创建云服务器

== 软件版本

本操作示例使用如下软件版本，在实际操作时，请以实际软件版本为准。

* 操作系统： CentOS Stream 9 64bit
* Nginx 版本： 1.20.1

== 操作步骤
=== 安装 Nginx
1. 在云服务器列表，点击名称列点击 Web 终端图标，以 root 用户登录云服务器。
具体操作方法，请参见通过 http://test.com[Web 终端连接云服务器]。

2. 执行以下命令安装 Nginx。
+
[source,shell]
----
# 安装 Nginx
yum -y install nginx
# 查看 Nginx 版本
nginx -v
----
