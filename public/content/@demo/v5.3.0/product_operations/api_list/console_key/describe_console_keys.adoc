---
title: 查看Console（DescribeConsoleKeys）
weight: 3
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| console
| String
| false
| Console

| console_key_name
| Array
| false
| Console公钥名称的列表

| console_keys
| Array
| false
| Console公钥的列表

| limit
| Int
| false
| 偏移量

| offset
| Int
| false
| 起始坐标

| owner
| String
| false
| Console公钥的所有者ID

| search_word
| String
| false
| 搜索关键字
|===

== 返回数据
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| console_id
| String
| false
| Console的ID

| console_key_id
| String
| false
| Console公钥

| console_key_name
| String
| false
| Console公钥的名称

| create_time
| String
| false
| Console公钥的创建时间

| description
| String
| false
| Console公钥的描述

| owner
| String
| false
| Console公钥的所有者

| secret_console_key
| String
| false
| Console公钥的私钥
|===

== 请求示例
[,javascript]
----
{
  "action": "DescribeConsoleKeys",
  "console_id": "testingcloud",
  "console_key_name": [
    "zzzz"
  ],
  "console_keys": [
    "FSTMGTOMPKSZJHCTHZHC"
  ],
  "limit": 20,
  "offset": 0,
  "owner": "usr-xCrXIiXv",
  "search_word": "zzzz"
}
----
