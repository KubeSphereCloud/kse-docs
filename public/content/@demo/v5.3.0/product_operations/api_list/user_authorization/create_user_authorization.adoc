---
title: 创建认证授权（CreateUserAuthorization）
weight: 1
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| client_id
| String
| true
| 

| user_id
| String
| true
| 用户id
|===

== 返回数据
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| client_id
| String
| false
| 

| user_authorization_id
| String
| false
| 授权的id

| user_id
| String
| false
| 用户id
|===

== 请求示例
[,javascript]
----
{
  "action": "CreateUserAuthorization",
  "client_id": "testing",
  "user_id": "usr-HsF6MOoB"
}
----

== 返回示例
[,javascript]
----
{
  "action": "CreateUserAuthorizationResponse",
  "client_id": "testing",
  "ret_code": 0,
  "user_authorization_id": "uauth-TvQelSs3",
  "user_id": "usr-HsF6MOoB"
}
----
