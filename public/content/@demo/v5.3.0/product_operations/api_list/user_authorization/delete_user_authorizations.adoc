---
title: 删除/取消用户授权、认证（DeleteUserAuthorizations）
weight: 2
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| user_authorization_ids
| Array
| true
| 用户认证id
|===

== 返回数据
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| user_authorization_ids
| String
| false
| 用户认证id
|===

== 请求示例
[,javascript]
----
{
  "user_authorization_ids": [
    "uauth-TvQelSs3"
  ]
}
----

== 返回示例
[,javascript]
----
{
  "action": "DeleteUserAuthorizationsResponse",
  "ret_code": 0,
  "user_authorization_ids": [
    "uauth-TvQelSs3"
  ]
}
----
