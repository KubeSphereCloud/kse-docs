---
title: 删除子用户（DeleteSubUsers）
weight: 3
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| status
| String
| false
| 子用户的状态

| users
| Array
| true
| 子用户ID的列表
|===

== 返回数据
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| users
| Array
| false
| 子用户ID的列表
|===

== 请求示例
[,javascript]
----
{
  "action": "DeleteSubUsers",
  "status": "active",
  "users": [
    "usr-xqm5ieFD"
  ]
}
----

== 返回示例
[,javascript]
----
{
  "action": "DeleteSubUsersResponse",
  "ret_code": 0,
  "users": [
    "usr-xqm5ieFD"
  ]
}
----
