== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| user
| String
| true
| 用户

| zones
| Array
| false
| 区域列表
|===

== 返回数据
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| user_id
| String
| false
| 用户ID
|===

== 请求示例
[,javascript]
----
{
  "user": "admin",
  "zones": [
    "pk3"
  ]
}
----

== 返回示例
[,javascript]
----
{
  "action": "ProhibitZoneAccessResponse",
  "ret_code": 0,
  "user_id": "admin"
}
----
