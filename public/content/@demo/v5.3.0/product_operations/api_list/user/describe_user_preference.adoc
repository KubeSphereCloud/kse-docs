---
title: 获取用户偏好（DescribeUserPreference）
weight: 17
---

== 基本信息

=== Path
/iaas

=== Method
GET

== 请求参数
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| users
| Array
| true
| 用户id列表
|===

== 返回数据
[cols="1,1,1,2", options="header"]
|===
| 参数名 | 类型 | 是否必须 | 描述

| user_preference_set
| Array
| false
| 用户偏好列表
|===

== 请求示例
[,javascript]
----
{
  "users": "admin"
}
----

== 返回示例
[,javascript]
----
{
  "action": "DescribeUserPreferenceResponse",
  "ret_code": 0,
  "total_count": 0,
  "user_preference_set": [
    {
      "indep_quota": "vpn",
      "user_id": "admin"
    }
  ]
}
----
