title: 查看服务日志
weight: 3

{product_name} 服务端由若干服务模块组成。

当用户请求传输到 {product_name} 服务端时，{product_name} 服务端首先会根据 VIP，分配该用户请求至某网关节点的 Nginx 服务进行处理，Nginx 服务再根据请求类型将该请求发送至 `global-gateway`，`qs-gateway3` 或 `qs-cache` 进行处理。


在查看具体日志之前，用户需对各模块的功能，以及日志文件进行了解，以便快速定位问题。同时，由于 {product_name} 日志内容较多，用户可根据 Request ID 来查找具体请求的相关日志内容。
