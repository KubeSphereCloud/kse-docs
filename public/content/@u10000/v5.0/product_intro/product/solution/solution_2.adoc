== 用户概况

慈铭体检是中国体检行业 A 股上市企业，以健康体检为主营业务，涵盖深度体检、私人医生定制、精准健康管理服务、绿色就医转诊等集团化、连锁式经营机构。旗下拥有 `慈铭体检`、`慈铭奥亚健康管理医院` 等子品牌。

== 业务挑战

* 随着业务拓展和体检中心的快速扩张，每天都有大量 PACS 影像数据产生。
* 医疗行业规定：门诊数据，保存不得低于 15 年；住院数据，保存不得低于 30 年。这些不断累计的数据对存储产生了挑战。
* 数据存储在公有云必须脱敏，需要将其迁回到私有化环境。
* 各分院影像系统均为独立建设，各中心数据共享和应用融合无法实现，信息孤岛的存在直接导致阅片效率极低。


== 解决方案

image::/images/u10000/product_13.png[,100%]

* 6 节点 U10000 一体机作为医疗影像平台的分布式存储资源池。
* U10000 存储系统通过 S3 API 对接 PACS 影像业务系统。
* 各个体检中心通过专线将前端服务器临时存储的 PACS 数据上传至总部数据中心。


== 客户收益

* 企业级分布式存储系统扩展性和数据处理能力提升巨大，能够满足不断增长的数据存储需求。
* 相比公有云对象存储，私有化部署完全解决行业合规问题；同时，在长期成本和运维成本上，对象存储私有化部署相比公有云方案拥有更低成本。
* PACS 系统通过 AWS S3 接口对接 U10000，实现 DICOM 影像高效安全的存储调阅。还实现集团内各分院 CT 检查患者信息的统一管理、统一存储和统一服务。
