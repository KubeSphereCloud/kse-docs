== 用户概况

洋河公司是行业内唯一拥有洋河、双沟两个中国名酒、中华老字号，以及洋河、蓝色经典、双沟、双沟珍宝坊、梦之蓝五个中国驰名商标的企业。英国著名的权威品牌评估机构（BF）发布排行榜，洋河酒业位列中国第二、全球第三位。

== 业务挑战

* 洋河股份的企业信息化需要从传统的 ERP 跳转到基于智慧的、基于互联网云计算的平台进行核心建设。
* 现有 IT 架构无法支撑企业的快速发展，运行效率低下。
* 现有 IT 架构无法支撑企业对新技术的试点。


== 解决方案

image::/images/u10000/product_15.png[,100%]

* U10000 一体机搭建存储集群，存储系统通过 API 支持 iOS、Andriod、Browser 客户端接入。
* U10000 通过接入服务接入客户端，将业务 IO 通过负载均衡服务进行分发处理，对冷热数据进行分级存储。
* 通过无缝对接大数据平台，将客户画像进行刻画分析，实时将客户信息进行存储分类。


== 客户收益

* 大数据 AI 应用在智慧渠道中的精准画像，对整个终端的流量、客户画像、商圈画像进行分析，通过分析决定洋河股份市场政策投入的方向和策略。
* AI 分析的关键数据，渠道图片和视频均存储在青云存储 U10000 平台内，并和大数据应用无缝对接，实时的进行数据分析挖掘，提高营销成果的快速分析能力。
* 极大提高渠道进销存的运转效率，以及营销系统的投放精准性。节省营销费用超 60%。
