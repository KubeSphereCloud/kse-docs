== 用户概况

江苏省重点交通基础设施建设项目省级投融资平台，负责江苏省高速公路、铁路、机场、港口、码头等重点交通基础设施建设的投融资，江苏省高速公路的运营和管理，以及相关竞争性企业的资产和市场经营管理。

== 业务挑战

* 旧有系统无法应对每天数千万张路况图片的存储处理能力。
* 系统的数据和信息交换及融合困难，流程和功能无法实现协同统一。
* 缺乏从业务出发结合信息化的特点进行业务流程改进的角度来研发应用系统，造成信息化成果落地不足。


== 解决方案

image::/images/u10000/product_16.png[,100%]

* 搭建 U10000 集群承接交控高速各种音视频、监控、收费收据。无须任何改造，上层业务系统可标准化地对接 U10000。
* U10000 中存储超过数百亿的图片文件，并利用自身异步数据处理框架，作为整个车牌识别方案的数据中枢来进行数据存储与流转。


== 客户收益

* 实现全路网内路况分析的实时化和业务联动一体化。大幅提高辖区内路网的精细化管理能力，使得成本大幅降低，包括采购成本、运维成本等。
* 实现数据快速存储和实时分析能力，相比于传统监控分析方案，将车牌分析的响应完成时间，由小时级大幅降低到分钟级。
* 构建灵活、开放的数据分析处理体系，提升 IT 系统的持续运营能力。
