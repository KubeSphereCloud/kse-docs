U10000 是一款软件定义的可大规模横向扩展的非结构化数据统一存储产品，架构上遵循业界先进的 Scale-out、服务化、微服务化等设计原则。

image::/images/u10000/product_2.png[,100%]

如上图所示，U10000 作为一款分布式存储产品，从软件架构上主要分为三层：

* 接入层（Access Layer）
* 存储层（Storage Layer）
* 统一运维管理平台（Intelligent Portal）


== Access Layer 接入层

支持兼容标准 POSIX 存储访问接口和标准的 CIFS、NFS 接口，同时提供业界标准的 S3 对象和大数据存储 HDFS 存储接口，可无缝与多种业务系统应用适配，承载多种企业复杂业务应用。

== Storage Layer 存储层

负责存储保存数据，处理数据创建、删除、读写等基本请求，也对外提供丰富的高级功能特性、多维度的资源管理能力以及独特的存储引擎。

通过副本模式或 EC 模式对外提供可靠的、分布式化的数据管理能力，存储容量支持水平扩展，灵活可靠地支撑业务需求。

== Intelligent Portal 统一管理运维平台

为用户提供管理服务，包括资源管理、服务管理、系统管理、用户管理、监控告警、日志信息收集等，实现对存储集群的各种配置、操作和监控，友好的可视化操作界面能有效降低运维难度和人力成本。
