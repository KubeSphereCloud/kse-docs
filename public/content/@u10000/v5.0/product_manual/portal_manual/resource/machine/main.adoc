服务器主页面，显示当前环境中的所有服务器列表。

. 成功登录 Portal。

. 点击**存储资源 > 服务器**，进入服务器列表页面。用户可在服务器搜索框内，输入待查询的服务器名，来搜索目标服务器。

**页面信息说明**

* 如图所示，服务器 ``storage0-cluster1`` 的 IP 地址为 ``101.100.11.78``，在整个系统中充当``全能节点``的角色，当前状态正常。挂载的索引盘 ``1`` 个在线且无异常离线索引盘，索引盘容量已使用 ``28.09%``。挂载的数据盘有 ``12`` 个，且无异常数据盘，存储容量已使用 ``31.94%``。
* 服务器在系统中充当的**角色**，目前可分为``网关节点``、``存储节点``、``全能节点``。
* 服务器的**状态**，目前可分为``在线``与``离线``，分别表示服务器运行正常与服务器运行异常。
