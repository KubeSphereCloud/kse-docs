对象存储模块提供 Bucket 相关的基本操作，配置与监控相关内容。
用户可在该页面创建、删除、禁用 Bucket；也可对 Bucket 的容量配额进行调整，开启关闭生命周期等子资源功能。
除此之外，该页面还提供 Bucket 的 API 请求，流量以及存储容量的实时监控功能。

include::../../../../../../_components/note.adoc[]

该页面支持查看 Bucket 内的 Object 属性，但不支持 Object 相关操作，包括对 Object 进行上传、下载与删除操作。
