该页面展示集群 API 请求以及网关的不同 HTTP 返回码的统计信息。运维人员需重点关注 `4xx` 和 `5xx` 的错误请求的 OPS。

用户在成功登录 Portal 后，点击**对象存储 > 业务指标 > API 统计**。

image::/images/u10000/stor_bucket_business_api.png[stor_bucket_business_api,100%]

**页面信息说明**

* 用户可将鼠标移动至图例相应的时间点，以查看详细信息。
* 如页面所示，Portal 支持用户根据需求，对监控项的时间范围进行设定。如：实时、3小时、1天的监控数据是从当前时间往前取指定时间长度得到的数据，而自定义的数据是限定了起始时间和截止时间得到的数据。
* {product_name} 系统对各指标的数据采集时间间隔可参考《{product_name} 常见问题》。
