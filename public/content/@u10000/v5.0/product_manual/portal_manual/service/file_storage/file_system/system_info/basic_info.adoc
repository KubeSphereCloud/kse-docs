对已成功创建的文件系统，用户可查看其详细配置信息，包括配额，数据块大小等。详细操作步骤如下。

. 成功登录 Portal。

. 点击**存储服务 > 文件存储 > 文件系统**，进入文件系统列表页面。该页面列表显示当前环境中已成功创建的文件系统。

. 在文件系统列表页面点击目标文件系统名，进入文件系统详情页面，默认显示**基本信息**标签页。
+
image::/images/u10000/stor_file_system_basic.png[stor_file_system_basic,50%]
+
include::../../../../../../../../_components/note.adoc[]

* 文件系统**状态**不为 `正常` 时，需及时调查原因并解决，以免文件系统不能正常对外提供访问。
* **配额** `限制` 的容量，为用户实际可用容量。如**配额**设置为 `10GB`，则说明用户当前文件系统可用容量为 `10GB`。
