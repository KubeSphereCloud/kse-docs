从用户组移除用户，可参考如下步骤。


. 成功登录 Portal。

. 点击**存储服务 > 文件存储 > 文件用户**，默认进入用户列表页面。

. 点击**用户组**，进入用户组列表页面。

. 在用户组列表页面，点击目标用户组的**更多 > 管理用户**。
+
image::/images/u10000/stor_file_user_group_add.png[stor_file_user_group_add,100%]

. 弹出**管理用户**对话框，该页面显示当前已添加至用户组的用户列表。点击目标用户的**移除**操作，可将目标用户从用户组移除。
+
image::/images/u10000/stor_file_user_group_add_1.png[stor_file_user_group_add_1,40%]

. 用户可在**管理用户**对话框内确认，成功移除的用户，不在**管理用户**对话框的用户列表显示。
