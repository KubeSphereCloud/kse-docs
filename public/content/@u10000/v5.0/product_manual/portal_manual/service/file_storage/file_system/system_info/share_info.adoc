description: 
weight: 2

用户可查看指定文件系统当前已创建的挂载点信息，包括共享目录、挂载路径、挂载状态等。详细内容参考下文。

. 成功登录 Portal。

. 点击**存储服务 > 文件存储 > 文件系统**，进入文件系统列表页面。

. 在文件系统列表页面点击目标文件系统名，进入文件系统详情页面。

. 在文件系统详情页面，点击**挂载点**标签，进入挂载点列表页面。该页面列表显示当前文件系统已创建的所有挂载点。
