对于已创建的文件系统，用户通过权限组的配置和使用，来实现对客户端或用户的访问及权限控制。

* NFS 协议通过 IP 与 IP 段来限制访问的客户端及其读写权限。
* SMB 与 FTP 协议通过用户和用户组来限制访问的用户及其读写权限。
+
include::../../../../../../../_components/note.adoc[]

若单个用户的权限和其所在用户组的权限冲突，优先匹配单个用户的权限。
