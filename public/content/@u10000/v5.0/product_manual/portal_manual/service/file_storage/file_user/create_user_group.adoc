用户组是一组用户的合集。用户组创建完成后，可添加用户至该用户组。

. 成功登录 Portal。

. 点击**存储服务 > 文件存储 > 文件用户**，默认进入用户列表页面。

. 点击**用户组**，进入用户组列表页面。

. 在用户组列表页面，点击**新增用户组**。

. 弹出**新增用户组**对话框，用户根据页面提示信息，输入相应参数即可。
+
image::/images/u10000/stor_file_user_group_create.png[stor_file_user_group_create,40%]
+
include::../../../../../../../_components/note.adoc[]

系统将默认在用户名组名称前添加 `nas-` 前缀。

include::../../../../../../../_components/table_footer.adoc[]

. 可在用户组列表页面，查看新增用户组信息。
+
include::../../../../../../../_components/note.adoc[]

新增的用户组，在绑定用户之前，**用户数**为 `0`。
