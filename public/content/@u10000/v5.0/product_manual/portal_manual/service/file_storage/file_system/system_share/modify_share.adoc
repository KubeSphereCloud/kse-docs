title: 修改挂载点
description: 
weight: 3

对于已创建的挂载点，Portal 支持对其权限组进行修改。权限组的修改，立马生效。如：当前客户端正在写，若此时修改权限为 `只读`，则当前正在进行中的 `写` 操作，会立马中断。



. 成功登录 Portal。

. 点击**存储服务 > 文件存储 > 文件系统**，进入文件系统列表页面。

. 点击目标文件系统名，进入文件系统详情页面后，点击**挂载点**标签，进入文件系统挂载点列表页面。

. 在挂载点列表页面，点击目标挂载点的**更多 > 修改**。
+
image::/images/u10000/stor_file_system_share_more.png[stor_file_system_share_more,100%]

. 在弹出的**修改挂载点**对话框内，重新选择**权限组**，确认无误后，点击**确定**。
