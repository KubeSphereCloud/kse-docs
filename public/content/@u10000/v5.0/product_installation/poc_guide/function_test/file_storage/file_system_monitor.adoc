| 用例编号 | 01

|测试目的
|验证监控的相关功能，支持文件系统的流量监控和容量监控。

|预置条件
a|
. {product_name} 已经安装部署完毕，且运行正常。
. 文件系统 FS1 已成功创建，并创建共享。

|测试步骤
a|
. 使用正确的用户名和密码登录 Portal。
. 将文件系统 FS1 的共享挂载在客户端，在挂载目录下写入 10GB 数据，读取 10GB 数据。
. 15min 后，在 Portal 页面上点击文件系统 FS1 详情链接，查看文件系统 FS1 的容量监控和流量监控数据。


|预期结果
a|
. 成功登录 Portal。
. 写入数据成功、读取数据成功。
. 查看文件系统 FS1 的监控数据成功，容量使用量为 10GB；写入流量为 10GB；由于本地缓存，读取流量略小于 10GB。
