== 参考文档

环境规划可参考文档《{product_name} 环境规划指南》。

== 操作系统

=== 系统要求

* 目前 {product_name} 在 x86 服务器上支持的操作系统有 `UOS 1050e`、`ubuntu 20.04`、`kylin 10-SP2-x86-Release-Build09-20210524`。

* ARM 服务器上支持的操作系统为 `ubuntu 18.04`。

* {product_name} 推荐用户使用 `ubuntu 20.04`。
* 操作系统需包含 Iptables 程序，且 `mkfs.xfs` 或 `mkfs.ext4` 程序至少包含一个。

=== 操作步骤


. 从 {product_name} 的 Publisher 服务器上获取操作系统镜像。

+
[source,shell]
----
ftp://ins.qingcloud.com/
----

. 将操作系统镜像上传至各节点服务器。所有节点服务器均需配置为 `用户名密码` 方式登录，且用户名密码要求所有节点服务器保持一致。

. 安装操作系统时，系统盘分区需遵循如下规则要求。
+
* 规则一：系统盘划分为两个分区，分区格式为 `xfs` 或 `ext4`，不使用 `lvm`。
* 规则二：第一个分区为交换分区，大小为 `32G`；第二个分区为系统分区，将剩余空间划分至根分区 `/` 下即可。
