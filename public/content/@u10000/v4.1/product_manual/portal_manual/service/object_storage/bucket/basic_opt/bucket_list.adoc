用户点击**对象存储 > Bucket**，进入 Bucket 列表页面：

image::/images/u10000/stor_bucket_main.png[stor_bucket_main,100%]

如上图所示，该页面列表显示当前区域下，所有用户创建的 Bucket 信息。页面的详细信息说明如下：

* `用户 ID` 及 `邮箱`，为创建该 Bucket 的用户信息。
* `空间大小` 是指当前 Bucket 已使用空间大小。
* 该页面支持用户导出 Bucket 列表至本地，支持搜索与过滤功能，方便用户快速查找目标 Bucket，以便快速排查问题。
* 该页面提供 Bucket 实时带宽排名，列表显示带宽使用量排名前 20 的 Bucket。
* 当前支持的 Bucket 状态有：`活跃`，`已禁用`，`已删除`，`已销毁`，`正在创建`。各状态说明如下：
+
[cols="2,8", options="header"]
|===
| 名词 |解释

|活跃
|表示当前 Bucket 可正常使用。

|已禁用
|表示当前 Bucket 不可使用。只支持删除文件的操作，其他操作不可执行。

|已删除
|表示当前 Bucket 已删除，系统正在清理资源。

|已销毁
|表示当前 Bucket 已销毁，系统已完成相关的资源清理。

|正在创建
|表示当前 Bucket 正在创建过程中。
|===
