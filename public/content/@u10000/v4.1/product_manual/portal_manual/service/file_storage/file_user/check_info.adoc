. 成功登录 Portal。

. 点击**存储服务 > 文件存储 > 文件用户**，默认进入用户列表页面。
+
image::/images/u10000/stor_file_user_main.png[stor_file_user_main,100%]




. 点击**用户组**，进入用户组列表页面。用户组列表信息中，显示当前用户组内已加入的用户数。
+
如，**用户数**为 `1`，则表示当前用户组内添加了一个用户；**用户数**为 `0`，则表示当前用户组内没有添加用户。
+
image::/images/u10000/stor_file_user_group_main.png[stor_file_user_group_main,100%]
+
include::../../../../../../../_components/note.adoc[]


* `DEFAULT_USER_GROUP_aKBwgj7a` 是系统创建的默认用户组。其中 `aKBwgj7a` 是创建该用户组的 User ID。

* 新增用户默认会加入到该默认用户组中。
