== 功能概述

{product_name} 兼容标准的 POSIX 语义，对外提供分布式文件存储服务，支持 NFS、SMB、FTP、CIFS 等访问协议，可承载多种业务场景，存储各种非结构化数据，如日志、文档、邮件、图片、音视频等。
