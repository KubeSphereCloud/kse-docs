删除文件系统前，用户需确保该文件系统内无数据存储。

include::../../../../../../../../_components/note.adoc[]

对使用中的文件系统进行删除操作，系统会返回操作失败。因此，删除文件系统前，用户需确保该文件系统内无数据存储。

include::../../../../../../../../_components/table_footer.adoc[]


详细操作如下。

. 成功登录 Portal。

. 点击**存储服务 > 文件存储 > 文件系统**，进入文件系统列表页面。

. 在文件系统列表页面点击目标文件系统的**更多 > 删除**。
+
image::/images/u10000/stor_file_system_more.png[stor_file_system_more,100%]

. 在弹出的确认对话框内，确认操作无误后，点击**确认**即可删除。
