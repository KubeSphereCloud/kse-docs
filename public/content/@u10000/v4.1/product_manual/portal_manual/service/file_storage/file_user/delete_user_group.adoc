include::../../../../../../../_components/note.adoc[]

删除用户组之前，需确认当前用户组内无用户。

include::../../../../../../../_components/table_footer.adoc[]


. 成功登录 Portal。

. 点击**存储服务 > 文件存储 > 文件用户**，默认进入用户列表页面。

. 点击**用户组**，进入用户组列表页面。


. 在用户组列表页面，点击目标用户组的**更多 > 删除**。
+
image::/images/u10000/stor_file_user_group_add.png[stor_file_user_group_add,100%]


. 弹出确认对话框，用户确认操作无误后，点击**删除**。
+
include::../../../../../../../_components/note.adoc[]

若当前用户组内还有用户，则删除操作会失败。

include::../../../../../../../_components/table_footer.adoc[]

. 用户可在用户组列表页面确认，成功删除的用户组不在列表显示。
