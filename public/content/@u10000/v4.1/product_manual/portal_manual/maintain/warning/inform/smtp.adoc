SMTP 即邮件通知方式。在配置邮件发送告警信息时，需先配置link:#_告警模版[告警模版]。SMTP 配置包括邮件发送服务器信息以及邮件接收服务器相关信息。

详细的操作步骤如下。

. 成功link:../../../../login[登录 Portal]。

. 点击**运维管理 > 告警管理 > 告警通知 > SMTP**，进入 SMTP 设置页面。
+
image::/images/u10000/maintain_warning_info_smtp.png[maintain_warning_info_smtp,100%]
+
**页面信息说明：**

* 如图所示，该页面分为两个卡片，分别显示** SMTP 发件邮箱设置**与**告警接收组**相关信息。
* 发件箱仅需配置一个，如需修改发件箱配置或删除已有邮件箱配置，可点击** SMTP 发件邮箱设置**卡片上的**删除**或**修改**按钮。在弹出的对话框内，根据提示信息完成相应操作。
* 为方便用户快速查看告警接收组，页面支持筛选功能。用户可通过输入关键字进行筛选查询。


. 配置 SMTP 发件邮箱。用户点击** SMTP 发件邮箱设置**卡片的**配置邮箱**，弹出**配置 SMTP 发件邮箱**对话框，用户根据页面提示信息，填写相关参数后，点击**保存**。
+
image::/images/u10000/maintain_warning_info_smtp_create.png[maintain_warning_info_smtp_create,100%]
+
**页面信息说明：**

* 该页面参数用于配置 SMTP 发件邮箱相关信息，后续告警邮件均由此发件箱发送。
* 页面配置参数说明如下：
+
[cols="2*", options="header"]
|===
| 参数 |说明

|邮箱地址
|发件箱地址

|邮箱服务器
|发件箱服务器

|端口
|发件箱端口

|密码
|发件箱密码
|===

. 配置告警接收组。用户点击**告警接收组**卡片的**创建告警接收组**，进入创建告警接收组页面，根据页面信息填写相关参数后，点击**确定**。
+
image::/images/u10000/maintain_warning_receive_create.png[maintain_warning_receive_create,100%]
+
**页面信息说明：**

* 该页面参数用于配置告警邮件接收人相关信息，后续告警邮件的接收人以及邮件内容。
* 页面配置参数说明如下：
+
[cols="2*", options="header"]
|===
| 参数 |说明

|接收组名称
|即为告警信息接收群组，与其他信息接收群组含义相同，群组里可有多个联系人，也可以仅包含一个联系人。群组里所有联系人，均可收到告警信息邮件。

|备注
|可填写当前接收组相关信息，非必填项

|关联告警用户
|选择告警邮件接收人。用户可根据需求选择一个或多个用户添加至该告警群组。若当前页面无可选告警用户，或当前页面列举的告警用户不满足需求，用户可先自行link:#_告警用户[创建告警用户]后，再做关联。

|关联告警模板
|选择告警邮件发送模板。用户可根据需求选择一个或多个告警模板。若当前页面无可选告警模板，或当前页面列举的告警模板不满足需求，用户可先自行link:#_告警模版[创建告警模板]后，再做关联。
|===

. 配置后的 SMTP 发件箱信息与告警接收组信息均可修改与删除。
+
* 点击** SMTP 发件邮箱设置**卡片上的**删除**或**修改**按钮。在弹出的对话框内，根据提示信息即可完成 SMTP 发件邮箱的删除或修改。
* 点击**告警接收组**卡片上的待删除或修改的告警接受组的**更多操作**，点击**删除**或**修改**按钮。在弹出的对话框内，根据提示信息即可完成告警接收组的删除或修改。
+
