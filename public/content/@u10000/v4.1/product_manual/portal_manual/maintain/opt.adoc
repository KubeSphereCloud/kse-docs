== 功能概述

Portal 提供的操作记录功能，是指在 Portal 上进行的操作将被记录并展示。其他操作，如 Console 上进行的操作，通过 API 进行的相关操作等，将不在此显示。


== 查看操作记录

. 成功link:../../login[登录 Portal]。

. 点击**运维管理 > 操作记录**，进入操作记录列表页面。
+
image::/images/u10000/maintain_opt_main.png[maintain_opt_main,100%]
+
**页面信息说明：**

* 如页面所示，操作记录详细记录了操作时间、操作人、操作内容以及操作者的 IP 地址等相关信息。
* 为快速定位待查看的操作记录，该页面提供筛选功能，用户可按照日期与关键字对操作记录进行筛选查看。
