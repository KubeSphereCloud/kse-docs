== 用户概况

亿咖通科技是浙江吉利控股集团战略投资独立运营的科技生态企业，专注于汽车智能化与网联化技术，汽车芯片、智能座舱、智能驾驶、高精度地图、大数据及车联网云平台等核心技术产品，持续打造行业领先的智能网联开放平台。

== 业务挑战

* 无人驾驶深度依赖于高精度地图。其中单台数据采集车每天能采集 1TB 的数据，整个车队共享的数据将是海量的，这对存储系统的扩展性提出了要求。
* 在地图制作的过程中，涉及到多个地图制作平台和软件。包括与对接地图制作的多种平台，地图信息数据输入平台、标注平台、深度学习一体机，涉及多种语言接口等。这要求存储系统需要支持各类接口和生态兼容。
* 存储在公有云的数据，和应用之间相隔太远，网络成为其瓶颈。另外数据隐私性要求，亟需将这些数据迁回私有云环境中。


== 解决方案

image::/images/u10000/product_14.png[,100%]

* 计算分析：部署 DGX-1 英伟达机器学习一体机、大数据分析应用。
* 存储中枢：采用 3+N 标准部署模式的 U10000 作为数据的存储和流转中枢，构建高精度地图服务。


== 客户收益

* U10000 分层架构设计，水平扩展，根据业务规模精准扩容，无资源浪费。
* 完善的数据安全和访问安全保证，确保采集数据的安全和隐私性。
* 公有云和私有云同一架构，超大规模生产验证。稳定可用，让用户更加专注业务。
* 开放 API 接口，兼容 S3 生态，覆盖各类语言 SDK 完善的生态体系（大数据分析、数据仓库、物联网、车联网）。
* 极低运维成本，只需一位运维人员即可实现整体操作管理。
