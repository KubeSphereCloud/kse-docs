== 测试目的

{product_name} 的 Console 能正常提供对象存储服务。



== 预置条件

{product_name} 存储系统已经部署完毕，且对象存储相关服务运行正常。


== 测试过程

. 登录 Console。

. 创建 Bucket。

. 上传文件至 Bucket。

. 从指定 Bucket 下载文件。

. 删除文件。

. 删除 Bucket。

. 独立部署环境，需创建并下载 API Key，删除 API Key。

== 预期结果

. 步骤1，融合部署环境中，在 IaaS 的 Console 界面，存储栏目下有 {product_name} 对象存储的菜单；独立部署环境中，能正常登录 Console。

. 步骤2，创建 Bucket 成功，且显示在 Bucket 列表。

. 步骤3，上传文件成功，且显示在相应的 Bucket 文件列表。

. 步骤4，下载文件成功。

. 步骤5，删除文件成功，该文件不在 Bucket 文件列表显示。

. 步骤6，删除 Bucket 成功。

. 步骤7，独立部署环境中，创建下载删除 API Key 成功。
